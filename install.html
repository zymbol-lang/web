Using the static binary

No installation needed — download, make executable, run.

# Make it executable
chmod +x zymbol_lang_v0.0.2_x86_64_linux

# Run a file
./zymbol_lang_v0.0.2_x86_64_linux run hello.zy

# (Optional) Install system-wide
sudo install -m755 zymbol_lang_v0.0.2_x86_64_linux /usr/local/bin/zymbol

Compile from Source

Works on Linux, macOS, and Windows. Requires Rust stable (rustup).

1
# Clone the interpreter repository
git clone https://github.com/zymbol-lang/interpreter
cd interpreter
2
# Build optimized release binary
cargo build --release
# Binary: target/release/zymbol
3
# (Optional) Install system-wide on Linux / macOS
sudo install -m755 target/release/zymbol /usr/local/bin/zymbol
4
# Verify
zymbol --version
