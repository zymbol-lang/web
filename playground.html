Browser Playground zymbol.js v0.0.5

This playground runs a browser-only build of the Zymbol interpreter (zymbol.js) compiled directly to JavaScript. It is a preview environment intended to explore the language without installing anything.

CLI argument capture >< args is supported — use the args… field next to the Run button.

Shell execution <\ cmd \> returns a high-resolution timestamp (entropy source for random seeds). Script inclusion </ file.zy /> is not available in the browser runtime.

TUI mode (>>|) works in the browser with a canvas renderer. On mobile: swipe for directional input (↑↓←→), tap for Enter, and the ✕ Stop button to exit. A physical keyboard is recommended for the best experience.

Code runs without prior semantic checking — runtime errors are displayed in the output panel. For the full language experience use the native interpreter.
