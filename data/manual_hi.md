# Zymbol-Lang की संक्षिप्त मार्गदर्शिका

**Zymbol-Lang** एक प्रतीकात्मक प्रोग्रामिंग भाषा है। इसमें कोई कीवर्ड नहीं है — सब कुछ एक प्रतीक है। यह किसी भी मानव भाषा में एक समान कार्य करती है।

- कोई कीवर्ड नहीं (`if`, `while`, `return` मौजूद नहीं — केवल प्रतीक `?`, `@`, `<~`)
- पूर्ण Unicode — किसी भी भाषा या इमोजी में पहचानकर्ता 👋
- मानव-भाषा तटस्थ — कोड हर भाषा में समान है

---

## चर और स्थिरांक

```zymbol
च = १०              // चर (परिवर्तनीय)
पाई := ३.१४१५९      // स्थिरांक — पुनः असाइन करने पर रनटाइम त्रुटि
नाम = "अनु"
नशट = #१            // बूलियन सत्य
👋 := "नमस्ते"
```

```zymbol
च = १०
च += ५    // १५
च -= ३    // १२
च *= २    // २४
च /= ३    // ८
च %= ३    // २
च ^= २    // ४
च++       // ५
च--       // ४
```

## डेटा प्रकार

| प्रकार | उदाहरण | `#?` प्रतीक | नोट |
|--------|--------|-------------|-----|
| पूर्णांक (Int) | `४२`, `-७` | `###` | 64-bit चिह्नित |
| दशमलव (Float) | `३.१४`, `१.५e१०` | `##.` | वैज्ञानिक संकेतन समर्थित |
| स्ट्रिंग (String) | `"नमस्ते"` | `##"` | इंटरपोलेशन: `"नमस्ते {नाम}"` |
| वर्ण (Char) | `'अ'` | `##'` | एक Unicode वर्ण |
| बूलियन (Bool) | `#१`, `#०` | `##?` | संख्यात्मक नहीं — `#१ ≠ १` |
| सरणी (Array) | `[१, २, ३]` | `##]` | समरूप तत्व |
| टुपल (Tuple) | `(क, ख)` | `##)` | स्थितीय |
| नामित टुपल | `(x: १, y: २)` | `##)` | नामित फ़ील्ड |

```zymbol
// प्रकार परीक्षण — (प्रकार, अंक, मान) लौटाता है
meta = ४२#?
>> meta ¶         // → (###, २, ४२)
t = meta[०]
>> t ¶            // → ###
```

## आउटपुट और इनपुट

```zymbol
>> "नमस्ते" ¶                      // ¶ या \\ स्पष्ट नई पंक्ति
>> "क=" च " ख=" न ¶                // जुक्सटापोजिशन — कई मान
>> (सरणी$#) ¶                      // पोस्टफ़िक्स ऑपरेटर को >> में कोष्ठक चाहिए

<< नाम                              // बिना प्रॉम्प्ट — चर में पढ़ें
<< "आपका नाम? " नाम                // प्रॉम्प्ट के साथ
```

> `¶` (स्पेनिश कीबोर्ड पर AltGr+R) और `\\` समतुल्य नई पंक्तियाँ हैं।

## ऑपरेटर

```zymbol
// अंकगणित — असाइनमेंट में उपयोग करें; >> में सीधे कुछ ऑपरेटरों में विशेषताएं हैं
क = १०
ख = ३
r१ = क + ख    // १३     r२ = क - ख    // ७
r३ = क * ख    // ३०     r४ = क / ख    // ३  (पूर्णांक भाग)
r५ = क % ख    // १      r६ = क ^ ख    // १०००  (घातांक)

// तुलना
क == ख    // #०    क <> ख    // #१    क < ख    // #०
क <= ख    // #०   क > ख     // #१    क >= ख   // #१

// तार्किक
#१ && #०    // #०
#१ || #०    // #१
!#१         // #०
```

## स्ट्रिंग

```zymbol
// तीन संयोजन रूप
नाम = "अनु"
न = ४२

संदेश = "नमस्ते ", नाम, "!"            // अल्पविराम — असाइनमेंट में
>> "नमस्ते " नाम " आपके पास " न ¶     // जुक्सटापोजिशन — >> में
विवरण = "नमस्ते {नाम}, आपके पास {न}"  // इंटरपोलेशन — कहीं भी
```

```zymbol
s = "नमस्ते दुनिया"
len = s$#                  // लंबाई
sub = s$[०..७]             // "नमस्ते"  (अंत बहिष्कृत)
has = s$? "दुनिया"         // #१
parts = "क,ख,ग,घ" / ','    // [क, ख, ग, घ]
rep = s$~~["ते":"ता"]      // प्रतिस्थापन
rep१ = s$~~["ते":"ता":१]   // पहले N प्रतिस्थापन
```

> `+` केवल संख्याओं के लिए है। स्ट्रिंग के लिए `,`, जुक्सटापोजिशन या इंटरपोलेशन का उपयोग करें।

## प्रवाह नियंत्रण

```zymbol
च = ७

? च > ० { >> "धनात्मक" ¶ }

    >> "बड़ा" ¶
```

## मिलान

```zymbol
// रेंज
अंक = ८५
श्रेणी = ?? अंक {
    ९०..१०० : 'A'
    ८०..८९  : 'B'
    ७०..७९  : 'C'
    _       : 'F'
}
>> श्रेणी ¶    // → B

// स्ट्रिंग
रंग = "लाल"
कोड = ?? रंग {
    "लाल"   : "#FF००००"
    "हरा"   : "#००FF००"
    _       : "#००००००"
}

// गार्ड
तापमान = -५
स्थिति = ?? तापमान {
    _? तापमान < ०  : "बर्फ"
    _? तापमान < २० : "ठंडा"
    _? तापमान < ३५ : "गर्म"
    _              : "बहुत गर्म"
}
>> स्थिति ¶    // → बर्फ

// कथन रूप (ब्लॉक शाखाएं)
?? न {
    ०       : { >> "शून्य" ¶ }
    _? न < ०: { >> "ऋणात्मक" ¶ }
    _       : { >> "धनात्मक" ¶ }
}
```

## लूप

```zymbol
@ न:०..४  { >> न " " }        // समावेशी रेंज:  ० १ २ ३ ४
@ न:१..९:२ { >> न " " }       // स्टेप के साथ:   १ ३ ५ ७ ९
@ न:५..०:१ { >> न " " }       // उल्टा:          ५ ४ ३ २ १ ०

न = १
@ न <= ६४ { न *= २ }
>> न ¶                        // → १२८  (while)

फल = ["सेब", "नाशपाती", "अंगूर"]
@ फ:फल { >> फ ¶ }             // सरणी पर for-each

@ व:"नमस्ते" { >> व "-" }
>> ¶                          // → न-म-स-्-त-े-  (स्ट्रिंग पर)

@ न:१..१० {
    ? न % २ == ० { @> }       // @> जारी रखें
    ? न > ७ { @! }             // @! रुकें
    >> न " "
}
>> ¶                          // → १ ३ ५ ७

// अनंत लूप
i = ०
@ {
    i++
    ? i >= ५ { @! }
    >> i " "
}
>> ¶                          // → १ २ ३ ४

// लेबल वाला लूप (नेस्टेड ब्रेक)
count = ०
@ @outer {
    count++
    ? count >= ३ { @! outer }
}
>> count ¶                    // → ३
```

## फ़ंक्शन

```zymbol
जोड़(क, ख) { <~ क + ख }
>> जोड़(३, ४) ¶    // → ७

गुणनखंड(न) {
    ? न <= १ { <~ १ }
    <~ न * गुणनखंड(न - १)
}
>> गुणनखंड(५) ¶    // → १२०
```

फ़ंक्शनों का **पृथक स्कोप** होता है — वे बाहरी चर नहीं पढ़ सकते। आउटपुट पैरामीटर `<~` से कॉलर चर बदलें:

```zymbol
swap(क<~, ख<~) {
    tmp = क
    क = ख
    ख = tmp
}
च = १०
य = २०
swap(च, य)
>> "च=" च " य=" य ¶    // → च=२० य=१०
```

> नामित फ़ंक्शन प्रथम-श्रेणी नहीं हैं। तर्क के रूप में पास करने के लिए लपेटें: `x -> जोड़(x)`।

## लैम्बडा और क्लोज़र

```zymbol
दोगुना = च -> च * २
जोड़ = (क, ख) -> क + ख
>> दोगुना(५) ¶    // → १०
>> जोड़(३, ७) ¶   // → १०

// ब्लॉक लैम्बडा
वर्गीकरण = च -> {
    ? च > ० { <~ "धनात्मक" }
    _? च < ० { <~ "ऋणात्मक" }
    <~ "शून्य"
}

// क्लोज़र — बाहरी स्कोप कैप्चर करता है
गुणांक = ३
तिगुना = च -> च * गुणांक
>> तिगुना(७) ¶    // → २१

// फ़ैक्टरी
संचालन(न) { <~ च -> च + न }
दस_जोड़ = संचालन(१०)
>> दस_जोड़(५) ¶    // → १५

// सरणी में
क्रियाएं = [च -> च+१, च -> च*२, च -> च*च]
>> क्रियाएं[२](५) ¶    // → २५
```

## सरणी

सरणियाँ **परिवर्तनीय** होती हैं और **एक ही प्रकार** के तत्वों को रखती हैं।

```zymbol
सरणी = [१, २, ३, ४, ५]

सरणी[०]          // १ — एक्सेस (० से अनुक्रमित)
सरणी[-१]         // ५ — ऋणात्मक अनुक्रमांक (अंतिम)
सरणी$#           // ५ — लंबाई (>> में (सरणी$#) का उपयोग करें)

सरणी = सरणी$+ ६            // जोड़ें → [१,२,३,४,५,६]
arr२ = सरणी$+[२] ९९         // अनुक्रमांक २ पर डालें
arr३ = सरणी$- ३             // मान की पहली घटना हटाएं
arr४ = सरणी$-- ३            // सभी घटनाएं हटाएं
arr५ = सरणी$-[०]            // अनुक्रमांक पर हटाएं
arr६ = सरणी$-[१..३]         // रेंज हटाएं (अंत बहिष्कृत)

has = सरणी$? ३              // #१ — शामिल है
pos = सरणी$?? ३             // [२] — मान के सभी अनुक्रमांक
sl = सरणी$[०..३]            // [१,२,३] — स्लाइस (अंत बहिष्कृत)
sl२ = सरणी$[०:३]            // [१,२,३] — गिनती-आधारित वाक्यरचना

asc = सरणी$^+               // आरोही क्रम (केवल प्रिमिटिव)
desc = सरणी$^-              // अवरोही क्रम (केवल प्रिमिटिव)

// नामित/स्थितीय टुपल सरणियां — $^ लैम्बडा के साथ
db = [(name: "कार्ला", age: २८), (name: "अना", age: २५), (name: "बॉब", age: ३०)]
by_age  = db$^ (क, ख -> क.age < ख.age)    // आयु के अनुसार आरोही (<)
by_name = db$^ (क, ख -> क.name > ख.name)  // नाम के अनुसार अवरोही (>)
>> by_age[०].name ¶     // → अना
>> by_name[०].name ¶    // → कार्ला

// प्रत्यक्ष तत्व अपडेट (केवल सरणियाँ)
सरणी[१] = ९९              // असाइन करें
सरणी[०] += ५              // संयुक्त: +=  -=  *=  /=  %=  ^=

// फ़ंक्शनल अपडेट — नई सरणी लौटाता है; मूल अपरिवर्तित रहता है
arr२ = सरणी[१]$~ ९९

```

> सभी संग्रह ऑपरेटर **नई सरणी** लौटाते हैं। वापस असाइन करें: `सरणी = सरणी$+ ४`।
> ऑपरेटर चेन नहीं किए जा सकते — मध्यवर्ती असाइनमेंट का उपयोग करें।
> `$^+` / `$^-` **प्रिमिटिव सरणियों** (संख्याएं, स्ट्रिंग) को सॉर्ट करते हैं। टुपल सरणियों के लिए `$^` और लैम्बडा उपयोग करें — दिशा लैम्बडा में एन्कोड है (`<` = आरोही, `>` = अवरोही)।

**मूल्य अर्थशास्त्र** — किसी सरणी को दूसरे चर में असाइन करने से एक स्वतंत्र प्रति बनती है:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b अप्रभावित रहता है
```

```zymbol
// नेस्टेड सरणियां
matrix = [[१,२,३],[४,५,६],[७,८,९]]
>> matrix[१][२] ¶    // → ६
```

## टुपल

टुपल **अपरिवर्तनीय** क्रमबद्ध कंटेनर हैं जो **विभिन्न प्रकारों** के मान रख सकते हैं। सरणियों के विपरीत, निर्माण के बाद तत्वों को बदला नहीं जा सकता।

```zymbol
// स्थितीय
point = (१०, २०)
>> point[०] ¶    // → १०

डेटा = (४२, "नमस्ते", #१, ३.१४)
>> डेटा[२] ¶     // → #१

// नामित
व्यक्ति = (name: "अनु", age: २५)
>> व्यक्ति.name ¶    // → अनु
>> व्यक्ति[०] ¶      // → अनु  (अनुक्रमांक भी काम करता है)

// नेस्टेड
pos = (x: १०, y: २०)
p = (pos: pos, label: "मूल")
>> p.pos.x ¶        // → १०
```

**अपरिवर्तनीयता** — टुपल तत्व को बदलने का कोई भी प्रयास रनटाइम त्रुटि है:

```zymbol
t = (१०, २०, ३०)
// t[०] = ९९    // ❌ रनटाइम त्रुटि: टुपल अपरिवर्तनीय हैं
// t[०] += ५    // ❌ समान त्रुटि
```

परिवर्तित मान प्राप्त करने के लिए `$~` (फ़ंक्शनल अपडेट) का उपयोग करें — एक **नया** टुपल लौटाता है:

```zymbol
t = (१०, २०, ३०)
t२ = t[१]$~ ९९९
>> t ¶     // → (१०, २०, ३०)   ← मूल अपरिवर्तित
>> t२ ¶    // → (१०, ९९९, ३०)

// नामित टुपल — स्पष्ट रूप से पुनर्निर्माण करें
व्यक्ति = (name: "Alice", age: २५)
बड़ा  = (name: व्यक्ति.name, age: २६)
>> व्यक्ति.age ¶    // → २५
>> बड़ा.age ¶       // → २६
```

## उच्च-क्रम फ़ंक्शन

> HOF ऑपरेटरों को **इनलाइन लैम्बडा** चाहिए — लैम्बडा चर सीधे काम नहीं करते।

```zymbol
संख्याएं = [१, २, ३, ४, ५, ६, ७, ८, ९, १०]

doubled  = संख्याएं$> (च -> च * २)                // map  → [२,४,६…२०]
evens    = संख्याएं$| (च -> च % २ == ०)            // filter → [२,४,६,८,१०]
total    = संख्याएं$< (०, (acc, च) -> acc + च)     // reduce → ५५

// मध्यवर्ती असाइनमेंट से चेन करें
step१ = संख्याएं$| (च -> च > ३)
step२ = step१$> (च -> च * च)
>> step२ ¶    // → [१६, २५, ३६, ४९, ६४, ८१, १००]

// HOF में नामित फ़ंक्शन — लैम्बडा में लपेटें
दोगुना(च) { <~ च * २ }
r = संख्याएं$> (च -> दोगुना(च))    // ✅
```

## पाइप ऑपरेटर

RHS को हमेशा पाइप किए गए मान के लिए `_` प्लेसहोल्डर चाहिए:

```zymbol
double = च -> च * २
add = (क, ख) -> क + ख
inc = च -> च + १

५ |> double(_)        // → १०
१० |> add(_, ५)       // → १५
५ |> add(२, _)        // → ७

// चेन
r = ५ |> double(_) |> inc(_) |> double(_)
>> r ¶    // → २२  (५→१०→११→२२)
```

## अंक प्रणालियाँ

Zymbol **69 Unicode अंक ब्लॉकों** में संख्याएँ प्रदर्शित कर सकता है — देवनागरी, अरबी-भारतीय, थाई, क्लिंगन pIqaD, गाढ़ा गणितीय, LCD अंक और अधिक। सक्रिय मोड केवल `>>` आउटपुट को प्रभावित करता है; आंतरिक अंकगणित हमेशा बाइनरी रहता है।

### सिस्टम सक्रिय करें

लक्ष्य लिपि के `०` और `९` अंकों को `#…#` के बीच लिखें:

```zymbol
#०९#    // देवनागरी   (U+0966–U+096F)
#٠٩#    // अरबी-भारतीय (U+0660–U+0669)
#๐๙#    // थाई        (U+0E50–U+0E59)
#09#    // ASCII रीसेट
```

### आउटपुट और बूलियन

```zymbol
x = ४२
>> x ¶          // → ४२   (डिफ़ॉल्ट ASCII)

#०९#
>> x ¶          // → ४२
>> ३.१४ ¶       // → ३.१४   (दशमलव बिंदु हमेशा ASCII)
>> १ + २ ¶      // → ३

// बूलियन: # उपसर्ग हमेशा ASCII, अंक मोड के अनुसार
>> #१ ¶         // → #१   (देवनागरी में सत्य)
>> #० ¶         // → #०   (असत्य — ०  शून्य से भिन्न)

x = २८ > ४
>> x ¶          // → #१   (तुलना परिणाम सक्रिय मोड का अनुसरण करता है)
```

### स्रोत में नेटिव लिटरल

किसी भी समर्थित सिस्टम के अंक रेंज, मॉड्यूलो, तुलना में वैध लिटरल हैं:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### किसी भी सिस्टम में बूलियन लिटरल

किसी भी समर्थित ब्लॉक का `#` + `०` या `१` अंक एक वैध बूलियन लिटरल है:

```zymbol
#٠٩#
نشط = #١        // #१ के समान
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **हमेशा ASCII** है। `#०` (असत्य) किसी भी सिस्टम में `०` (शून्य पूर्णांक) से दृश्यतः भिन्न है।

## डेटा ऑपरेटर

```zymbol
// स्ट्रिंग को संख्या में पार्स करें
v१ = #|"४२"|      // → ४२  (Int)
v२ = #|"३.१४"|    // → ३.१४  (Float)
v३ = #|"abc"|     // → "abc"  (सुरक्षित विफलता, कोई त्रुटि नहीं)

// गोल / काटना
pi = ३.१४१५९२६५
r२ = #.२|pi|      // → ३.१४  (२ दशमलव स्थान पर गोल)
r४ = #.४|pi|      // → ३.१४१६
t२ = #!२|pi|      // → ३.१४  (काटना)

// संख्या स्वरूपण
fmt = #,|१२३४५६७|  // → १,२३४,५६७  (अल्पविराम-पृथक)
sci = #^|१२३४५.६७८|    // → १.२३४५६७८e४  (वैज्ञानिक)

// आधार लिटरल
a = 0x41         // → 'A'  (हेक्स)
b = 0b01000001   // → 'A'  (बाइनरी)
c = 0o101        // → 'A'  (ऑक्टल)

// आधार रूपांतरण आउटपुट
hex = 0x|२५५|    // → "0x00FF"
bin = 0b|६५|     // → "0b1000001"
oct = 0o|८|      // → "0o10"
dec = 0d|२५५|    // → "0d0255"
```

## प्रतीक संदर्भ

| प्रतीक | क्रिया | प्रतीक | क्रिया |
|--------|--------|--------|--------|
| `=` | चर | `$#` | लंबाई |
| `:=` | स्थिरांक | `$+` | जोड़ें |
| `>>` | आउटपुट | `$+[i]` | अनुक्रमांक पर डालें |
| `<<` | इनपुट | `$--` | मान की सभी घटनाएं हटाएं |
| `¶` / `\` | नई पंक्ति | `$-[i]` | अनुक्रमांक पर हटाएं |
| `?` | if | `$-[i..j]` | रेंज हटाएं |
| `_?` | else-if | `$?` | शामिल है |
| `_` | else / वाइल्डकार्ड | `$??` | सभी अनुक्रमांक खोजें |
| `??` | मिलान | `$[s..e]` | स्लाइस |
| `@` | लूप | `$>` | map |
| `@!` | रुकें | `$\|` | filter |
| `@>` | जारी रखें | | |
| `->` | लैम्बडा | `$<` | reduce |
| `arr[i] = val` | तत्व अपडेट (केवल सरणियाँ) | `arr[i] += val` | संयुक्त अपडेट |
| `arr[i]$~` | फ़ंक्शनल अपडेट (नई प्रति) | `$^+` | आरोही क्रम |
| `$^-` | अवरोही क्रम | `$^` | लैम्बडा से क्रम |
| `\|>` | pipe | `<~` | return |
| `#१` | सत्य | `$!` | त्रुटि है |
| `#०` | असत्य | `$!!` | त्रुटि प्रसार |
| `!?` | try | `#` | मॉड्यूल घोषित करें |
| `:!` | catch | `#>` | निर्यात |
| `:>` | finally | `<#` | आयात |
| `.` | फ़ील्ड एक्सेस | `::` | मॉड्यूल कॉल |
| `#\|..\|` | संख्या पार्स | `#.N\|..\|` | गोल करना |
| `#!N\|..\|` | काटना | `#,\|..\|` | अल्पविराम स्वरूप |
| `#d0d9#` | अंक मोड स्विच | | |
| `#^\|..\|` | वैज्ञानिक | `<\ ..\>` | शेल कमांड |
| `><` | CLI आर्गुमेंट | `$~~[..]` | स्ट्रिंग प्रतिस्थापन |
| `[क,ख]=arr` | सरणी विनाशीकरण | `(x,y)=tup` | टुपल विनाशीकरण |
