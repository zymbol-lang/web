## വേരിയബിളുകളും സ്ഥിരാങ്കങ്ങളും

```zymbol
x = ൧൦              // മാറ്റാവുന്ന വേരിയബിൾ
π := ൩.൧൪൧൫൯        // സ്ഥിരാങ്കം — വീണ്ടും അസൈൻമെന്റ് റൺടൈം പിശകാണ്
പേര് = "ആലീസ്"
സജീവം = #൧         // ബൂളിയൻ സത്യം
👋 := "ഹലോ"
```

```zymbol
x = ൧൦    // ൧൦
x += ൫    // ൧൫
x -= ൩    // ൧൨
x *= ൨    // ൨൪
x /= ൩    // ൮
x %= ൩    // ൨
x ^= ൨    // ൪
x++        // ൫
x--        // ൪
```

`°` (ഡിഗ്രി ചിഹ്നം, U+00B0) ആദ്യ ഉപയോഗത്തിൽ ഒരു വേരിയബിളിനെ അതിന്റെ നിഷ്പക്ഷ മൂല്യത്തിലേക്ക് യാന്ത്രികമായി ആരംഭിക്കുന്നു:

```zymbol
സംഖ്യകൾ = [൩, ൧, ൪, ൧, ൫]
@ n:സംഖ്യകൾ {
    °ആകെ += n    // ലൂപ്പിന് മുകളിൽ യാന്ത്രികമായി ൦ ലേക്ക് ആരംഭിക്കുന്നു; @ കഴിഞ്ഞും നിലനിൽക്കുന്നു
}
>> ആകെ ¶         // → ൧൪
```

> `°x` (പൂർവ്വപ്രത്യയം) ലൂപ്പിന് മുകളിൽ നങ്കൂരമിടുന്നു — ഫലം `@` കഴിഞ്ഞ് ആക്സസ് ചെയ്യാം.
> `x°` (പിൻപ്രത്യയം) ലൂപ്പിനുള്ളിൽ നങ്കൂരമിടുന്നു — ലൂപ്പ് അവസാനിക്കുമ്പോൾ മരിക്കുന്നു.

| തരം | ലിറ്ററൽ | `#?` ടാഗ് | കുറിപ്പുകൾ |

| ഫ്ലോട്ടിംഗ് | `3.14`, `1.5e10` | `##.` | ശാസ്ത്രീയ നൊട്ടേഷൻ അനുവദനീയം |
| സ്ട്രിംഗ് | `"ടെക്സ്റ്റ്"` | `##"` | ഇന്റർപോളേഷൻ: `"ഹലോ {പേര്}"` |
| കാരക്ടർ | `'അ'` | `##'` | ഒരൊറ്റ യൂണികോഡ് കാരക്ടർ |

| അറേ | `[1, 2, 3]` | `##]` | സജാതീയ ഘടകങ്ങൾ |
| ടപ്പിൾ | `(a, b)` | `##)` | സ്ഥാനീയ |

## ഔട്ട്പുട്ടും ഇൻപുട്ടും

```zymbol
>> "ഹലോ" ¶                       // ¶ അല്ലെങ്കിൽ \\ വ്യക്തമായ പുതിയ വരിക്ക്
>> "a=" a " b=" b ¶               // juxtaposition — ഒന്നിലധികം മൂല്യങ്ങൾ
>> (arr$#) ¶                      // പോസ്റ്റ്ഫിക്സ് ഓപ്പറേറ്റർമാർക്ക് >> ൽ ( ) ആവശ്യമാണ്

>> പേര്                           // വേരിയബിളിലേക്ക് വായിക്കുക (പ്രോംപ്റ്റ് ഇല്ലാതെ)
>> "പേര് നൽകുക: " പേര്            // പ്രോംപ്റ്റോടെ
```

> `¶` (സ്പാനിഷ് കീബോർഡിൽ AltGr+R) ഉം `\\` ഉം തത്തുല്യമായ പുതിയ വരികളാണ്.

## ടിയുഐ പ്രിമിറ്റീവുകൾ

സംവേദനാത്മക പ്രോഗ്രാമുകൾക്കായുള്ള ടെർമിനൽ യൂസർ ഇന്റർഫേസ് ഓപ്പറേറ്റർമാർ. മിക്കതിനും `>>| { }` ബ്ലോക്ക് ആവശ്യമാണ് (ഇതര സ്ക്രീൻ + റോ മോഡ്).

```zymbol
>>| {
    >>!                             // ഇതര സ്ക്രീൻ മായ്ക്കുക
    >>~ (൧, ൧, ൦, ൧൦) > "പ്രവർത്തിക്കുന്നു"  // വരി ൧, കോളം ൧, fg=൧൦ (പച്ച)
    @~ ൧൦൦൦                         // ൧ സെക്കൻഡ് ഇടവേള (൧൦൦൦ ms)
    >>~ (൨, ൧) > "പൂർത്തിയായി."
}
// പുറത്തുകടക്കുമ്പോൾ ടെർമിനൽ യാന്ത്രികമായി പുനഃസ്ഥാപിക്കപ്പെടുന്നു
```

```zymbol
// കീ പ്രസ്സും ടെർമിനൽ വലുപ്പവും
>>| {
    [വരികൾ, കോളങ്ങൾ] = >>?              // ടെർമിനലിന്റെ അളവുകൾ ചോദിക്കുക
    >>~ (൧, ൧) > "ടെർമിനൽ: " വരികൾ " x " കോളങ്ങൾ
    <<| കീ                         // ബ്ലോക്കിംഗ് കീ പ്രസ്സ് വായിക്കുക
    >>~ (൨, ൧) > "അമർത്തിയത്: " കീ
}
```

> `>>!` സ്ക്രീൻ മായ്ക്കുന്നു. `>>?` `[വരികൾ, കോളങ്ങൾ]` തിരികെ നൽകുന്നു. `@~ N` N മില്ലിസെക്കൻഡ് ഉറങ്ങുന്നു.
> `<<|` ഒരു കീ പ്രസ്സ് വായിക്കുന്നു (ബ്ലോക്കിംഗ്); `<<|?` ബ്ലോക്ക് ചെയ്യാതെ പോൾ ചെയ്യുന്നു (ഒന്നുമില്ലെങ്കിൽ `'\0'` തിരികെ നൽകുന്നു).
> സ്ഥാനീയ ഔട്ട്പുട്ട് ടപ്പിൾ: `(വരി, കോളം, BKS, fg, bg)` — ഏത് സ്ലോട്ടും കോമ ഉപയോഗിച്ച് ഒഴിവാക്കാം (`>>~ (,,, 196) > "ചുവപ്പ്"`).
> BKS ബിറ്റ്മാസ്ക്: `1`=ബോൾഡ്, `2`=ഇറ്റാലിക്, `4`=അണ്ടർലൈൻ. ANSI 256 നിറങ്ങളുടെ പാലറ്റ് (`0`=ടെർമിനൽ ഡിഫോൾട്ട്).
> tree-walker മാത്രം (`>>!`, `>>?`, `@~`, `>>~` ഒഴികെ ഇവ `--vm` ലും പ്രവർത്തിക്കുന്നു).

## ലൂപ്പുകൾ

```zymbol
@ i:൦..൪  { >> i " " }        // ശ്രേണി ഉൾപ്പെടുന്നു:  ൦ ൧ ൨ ൩ ൪
@ i:൧..൯:൨ { >> i " " }       // ഘട്ടത്തോടെ:         ൧ ൩ ൫ ൭ ൯
@ i:൫..൦:൧ { >> i " " }       // വിപരീതം:           ൫ ൪ ൩ ൨ ൧ ൦

n = ൧
@ n <= ൬൪ { n *= ൨ }
>> n ¶                        // → ൧൨൮  (while)

പഴങ്ങൾ = ["ആപ്പിൾ", "പിയർ", "മുന്തിരി"]
@ പ:പഴങ്ങൾ { >> പ ¶ }         // അറേയിലെ ഓരോ ഘടകത്തിനും

@ അ:"ഹലോ" { >> അ "-" }
>> ¶                          // → ഹ-ല-ോ-  (സ്ട്രിംഗിലെ ഓരോ കാരക്ടറിനും)

@ i:൧..൧൦ {
    ? i % ൨ == ൦ { @> }       // @> തുടരുക
    ? i > ൭ { @! }             // @! തകർക്കുക
    >> i " "
}
>> ¶                          // → ൧ ൩ ൫ ൭

// അനന്ത ലൂപ്പ്
i = ൦
@ {
    i++
    ? i >= ൫ { @! }
    >> i " "
}
>> ¶                          // → ൧ ൨ ൩ ൪

// ലേബൽ ചെയ്ത ലൂപ്പ് (നെസ്റ്റഡ് തകർക്കൽ)
എണ്ണം = ൦
@:ബാഹ്യ {
    എണ്ണം++
    ? എണ്ണം >= ൩ { @:ബാഹ്യ! }
}
>> എണ്ണം ¶                    // → ൩
```

## ഫംഗ്ഷനുകൾ

```zymbol
സങ്കലനം(a, b) { <~ a + b }
>> സങ്കലനം(൩, ൪) ¶    // → ൭

ഫാക്ടോറിയൽ(n) {
    ? n <= ൧ { <~ ൧ }
    <~ n * ഫാക്ടോറിയൽ(n - ൧)
}
>> ഫാക്ടോറിയൽ(൫) ¶    // → ൧൨൦
```

ഫംഗ്ഷനുകൾക്ക് **ഒറ്റപ്പെട്ട സ്കോപ്പ്** ഉണ്ട് — അവയ്ക്ക് ബാഹ്യ വേരിയബിളുകൾ വായിക്കാൻ കഴിയില്ല. കോളർ വേരിയബിളുകൾ മാറ്റുന്നതിന് ഔട്ട്പുട്ട് പാരാമീറ്ററുകൾ `<~` ഉപയോഗിക്കുക:

```zymbol
സ്വാപ്പ്(a<~, b<~) {
    താത്കാലിക = a
    a = b
    b = താത്കാലിക
}
x = ൧൦
y = ൨൦
സ്വാപ്പ്(x, y)
>> "x=" x " y=" y ¶    // → x=൨൦ y=൧൦
```

> നാമകരണം ചെയ്ത ഫംഗ്ഷനുകൾ **ഒന്നാം-ശ്രേണിയിലെ മൂല്യങ്ങളാണ്** — നേരിട്ട് പാസ് ചെയ്യുക: `സംഖ്യകൾ$> ഇരട്ടി`. പൊതിയുന്നതിന്: `x -> fn(x)` ഉം സാധുവാണ്.

## ലാംഡകളും ക്ലോഷറുകളും

```zymbol
ഇരട്ടി = x -> x * ൨
സങ്കലനം = (a, b) -> a + b
>> ഇരട്ടി(൫) ¶    // → ൧൦
>> സങ്കലനം(൩, ൭) ¶  // → ൧൦

// ബ്ലോക്ക് ലാംഡ
തരംതിരിക്കുക = x -> {
    ? x > ൦ { <~ "പോസിറ്റീവ്" }
    _? x < ൦ { <~ "നെഗറ്റീവ്" }
    <~ "പൂജ്യം"
}

// ക്ലോഷർ — ബാഹ്യ സ്കോപ്പ് പിടിച്ചെടുക്കുന്നു
ഘടകം = ൩
മൂന്നിരട്ടി = x -> x * ഘടകം
>> മൂന്നിരട്ടി(൭) ¶    // → ൨൧

// ഫാക്ടറി
സങ്കലന_നിർമ്മാതാവ്(n) { <~ x -> x + n }
പത്തിന്റെ_സങ്കലനം = സങ്കലന_നിർമ്മാതാവ്(൧൦)
>> പത്തിന്റെ_സങ്കലനം(൫) ¶    // → ൧൫

// അറേയിൽ
ഓപ്പറേറ്റർമാർ = [x -> x+൧, x -> x*൨, x -> x*x]
>> ഓപ്പറേറ്റർമാർ[൩](൫) ¶    // → ൨൫
```

## അറേ

അറേകൾ **മാറ്റാവുന്ന**വയും **ഒരേ തരത്തിലുള്ള** ഘടകങ്ങളെ ഉൾക്കൊള്ളുന്നവയുമാണ്.

```zymbol
arr = [൧, ൨, ൩, ൪, ൫]

x = arr[൧]      // ൧ — ആക്സസ് (൧-അടിസ്ഥാനം: ആദ്യ ഘടകം)
x = arr[-൧]     // ൫ — നെഗറ്റീവ് സൂചിക (അവസാന ഘടകം)
x = arr$#       // ൫ — നീളം (>> ൽ (arr$#) ഉപയോഗിക്കുക)

arr = arr$+ ൬            // ചേർക്കുക → [൧,൨,൩,൪,൫,൬]
arr൨ = arr$+[൨] ൯൯       // സ്ഥാനം ൨ ൽ തിരുകുക (൧-അടിസ്ഥാനം)
arr൩ = arr$- ൩           // മൂല്യത്തിന്റെ ആദ്യ സംഭവം നീക്കം ചെയ്യുക
arr൪ = arr$-- ൩          // എല്ലാ സംഭവങ്ങളും നീക്കം ചെയ്യുക
arr൫ = arr$-[൧]          // സൂചിക ൧ ൽ നീക്കം ചെയ്യുക (ആദ്യ ഘടകം)
arr൬ = arr$-[൨..൩]       // ശ്രേണി നീക്കം ചെയ്യുക (൧-അടിസ്ഥാനം, അവസാനം ഉൾപ്പെടുന്നു)

ഉണ്ട് = arr$? ൩            // #൧ — ഉൾക്കൊള്ളുന്നു
സ്ഥാനങ്ങൾ = arr$?? ൩           // [൩] — മൂല്യത്തിന്റെ എല്ലാ സൂചികകളും (൧-അടിസ്ഥാനം)
സ്ലൈസ് = arr$[൧..൩]          // [൧,൨,൩] — സ്ലൈസ് (൧-അടിസ്ഥാനം, അവസാനം ഉൾപ്പെടുന്നു)
സ്ലൈസ്൨ = arr$[൧:൩]          // [൧,൨,൩] — സമാനം, എണ്ണം-അടിസ്ഥാനമാക്കിയുള്ള വാക്യഘടന

ആരോഹണ = arr$^+             // ആരോഹണ ക്രമത്തിൽ അടുക്കുക (പ്രാകൃതങ്ങൾ മാത്രം)
അവരോഹണ = arr$^-            // അവരോഹണ ക്രമത്തിൽ അടുക്കുക (പ്രാകൃതങ്ങൾ മാത്രം)

// നാമകരണം ചെയ്ത/സ്ഥാനീയ ടപ്പിൾ അറേ — താരതമ്യ ലാംഡ ഉപയോഗിച്ച് $^ ഉപയോഗിക്കുക
ഡാറ്റാബേസ് = [(പേര്: "കാർല", പ്രായം: ൨൮), (പേര്: "അന", പ്രായം: ൨൫), (പേര്: "ബോബ്", പ്രായം: ൩൦)]
പ്രായം_അനുസരിച്ച്  = ഡാറ്റാബേസ്$^ (a, b -> a.പ്രായം < b.പ്രായം)    // പ്രായം അനുസരിച്ച് ആരോഹണ (<)
പേര്_അനുസരിച്ച് = ഡാറ്റാബേസ്$^ (a, b -> a.പേര് > b.പേര്)   // പേര് അനുസരിച്ച് അവരോഹണ (>)
>> പ്രായം_അനുസരിച്ച്[൧].പേര് ¶     // → അന
>> പേര്_അനുസരിച്ച്[൧].പേര് ¶    // → കാർല

// നേരിട്ടുള്ള ഘടക അപ്ഡേറ്റ് (അറേകൾ മാത്രം)
arr[൧] = ൯൯              // അസൈൻ ചെയ്യുക
arr[൨] += ൫              // സംയുക്തം: +=  -=  *=  /=  %=  ^=

// ഫംഗ്ഷണൽ അപ്ഡേറ്റ് — ഒരു പുതിയ അറേ തിരികെ നൽകുന്നു; യഥാർത്ഥം മാറ്റമില്ലാതെ
arr൨ = arr[൨]$~ ൯൯
```

> എല്ലാ ശേഖരണ ഓപ്പറേറ്റർമാരും ഒരു **പുതിയ അറേ** തിരികെ നൽകുന്നു. തിരികെ അസൈൻ ചെയ്യുക: `arr = arr$+ 4`.
> `$+` ശൃംഖലിപ്പിക്കാൻ കഴിയും: `arr = arr$+ 5$+ 6$+ 7`. മറ്റ് ഓപ്പറേറ്റർമാർ ഇന്റർമീഡിയറ്റ് അസൈൻമെന്റുകൾ ഉപയോഗിക്കുന്നു.
> **സൂചികവൽക്കരണം 1-അടിസ്ഥാനമാണ്**: `arr[1]` ആദ്യ ഘടകമാണ്; `arr[0]` ഒരു റൺടൈം പിശകാണ്.
> `$^+` / `$^-` **പ്രാകൃത അറേകളെ** (സംഖ്യകൾ, സ്ട്രിംഗുകൾ) അടുക്കുന്നു. ടപ്പിൾ അറേകൾക്ക് താരതമ്യ ലാംഡ ഉപയോഗിച്ച് `$^` ഉപയോഗിക്കുക — ദിശ ലാംഡയിൽ എൻകോഡ് ചെയ്തിരിക്കുന്നു (`<` = ആരോഹണ, `>` = അവരോഹണ).

**മൂല്യ സെമാന്റിക്സ്** — ഒരു അറേ മറ്റൊരു വേരിയബിളിലേക്ക് അസൈൻ ചെയ്യുന്നത് ഒരു സ്വതന്ത്ര പകർപ്പ് സൃഷ്ടിക്കുന്നു:

```zymbol
a = [൧, ൨, ൩]
b = a
a[൧] = ൯൯
>> a ¶    // → [൯൯, ൨, ൩]
>> b ¶    // → [൧, ൨, ൩]   ← b ബാധിക്കപ്പെട്ടിട്ടില്ല
```

```zymbol
// നെസ്റ്റഡ് അറേകൾ (൧-അടിസ്ഥാന സൂചികവൽക്കരണം)
മാട്രിക്സ് = [[൧,൨,൩],[൪,൫,൬],[൭,൮,൯]]
>> മാട്രിക്സ്[൨][൩] ¶    // → ൬  (വരി ൨, കോളം ൩)
```

```zymbol
// അറേ
arr = [൧൦, ൨൦, ൩൦, ൪൦, ൫൦]
[a, b, c] = arr              // a=൧൦  b=൨൦  c=൩൦
[ആദ്യത്തെ, *ബാക്കി] = arr         // ആദ്യത്തെ=൧൦  ബാക്കി=[൨൦,൩൦,൪൦,൫൦]

ബിന്ദു = (൧൦൦, ൨൦൦)

(പേര്: n, പ്രായം: p) = വ്യക്തി   // n="അന"  p=൨൫
```

## ടപ്പിൾ

ടപ്പിളുകൾ **മാറ്റാനാകാത്ത** ക്രമപ്പെടുത്തിയ കണ്ടെയ്നറുകളാണ്, അവയ്ക്ക് **വ്യത്യസ്ത തരത്തിലുള്ള** മൂല്യങ്ങൾ ഉൾക്കൊള്ളാൻ കഴിയും.
അറേകളിൽ നിന്ന് വ്യത്യസ്തമായി, സൃഷ്ടിച്ചതിന് ശേഷം ഘടകങ്ങൾ മാറ്റാൻ കഴിയില്ല.

```zymbol
// സ്ഥാനീയ — മിശ്രിത തരങ്ങൾ അനുവദനീയമാണ്
ബിന്ദു = (൧൦, ൨൦)
>> ബിന്ദു[൧] ¶    // → ൧൦

ഡാറ്റ = (൪൨, "ഹലോ", #൧, ൩.൧൪)
>> ഡാറ്റ[൩] ¶     // → #൧

// നാമകരണം ചെയ്തത്
വ്യക്തി = (പേര്: "ആലീസ്", പ്രായം: ൨൫)
>> വ്യക്തി.പേര് ¶    // → ആലീസ്
>> വ്യക്തി[൧] ¶      // → ആലീസ്  (സൂചികയും പ്രവർത്തിക്കുന്നു, ൧-അടിസ്ഥാനം)

// നെസ്റ്റഡ്
സ്ഥാനം = (x: ൧൦, y: ൨൦)
p = (സ്ഥാനം: സ്ഥാനം, ലേബൽ: "ഉത്ഭവം")
>> p.സ്ഥാനം.x ¶        // → ൧൦
```

**മാറ്റാനാകായ്മ** — ഒരു ടപ്പിൾ ഘടകം മാറ്റാനുള്ള ഏതൊരു ശ്രമവും റൺടൈം പിശകാണ്:

```zymbol
t = (൧൦, ൨൦, ൩൦)
// t[൧] = ൯൯    // ❌ റൺടൈം പിശക്: ടപ്പിളുകൾ മാറ്റാനാകാത്തവയാണ്
// t[൧] += ൫    // ❌ അതേ പിശക്
```

പരിഷ്കരിച്ച മൂല്യം നേടുന്നതിന് `$~` (ഫംഗ്ഷണൽ അപ്ഡേറ്റ്) ഉപയോഗിക്കുക — ഒരു **പുതിയ** ടപ്പിൾ തിരികെ നൽകുന്നു:

```zymbol
t = (൧൦, ൨൦, ൩൦)
t൨ = t[൨]$~ ൯൯൯
>> t ¶     // → (൧൦, ൨൦, ൩൦)   ← യഥാർത്ഥം മാറ്റമില്ല
>> t൨ ¶    // → (൧൦, ൯൯൯, ൩൦)

// നാമകരണം ചെയ്ത ടപ്പിൾ — വ്യക്തമായി പുനർനിർമ്മിക്കുക
വ്യക്തി = (പേര്: "ആലീസ്", പ്രായം: ൨൫)
വലുത്  = (പേര്: വ്യക്തി.പേര്, പ്രായം: ൨൬)
>> വ്യക്തി.പ്രായം ¶    // → ൨൫
>> വലുത്.പ്രായം ¶     // → ൨൬
```

## ഉയർന്ന-ക്രമ ഫംഗ്ഷനുകൾ

```zymbol
സംഖ്യകൾ = [൧, ൨, ൩, ൪, ൫, ൬, ൭, ൮, ൯, ൧൦]

ഇരട്ടിപ്പിച്ചവ  = സംഖ്യകൾ$> (x -> x * ൨)                  // map  → [൨,൪,൬…൨൦]
ഇരട്ട    = സംഖ്യകൾ$| (x -> x % ൨ == ൦)           // filter → [൨,൪,൬,൮,൧൦]
ആകെ    = സംഖ്യകൾ$< (൦, (ശേഖരണം, x) -> ശേഖരണം + x)     // reduce → ൫൫

// ഇന്റർമീഡിയറ്റുകളിലൂടെ ശൃംഖലിപ്പിക്കുക
പടി൧ = സംഖ്യകൾ$| (x -> x > ൩)
പടി൨ = പടി൧$> (x -> x * x)
>> പടി൨ ¶    // → [൧൬, ൨൫, ൩൬, ൪൯, ൬൪, ൮൧, ൧൦൦]

// നാമകരണം ചെയ്ത ഫംഗ്ഷനുകൾ നേരിട്ട് HOF ലേക്ക് പാസ് ചെയ്യാൻ കഴിയും
ഇരട്ടി(x) { <~ x * ൨ }
വലുതാണ്(x) { <~ x > ൫ }
r = സംഖ്യകൾ$> ഇരട്ടി       // ✅ നേരിട്ടുള്ള റഫറൻസ്
r = സംഖ്യകൾ$| വലുതാണ്       // ✅ നേരിട്ടുള്ള റഫറൻസ്
```

## പൈപ്പ് ഓപ്പറേറ്റർ

വലതുവശത്ത് എല്ലായ്പ്പോഴും പൈപ്പ് ചെയ്ത മൂല്യത്തിനായി `_` പ്ലേസ്ഹോൾഡർ ആവശ്യമാണ്:

```zymbol
ഇരട്ടി = x -> x * ൨
സങ്കലനം = (a, b) -> a + b
വർദ്ധന = x -> x + ൧

ഫ൧ = ൫ |> ഇരട്ടി(_)        // → ൧൦
ഫ൨ = ൧൦ |> സങ്കലനം(_, ൫)       // → ൧൫
ഫ൩ = ൫ |> സങ്കലനം(൨, _)        // → ൭

// ശൃംഖലിപ്പിച്ചത്
ഫ = ൫ |> ഇരട്ടി(_) |> വർദ്ധന(_) |> ഇരട്ടി(_)
>> ഫ ¶    // → ൨൨  (൫→൧൦→൧൧→൨൨)
```

## പിശക് കൈകാര്യം ചെയ്യൽ

```zymbol
!? {
    x = ൧൦ / ൦
} :! ##Div {
    >> "പൂജ്യം കൊണ്ടുള്ള ഹരണം" ¶
} :! {
    >> "മറ്റുള്ളവ: " _err ¶    // _err പിശക് സന്ദേശം ഉൾക്കൊള്ളുന്നു
} :> {
    >> "എപ്പോഴും പ്രവർത്തിക്കുന്നു" ¶
}
```

| തരം | എപ്പോൾ |
|------|------|
| `##Div` | പൂജ്യം കൊണ്ടുള്ള ഹരണം |
| `##IO` | ഫയൽ / സിസ്റ്റം |
| `##Index` | പരിധിക്ക് പുറത്തുള്ള സൂചിക |
| `##Type` | തരം പൊരുത്തക്കേട് |
| `##Parse` | ഡാറ്റ പാഴ്സിംഗ് |
| `##Network` | നെറ്റ്വർക്ക് പിശകുകൾ |
| `##_` | ഏത് പിശകും (എല്ലാം-പിടിക്കുന്നത്) |

## മൊഡ്യൂളുകൾ

```zymbol
// lib/calc.zy — മൊഡ്യൂൾ ബോഡി ബ്രെയ്സുകളിൽ ഉൾക്കൊള്ളിച്ചിരിക്കുന്നു
# calc {
    #> { സങ്കലനം, get_PI }

    _π := ൩.൧൪൧൫൯
    സങ്കലനം(a, b) { <~ a + b }
    get_PI() { <~ _π }
}
```

```zymbol
// main.zy
<# ./lib/calc => സി    // അപരനാമം ആവശ്യമാണ്

>> സി::സങ്കലനം(൫, ൩) ¶     // → ൮
π = സി::get_PI()
>> π ¶               // → ൩.൧൪൧൫൯
```

```zymbol
// വ്യത്യസ്ത പൊതു നാമത്തിൽ കയറ്റുമതി ചെയ്യുക
# mylib {
    #> { _ആന്തരിക_സങ്കലനം => തുക }

    _ആന്തരിക_സങ്കലനം(a, b) { <~ a + b }
}
```

```zymbol
<# ./mylib => എം

>> എം::തുക(൩, ൪) ¶    // → ൭  (ആന്തരിക നാമം _ആന്തരിക_സങ്കലനം മറച്ചിരിക്കുന്നു)
```

> **മൊഡ്യൂൾ നിയമങ്ങൾ**: `# പേര് { }` എന്നതിനുള്ളിൽ `#>`, ഫംഗ്ഷൻ നിർവചനങ്ങൾ, അക്ഷരാർത്ഥ വേരിയബിൾ/സ്ഥിരാങ്കം ആരംഭകങ്ങൾ മാത്രമേ അനുവദനീയമുള്ളൂ. എക്സിക്യൂട്ടബിൾ പ്രസ്താവനകൾ (`>>`, `<<`, ലൂപ്പുകൾ, മുതലായവ) പിശക് E013 ഉയർത്തുന്നു.

## സംഖ്യാ മോഡുകൾ

Zymbol ന് സംഖ്യകൾ **69 യൂണികോഡ് അക്ക ലിപികളിൽ** പ്രദർശിപ്പിക്കാൻ കഴിയും — ദേവനാഗരി, അറബി-ഇന്ത്യൻ, തായ്, ക്ലിംഗോൺ pIqaD, ഗണിത ബോൾഡ്, എൽസിഡി സെഗ്മെന്റുകൾ, കൂടാതെ മറ്റും. സജീവ മോഡ് `>>` ഔട്ട്പുട്ടിനെ മാത്രം ബാധിക്കുന്നു; ആന്തരിക ഗണിതം എല്ലായ്പ്പോഴും ബൈനറിയാണ്.

### ഒരു ലിപി സജീവമാക്കുന്നു

ലക്ഷ്യ ലിപിയുടെ `0` ഉം `9` ഉം അക്കങ്ങൾ `#…#` എന്നതിനുള്ളിൽ എഴുതുക:

```zymbol
#०९#    // ദേവനാഗരി   (U+0966–U+096F)
#٠٩#    // അറബി-ഇന്ത്യൻ (U+0660–U+0669)
#๐๙#    // തായ്         (U+0E50–U+0E59)
#09#    // എഎസ്സിഐഐ-യിലേക്ക് പുനഃസജ്ജമാക്കുക
```

### ഔട്ട്പുട്ടും ബൂളിയനുകളും

```zymbol
x = ൪൨
>> x ¶          // → ൪൨   (എഎസ്സിഐഐ ഡിഫോൾട്ട്)

#०९#
>> x ¶          // → ४२
>> ൩.൧൪ ¶       // → ३.१४   (ദശാംശ ബിന്ദു എല്ലായ്പ്പോഴും എഎസ്സിഐഐ)
>> ൧ + ൨ ¶      // → ३

// ബൂളിയനുകൾ: # പൂർവ്വപ്രത്യയം എല്ലായ്പ്പോഴും എഎസ്സിഐഐ, അക്കം പൊരുത്തപ്പെടുന്നു
>> #൧ ¶         // → #१   (ദേവനാഗരിയിൽ സത്യം)
>> #൦ ¶         // → #०   (തെറ്റ് — ൦ പൂർണ്ണസംഖ്യ പൂജ്യത്തിൽ നിന്ന് വ്യത്യസ്തമാണ്)

x = ൨൮ > ൪
>> x ¶          // → #१   (താരതമ്യ ഫലം സജീവ മോഡ് പിന്തുടരുന്നു)
```

### ഉറവിടത്തിൽ നേറ്റീവ് അക്ക ലിറ്ററലുകൾ

പിന്തുണയ്ക്കുന്ന ഏത് ലിപിയുടെയും അക്കങ്ങൾ സാധുവായ ലിറ്ററലുകളാണ് — ശ്രേണികളിൽ, മോഡുലോയിൽ, താരതമ്യങ്ങളിൽ:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### ഏത് ലിപിയിലും ബൂളിയൻ ലിറ്ററലുകൾ

`#` + ഏത് ബ്ലോക്കിൽ നിന്നുമുള്ള അക്കം `0` അല്ലെങ്കിൽ `1` ഒരു സാധുവായ ബൂളിയൻ ലിറ്ററലാണ്:

```zymbol
#٠٩#
സജീവം = #١        // #൧ ന് തുല്യമാണ്
>> സജീവം ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **എല്ലായ്പ്പോഴും എഎസ്സിഐഐ** ആണ്. `#0` (തെറ്റ്) എല്ലായ്പ്പോഴും ഓരോ ലിപിയിലും `0` (പൂർണ്ണസംഖ്യ പൂജ്യം) എന്നതിൽ നിന്ന് ദൃശ്യപരമായി വ്യത്യസ്തമാണ്.

## ഡാറ്റാ ഓപ്പറേറ്റർമാർ

```zymbol
// തരം പരിവർത്തന കാസ്റ്റുകൾ
f = ##.൪൨         // → ൪൨.൦  (ഫ്ലോട്ടിംഗിലേക്ക്)
i = ###൩.൭        // → ൪     (പൂർണ്ണസംഖ്യയിലേക്ക്, റൗണ്ട്)
t = ##!൩.൭        // → ൩     (പൂർണ്ണസംഖ്യയിലേക്ക്, ട്രങ്കേറ്റ്)

// സ്ട്രിംഗിനെ സംഖ്യയിലേക്ക് പാഴ്സ് ചെയ്യുക
v൧ = #|"൪൨"|      // → ൪൨  (പൂർണ്ണസംഖ്യ)
v൨ = #|"൩.൧൪"|    // → ൩.൧൪  (ഫ്ലോട്ടിംഗ്)
v൩ = #|"abc"|     // → "abc"  (സുരക്ഷിതം, പിശകില്ല)

// റൗണ്ട് / ട്രങ്കേറ്റ്
π = ൩.൧൪൧൫൯൨൬൫
റൗണ്ട്൨ = #.൨|π|      // → ൩.൧൪  (൨ ദശാംശ സ്ഥാനങ്ങളിലേക്ക് റൗണ്ട്)
റൗണ്ട്൪ = #.൪|π|      // → ൩.൧൪൧൬
ട്രങ്ക്൨ = #!൨|π|      // → ൩.൧൪  (ട്രങ്കേറ്റ്)

// സംഖ്യ ഫോർമാറ്റിംഗ്
ഫോർമാറ്റ് = #,|൧൨൩൪൫൬൭|  // → ൧,൨൩൪,൫൬൭  (കോമ-വേർതിരിച്ചത്)
ശാസ്ത്രീയ = #^|൧൨൩൪൫.൬൭൮|    // → ൧.൨൩൪൫൬൭൮e൪  (ശാസ്ത്രീയ)

// അടിസ്ഥാന ലിറ്ററലുകൾ
a = 0x41         // → 'A'  (ഹെക്സ്)
b = 0b01000001   // → 'A'  (ബൈനറി)
c = 0o101        // → 'A'  (ഒക്ടൽ)

// അടിസ്ഥാന പരിവർത്തന ഔട്ട്പുട്ട്
ഹെക്സ് = 0x|൨൫൫|    // → "0x00FF"
ബൈനറി = 0b|൬൫|     // → "0b1000001"
ഒക്ടൽ = 0o|൮|      // → "0o10"
ദശാംശം = 0d|൨൫൫|    // → "0d0255"
```

## ഷെൽ സംയോജനം

```zymbol
തീയതി = <\ date +%Y-%m-%d \>     // stdout പിടിച്ചെടുക്കുന്നു (അവസാനത്തെ \n ഉൾപ്പെടെ)
>> "ഇന്ന്: " തീയതി

ഫയൽ = "data.txt"
ഉള്ളടക്കം = <\ cat {ഫയൽ} \>      // കമാൻഡുകളിൽ ഇന്റർപോളേഷൻ

ഔട്ട്പുട്ട് = </"./subscript.zy"/>   // മറ്റൊരു Zymbol സ്ക്രിപ്റ്റ് എക്സിക്യൂട്ട് ചെയ്യുക, ഔട്ട്പുട്ട് പിടിച്ചെടുക്കുക
>> ഔട്ട്പുട്ട്
```

> `><` CLI ആർഗ്യുമെന്റുകളെ സ്ട്രിംഗ് അറേയായി പിടിച്ചെടുക്കുന്നു (tree-walker മാത്രം).

## ചിഹ്ന റഫറൻസ്

| ചിഹ്നം | പ്രവർത്തനം | ചിഹ്നം | പ്രവർത്തനം |
|--------|-----------|--------|-----------|
| `=` | വേരിയബിൾ | `$#` | നീളം |
| `:=` | സ്ഥിരാങ്കം | `$+` | ചേർക്കുക (ശൃംഖലിപ്പിക്കാവുന്ന) |
| `>>` | ഔട്ട്പുട്ട് | `$+[i]` | സൂചികയിൽ തിരുകുക (1-അടിസ്ഥാനം) |
| `<<` | ഇൻപുട്ട് | `$-` | മൂല്യം ഉപയോഗിച്ച് ആദ്യത്തേത് നീക്കം ചെയ്യുക |
| `¶` / `\\` | പുതിയ വരി | `$--` | മൂല്യം ഉപയോഗിച്ച് എല്ലാം നീക്കം ചെയ്യുക |
| `?` | എങ്കിൽ | `$-[i]` | സൂചികയിൽ നീക്കം ചെയ്യുക (1-അടിസ്ഥാനം) |
| `_?` | അല്ലെങ്കിൽ-എങ്കിൽ | `$-[i..j]` | ശ്രേണി നീക്കം ചെയ്യുക (1-അടിസ്ഥാനം) |
| `_` | അല്ലെങ്കിൽ / വൈൽഡ്കാർഡ് | `$?` | ഉൾക്കൊള്ളുന്നു |
| `??` | പൊരുത്തം | `$??` | എല്ലാ സൂചികകളും കണ്ടെത്തുക (1-അടിസ്ഥാനം) |
| `@` | ലൂപ്പ് | `$[s..e]` | സ്ലൈസ് (1-അടിസ്ഥാനം) |
| `@ N { }` | N തവണ ലൂപ്പ് | `$>` | map |
| `@!` | തകർക്കുക | `$\|` | filter |
| `@>` | തുടരുക | `$<` | reduce |
| `@:പേര് { }` | ലേബൽ ചെയ്ത ലൂപ്പ് | `$/ ഡിലിമിറ്റർ` | സ്ട്രിംഗ് വിഭജിക്കുക |
| `@:പേര്!` | ലേബൽ തകർക്കുക | `$++ a b c` | കോൺകാറ്റ് നിർമ്മിക്കുക |
| `@:പേര്>` | ലേബൽ തുടരുക | `arr[i>j>k]` | നാവിഗേഷൻ സൂചിക |
| `->` | ലാംഡ | `arr[i] = മൂല്യം` | ഘടകം അപ്ഡേറ്റ് ചെയ്യുക (അറേകൾ മാത്രം) |
| `arr[i] += മൂല്യം` | സംയുക്ത അപ്ഡേറ്റ് | `arr[i]$~` | ഫംഗ്ഷണൽ അപ്ഡേറ്റ് (പുതിയ പകർപ്പ്) |
| `$^+` | ആരോഹണ അടുക്കുക (പ്രാകൃതം) | `$^-` | അവരോഹണ അടുക്കുക (പ്രാകൃതം) |
| `$^` | താരതമ്യ ഉപയോഗിച്ച് അടുക്കുക (ടപ്പിൾ) | `<~` | തിരികെ നൽകുക |
| `\|>` | പൈപ്പ് | `!?` | ശ്രമിക്കുക |
| `:!` | പിടിക്കുക | `:>` | ഒടുവിൽ |
| `#1` | സത്യം | `#0` | തെറ്റ് |
| `$!` | പിശകാണോ | `$!!` | പിശക് പ്രചരിപ്പിക്കുക |
| `<#` | ഇറക്കുമതി ചെയ്യുക | `#>` | കയറ്റുമതി ചെയ്യുക |
| `#` | മൊഡ്യൂൾ പ്രഖ്യാപിക്കുക | `::` | മൊഡ്യൂൾ വിളിക്കുക |
| `.` | ഫീൽഡ് ആക്സസ് | `#?` | തരം മെറ്റാഡാറ്റ |
| `#\|..\|` | സംഖ്യ പാഴ്സ് ചെയ്യുക | `##.` | ഫ്ലോട്ടിംഗിലേക്ക് പരിവർത്തനം ചെയ്യുക |
| `###` | പൂർണ്ണസംഖ്യയിലേക്ക് പരിവർത്തനം ചെയ്യുക (റൗണ്ട്) | `##!` | പൂർണ്ണസംഖ്യയിലേക്ക് പരിവർത്തനം ചെയ്യുക (ട്രങ്കേറ്റ്) |
| `#.N\|..\|` | റൗണ്ട് | `#!N\|..\|` | ട്രങ്കേറ്റ് |
| `#,\|..\|` | കോമ ഫോർമാറ്റ് | `#^\|..\|` | ശാസ്ത്രീയ |
| `#d0d9#` | സംഖ്യാ മോഡ് മാറ്റുക | `#09#` | എഎസ്സിഐഐ-യിലേക്ക് പുനഃസജ്ജമാക്കുക |
| `<\ ..\>` | ഷെൽ എക്സിക്യൂട്ട് ചെയ്യുക | `>\<` | CLI ആർഗ്യുമെന്റുകൾ |
| `\ വേരിയബിൾ` | വ്യക്തമായി വേരിയബിൾ നശിപ്പിക്കുക | `°x` / `x°` | ഹോട്ട് നിർവചനം (ഓട്ടോ-ആരംഭം) |
| `>>|` | ടിയുഐ ബ്ലോക്ക് (ഇതര സ്ക്രീൻ) | `>>~` | സ്ഥാനീയ ഔട്ട്പുട്ട് |
| `>>!` | സ്ക്രീൻ മായ്ക്കുക | `>>?` | ടെർമിനൽ വലുപ്പം ചോദിക്കുക |
| `<<\|` | ബ്ലോക്കിംഗ് കീ പ്രസ്സ് | `<<\|?` | നോൺ-ബ്ലോക്കിംഗ് കീ പ്രസ്സ് പോൾ |
| `@~ N` | N മില്ലിസെക്കൻഡ് ഉറങ്ങുക | `$*` | സ്ട്രിംഗ് N തവണ ആവർത്തിക്കുക |

### v0.0.5 — ടിയുഐ പ്രിമിറ്റീവുകൾ, ഹോട്ട് നിർവചനം & സ്ട്രിംഗ് ആവർത്തനം _(മേയ് 2026)_

- **ബ്രേക്കിംഗ്** മാച്ച് ആം സെപ്പറേറ്റർ: `പാറ്റേൺ : ഫലം` → `പാറ്റേൺ => ഫലം`
- **ബ്രേക്കിംഗ്** ഇറക്കുമതി അപരനാമം: `<# പാത്ത് <= അപരനാമം` → `<# പാത്ത് => അപരനാമം`
- **ബ്രേക്കിംഗ്** കയറ്റുമതി പുനർനാമകരണം: `#> { fn <= പബ് }` → `#> { fn => പബ് }`
- **ചേർത്തു** ടിയുഐ ബ്ലോക്ക് `>>| { }` — ഇതര സ്ക്രീൻ + റോ മോഡ്; പുറത്തുകടക്കുമ്പോൾ വൃത്തിയാക്കുന്നു
- **ചേർത്തു** സ്ഥാനീയ ഔട്ട്പുട്ട് `>>~ (വരി, കോളം, BKS, fg, bg) > ഇനങ്ങൾ` — അപൂർവ സ്ലോട്ടുകൾ, ANSI 256 നിറം
- **ചേർത്തു** കീ ഇൻപുട്ട് `<<| വേരിയബിൾ` (ബ്ലോക്കിംഗ്) കൂടാതെ `<<|? വേരിയബിൾ` (നോൺ-ബ്ലോക്കിംഗ് പോൾ)
- **ചേർത്തു** `>>!` സ്ക്രീൻ മായ്ക്കുക, `>>?` ടെർമിനൽ വലുപ്പം ചോദിക്കുക, `@~ N` N മില്ലിസെക്കൻഡ് ഉറങ്ങുക
- **ചേർത്തു** ഹോട്ട് നിർവചനം `°x` / `x°` — ലൂപ്പുകളിൽ ആദ്യ ഉപയോഗത്തിൽ വേരിയബിൾ യാന്ത്രിക ആരംഭം
- **ചേർത്തു** സ്ട്രിംഗ് ആവർത്തനം `സ്ട്രിംഗ് $* N` — ഒരു സ്ട്രിംഗ് N തവണ ആവർത്തിക്കുക
- **വിഎം** സമത്വം: 436/436 ടെസ്റ്റുകൾ വിജയിക്കുന്നു

### v0.0.4 — 1-അടിസ്ഥാന സൂചികവൽക്കരണം, ഒന്നാം-ശ്രേണി ഫംഗ്ഷനുകൾ & ബ്ലോക്ക് മൊഡ്യൂളുകൾ _(ഏപ്രിൽ 2026)_

- **ബ്രേക്കിംഗ്** എല്ലാ സൂചികവൽക്കരണവും **1-അടിസ്ഥാന** ആയി മാറ്റി — `arr[1]` ആദ്യ ഘടകമാണ്; `arr[0]` ഒരു റൺടൈം പിശകാണ്
- **ചേർത്തു** നാമകരണം ചെയ്ത ഫംഗ്ഷനുകൾ **ഒന്നാം-ശ്രേണിയിലെ മൂല്യങ്ങളാണ്** — നേരിട്ട് HOF ലേക്ക് പാസ് ചെയ്യുക: `സംഖ്യകൾ$> ഇരട്ടി`
- **ചേർത്തു** മൊഡ്യൂളുകൾക്കായുള്ള **ബ്ലോക്ക് വാക്യഘടന ആവശ്യമാണ്** — `# പേര് { ... }` — ഫ്ലാറ്റ് വാക്യഘടന നീക്കം ചെയ്തു
- **ചേർത്തു** ബഹു-മാന സൂചികവൽക്കരണം: `arr[i>j>k]` (നാവിഗേഷൻ), `arr[p ; q]` (ഫ്ലാറ്റ് എക്സ്ട്രാക്ഷൻ)
- **ചേർത്തു** തരം പരിവർത്തന കാസ്റ്റുകൾ: `##.എക്സ്പ്രെഷൻ` (ഫ്ലോട്ടിംഗ്), `###എക്സ്പ്രെഷൻ` (പൂർണ്ണസംഖ്യ റൗണ്ട്), `##!എക്സ്പ്രെഷൻ` (പൂർണ്ണസംഖ്യ ട്രങ്കേറ്റ്)
- **ചേർത്തു** സ്ട്രിംഗ് വിഭജനം: `സ്ട്രിംഗ്$/ ഡിലിമിറ്റർ` — `Array(സ്ട്രിംഗ്)` തിരികെ നൽകുന്നു
- **ചേർത്തു** കോൺകാറ്റ് നിർമ്മിക്കുക: `അടിസ്ഥാനം$++ a b c` — ഒന്നിലധികം ഇനങ്ങൾ ചേർക്കുന്നു- **ചേർത്തു** ടൈംസ് ലൂപ്പ്: `@ N { }` — കൃത്യമായി N തവണ ആവർത്തിക്കുക
- **ചേർത്തു** ലേബൽ ചെയ്ത ലൂപ്പ് വാക്യഘടന: `@:പേര് { }`, `@:പേര്!`, `@:പേര്>` — `@ @പേര്` / `@! പേര്` എന്നതിനെ മാറ്റിസ്ഥാപിക്കുന്നു
- **ചേർത്തു** വേരിയബിൾ സ്കോപ്പ് നിയമങ്ങൾ: `_പേര്` വേരിയബിളുകൾക്ക് കൃത്യമായ ബ്ലോക്ക് സ്കോപ്പ് ഉണ്ട്; `\ വേരിയബിൾ` നേരത്തെ നശിപ്പിക്കുന്നു
- **ചേർത്തു** മാച്ച് താരതമ്യ പാറ്റേണുകൾ: `< 0 =>`, `> 5 =>`, `== 42 =>` മുതലായവ
- **ചേർത്തു** മൊഡ്യൂൾ E013 പിശക്: മൊഡ്യൂൾ ബോഡിയിൽ എക്സിക്യൂട്ടബിൾ പ്രസ്താവനകൾ നിരോധിച്ചിരിക്കുന്നു
- **ശരിയാക്കി** `alias.CONST` ഇപ്പോൾ ശരിയായി പരിഹരിക്കുന്നു; `#>` ഫംഗ്ഷൻ നിർവചനങ്ങൾക്ക് ശേഷം വരാം
- **വിഎം** പൂർണ്ണ സമത്വം: 393/393 ടെസ്റ്റുകൾ വിജയിക്കുന്നു
