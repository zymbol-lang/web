# Zymbol-Lang ਦੀ ਸੰਖੇਪ ਗਾਈਡ

**Zymbol-Lang** ਇੱਕ ਪ੍ਰਤੀਕਾਤਮਕ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾ ਹੈ। ਇਹ ਕੋਈ ਵਿਸ਼ੇਸ਼ ਸ਼ਬਦ (keywords) ਨਹੀਂ ਵਰਤਦੀ — ਸਭ ਕੁਝ ਪ੍ਰਤੀਕ ਹੈ। ਇਹ ਹਰ ਮਨੁੱਖੀ ਭਾਸ਼ਾ ਵਿੱਚ ਇੱਕੋ ਤਰੀਕੇ ਨਾਲ ਕੰਮ ਕਰਦੀ ਹੈ।

- ਕੋਈ ਵਿਸ਼ੇਸ਼ ਸ਼ਬਦ ਨਹੀਂ (`if`, `while`, `return` ਨਹੀਂ — ਸਿਰਫ਼ ਪ੍ਰਤੀਕ `?`, `@`, `<~`)
- ਪੂਰੀ Unicode ਸਹਾਇਤਾ — ਕਿਸੇ ਵੀ ਭਾਸ਼ਾ ਜਾਂ ਇਮੋਜੀ ਤੋਂ ਪਛਾਣਕਰਤਾ 👋
- ਭਾਸ਼ਾ-ਨਿਰਪੱਖ — ਕੋਡ ਸਾਰੀਆਂ ਭਾਸ਼ਾਵਾਂ ਵਿੱਚ ਇੱਕਸਾਰ ਹੈ

## ਚਰ ਅਤੇ ਸਥਿਰਾਂਕ

```zymbol
ਚ = ੧੦            // ਚਰ (ਬਦਲਣਯੋਗ)
ਪਾਈ := ੩.੧੪੧੫੯    // ਸਥਿਰਾਂਕ (ਨਾ-ਬਦਲਣਯੋਗ — ਮੁੜ ਤਫ਼ਵੀਜ਼ ਗਲਤੀ ਹੈ)
ਨਾਮ = "ਅਨੂ"
ਸਕਿਰਿਆ = #੧      // Boolean ਸੱਚ
👋 := "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ"
```

```zymbol
ਚ = ੧੦
ਚ += ੫    // ੧੫
ਚ -= ੩    // ੧੨
ਚ *= ੨    // ੨੪
ਚ /= ੩    // ੮
ਚ %= ੩    // ੨
ਚ ^= ੨    // ੪
ਚ++       // ੫
ਚ--       // ੪
```

## ਡੇਟਾ ਕਿਸਮਾਂ ਦੀ ਸਾਰਣੀ

| ਕਿਸਮ            | ਉਦਾਹਰਨ              | `#?` ਪ੍ਰਤੀਕ | ਨੋਟ                                      |
|-----------------|---------------------|-------------|------------------------------------------|
| ਪੂਰਨ ਅੰਕ        | `੪੨`, `-੭`          | `###`       | 64-bit ਚਿੰਨ੍ਹ ਸਮੇਤ                      |
| ਦਸ਼ਮਲਵ          | `੩.੧੪`, `੧.੫e੧੦`    | `##.`       | ਵਿਗਿਆਨਕ ਸੰਕੇਤਨ ਮਾਨਯੋਗ                   |
| ਸਤਰ             | `"ਹੈਲੋ"`            | `##"`       | ਅੰਤਰ-ਪੋਲੇਸ਼ਨ: `"ਹੈਲੋ {ਨਾਮ}"`            |
| ਅੱਖਰ            | `'A'`               | `##'`       | ਇੱਕ Unicode ਅੱਖਰ                         |
| Boolean         | `#੧`, `#੦`          | `##?`       | ਅੰਕੀ 1 ਅਤੇ 0 ਨਹੀਂ                       |
| ਐਰੇ (Array)     | `[੧, ੨, ੩]`         | `##]`       | ਸਾਰੇ ਤੱਤ ਇੱਕੋ ਕਿਸਮ ਦੇ                   |
| ਟੂਪਲ           | `(a, b)`            | `##)`       | ਸਥਿਤੀ-ਅਧਾਰਿਤ                            |
| ਨਾਮੀ ਟੂਪਲ      | `(x: ੧, y: ੨)`      | `##)`       | ਨਾਮ ਜਾਂ ਸੂਚਕਾਂਕ ਤੋਂ ਪਹੁੰਚ               |

```zymbol
// ਕਿਸਮ ਜਾਂਚ — (type, digits, value) ਵਾਪਸ ਕਰਦਾ ਹੈ
meta = ੪੨#?
>> meta ¶         // → (###, ੨, ੪੨)
t = meta[੦]
>> t ¶            // → ###
```

## ਆਉਟਪੁੱਟ ਅਤੇ ਇਨਪੁੱਟ

```zymbol
// ਆਉਟਪੁੱਟ — ਆਪੇ ਨਵੀਂ ਲਾਈਨ ਨਹੀਂ ਆਉਂਦੀ
>> "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ, ਦੁਨੀਆ!" ¶        // ¶ ਜਾਂ \\ ਸਪੱਸ਼ਟ ਨਵੀਂ ਲਾਈਨ ਦਿੰਦਾ ਹੈ
>> "ਅ=" ਅ " ਬ=" ਬ ¶                  // ਕਈ ਮੁੱਲ ਨਾਲ-ਨਾਲ ਰੱਖ ਕੇ
>> "ਜੋੜ=" ਜੋੜ(੨, ੩) ¶               // ਫੰਕਸ਼ਨ ਕਾਲ ਕਿਸੇ ਵੀ ਥਾਂ
>> (ਸੂਚੀ$#) ¶                        // postfix ਆਪਰੇਟਰਾਂ ਨੂੰ >> ਵਿੱਚ ਬਰੈਕਟ ਚਾਹੀਦੇ ਹਨ

// ਇਨਪੁੱਟ
<< ਨਾਮ                               // ਸੰਕੇਤ ਤੋਂ ਬਿਨਾਂ — ਚਰ ਵਿੱਚ ਪੜ੍ਹਦਾ ਹੈ
<< "ਆਪਣਾ ਨਾਮ? " ਨਾਮ                 // ਸੰਕੇਤ ਸਮੇਤ
```

> `¶` ਅਤੇ `\\` ਨਵੀਂ ਲਾਈਨ ਲਈ ਬਰਾਬਰ ਹਨ।

## ਸਤਰ

```zymbol
// ਤਿੰਨ ਜੋੜਨ ਦੇ ਤਰੀਕੇ
ਨਾਮ = "ਅਨੂ"
ਉ = ੨੫

ਸੁਨੇਹਾ = "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ ", ਨਾਮ, "!"     // ਕੌਮਾ — ਅਸਾਈਨਮੈਂਟ ਵਿੱਚ
>> "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ " ਨਾਮ " ਤੁਸੀਂ " ਉ " ਸਾਲ ਦੇ ਹੋ" ¶    // juxtaposition — >> ਵਿੱਚ
ਜਾਣ-ਪਛਾਣ = "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ {ਨਾਮ}, ਤੁਸੀਂ {ਉ} ਸਾਲ ਦੇ ਹੋ"  // interpolation — ਕਿਤੇ ਵੀ
```

```zymbol
ਸ = "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ ਦੁਨੀਆ"
ਲ = ਸ$#                  // length
ਉ = ਸ$[੦..੫]             // slice (end exclusive)
ਹ = ਸ$? "ਦੁਨੀ"           // #੧
ਭ = "ਕ,ਖ,ਗ,ਘ" / ','      // [ਕ, ਖ, ਗ, ਘ]
ਬ = ਸ$~~["ਅ":"ਅ਼"]         // replace
ਬ੧ = ਸ$~~["ਅ":"ਅ਼":੧]      // first N
```

> `+` ਕੇਵਲ ਸੰਖਿਆਵਾਂ ਲਈ। Strings ਲਈ `,`, juxtaposition ਜਾਂ interpolation ਵਰਤੋ।

## ਪ੍ਰਵਾਹ ਨਿਯੰਤਰਣ

```zymbol
ਚ = ੭

// ਸਾਦਾ ਜੇ
? ਚ > ੦ { >> "ਸਕਾਰਾਤਮਕ" ¶ }

// ਜੇ / ਨਹੀਂ ਤਾਂ ਜੇ / ਨਹੀਂ ਤਾਂ
? ਚ > ੧੦੦ {
    >> "ਵੱਡਾ" ¶
} _? ਚ > ੦ {
    >> "ਸਕਾਰਾਤਮਕ" ¶
} _? ਚ == ੦ {
    >> "ਜ਼ੀਰੋ" ¶
} _ {
    >> "ਨਕਾਰਾਤਮਕ" ¶
}
```

`{ }` ਬਲਾਕ **ਜ਼ਰੂਰੀ ਹਨ**, ਭਾਵੇਂ ਸਿਰਫ਼ ਇੱਕ ਲਾਈਨ ਹੋਵੇ।

## ਮੇਲ (Match)

```zymbol
// ਅੰਕੀ ਰੇਂਜ ਨਾਲ ਮੇਲ
ਅੰਕ = ੮੫
ਦਰਜਾ = ?? ਅੰਕ {
    ੯੦..੧੦੦ : 'A'
    ੮੦..੮੯  : 'B'
    ੭੦..੭੯  : 'C'
    _       : 'F'
}
>> ਦਰਜਾ ¶    // → B

// ਗਾਰਡਾਂ ਨਾਲ ਮੇਲ (ਮਨਮਾਨੀਆਂ ਸ਼ਰਤਾਂ)
ਤਾਪਮਾਨ = -੫
ਹਾਲਤ = ?? ਤਾਪਮਾਨ {
    _? ਤਾਪਮਾਨ < ੦  : "ਬਰਫ਼"
    _? ਤਾਪਮਾਨ < ੨੦ : "ਠੰਡਾ"
    _? ਤਾਪਮਾਨ < ੩੫ : "ਗਰਮ"
    _               : "ਬਹੁਤ ਗਰਮ"
}
>> ਹਾਲਤ ¶    // → ਬਰਫ਼

// ਸਤਰਾਂ ਨਾਲ ਮੇਲ
ਰੰਗ = "ਲਾਲ"
ਕੋਡ = ?? ਰੰਗ {
    "ਲਾਲ"   : "#FF੦੦੦੦"
    "ਹਰਾ"   : "#੦੦FF੦੦"
    _        : "#੦੦੦੦੦੦"
}
>> ਕੋਡ ¶

// ਬਲਾਕ ਆਰਮਜ਼
?? ਨ {
    ੦       : { >> "ਜ਼ੀਰੋ" ¶ }
    _? ਨ < ੦: { >> "ਨਕਾਰਾਤਮਕ" ¶ }
    _       : { >> "ਸਕਾਰਾਤਮਕ" ¶ }
}
```

## ਲੂਪ

```zymbol
@ ਇ:੦..੪  { >> ਇ " " }        // ਰੇਂਜ ਸ਼ਾਮਲ:  ੦ ੧ ੨ ੩ ੪
@ ਇ:੧..੯:੨ { >> ਇ " " }       // ਕਦਮ ਨਾਲ:    ੧ ੩ ੫ ੭ ੯
@ ਇ:੫..੦:੧ { >> ਇ " " }       // ਉਲਟਾ:       ੫ ੪ ੩ ੨ ੧ ੦

ਨ = ੧
@ ਨ <= ੬੪ { ਨ *= ੨ }
>> ਨ ¶                        // → ੧੨੮  (while)

ਫਲ = ["ਸੇਬ", "ਨਾਸ਼ਪਾਤੀ", "ਅੰਗੂਰ"]
@ ਫ:ਫਲ { >> ਫ ¶ }             // for-each array

@ ਅ:"ਹੈਲੋ" { >> ਅ "-" }
>> ¶                          // → ਹ-ੈ-ਲ-ੋ-  (for-each string)

@ ਇ:੧..੧੦ {
    ? ਇ % ੨ == ੦ { @> }       // @> continue
    ? ਇ > ੭ { @! }             // @! break
    >> ਇ " "
}
>> ¶                          // → ੧ ੩ ੫ ੭

// ਅਨੰਤ ਲੂਪ
ਇ = ੦
@ {
    ਇ++
    ? ਇ >= ੫ { @! }
    >> ਇ " "
}
>> ¶                          // → ੧ ੨ ੩ ੪

// ਲੇਬਲਡ ਲੂਪ (nested break)
count = ੦
@ @outer {
    count++
    ? count >= ੩ { @! outer }
}
>> count ¶                    // → ੩
```

## ਫੰਕਸ਼ਨ

```zymbol
// ਘੋਸ਼ਣਾ ਅਤੇ ਕਾਲ
ਜੋੜ(ਅ, ਬ) { <~ ਅ + ਬ }
>> ਜੋੜ(੩, ੪) ¶    // → ੭

// ਪੁਨਰਾਵ੍ਰਿੱਤੀ (Recursion)
ਗੁਣਾ(ਨ) {
    ? ਨ <= ੧ { <~ ੧ }
    <~ ਨ * ਗੁਣਾ(ਨ - ੧)
}
>> ਗੁਣਾ(੫) ¶    // → ੧੨੦
```

ਫੰਕਸ਼ਨਾਂ ਦਾ **ਅਲੱਗ scope** ਹੁੰਦਾ ਹੈ — ਉਹ ਬਾਹਰੀ ਚਰ ਨਹੀਂ ਪੜ੍ਹ ਸਕਦੇ। ਕਾਲਰ ਚਰਾਂ ਨੂੰ ਬਦਲਣ ਲਈ ਆਉਟਪੁੱਟ ਪੈਰਾਮੀਟਰ `<~` ਵਰਤੋ:

```zymbol
ਅਦਲਾ_ਬਦਲੀ(ਅ<~, ਬ<~) {
    ਅਸਥਾਈ = ਅ
    ਅ = ਬ
    ਬ = ਅਸਥਾਈ
}
ਚ = ੧੦
ਯ = ੨੦
ਅਦਲਾ_ਬਦਲੀ(ਚ, ਯ)
>> "ਚ=" ਚ " ਯ=" ਯ ¶    // → ਚ=੨੦ ਯ=੧੦
```

> ਨਾਮਿਤ ਫੰਕਸ਼ਨ first-class ਨਹੀਂ ਹਨ। ਤਰਕ ਵਜੋਂ ਪਾਸ ਕਰਨ ਲਈ ਲਪੇਟੋ: `x -> ਜੋੜ(x)`.

## ਲੈਂਬਡਾ ਅਤੇ ਕਲੋਜ਼ਰ

```zymbol
// ਸਧਾਰਨ ਲੈਂਬਡਾ (ਅੰਤਰਨਿਹਿਤ return)
ਦੁੱਗਣਾ = ਚ -> ਚ * ੨
ਜੋੜ = (ਅ, ਬ) -> ਅ + ਬ
>> ਦੁੱਗਣਾ(੫) ¶    // → ੧੦
>> ਜੋੜ(੩, ੭) ¶    // → ੧੦

// ਬਲਾਕ ਲੈਂਬਡਾ (ਸਪਸ਼ਟ return)
ਸ਼੍ਰੇਣੀਕਰਨ = ਚ -> {
    ? ਚ > ੦ { <~ "ਸਕਾਰਾਤਮਕ" }
    _? ਚ < ੦ { <~ "ਨਕਾਰਾਤਮਕ" }
    <~ "ਜ਼ੀਰੋ"
}

// ਕਲੋਜ਼ਰ — ਲੈਂਬਡਾ ਬਾਹਰੀ scope ਚਰ ਕੈਪਚਰ ਕਰਦਾ ਹੈ
ਗੁਣਾਂਕ = ੩
ਤਿੱਗਣਾ = ਚ -> ਚ * ਗੁਣਾਂਕ
>> ਤਿੱਗਣਾ(੭) ¶    // → ੨੧

// ਫੰਕਸ਼ਨ ਫੈਕਟਰੀ
ਜੋੜ_ਬਣਾਓ(ਨ) { <~ ਚ -> ਚ + ਨ }
ਦਸ_ਜੋੜੋ = ਜੋੜ_ਬਣਾਓ(੧੦)
>> ਦਸ_ਜੋੜੋ(੫) ¶    // → ੧੫

// ਸੂਚੀ ਵਿੱਚ ਸੰਗ੍ਰਹਿਤ
ਕਾਰਵਾਈਆਂ = [ਚ -> ਚ+੧, ਚ -> ਚ*੨, ਚ -> ਚ*ਚ]
>> ਕਾਰਵਾਈਆਂ[੨](੫) ¶    // → ੨੫
```

## ਐਰੇ

ਐਰੇ **ਬਦਲਣਯੋਗ** ਹੁੰਦੇ ਹਨ ਅਤੇ **ਇੱਕੋ ਕਿਸਮ** ਦੇ ਤੱਤ ਰੱਖਦੇ ਹਨ।

```zymbol
ਐਰੇ = [੧, ੨, ੩, ੪, ੫]

ਐਰੇ[੦]          // ੧ — ਪਹੁੰਚ (੦-ਆਧਾਰਿਤ)
ਐਰੇ[-੧]         // ੫ — ਨਕਾਰਾਤਮਕ ਸੂਚਕਾਂਕ (ਆਖਰੀ)
ਐਰੇ$#           // ੫ — ਲੰਬਾਈ

ਐਰੇ = ਐਰੇ$+ ੬            // append → [੧,੨,੩,੪,੫,੬]
ਐਰੇ੨ = ਐਰੇ$+[੨] ੯੯       // ਸੂਚਕਾਂਕ ੨ 'ਤੇ ਪਾਓ
ਐਰੇ੩ = ਐਰੇ$- ੩           // ਪਹਿਲੀ ਘਟਨਾ ਹਟਾਓ
ਐਰੇ੪ = ਐਰੇ$-- ੩          // ਸਾਰੀਆਂ ਘਟਨਾਵਾਂ ਹਟਾਓ
ਐਰੇ੫ = ਐਰੇ$-[੦]          // ਸੂਚਕਾਂਕ ਨਾਲ ਹਟਾਓ
ਐਰੇ੬ = ਐਰੇ$-[੧..੩]       // ਰੇਂਜ ਹਟਾਓ (ਅੰਤ ਵਗੈਰਾ)

ਹੈ = ਐਰੇ$? ੩            // #੧
ਥਾਵਾਂ = ਐਰੇ$?? ੩         // [੨] — ਸਾਰੇ ਸੂਚਕਾਂਕ
ਟੁਕੜਾ = ਐਰੇ$[੦..੩]       // [੧,੨,੩]
ਟੁਕੜਾ੨ = ਐਰੇ$[੦:੩]       // [੧,੨,੩] — ਗਿਣਤੀ ਆਧਾਰਿਤ

ਚੜ੍ਹਦਾ = ਐਰੇ$^+          // ਚੜ੍ਹਦੇ ਕ੍ਰਮ ਵਿੱਚ (primitives)
ਉਤਰਦਾ = ਐਰੇ$^-           // ਉਤਰਦੇ ਕ੍ਰਮ ਵਿੱਚ (primitives)

// Named tuple arrays
ਡੇਟਾ = [(ਨਾਮ: "Carla", ਉ: ੨੮), (ਨਾਮ: "Ana", ਉ: ੨੫), (ਨਾਮ: "Bob", ਉ: ੩੦)]
ਉ_ਕ੍ਰ  = ਡੇਟਾ$^ (ਕ, ਖ -> ਕ.ਉ < ਖ.ਉ)
ਨ_ਕ੍ਰ = ਡੇਟਾ$^ (ਕ, ਖ -> ਕ.ਨਾਮ > ਖ.ਨਾਮ)
>> ਉ_ਕ੍ਰ[੦].ਨਾਮ ¶     // → Ana
>> ਨ_ਕ੍ਰ[੦].ਨਾਮ ¶    // → Carla

// ਸਿੱਧਾ element ਅੱਪਡੇਟ (ਕੇਵਲ ਐਰੇ)
ਐਰੇ[੧] = ੯੯              // ਤਫ਼ਵੀਜ਼ ਕਰੋ
ਐਰੇ[੦] += ੫              // ਸੰਯੁਕਤ: +=  -=  *=  /=  %=  ^=

// Functional ਅੱਪਡੇਟ — ਨਵਾਂ ਐਰੇ ਵਾਪਸ ਕਰਦਾ ਹੈ; ਮੂਲ ਅਪਰਿਵਰਤਿਤ
ਐਰੇ੨ = ਐਰੇ[੧]$~ ੯੯
```

> ਸਾਰੇ collection operators **ਨਵਾਂ array** ਵਾਪਸ ਕਰਦੇ ਹਨ। ਮੁੜ ਤਫ਼ਵੀਜ਼ ਕਰੋ: `ਐਰੇ = ਐਰੇ$+ ੪`.
> Chaining ਨਹੀਂ: ਵੱਖਰੀਆਂ ਤਫ਼ਵੀਜ਼ਾਂ ਵਰਤੋ।
> `$^+` / `$^-` **primitive arrays** sort ਕਰਦੇ ਹਨ। Tuple arrays ਲਈ comparator lambda ਨਾਲ `$^` ਵਰਤੋ।

**ਮੁੱਲ ਅਰਥ-ਵਿਗਿਆਨ** — ਐਰੇ ਨੂੰ ਦੂਜੇ ਚਰ ਨੂੰ ਤਫ਼ਵੀਜ਼ ਕਰਨ ਨਾਲ ਸੁਤੰਤਰ ਕਾਪੀ ਬਣਦੀ ਹੈ:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b ਅਪ੍ਰਭਾਵਿਤ ਰਹਿੰਦਾ ਹੈ
```

```zymbol
// Nested arrays
matrix = [[੧,੨,੩],[੪,੫,੬],[੭,੮,੯]]
>> matrix[੧][੨] ¶    // → ੬
```

## ਟੁਪਲ

Tuples are **immutable** ordered containers that can hold values of **different types**. Unlike arrays, elements cannot be changed after creation.

```zymbol
// ਸਥਿਤੀ-ਅਧਾਰਿਤ
ਬਿੰਦੂ = (੧੦, ੨੦)
>> ਬਿੰਦੂ[੦] ¶    // → ੧੦

data = (42, "hello", #1, 3.14)
>> data[2] ¶     // → #1

// ਨਾਮਿਤ
ਵਿਅਕਤੀ = (ਨਾਮ: "ਐਲਿਸ", ਉਮਰ: ੨੫)
>> ਵਿਅਕਤੀ.ਨਾਮ ¶    // → ਐਲਿਸ
>> ਵਿਅਕਤੀ[੦] ¶     // → ਐਲਿਸ  (ਸੂਚਕਾਂਕ ਵੀ ਕੰਮ ਕਰਦਾ ਹੈ)

// ਨੈਸਟਡ
ਸਥਾਨ = (ਚ: ੧੦, ਯ: ੨੦)
ਪੌਇੰਟ = (ਸਥਾਨ: ਸਥਾਨ, ਲੇਬਲ: "ਮੂਲ")
>> ਪੌਇੰਟ.ਸਥਾਨ.ਚ ¶    // → ੧੦
```

**Immutability** — any attempt to modify a tuple element is a runtime error:

```zymbol
t = (10, 20, 30)
// t[0] = 99    // ❌ runtime error: tuples are immutable
// t[0] += 5    // ❌ same error
```

To derive a modified value use `$~` (functional update) — returns a **new** tuple:

```zymbol
t = (10, 20, 30)
t2 = t[1]$~ 999
>> t ¶     // → (10, 20, 30)   ← original unchanged
>> t2 ¶    // → (10, 999, 30)

// Named tuple — rebuild explicitly
ਵਿਅਕਤੀ = (ਨਾਮ: "Alice", ਉਮਰ: 25)
ਵੱਡਾ = (ਨਾਮ: ਵਿਅਕਤੀ.ਨਾਮ, ਉਮਰ: 26)
>> ਵਿਅਕਤੀ.ਉਮਰ ¶    // → 25
>> ਵੱਡਾ.ਉਮਰ ¶       // → 26
```

## ਉੱਚ-ਕ੍ਰਮ ਫੰਕਸ਼ਨ

> HOF ਓਪਰੇਟਰਾਂ ਨੂੰ **ਇਨਲਾਈਨ ਲੈਂਬਡਾ** ਦੀ ਲੋੜ ਹੈ — ਲੈਂਬਡਾ ਚਰ ਸਿੱਧੇ ਕੰਮ ਨਹੀਂ ਕਰਦੇ।

```zymbol
ਸੰਖਿਆਵਾਂ = [੧, ੨, ੩, ੪, ੫, ੬, ੭, ੮, ੯, ੧੦]

ਦੁੱਗਣੇ  = ਸੰਖਿਆਵਾਂ$> (ਚ -> ਚ * ੨)                // map  → [੨,੪,੬…੨੦]
ਜੁਫ਼ਤ    = ਸੰਖਿਆਵਾਂ$| (ਚ -> ਚ % ੨ == ੦)           // filter → [੨,੪,੬,੮,੧੦]
ਕੁੱਲ    = ਸੰਖਿਆਵਾਂ$< (੦, (ਸੰਚ, ਚ) -> ਸੰਚ + ਚ)    // reduce → ੫੫

// ਵਿਚਕਾਰਲੀਆਂ ਜ਼ਰੀਏ ਜੰਜ਼ੀਰ
ਕਦਮ੧ = ਸੰਖਿਆਵਾਂ$| (ਚ -> ਚ > ੩)
ਕਦਮ੨ = ਕਦਮ੧$> (ਚ -> ਚ * ਚ)
>> ਕਦਮ੨ ¶    // → [੧੬, ੨੫, ੩੬, ੪੯, ੬੪, ੮੧, ੧੦੦]

// ਨਾਮਿਤ ਫੰਕਸ਼ਨ HOF ਵਿੱਚ — ਲੈਂਬਡਾ ਵਿੱਚ ਲਪੇਟੋ
ਦੁੱਗਣਾ_ਕਰੋ(ਚ) { <~ ਚ * ੨ }
ਨਤੀਜਾ = ਸੰਖਿਆਵਾਂ$> (ਚ -> ਦੁੱਗਣਾ_ਕਰੋ(ਚ))    // ✅
```

## ਪਾਈਪ ਅਪਰੇਟਰ

RHS ਨੂੰ ਹਮੇਸ਼ਾ `_` placeholder ਲੋੜੀਂਦਾ ਹੈ:

```zymbol

ਵਧਾ = ਚ -> ਚ + ੧

>> ਰ ¶    // → ੨੨  (੫→੧੦→੧੧→੨੨)
```

## ਅੰਕ ਪ੍ਰਣਾਲੀਆਂ

Zymbol **69 Unicode ਅੰਕ ਬਲਾਕਾਂ ਵਿੱਚ** ਸੰਖਿਆਵਾਂ ਦਿਖਾ ਸਕਦਾ ਹੈ — ਗੁਰਮੁਖੀ, ਦੇਵਨਾਗਰੀ, ਅਰਬੀ-ਭਾਰਤੀ, ਥਾਈ, ਕਲਿੰਗਨ pIqaD ਅਤੇ ਹੋਰ। ਸਕਿਰਿਆ ਮੋਡ ਸਿਰਫ਼ `>>` ਆਉਟਪੁੱਟ ਨੂੰ ਪ੍ਰਭਾਵਿਤ ਕਰਦਾ ਹੈ; ਅੰਦਰੂਨੀ ਗਣਨਾ ਹਮੇਸ਼ਾ ਬਾਈਨਰੀ।

### ਸਿਸਟਮ ਸਕਿਰਿਆ ਕਰੋ

ਟੀਚਾ ਲਿਪੀ ਦੇ `੦` ਅਤੇ `੯` ਅੰਕ `#…#` ਵਿਚਕਾਰ ਲਿਖੋ:

```zymbol
#੦੯#    // PA script (੦..੯)
#٠٩#    // Arabic-Indic  (U+0660–U+0669)
#09#    // reset to ASCII
```

### ਆਉਟਪੁੱਟ ਅਤੇ ਬੂਲੀਅਨ

```zymbol
x = ੪੨
>> x ¶          // → ੪੨   (ASCII ਡਿਫੌਲਟ)

#੦੯#
>> x ¶          // → ੪੨
>> ੩.੧੪ ¶       // → ੩.੧੨੩੪   (ਦਸ਼ਮਲਵ ਬਿੰਦੂ ਹਮੇਸ਼ਾ ASCII)
>> ੧ + ੨ ¶      // → ੩

>> #੧ ¶         // → #੧   (ਸੱਚ)
>> #੦ ¶         // → #੦   (ਝੂਠ — ੦ ਤੋਂ ਵੱਖਰਾ)

x = ੨੮ > ੪
>> x ¶          // → #੧   (ਤੁਲਨਾ ਨਤੀਜਾ)
```

### ਸਰੋਤ ਵਿੱਚ ਮੂਲ ਲਿਟਰਲ

ਕਿਸੇ ਵੀ ਸਮਰਥਿਤ ਸਿਸਟਮ ਦੇ ਅੰਕ ਰੇਂਜ, ਮਾਡੁਲੋ, ਤੁਲਨਾ ਵਿੱਚ ਵੈਧ ਲਿਟਰਲ:

```zymbol
#੦੯#

@ ਨ:੧..੧੫ {
    ? ਨ % ੧੫ == ੦ { >> "FizzBuzz" ¶ }
    _? ਨ % ੩  == ੦ { >> "Fizz" ¶ }
    _? ਨ % ੫  == ੦ { >> "Buzz" ¶ }
    _ { >> ਨ ¶ }
}
```

### ਕਿਸੇ ਵੀ ਸਿਸਟਮ ਵਿੱਚ ਬੂਲੀਅਨ ਲਿਟਰਲ

`#` + ਕਿਸੇ ਵੀ ਸਮਰਥਿਤ ਬਲਾਕ ਦਾ `੦` ਜਾਂ `੧` ਅੰਕ ਇੱਕ ਵੈਧ ਬੂਲੀਅਨ ਲਿਟਰਲ ਹੈ:

```zymbol
#٠٩#
نشط = #١        // same as #੧
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **ਹਮੇਸ਼ਾ ASCII** ਹੈ। `#੦` (ਝੂਠ) ਕਿਸੇ ਵੀ ਸਿਸਟਮ ਵਿੱਚ `੦` (ਸਿਫ਼ਰ ਪੂਰਨ ਅੰਕ) ਤੋਂ ਦ੍ਰਿਸ਼ਟੀਗੋਚਰ ਤੌਰ 'ਤੇ ਵੱਖਰਾ ਹੈ।
