# Zymbol-Lang ਦੀ ਸੰਖੇਪ ਗਾਈਡ

**Zymbol-Lang** ਇੱਕ ਪ੍ਰਤੀਕਾਤਮਕ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾ ਹੈ। ਇਹ ਕੋਈ ਵਿਸ਼ੇਸ਼ ਸ਼ਬਦ (keywords) ਨਹੀਂ ਵਰਤਦੀ — ਸਭ ਕੁਝ ਪ੍ਰਤੀਕ ਹੈ। ਇਹ ਹਰ ਮਨੁੱਖੀ ਭਾਸ਼ਾ ਵਿੱਚ ਇੱਕੋ ਤਰੀਕੇ ਨਾਲ ਕੰਮ ਕਰਦੀ ਹੈ।

- ਕੋਈ ਵਿਸ਼ੇਸ਼ ਸ਼ਬਦ ਨਹੀਂ (`if`, `while`, `return` ਨਹੀਂ — ਸਿਰਫ਼ ਪ੍ਰਤੀਕ `?`, `@`, `<~`)
- ਪੂਰੀ Unicode ਸਹਾਇਤਾ — ਕਿਸੇ ਵੀ ਭਾਸ਼ਾ ਜਾਂ ਇਮੋਜੀ ਤੋਂ ਪਛਾਣਕਰਤਾ 👋
- ਭਾਸ਼ਾ-ਨਿਰਪੱਖ — ਕੋਡ ਸਾਰੀਆਂ ਭਾਸ਼ਾਵਾਂ ਵਿੱਚ ਇੱਕਸਾਰ ਹੈ

## ਚਰ ਅਤੇ ਸਥਿਰਾਂਕ

```zymbol
ਚ = 10            // ਚਰ (ਬਦਲਣਯੋਗ)
ਪਾਈ := 3.14159    // ਸਥਿਰਾਂਕ (ਨਾ-ਬਦਲਣਯੋਗ — ਮੁੜ ਤਫ਼ਵੀਜ਼ ਗਲਤੀ ਹੈ)
ਨਾਮ = "ਅਨੂ"
ਸਕਿਰਿਆ = #1      // Boolean ਸੱਚ
👋 := "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ"
```

```zymbol
ਚ = 10
ਚ += 5    // 15
ਚ -= 3    // 12
ਚ *= 2    // 24
ਚ /= 3    // 8
ਚ %= 3    // 2
ਚ ^= 2    // 4
ਚ++       // 5
ਚ--       // 4
```

## ਡੇਟਾ ਕਿਸਮਾਂ ਦੀ ਸਾਰਣੀ

| ਕਿਸਮ            | ਉਦਾਹਰਨ              | `#?` ਪ੍ਰਤੀਕ | ਨੋਟ                                      |
|-----------------|---------------------|-------------|------------------------------------------|
| ਪੂਰਨ ਅੰਕ        | `42`, `-7`          | `###`       | 64-bit ਚਿੰਨ੍ਹ ਸਮੇਤ                      |
| ਦਸ਼ਮਲਵ          | `3.14`, `1.5e10`    | `##.`       | ਵਿਗਿਆਨਕ ਸੰਕੇਤਨ ਮਾਨਯੋਗ                   |
| ਸਤਰ             | `"ਹੈਲੋ"`            | `##"`       | ਅੰਤਰ-ਪੋਲੇਸ਼ਨ: `"ਹੈਲੋ {ਨਾਮ}"`            |
| ਅੱਖਰ            | `'A'`               | `##'`       | ਇੱਕ Unicode ਅੱਖਰ                         |
| Boolean         | `#1`, `#0`          | `##?`       | ਅੰਕੀ 1 ਅਤੇ 0 ਨਹੀਂ                       |
| ਐਰੇ (Array)     | `[1, 2, 3]`         | `##]`       | ਸਾਰੇ ਤੱਤ ਇੱਕੋ ਕਿਸਮ ਦੇ                   |
| ਟੂਪਲ           | `(a, b)`            | `##)`       | ਸਥਿਤੀ-ਅਧਾਰਿਤ                            |
| ਨਾਮੀ ਟੂਪਲ      | `(x: 1, y: 2)`      | `##)`       | ਨਾਮ ਜਾਂ ਸੂਚਕਾਂਕ ਤੋਂ ਪਹੁੰਚ               |

```zymbol
// ਕਿਸਮ ਜਾਂਚ — (type, digits, value) ਵਾਪਸ ਕਰਦਾ ਹੈ
meta = 42#?
>> meta ¶         // → (###, 2, 42)
t = meta[0]
>> t ¶            // → ###
```

## ਆਉਟਪੁੱਟ ਅਤੇ ਇਨਪੁੱਟ

```zymbol
// ਆਉਟਪੁੱਟ — ਆਪੇ ਨਵੀਂ ਲਾਈਨ ਨਹੀਂ ਆਉਂਦੀ
>> "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ, ਦੁਨੀਆ!" ¶        // ¶ ਜਾਂ \\ ਸਪੱਸ਼ਟ ਨਵੀਂ ਲਾਈਨ ਦਿੰਦਾ ਹੈ
>> "ਅ=" ਅ " ਬ=" ਬ ¶                  // ਕਈ ਮੁੱਲ ਨਾਲ-ਨਾਲ ਰੱਖ ਕੇ
>> "ਜੋੜ=" ਜੋੜ(2, 3) ¶               // ਫੰਕਸ਼ਨ ਕਾਲ ਕਿਸੇ ਵੀ ਥਾਂ
>> (ਸੂਚੀ$#) ¶                        // postfix ਆਪਰੇਟਰਾਂ ਨੂੰ >> ਵਿੱਚ ਬਰੈਕਟ ਚਾਹੀਦੇ ਹਨ

// ਇਨਪੁੱਟ
<< ਨਾਮ                               // ਸੰਕੇਤ ਤੋਂ ਬਿਨਾਂ — ਚਰ ਵਿੱਚ ਪੜ੍ਹਦਾ ਹੈ
<< "ਆਪਣਾ ਨਾਮ? " ਨਾਮ                 // ਸੰਕੇਤ ਸਮੇਤ
```

> `¶` ਅਤੇ `\\` ਨਵੀਂ ਲਾਈਨ ਲਈ ਬਰਾਬਰ ਹਨ।

## ਸਤਰ

```zymbol
// ਤਿੰਨ ਜੋੜਨ ਦੇ ਤਰੀਕੇ
ਨਾਮ = "ਅਨੂ"
ਉ = 25

ਸੁਨੇਹਾ = "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ ", ਨਾਮ, "!"     // ਕੌਮਾ — ਅਸਾਈਨਮੈਂਟ ਵਿੱਚ
>> "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ " ਨਾਮ " ਤੁਸੀਂ " ਉ " ਸਾਲ ਦੇ ਹੋ" ¶    // juxtaposition — >> ਵਿੱਚ
ਜਾਣ-ਪਛਾਣ = "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ {ਨਾਮ}, ਤੁਸੀਂ {ਉ} ਸਾਲ ਦੇ ਹੋ"  // interpolation — ਕਿਤੇ ਵੀ
```

```zymbol
ਸ = "ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ ਦੁਨੀਆ"
ਲ = ਸ$#                  // length
ਉ = ਸ$[0..5]             // slice (end exclusive)
ਹ = ਸ$? "ਦੁਨੀ"           // #1
ਭ = "ਕ,ਖ,ਗ,ਘ" / ','      // [ਕ, ਖ, ਗ, ਘ]
ਬ = ਸ$~~["ਅ":"ਅ਼"]         // replace
ਬ1 = ਸ$~~["ਅ":"ਅ਼":1]      // first N
```

> `+` ਕੇਵਲ ਸੰਖਿਆਵਾਂ ਲਈ। Strings ਲਈ `,`, juxtaposition ਜਾਂ interpolation ਵਰਤੋ।

## ਪ੍ਰਵਾਹ ਨਿਯੰਤਰਣ

```zymbol
ਚ = 7

// ਸਾਦਾ ਜੇ
? ਚ > 0 { >> "ਸਕਾਰਾਤਮਕ" ¶ }

// ਜੇ / ਨਹੀਂ ਤਾਂ ਜੇ / ਨਹੀਂ ਤਾਂ
? ਚ > 100 {
    >> "ਵੱਡਾ" ¶
} _? ਚ > 0 {
    >> "ਸਕਾਰਾਤਮਕ" ¶
} _? ਚ == 0 {
    >> "ਜ਼ੀਰੋ" ¶
} _ {
    >> "ਨਕਾਰਾਤਮਕ" ¶
}
```

`{ }` ਬਲਾਕ **ਜ਼ਰੂਰੀ ਹਨ**, ਭਾਵੇਂ ਸਿਰਫ਼ ਇੱਕ ਲਾਈਨ ਹੋਵੇ।

## ਮੇਲ (Match)

```zymbol
// ਅੰਕੀ ਰੇਂਜ ਨਾਲ ਮੇਲ
ਅੰਕ = 85
ਦਰਜਾ = ?? ਅੰਕ {
    90..100 : 'A'
    80..89  : 'B'
    70..79  : 'C'
    _       : 'F'
}
>> ਦਰਜਾ ¶    // → B

// ਗਾਰਡਾਂ ਨਾਲ ਮੇਲ (ਮਨਮਾਨੀਆਂ ਸ਼ਰਤਾਂ)
ਤਾਪਮਾਨ = -5
ਹਾਲਤ = ?? ਤਾਪਮਾਨ {
    _? ਤਾਪਮਾਨ < 0  : "ਬਰਫ਼"
    _? ਤਾਪਮਾਨ < 20 : "ਠੰਡਾ"
    _? ਤਾਪਮਾਨ < 35 : "ਗਰਮ"
    _               : "ਬਹੁਤ ਗਰਮ"
}
>> ਹਾਲਤ ¶    // → ਬਰਫ਼

// ਸਤਰਾਂ ਨਾਲ ਮੇਲ
ਰੰਗ = "ਲਾਲ"
ਕੋਡ = ?? ਰੰਗ {
    "ਲਾਲ"   : "#FF0000"
    "ਹਰਾ"   : "#00FF00"
    _        : "#000000"
}
>> ਕੋਡ ¶

// ਬਲਾਕ ਆਰਮਜ਼
?? ਨ {
    0       : { >> "ਜ਼ੀਰੋ" ¶ }
    _? ਨ < 0: { >> "ਨਕਾਰਾਤਮਕ" ¶ }
    _       : { >> "ਸਕਾਰਾਤਮਕ" ¶ }
}
```

## ਲੂਪ

```zymbol
@ ਇ:0..4  { >> ਇ " " }        // ਰੇਂਜ ਸ਼ਾਮਲ:  0 1 2 3 4
@ ਇ:1..9:2 { >> ਇ " " }       // ਕਦਮ ਨਾਲ:    1 3 5 7 9
@ ਇ:5..0:1 { >> ਇ " " }       // ਉਲਟਾ:       5 4 3 2 1 0

ਨ = 1
@ ਨ <= 64 { ਨ *= 2 }
>> ਨ ¶                        // → 128  (while)

ਫਲ = ["ਸੇਬ", "ਨਾਸ਼ਪਾਤੀ", "ਅੰਗੂਰ"]
@ ਫ:ਫਲ { >> ਫ ¶ }             // for-each array

@ ਅ:"ਹੈਲੋ" { >> ਅ "-" }
>> ¶                          // → ਹ-ੈ-ਲ-ੋ-  (for-each string)

@ ਇ:1..10 {
    ? ਇ % 2 == 0 { @> }       // @> continue
    ? ਇ > 7 { @! }             // @! break
    >> ਇ " "
}
>> ¶                          // → 1 3 5 7

// ਅਨੰਤ ਲੂਪ
ਇ = 0
@ {
    ਇ++
    ? ਇ >= 5 { @! }
    >> ਇ " "
}
>> ¶                          // → 1 2 3 4

// ਲੇਬਲਡ ਲੂਪ (nested break)
count = 0
@ @outer {
    count++
    ? count >= 3 { @! outer }
}
>> count ¶                    // → 3
```

## ਫੰਕਸ਼ਨ

```zymbol
// ਘੋਸ਼ਣਾ ਅਤੇ ਕਾਲ
ਜੋੜ(ਅ, ਬ) { <~ ਅ + ਬ }
>> ਜੋੜ(3, 4) ¶    // → 7

// ਪੁਨਰਾਵ੍ਰਿੱਤੀ (Recursion)
ਗੁਣਾ(ਨ) {
    ? ਨ <= 1 { <~ 1 }
    <~ ਨ * ਗੁਣਾ(ਨ - 1)
}
>> ਗੁਣਾ(5) ¶    // → 120
```

ਫੰਕਸ਼ਨਾਂ ਦਾ **ਅਲੱਗ scope** ਹੁੰਦਾ ਹੈ — ਉਹ ਬਾਹਰੀ ਚਰ ਨਹੀਂ ਪੜ੍ਹ ਸਕਦੇ। ਕਾਲਰ ਚਰਾਂ ਨੂੰ ਬਦਲਣ ਲਈ ਆਉਟਪੁੱਟ ਪੈਰਾਮੀਟਰ `<~` ਵਰਤੋ:

```zymbol
ਅਦਲਾ_ਬਦਲੀ(ਅ<~, ਬ<~) {
    ਅਸਥਾਈ = ਅ
    ਅ = ਬ
    ਬ = ਅਸਥਾਈ
}
ਚ = 10
ਯ = 20
ਅਦਲਾ_ਬਦਲੀ(ਚ, ਯ)
>> "ਚ=" ਚ " ਯ=" ਯ ¶    // → ਚ=20 ਯ=10
```

> ਨਾਮਿਤ ਫੰਕਸ਼ਨ first-class ਨਹੀਂ ਹਨ। ਤਰਕ ਵਜੋਂ ਪਾਸ ਕਰਨ ਲਈ ਲਪੇਟੋ: `x -> ਜੋੜ(x)`.

## ਲੈਂਬਡਾ ਅਤੇ ਕਲੋਜ਼ਰ

```zymbol
// ਸਧਾਰਨ ਲੈਂਬਡਾ (ਅੰਤਰਨਿਹਿਤ return)
ਦੁੱਗਣਾ = ਚ -> ਚ * 2
ਜੋੜ = (ਅ, ਬ) -> ਅ + ਬ
>> ਦੁੱਗਣਾ(5) ¶    // → 10
>> ਜੋੜ(3, 7) ¶    // → 10

// ਬਲਾਕ ਲੈਂਬਡਾ (ਸਪਸ਼ਟ return)
ਸ਼੍ਰੇਣੀਕਰਨ = ਚ -> {
    ? ਚ > 0 { <~ "ਸਕਾਰਾਤਮਕ" }
    _? ਚ < 0 { <~ "ਨਕਾਰਾਤਮਕ" }
    <~ "ਜ਼ੀਰੋ"
}

// ਕਲੋਜ਼ਰ — ਲੈਂਬਡਾ ਬਾਹਰੀ scope ਚਰ ਕੈਪਚਰ ਕਰਦਾ ਹੈ
ਗੁਣਾਂਕ = 3
ਤਿੱਗਣਾ = ਚ -> ਚ * ਗੁਣਾਂਕ
>> ਤਿੱਗਣਾ(7) ¶    // → 21

// ਫੰਕਸ਼ਨ ਫੈਕਟਰੀ
ਜੋੜ_ਬਣਾਓ(ਨ) { <~ ਚ -> ਚ + ਨ }
ਦਸ_ਜੋੜੋ = ਜੋੜ_ਬਣਾਓ(10)
>> ਦਸ_ਜੋੜੋ(5) ¶    // → 15

// ਸੂਚੀ ਵਿੱਚ ਸੰਗ੍ਰਹਿਤ
ਕਾਰਵਾਈਆਂ = [ਚ -> ਚ+1, ਚ -> ਚ*2, ਚ -> ਚ*ਚ]
>> ਕਾਰਵਾਈਆਂ[2](5) ¶    // → 25
```

## ਐਰੇ

```zymbol
ਐਰੇ = [1, 2, 3, 4, 5]

ਐਰੇ[0]          // 1 — ਪਹੁੰਚ (0-ਆਧਾਰਿਤ)
ਐਰੇ[-1]         // 5 — ਨਕਾਰਾਤਮਕ ਸੂਚਕਾਂਕ (ਆਖਰੀ)
ਐਰੇ$#           // 5 — ਲੰਬਾਈ

ਐਰੇ = ਐਰੇ$+ 6            // append → [1,2,3,4,5,6]
ਐਰੇ2 = ਐਰੇ$+[2] 99       // ਸੂਚਕਾਂਕ 2 'ਤੇ ਪਾਓ
ਐਰੇ3 = ਐਰੇ$- 3           // ਪਹਿਲੀ ਘਟਨਾ ਹਟਾਓ
ਐਰੇ4 = ਐਰੇ$-- 3          // ਸਾਰੀਆਂ ਘਟਨਾਵਾਂ ਹਟਾਓ
ਐਰੇ5 = ਐਰੇ$-[0]          // ਸੂਚਕਾਂਕ ਨਾਲ ਹਟਾਓ
ਐਰੇ6 = ਐਰੇ$-[1..3]       // ਰੇਂਜ ਹਟਾਓ (ਅੰਤ ਵਗੈਰਾ)

ਹੈ = ਐਰੇ$? 3            // #1
ਥਾਵਾਂ = ਐਰੇ$?? 3         // [2] — ਸਾਰੇ ਸੂਚਕਾਂਕ
ਟੁਕੜਾ = ਐਰੇ$[0..3]       // [1,2,3]
ਟੁਕੜਾ2 = ਐਰੇ$[0:3]       // [1,2,3] — ਗਿਣਤੀ ਆਧਾਰਿਤ

ਚੜ੍ਹਦਾ = ਐਰੇ$^+          // ਚੜ੍ਹਦੇ ਕ੍ਰਮ ਵਿੱਚ (primitives)
ਉਤਰਦਾ = ਐਰੇ$^-           // ਉਤਰਦੇ ਕ੍ਰਮ ਵਿੱਚ (primitives)

// Named tuple arrays
ਡੇਟਾ = [(ਨਾਮ: "Carla", ਉ: 28), (ਨਾਮ: "Ana", ਉ: 25), (ਨਾਮ: "Bob", ਉ: 30)]
ਉ_ਕ੍ਰ  = ਡੇਟਾ$^ (ਕ, ਖ -> ਕ.ਉ < ਖ.ਉ)
ਨ_ਕ੍ਰ = ਡੇਟਾ$^ (ਕ, ਖ -> ਕ.ਨਾਮ > ਖ.ਨਾਮ)
>> ਉ_ਕ੍ਰ[0].ਨਾਮ ¶     // → Ana
>> ਨ_ਕ੍ਰ[0].ਨਾਮ ¶    // → Carla

ਐਰੇ[1] = 99              // ਜਗ੍ਹਾਂ ਅੱਪਡੇਟ
ਐਰੇ = ਐਰੇ[1]$~ 99        // functional ਅੱਪਡੇਟ
```

> ਸਾਰੇ collection operators **ਨਵਾਂ array** ਵਾਪਸ ਕਰਦੇ ਹਨ। ਮੁੜ ਤਫ਼ਵੀਜ਼ ਕਰੋ: `ਐਰੇ = ਐਰੇ$+ 4`.
> Chaining ਨਹੀਂ: ਵੱਖਰੀਆਂ ਤਫ਼ਵੀਜ਼ਾਂ ਵਰਤੋ।
> `$^+` / `$^-` **primitive arrays** sort ਕਰਦੇ ਹਨ। Tuple arrays ਲਈ comparator lambda ਨਾਲ `$^` ਵਰਤੋ।

```zymbol
// Nested arrays
matrix = [[1,2,3],[4,5,6],[7,8,9]]
>> matrix[1][2] ¶    // → 6
```

## ਡਿਸਟ੍ਰਕਚਰਿੰਗ

```zymbol
// Array
ਐਰੇ = [10, 20, 30, 40, 50]
[ਕ, ਖ, ਗ] = ਐਰੇ              // ਕ=10  ਖ=20  ਗ=30
[ਪਹ, *ਬਾ] = ਐਰੇ              // ਪਹ=10  ਬਾ=[20,30,40,50]
[ਇ, _, ਊ] = [1, 2, 3]        // _ ਛੱਡਦਾ ਹੈ

// Positional tuple
ਬਿੰਦੂ = (100, 200)
(bx, by) = ਬਿੰਦੂ             // bx=100  by=200

// Named tuple
ਵਿਅਕਤੀ = (ਨਾਮ: "Ana", ਉ: 25, ਸ਼: "Amritsar")
(ਨਾਮ: ਨ, ਉ: ਉ_ਵ) = ਵਿਅਕਤੀ    // ਨ="Ana"  ਉ_ਵ=25
```

## ਟੁਪਲ

```zymbol
// ਸਥਿਤੀ-ਅਧਾਰਿਤ
ਬਿੰਦੂ = (10, 20)
>> ਬਿੰਦੂ[0] ¶    // → 10

// ਨਾਮਿਤ
ਵਿਅਕਤੀ = (ਨਾਮ: "ਐਲਿਸ", ਉਮਰ: 25)
>> ਵਿਅਕਤੀ.ਨਾਮ ¶    // → ਐਲਿਸ
>> ਵਿਅਕਤੀ[0] ¶     // → ਐਲਿਸ  (ਸੂਚਕਾਂਕ ਵੀ ਕੰਮ ਕਰਦਾ ਹੈ)

// ਨੈਸਟਡ
ਸਥਾਨ = (ਚ: 10, ਯ: 20)
ਪੌਇੰਟ = (ਸਥਾਨ: ਸਥਾਨ, ਲੇਬਲ: "ਮੂਲ")
>> ਪੌਇੰਟ.ਸਥਾਨ.ਚ ¶    // → 10
```
