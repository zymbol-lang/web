# Приручник Zymbol-Lang

> **Прегледано за v0.0.5 — 2026-05-12**

**Zymbol-Lang** је симболички програмски језик. Без кључних речи — све је симбол. Ради идентично на сваком људском језику.

- Без `if`, `while`, `return` — само `?`, `@`, `<~`
- Потпуни Unicode — идентификатори на било ком језику или емоџи
- Независан од људског језика — код је свуда исти

**Верзија тумача**: v0.0.5 | **Покривеност тестовима**: 436/436 (TW ↔ VM паритет)

## Променљиве и Константе

```zymbol
x = 10              // променљива
PI := 3.14159       // константа — поновно додељивање је грешка при извршавању
ime = "Алиса"
aktivan = #1        // логичка истина
👋 := "Здраво"
```

```zymbol
x = 10    // 10
x += 5    // 15
x -= 3    // 12
x *= 2    // 24
x /= 3    // 8
x %= 3    // 2
x ^= 2    // 4
x++        // 5
x--        // 4
```

`°` (знак степена, U+00B0) аутоматски иницијализује променљиву на њену неутралну вредност при првој употреби:

```zymbol
brojevi = [3, 1, 4, 1, 5]
@ n:brojevi {
    °ukupno += n    // ауто-иницијализација на 0 изнад петље; доступна nakon @
}
>> ukupno ¶         // → 14
```

> `°x` (префикс) сидри изнад петље — резултат је доступан после `@`.
> `x°` (суфикс) сидри унутар петље — нестаје кад петља заврши.
> Само tree-walker.

## Типови Података

| Тип | Литерал | Ознака `#?` | Напомене |
|-----|---------|-------------|---------|
| Цео | `42`, `-7` | `###` | 64-битни са предзнаком |
| Децимални | `3.14`, `1.5e10` | `##.` | Научна нотација OK |
| Низ | `"текст"` | `##"` | Интерполација: `"Здраво {ime}"` |
| Знак | `'A'` | `##'` | Један Unicode знак |
| Логички | `#1`, `#0` | `##?` | НИЈЕ нумеричко — `#1 ≠ 1` |
| Низ | `[1, 2, 3]` | `##]` | Хомогени елементи |
| Торка | `(a, b)` | `##)` | Позициона |
| Именована Торка | `(x: 1, y: 2)` | `##)` | Именована поља |
| Функција | именована референца | `##()` | Прва класа; приказује `<funct/N>` |
| Ламбда | `x -> x * 2` | `##->` | Прва класа; приказује `<lambd/N>` |

```zymbol
// Интроспекција типа — враћа (тип, цифре, вредност)
meta = 42#?
>> meta ¶         // → (###, 2, 42)
t = meta[1]
>> t ¶            // → ###
```

## TUI Примитиви

Терминални UI оператори за интерактивне програме. Већина захтева блок `>>| { }` (алтернативни екран + сиров режим).

```zymbol
>>| {
    >>!                             // брисање алтернативног екрана
    >>~ (1, 1, 0, 10) > "Ради"     // ред 1, колона 1, fg=10 (зелена)
    @~ 1000                         // пауза 1 секунда (1000 мс)
    >>~ (2, 1) > "Готово."
}
// терминал аутоматски обновљен при излазу
```

```zymbol
// Притисак тастера и величина терминала
>>| {
    [redovi, kolone] = >>?              // упит за димензије терминала
    >>~ (1, 1) > "Терминал: " redovi " x " kolone
    <<| taster                          // блокирајуће читање тастера
    >>~ (2, 1) > "Притиснуто: " taster
}
```

> `>>!` брише екран. `>>?` враћа `[redovi, kolone]`. `@~ N` спава N милисекунди.
> `<<|` чита један притисак тастера (блокирајуће); `<<|?` проверава без блокирања (враћа `'\0'` ако нема).
> Торка позиционираног излаза: `(red, kolona, SDM, fg, bg)` — сваки слот може се изоставити зарезом (`>>~ (,,, 196) > "црвена"`).
> Маска SDM: `1`=Подебљано, `2`=Курзив, `4`=Подвучено. ANSI 256-бојна палета (`0`=подразумевани терминал).
> Само tree-walker (осим `>>!`, `>>?`, `@~`, `>>~` који раде и са `--vm`).

## Петље

```zymbol
@ i:0..4  { >> i " " }        // опсег укључен:  0 1 2 3 4
@ i:1..9:2 { >> i " " }       // са кораком:      1 3 5 7 9
@ i:5..0:1 { >> i " " }       // обрнуто:         5 4 3 2 1 0

n = 1
@ n <= 64 { n *= 2 }
>> n ¶                        // → 128  (while)

voce = ["јабука", "крушка", "грожђе"]
@ v:voce { >> v ¶ }           // for-each низ

@ c:"zdravo" { >> c "-" }
>> ¶                          // → z-d-r-a-v-o-  (for-each стринг)

@ i:1..10 {
    ? i % 2 == 0 { @> }       // @> настави
    ? i > 7 { @! }             // @! прекини
    >> i " "
}
>> ¶                          // → 1 3 5 7

// Бесконачна петља
i = 0
@ {
    i++
    ? i >= 5 { @! }
    >> i " "
}
>> ¶                          // → 1 2 3 4

// Означена петља (угнеждено прекидање)
brojac = 0
@:spoljna {
    brojac++
    ? brojac >= 3 { @:spoljna! }
}
>> brojac ¶                   // → 3
```

## Функције

```zymbol
saberi(a, b) { <~ a + b }
>> saberi(3, 4) ¶    // → 7

faktorijel(n) {
    ? n <= 1 { <~ 1 }
    <~ n * faktorijel(n - 1)
}
>> faktorijel(5) ¶    // → 120
```

Функције имају **изолован опсег** — не могу читати спољне променљиве. Користите излазне параметре `<~` за измену променљивих позиваоца:

```zymbol
zameni(a<~, b<~) {
    priv = a
    a = b
    b = priv
}
x = 10
y = 20
zameni(x, y)
>> "x=" x " y=" y ¶    // → x=20 y=10
```

> Именоване функције су **вредности прве класе** — прослеђујте директно: `brojevi$> udvostruci`. За омотавање: `x -> fn(x)` је такође валидно.

## Низови

Низови су **променљиви** и садрже елементе **истог типа**.

```zymbol
niz = [1, 2, 3, 4, 5]

x = niz[1]      // 1 — приступ (1-базно: први елемент)
x = niz[-1]     // 5 — негативни индекс (последnji елемент)
x = niz$#       // 5 — дужина (користите (niz$#) у >>)

niz = niz$+ 6            // додавање → [1,2,3,4,5,6]
niz2 = niz$+[2] 99       // уметање на позицији 2 (1-базно)
niz3 = niz$- 3           // уклањање првог јављања вредности
niz4 = niz$-- 3          // уклањање свих јављања
niz5 = niz$-[1]          // уклањање на индексу 1 (први елемент)
niz6 = niz$-[2..3]       // уклањање опсега (1-базно, крај укључен)

ima = niz$? 3            // #1 — садржи
poz = niz$?? 3           // [3] — сви индекси вредности (1-базно)
rez = niz$[1..3]         // [1,2,3] — исечак (1-базно, крај укључен)
rez2 = niz$[1:3]         // [1,2,3] — исто, синтакса са бројем

rast = niz$^+            // сортирано растуће (само примитиви)
opad = niz$^-            // сортирано опадајуће (само примитиви)

// Низови именованих/позиционих торки — користите $^ са ламбда компаратором
bp = [(ime: "Карла", god: 28), (ime: "Ана", god: 25), (ime: "Боб", god: 30)]
po_god = bp$^ (a, b -> a.god < b.god)    // растуће по годинама  (<)
po_ime = bp$^ (a, b -> a.ime > b.ime)    // опадајуће по имену (>)
>> po_god[1].ime ¶     // → Ана
>> po_ime[1].ime ¶     // → Карла

// Директно ажурирање елемента (само низови)
niz[1] = 99              // додељивање
niz[2] += 5              // сложено: +=  -=  *=  /=  %=  ^=

// Функционално ажурирање — враћа нови низ; оригинал непромењен
niz2 = niz[2]$~ 99
```

> Сви оператори колекција враћају **нови низ**. Доделите назад: `niz = niz$+ 4`.
> `$+` може да се повеже у ланац: `niz = niz$+ 5$+ 6$+ 7`. Остали оператори користе средња додељивања.
> **Индексирање је 1-базно**: `niz[1]` је prvi елемент; `niz[0]` је грешка при извршавању.
> `$^+` / `$^-` сортирају **примитивне низове** (бројеви, стрингови). За низове торки користите `$^` са ламбда компаратором — смер је кодиран у ламбди (`<` = растуће, `>` = опадајуће).

**Семантика вредности** — додељивање низа другој променљивој ствара независну копију:

```zymbol
a = [1, 2, 3]
b = a
a[1] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b није под утицајем
```

```zymbol
// Угнежден низови (1-базно индексирање)
matrica = [[1,2,3],[4,5,6],[7,8,9]]
>> matrica[2][3] ¶    // → 6  (ред 2, колона 3)
```

## Торке

Торке су **непромењиви** уређени контејнери који могу да садрже вредности **различитих типова**.
За разлику од низова, елементи не могу да се мењају после стварања.

```zymbol
// Позиционе — дозвољени мешани типови
tacka = (10, 20)
>> tacka[1] ¶    // → 10

podaci = (42, "zdravo", #1, 3.14)
>> podaci[3] ¶   // → #1

// Именоване
osoba = (ime: "Алиса", god: 25)
>> osoba.ime ¶    // → Алиса
>> osoba[1] ¶     // → Алиса  (индекс такође ради, 1-базно)

// Угнежден
poz = (x: 10, y: 20)
p = (poz: poz, oznaka: "pocetak")
>> p.poz.x ¶      // → 10
```

**Непроменљивост** — сваки покушај измене елемента торке је грешка при извршавању:

```zymbol
t = (10, 20, 30)
// t[1] = 99    // ❌ грешка при извршавању: торке су непромењиве
// t[1] += 5    // ❌ иста грешка
```

За извођење измењене вредности користите `$~` (функционално ажурирање) — враћа **нову** торку:

```zymbol
t = (10, 20, 30)
t2 = t[2]$~ 999
>> t ¶     // → (10, 20, 30)   ← оригинал непромењен
>> t2 ¶    // → (10, 999, 30)

// Именована торка — експлицитна реконструкција
osoba = (ime: "Алиса", god: 25)
starija = (ime: osoba.ime, god: 26)
>> osoba.god ¶    // → 25
>> starija.god ¶  // → 26
```

## Функције Вишег Реда

## Оператор Цеви

Десна страна увек захтева `_` као заменик за прослеђену вредност:

```zymbol
udvostruci = x -> x * 2
saberi = (a, b) -> a + b
povecaj = x -> x + 1

r1 = 5 |> udvostruci(_)        // → 10
r2 = 10 |> saberi(_, 5)        // → 15
r3 = 5 |> saberi(2, _)         // → 7

// Повезано у ланац
r = 5 |> udvostruci(_) |> povecaj(_) |> udvostruci(_)
>> r ¶    // → 22  (5→10→11→22)
```

## Управљање Грешкама

```zymbol
!? {
    x = 10 / 0
} :! ##Div {
    >> "дељење нулом" ¶
} :! {
    >> "остало: " _err ¶    // _err садржи поруку грешке
} :> {
    >> "увек се извршава" ¶
}
```

| Тип | Кад |
|-----|-----|
| `##Div` | Дељење нулом |
| `##IO` | Датотека / систем |
| `##Index` | Индекс ван граница |
| `##Type` | Неускладивост типова |
| `##Parse` | Парсирање података |
| `##Network` | Мрежне грешке |
| `##_` | Свака грешка (catch-all) |

```zymbol
>> m::suma(3, 4) ¶    // → 7  (унутрашње ime _unutrasnji_zbir је скривено)
```

> **Правила модула**: само `#>`, дефиниције функција и литерални иницијализатори променљивих/константи су дозвољени унутар `# ime { }`. Извршне наредбе (`>>`, `<<`, петље итд.) изазивају грешку E013.

## Нумерички Режими

Zymbol може да приказује бројеве у **69 Unicode скрипти цифара** — Деванагари, Арапско-Индијски, Тајландски, Клингонски pIqaD, Математички подебљани, LCD сегменти и још. Активни режим утиче само на `>>` излаз; унутрашња аритметика је увек бинарна.

### Активирање скрипте

Запишите цифре `0` и `9` циљне скрипте затворене у `#…#`:

```zymbol
#०९#    // Деванагари   (U+0966–U+096F)
#٠٩#    // Арапско-Индијски (U+0660–U+0669)
#๐๙#    // Тајландски   (U+0E50–U+0E59)
#09#    // ресет на ASCII
```

### Излаз и логичке вредности

```zymbol
x = 42
>> x ¶          // → 42   (ASCII подразумевано)

#०९#
>> x ¶          // → ४२
>> 3.14 ¶       // → ३.१४   (децимална тачка увек ASCII)
>> 1 + 2 ¶      // → ३

// Логичко: префикс # увек ASCII, цифра се прилагођава
>> #1 ¶         // → #१   (истина у Деванагари)
>> #0 ¶         // → #०   (лаж — различито од ०  цео број нула)

x = 28 > 4
>> x ¶          // → #१   (резултат поређења прати активни режим)
```

### Домаћи цифровни литерали у изворном коду

Цифре сваке подржане скрипте су валидни литерали — у опсезима, модуло, поређењима:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### Логички литерали у свакој скрипти

`#` + цифра `0` или `1` из сваког блока је валидни логички литерал:

```zymbol
#٠٩#
aktivan = #١        // исто као #1
>> aktivan ¶        // → #١
>> (#١ && #٠) ¶     // → #٠
```

> `#` је **увек ASCII**. `#0` (лаж) је увек визуелно различито од `0` (цео број нула) у свакој скрипти.

## Оператори Података

```zymbol
// Конверзије типова
f = ##.42         // → 42.0  (у Децимални)
i = ###3.7        // → 4     (у Цео, заокруживање)
t = ##!3.7        // → 3     (у Цео, одсецање)

// Парсирање стринга у број
v1 = #|"42"|      // → 42  (Цео)
v2 = #|"3.14"|    // → 3.14  (Децимални)
v3 = #|"abc"|     // → "abc"  (безбедан неуспех, без грешке)

// Заокруживање / одсецање
pi = 3.14159265
v2 = #.2|pi|      // → 3.14  (заокружи на 2 децималне места)
v4 = #.4|pi|      // → 3.1416
t2 = #!2|pi|      // → 3.14  (одсецање)

// Форматирање бројева
fmt = #,|1234567|  // → 1,234,567  (одвојено зарезима)
nau = #^|12345.678|    // → 1.2345678e4  (научна нотација)

// Литерали основа
a = 0x41         // → 'A'  (хексадецимални)
b = 0b01000001   // → 'A'  (бинарни)
c = 0o101        // → 'A'  (октални)

// Излаз конверзије основе
hex = 0x|255|    // → "0x00FF"
bin = 0b|65|     // → "0b1000001"
oct = 0o|8|      // → "0o10"
dec = 0d|255|    // → "0d0255"
```

## Интеграција Shell-а

```zymbol
datum = <\ date +%Y-%m-%d \>     // хвата stdout (укључујући крајњи \n)
>> "Данас: " datum

fajl = "podaci.txt"
sadrzaj = <\ cat {fajl} \>       // интерполација у командама

izlaz = </"./podskript.zy"/>     // извршава други Zymbol скрипт, хвата излаз
>> izlaz
```

> `><` хвата CLI аргументе као низ стрингова (само tree-walker).

## Потпуни Пример: FizzBuzz

```zymbol
klasifikuj(broj) {
    ? broj % 15 == 0 { <~ "FizzBuzz" }
    _? broj % 3  == 0 { <~ "Fizz" }
    _? broj % 5  == 0 { <~ "Buzz" }
    _ { <~ broj }
}

@ i:1..20 { >> klasifikuj(i) ¶ }
```

## Референца Симбола

| Симбол | Операција | Симбол | Операција |
|--------|-----------|--------|-----------|
| `=` | променљива | `$#` | дужина |
| `:=` | константа | `$+` | додавање (повезиво) |
| `>>` | излаз | `$+[i]` | уметање на индекс (1-базно) |
| `<<` | улаз | `$-` | уклањање првог по вредности |
| `¶` / `\\` | нови ред | `$--` | уклањање свих по вредности |
| `?` | ако | `$-[i]` | уклањање на индексу (1-базно) |
| `_?` | иначе-ако | `$-[i..j]` | уклањање опсега (1-базно) |
| `_` | иначе / шаблон | `$?` | садржи |
| `??` | подударање | `$??` | налажење свих индекса (1-базно) |
| `@` | петља | `$[s..e]` | исечак (1-базно) |
| `@ N { }` | петља N пута | `$>` | map |
| `@!` | прекидање | `$\|` | filter |
| `@>` | настави | `$<` | reduce |
| `@:ime { }` | означена петља | `$/ razd` | раздвајање стринга |
| `@:ime!` | прекини означену | `$++ a b c` | конкатенација |
| `@:ime>` | настави означену | `niz[i>j>k]` | навигациони индекс |
| `->` | ламбда | `niz[i] = vred` | ажурирање елемента (само низови) |
| `niz[i] += vred` | сложено ажур. | `niz[i]$~` | функционално ажур. (нова копија) |
| `$^+` | сортирање растуће (прим.) | `$^-` | сортирање опадајуће (прим.) |
| `$^` | сортирање са компаратором (торке) | `<~` | враћање |
| `\|>` | цев | `!?` | покушај |
| `:!` | хватање | `:>` | на крају |
| `#1` | истина | `#0` | лаж |
| `$!` | је грешка | `$!!` | ширење грешке |
| `<#` | увоз | `#>` | извоз |
| `#` | декларација модула | `::` | позив модула |
| `.` | приступ пољу | `#?` | метаподаци типа |
| `#\|..\|` | парсирање броја | `##.` | конверзија у Децимални |
| `###` | конверзија у Цео (заокр.) | `##!` | конверзија у Цео (одсец.) |
| `#.N\|..\|` | заокруживање | `#!N\|..\|` | одсецање |
| `#,\|..\|` | формат са зарезима | `#^\|..\|` | научна нотација |
| `#d0d9#` | промена нумеричког режима | `#09#` | ресет на ASCII |
| `<\ ..\>` | извршавање shell-а | `>\<` | CLI аргументи |
| `\ prom` | изричито уклањање | `°x` / `x°` | врућа дефиниција (ауто-инит) |
| `>>|` | TUI блок (алт. екран) | `>>~` | позиционирани излаз |
| `>>!` | брисање екрана | `>>?` | упит о величини терминала |
| `<<\|` | блокирајућа тастатура | `<<\|?` | неблокирајућа тастатура |
| `@~ N` | спавање N милисекунди | `$*` | понављање стринга N пута |

- **Промена** Раздвојник гране подударања: `шаблон : резултат` → `шаблон => резултат`
