# Zymbol-Lang လက်စွဲ

> **v0.0.5 အတွက် ပြန်လည်သုံးသပ်ပြီး — 2026-05-14**

**Zymbol-Lang** သည် သင်္ကေတဆိုင်ရာ ပရိုဂရမ်းမင်းဘာသာစကားတစ်ခုဖြစ်သည်။ သော့ချက်စာလုံးများ မရှိပါ — အရာအားလုံးသည် သင်္ကေတဖြစ်သည်။ မည်သည့်လူ့ဘာသာစကားတွင်မဆို တစ်ထပ်တည်းအလုပ်လုပ်သည်။

## ပြောင်းလဲနိုင်သော တန်ဖိုးများနှင့် ကိန်းသေများ

```zymbol
x = ၁၀              // ပြောင်းလဲနိုင်သော ပြောင်းလဲတန်ဖိုး
π := ၃.၁၄၁၅၉        // ကိန်းသေ — ပြန်လည်သတ်မှတ်ခြင်းသည် အလုပ်လုပ်ချိန်အမှားဖြစ်သည်
အမည် = "အဲလစ်"
သက်ဝင် = #၁         // ဘူလီယန် မှန်သည်
👋 := "မင်္ဂလာပါ"
```

```zymbol
x = ၁၀    // ၁၀
x += ၅    // ၁၅
x -= ၃    // ၁၂
x *= ၂    // ၂၄
x /= ၃    // ၈
x %= ၃    // ၂
x ^= ၂    // ၄
x++        // ၅
x--        // ၄
```

`°` (ဒီဂရီသင်္ကေတ, U+00B0) သည် ပထမအသုံးပြုမှုတွင် ပြောင်းလဲတန်ဖိုးကို ၎င်း၏ ကြားနေတန်ဖိုးသို့ အလိုအလျောက် စတင်ပေးသည်:

```zymbol
ဂဏန်းများ = [၃, ၁, ၄, ၁, ၅]
@ n:ဂဏန်းများ {
    °စုစုပေါင်း += n    // ကွင်းဆက်၏အပေါ်တွင် အလိုအလျောက် ၀ သို့စတင်; @ ပြီးနောက် ရှင်သန်သည်
}
>> စုစုပေါင်း ¶         // → ၁၄
```

> `°x` (ရှေ့ဆက်) ကွင်းဆက်၏အပေါ်တွင် ကျောက်ချသည် — ရလဒ်ကို `@` ပြီးနောက် ဝင်ရောက်နိုင်သည်။
> `x°` (နောက်ဆက်) ကွင်းဆက်၏အတွင်းတွင် ကျောက်ချသည် — ကွင်းဆက်ပြီးဆုံးသောအခါ သေဆုံးသည်။
> tree-walker သာလျှင်။

## TUI အခြေခံအပိုင်းများ

အပြန်အလှန်တုံ့ပြန်နိုင်သော ပရိုဂရမ်များအတွက် ဂိတ်အသုံးပြုသူ ကြားခံအော်ပရေတာများ။ အများစုမှာ `>>| { }` ဘလောက် လိုအပ်သည် (အခြားမျက်နှာပြင် + ကုန်ကြမ်းမုဒ်)။

```zymbol
>>| {
    >>!                             // အခြားမျက်နှာပြင်ကို ရှင်းလင်းရန်
    >>~ (၁, ၁, ၀, ၁၀) > "လုပ်ဆောင်နေသည်"   // အတန်း ၁၊ ကော်လံ ၁၊ fg=၁၀ (အစိမ်း)
    @~ ၁၀၀၀                         // ၁ စက္ကန့် ခေတ္တရပ်ရန် (၁၀၀၀ ms)
    >>~ (၂, ၁) > "ပြီးစီးပြီ။"
}
// ထွက်သောအခါ ဂိတ်သည် အလိုအလျောက် ပြန်လည်ရယူသည်
```

```zymbol
// ကီးနှိပ်ခြင်းနှင့် ဂိတ်အရွယ်အစား
>>| {
    [အတန်းများ, ကော်လံများ] = >>?              // ဂိတ်၏ အတိုင်းအတာများကို မေးမြန်းရန်
    >>~ (၁, ၁) > "ဂိတ်: " အတန်းများ " x " ကော်လံများ
    <<| ကီး                         // ပိတ်ဆို့သည့် ကီးနှိပ်ခြင်းကို ဖတ်ရန်
    >>~ (၂, ၁) > "နှိပ်သည်: " ကီး
}
```

> `>>!` မျက်နှာပြင်ကို ရှင်းလင်းသည်။ `>>?` `[အတန်းများ, ကော်လံများ]` ပြန်ပေးသည်။ `@~ N` N မီလီစက္ကန့် အိပ်စက်သည်။
> `<<|` ကီးနှိပ်ခြင်းတစ်ခုကို ဖတ်သည် (ပိတ်ဆို့ခြင်း); `<<|?` ပိတ်ဆို့ခြင်းမရှိဘဲ စစ်ဆေးသည် (မရှိပါက `'\0'` ပြန်ပေးသည်)။
> နေရာအ�ိုက်အထွက် တပ်ပယ်: `(အတန်း, ကော်လံ, BKS, fg, bg)` — မည်သည့်နေရာကိုမဆို ကော်မာဖြင့် ချန်လှပ်နိုင်သည် (`>>~ (,,, 196) > "အနီရောင်"`)။
> BKS ဘစ်အလွှာ: `1`=အသားထူ, `2`=စောင်းစာလုံး, `4`=မျဉ်းအောက်။ ANSI 256 အရောင်အကွက် (`0`=ဂိတ်မူရင်း)။
> tree-walker သာလျှင် (`>>!`, `>>?`, `@~`, `>>~` မှလွဲ၍ ၎င်းတို့သည် `--vm` တွင်လည်း အလုပ်လုပ်သည်)။

```zymbol
@ i:၀..၄  { >> i " " }        // အတိုင်းအသွား ပါဝင်သည်:  ၀ ၁ ၂ ၃ ၄
@ i:၁..၉:၂ { >> i " " }       // ခြေလှမ်းဖြင့်:         ၁ ၃ ၅ ၇ ၉
@ i:၅..၀:၁ { >> i " " }       // ပြောင်းပြန်:           ၅ ၄ ၃ ၂ ၁ ၀

n = ၁
@ n <= ၆၄ { n *= ၂ }
>> n ¶                        // → ၁၂၈  (while)

အသီးများ = ["ပန်းသီး", "သစ်တော်သီး", "စပျစ်သီး"]
@ အသီး:အသီးများ { >> အသီး ¶ }         // ခင်းကျင်းမှု၏ ဒြပ်စင်တစ်ခုစီအတွက်

@ စ:"မင်္ဂလာပါ" { >> စ "-" }
>> ¶                          // → မ-င-်-ဂ-လ-ာ-ပ-ါ-  (စာကြောင်း၏ စာလုံးတစ်ခုစီအတွက်)

@ i:၁..၁၀ {
    ? i % ၂ == ၀ { @> }       // @> ဆက်လုပ်ရန်
    ? i > ၇ { @! }             // @! ဖျက်ရန်
    >> i " "
}
>> ¶                          // → ၁ ၃ ၅ ၇

// အဆုံးမရှိ ကွင်းဆက်
i = ၀
@ {
    i++
    ? i >= ၅ { @! }
    >> i " "
}
>> ¶                          // → ၁ ၂ ၃ ၄

// အညွှန်းတပ်ထားသော ကွင်းဆက် (အသိုက်လိုက် ဖျက်ခြင်း)
အရေအတွက် = ၀
@:ပြင်ပ {
    အရေအတွက်++
    ? အရေအတွက် >= ၃ { @:ပြင်ပ! }
}
>> အရေအတွက် ¶                    // → ၃
```

## လုပ်ဆောင်ချက်များ

```zymbol
ပေါင်း(a, b) { <~ a + b }
>> ပေါင်း(၃, ၄) ¶    // → ၇

ဆခွဲကိန်း(n) {
    ? n <= ၁ { <~ ၁ }
    <~ n * ဆခွဲကိန်း(n - ၁)
}
>> ဆခွဲကိန်း(၅) ¶    // → ၁၂၀
```

လုပ်ဆောင်ချက်များသည် **သီးခြားနယ်ပယ်** ရှိသည် — ၎င်းတို့သည် ပြင်ပပြောင်းလဲတန်ဖိုးများကို ဖတ်၍မရပါ။ ခေါ်ဆိုသူ၏ ပြောင်းလဲတန်ဖိုးများကို ပြောင်းလဲရန် အထွက်သတ်မှတ်ချက်များ `<~` ကို သုံးပါ:

```zymbol
လဲလှယ်(a<~, b<~) {
    ယာယီ = a
    a = b
    b = ယာယီ
}
x = ၁၀
y = ၂၀
လဲလှယ်(x, y)
>> "x=" x " y=" y ¶    // → x=၂၀ y=၁၀
```

> အမည်ပေးထားသော လုပ်ဆောင်ချက်များသည် **ပထမအဆင့် တန်ဖိုးများ** ဖြစ်သည် — တိုက်ရိုက်ပေးပို့ပါ: `ဂဏန်းများ$> နှစ်ဆ`။ ထုပ်ရန်: `x -> fn(x)` လည်း ခိုင်လုံသည်။

## လမ်ဘဒါများနှင့် ပိတ်ဆို့မှုများ

```zymbol
နှစ်ဆ = x -> x * ၂
ပေါင်း = (a, b) -> a + b
>> နှစ်ဆ(၅) ¶    // → ၁၀
>> ပေါင်း(၃, ၇) ¶  // → ၁၀

// ဘလောက် လမ်ဘဒါ
ခွဲခြားသတ်မှတ် = x -> {
    ? x > ၀ { <~ "အပေါင်း" }
    _? x < ၀ { <~ "အနုတ်" }
    <~ "သုည"
}

// ပိတ်ဆို့မှု — ပြင်ပနယ်ပယ်ကို ဖမ်းယူသည်
အကြောင်းရင်း = ၃
သုံးဆ = x -> x * အကြောင်းရင်း
>> သုံးဆ(၇) ¶    // → ၂၁

// စက်ရုံ
ပေါင်း_ဖန်တီးသူ(n) { <~ x -> x + n }
ဆယ်ပေါင်း = ပေါင်း_ဖန်တီးသူ(၁၀)
>> ဆယ်ပေါင်း(၅) ¶    // → ၁၅

// ခင်းကျင်းမှုတွင်
အော်ပရေတာများ = [x -> x+၁, x -> x*၂, x -> x*x]
>> အော်ပရေတာများ[၃](၅) ¶    // → ၂၅
```

## ခင်းကျင်းမှုများ

ခင်းကျင်းမှုများသည် **ပြောင်းလဲနိုင်** ပြီး **တူညီသောအမျိုးအစား** ဒြပ်စင်များကို သိမ်းဆည်းသည်။

```zymbol
arr = [၁, ၂, ၃, ၄, ၅]

x = arr[၁]      // ၁ — ဝင်ရောက်ခြင်း (၁-အခြေခံ: ပထမဒြပ်စင်)
x = arr[-၁]     // ၅ — အနုတ်ညွှန်းကိန်း (နောက်ဆုံးဒြပ်စင်)
x = arr$#       // ၅ — အလျား (>> တွင် (arr$#) သုံးပါ)

arr = arr$+ ၆            // ပေါင်းထည့်ရန် → [၁,၂,၃,၄,၅,၆]
arr၂ = arr$+[၂] ၉၉       // နေရာ ၂ တွင် ထည့်သွင်းရန် (၁-အခြေခံ)
arr၃ = arr$- ၃           // တန်ဖိုး၏ ပထမအကြိမ်ကို ဖယ်ရှားရန်
arr၄ = arr$-- ၃          // တန်ဖိုး၏ အကြိမ်အားလုံးကို ဖယ်ရှားရန်
arr၅ = arr$-[၁]          // ညွှန်းကိန်း ၁ တွင် ဖယ်ရှားရန် (ပထမဒြပ်စင်)
arr၆ = arr$-[၂..၃]       // အတိုင်းအသွားကို ဖယ်ရှားရန် (၁-အခြေခံ၊ အဆုံးပါဝင်သည်)

ရှိသည် = arr$? ၃            // #၁ — ပါဝင်သည်
နေရာများ = arr$?? ၃           // [၃] — တန်ဖိုး၏ ညွှန်းကိန်းအားလုံး (၁-အခြေခံ)
အတိုပိုင်း = arr$[၁..၃]          // [၁,၂,၃] — အတိုပိုင်း (၁-အခြေခံ၊ အဆုံးပါဝင်သည်)
အတိုပိုင်း၂ = arr$[၁:၃]          // [၁,၂,၃] — တူညီသည်၊ အရေအတွက်-အခြေခံ သဒ္ဒါ

အစီအစဉ်ကျ = arr$^+             // အစီအစဉ်ကျအောင် စီရန် (အခြေခံများသာ)
အစီအစဉ်ကျ_ပြန် = arr$^-            // အစီအစဉ်ကျပြန်အောင် စီရန် (အခြေခံများသာ)

// အမည်ပေး/နေရာအလိုက် တပ်ပယ် ခင်းကျင်းမှုများ — နှိုင်းယှဉ် လမ်ဘဒါဖြင့် $^ သုံးပါ
ဒေတာဘေ့စ် = [(အမည်: "ကာလာ", အသက်: ၂၈), (အမည်: "အနာ", အသက်: ၂၅), (အမည်: "ဘော့", အသက်: ၃၀)]
အသက်အလိုက်  = ဒေတာဘေ့စ်$^ (a, b -> a.အသက် < b.အသက်)    // အသက်အလိုက် အစီအစဉ်ကျ (<)
အမည်အလိုက် = ဒေတာဘေ့စ်$^ (a, b -> a.အမည် > b.အမည်)   // အမည်အလိုက် အစီအစဉ်ကျပြန် (>)
>> အသက်အလိုက်[၁].အမည် ¶     // → အနာ
>> အမည်အလိုက်[၁].အမည် ¶    // → ကာလာ

// တိုက်ရိုက်ဒြပ်စင် မွမ်းမံခြင်း (ခင်းကျင်းမှုများသာ)
arr[၁] = ၉၉              // သတ်မှတ်ရန်
arr[၂] += ၅              // ပေါင်းစပ်: +=  -=  *=  /=  %=  ^=

// လုပ်ဆောင်ချက်ဆိုင်ရာ မွမ်းမံခြင်း — ခင်းကျင်းမှုအသစ်ကို ပြန်ပေးသည်; မူရင်းမပြောင်းလဲ
arr၂ = arr[၂]$~ ၉၉
```

> စုစည်းမှုအော်ပရေတာအားလုံးသည် **ခင်းကျင်းမှုအသစ်** ကို ပြန်ပေးသည်။ ပြန်သတ်မှတ်ပါ: `arr = arr$+ 4`။
> `$+` ကို ဆက်စပ်နိုင်သည်: `arr = arr$+ 5$+ 6$+ 7`။ အခြားအော်ပရေတာများသည် ကြားခံသတ်မှတ်ချက်များကို သုံးသည်။
> **ညွှန်းကိန်းသတ်မှတ်ခြင်း 1-အခြေခံ**: `arr[1]` သည် ပထမဒြပ်စင်ဖြစ်သည်; `arr[0]` သည် အလုပ်လုပ်ချိန်အမှားဖြစ်သည်။
> `$^+` / `$^-` သည် **အခြေခံ ခင်းကျင်းမှုများ** (ဂဏန်းများ, စာကြောင်းများ) ကို စီသည်။ တပ်ပယ် ခင်းကျင်းမှုများအတွက် နှိုင်းယှဉ်လမ်ဘဒါဖြင့် `$^` သုံးပါ — ဦးတည်ချက်ကို လမ်ဘဒါတွင် ကုဒ်လုပ်ထားသည် (`<` = အစီအစဉ်ကျ, `>` = အစီအစဉ်ကျပြန်)။

**တန်ဖိုး အဓိပ္ပာယ်ဖွင့်ဆိုချက်** — ခင်းကျင်းမှုတစ်ခုကို အခြားပြောင်းလဲတန်ဖိုးသို့ သတ်မှတ်ခြင်းသည် သီးခြားမိတ္ထိတစ်ခုကို ဖန်တီးသည်:

```zymbol
a = [၁, ၂, ၃]
b = a
a[၁] = ၉၉
>> a ¶    // → [၉၉, ၂, ၃]
>> b ¶    // → [၁, ၂, ၃]   ← b သည် သက်ရောက်မှုမရှိပါ
```

```zymbol
// အသိုက်လိုက် ခင်းကျင်းမှုများ (၁-အခြေခံ ညွှန်းကိန်းသတ်မှတ်ခြင်း)
မက်ထရစ် = [[၁,၂,၃],[၄,၅,၆],[၇,၈,၉]]
>> မက်ထရစ်[၂][၃] ¶    // → ၆  (အတန်း ၂, ကော်လံ ၃)
```

## ဖွဲ့စည်းပုံဖျက်သိမ်းခြင်း

```zymbol
// ခင်းကျင်းမှု
arr = [၁၀, ၂၀, ၃၀, ၄၀, ၅၀]
[a, b, c] = arr              // a=၁၀  b=၂၀  c=၃၀
[ပထမ, *ကျန်သည်] = arr         // ပထမ=၁၀  ကျန်သည်=[၂၀,၃၀,၄၀,၅၀]
[x, _, z] = [၁, ၂, ၃]        // _ စွန့်ပစ်သည်

// နေရာအလိုက် တပ်ပယ်
အမှတ် = (၁၀၀, ၂၀၀)
(px, py) = အမှတ်             // px=၁၀၀  py=၂၀၀

// အမည်ပေးထားသော တပ်ပယ်
ပုဂ္ဂိုလ် = (အမည်: "အနာ", အသက်: ၂၅, မြို့: "မဒရစ်")
(အမည်: n, အသက်: a) = ပုဂ္ဂိုလ်   // n="အနာ"  a=၂၅
```

## တပ်ပယ်များ

တပ်ပယ်များသည် **ပြောင်းလဲမရသော** အစဉ်လိုက်ကွန်တိန်နာများဖြစ်ပြီး ၎င်းတို့သည် **မတူညီသောအမျိုးအစားများ၏** တန်ဖိုးများကို သိမ်းဆည်းနိုင်သည်။
ခင်းကျင်းမှုများနှင့်မတူဘဲ၊ ဖန်တီးပြီးနောက် ဒြပ်စင်များကို ပြောင်းလဲ၍မရပါ။

```zymbol
// နေရာအလိုက် — ရောနှောထားသောအမျိုးအစားများကို ခွင့်ပြုသည်
အမှတ် = (၁၀, ၂၀)
>> အမှတ်[၁] ¶    // → ၁၀

ဒေတာ = (၄၂, "မင်္ဂလာပါ", #၁, ၃.၁၄)
>> ဒေတာ[၃] ¶     // → #၁

// အမည်ပေးထားသော
ပုဂ္ဂိုလ် = (အမည်: "အဲလစ်", အသက်: ၂၅)
>> ပုဂ္ဂိုလ်.အမည် ¶    // → အဲလစ်
>> ပုဂ္ဂိုလ်[၁] ¶      // → အဲလစ်  (ညွှန်းကိန်းလည်း အလုပ်လုပ်သည်၊ ၁-အခြေခံ)

// အသိုက်လိုက်
နေရာ = (x: ၁၀, y: ၂၀)
p = (နေရာ: နေရာ, အညွှန်း: "မူလအစ")
>> p.နေရာ.x ¶        // → ၁၀
```

**ပြောင်းလဲမရခြင်း** — တပ်ပယ်ဒြပ်စင်တစ်ခုကို ပြောင်းလဲရန် ကြိုးစားမှုတိုင်းသည် အလုပ်လုပ်ချိန်အမှားဖြစ်သည်:

```zymbol
t = (၁၀, ၂၀, ၃၀)
// t[၁] = ၉၉    // ❌ အလုပ်လုပ်ချိန်အမှား: တပ်ပယ်များသည် ပြောင်းလဲမရပါ
// t[၁] += ၅    // ❌ တူညီသောအမှား
```

ပြုပြင်ထားသောတန်ဖိုးကိုရရှိရန် `$~` (လုပ်ဆောင်ချက်ဆိုင်ရာ မွမ်းမံခြင်း) သုံးပါ — **တပ်ပယ်အသစ်** ကို ပြန်ပေးသည်:

```zymbol
t = (၁၀, ၂၀, ၃၀)
t၂ = t[၂]$~ ၉၉၉
>> t ¶     // → (၁၀, ၂၀, ၃၀)   ← မူရင်းမပြောင်းလဲ
>> t၂ ¶    // → (၁၀, ၉၉၉, ၃၀)

// အမည်ပေးထားသော တပ်ပယ် — ရှင်းလင်းစွာ ပြန်လည်တည်ဆောက်ပါ
ပုဂ္ဂိုလ် = (အမည်: "အဲလစ်", အသက်: ၂၅)
ကြီးသည်  = (အမည်: ပုဂ္ဂိုလ်.အမည်, အသက်: ၂၆)
>> ပုဂ္ဂိုလ်.အသက် ¶    // → ၂၅
>> ကြီးသည်.အသက် ¶     // → ၂၆
```

## မြင့်မားသောအစဉ်လိုက် လုပ်ဆောင်ချက်များ

```zymbol
ဂဏန်းများ = [၁, ၂, ၃, ၄, ၅, ၆, ၇, ၈, ၉, ၁၀]

နှစ်ဆ_လုပ်ထားသော  = ဂဏန်းများ$> (x -> x * ၂)                  // map  → [၂,၄,၆…၂၀]
စုံ    = ဂဏန်းများ$| (x -> x % ၂ == ၀)           // filter → [၂,၄,၆,၈,၁၀]
စုစုပေါင်း    = ဂဏန်းများ$< (၀, (စုဆောင်းသူ, x) -> စုဆောင်းသူ + x)     // reduce → ၅၅

// ကြားခံများမှတစ်ဆင့် ဆက်စပ်ရန်
အဆင့်၁ = ဂဏန်းများ$| (x -> x > ၃)
အဆင့်၂ = အဆင့်၁$> (x -> x * x)
>> အဆင့်၂ ¶    // → [၁၆, ၂၅, ၃၆, ၄၉, ၆၄, ၈၁, ၁၀၀]

// အမည်ပေးထားသော လုပ်ဆောင်ချက်များကို HOF သို့ တိုက်ရိုက်ပေးပို့နိုင်သည်
နှစ်ဆ(x) { <~ x * ၂ }
ကြီးသည်_လား(x) { <~ x > ၅ }
r = ဂဏန်းများ$> နှစ်ဆ       // ✅ တိုက်ရိုက်ကိုးကားချက်
r = ဂဏန်းများ$| ကြီးသည်_လား       // ✅ တိုက်ရိုက်ကိုးကားချက်
```

## ပိုက်အော်ပရေတာ

ညာဘက်ခြမ်းသည် အမြဲတမ်း ပိုက်သွင်းထားသောတန်ဖိုးအတွက် `_` နေရာထားသူ လိုအပ်သည်:

```zymbol
နှစ်ဆ = x -> x * ၂
ပေါင်း = (a, b) -> a + b
တိုး = x -> x + ၁

ရ၁ = ၅ |> နှစ်ဆ(_)        // → ၁၀
ရ၂ = ၁၀ |> ပေါင်း(_, ၅)       // → ၁၅
ရ၃ = ၅ |> ပေါင်း(၂, _)        // → ၇

// ဆက်စပ်ထားသော
ရ = ၅ |> နှစ်ဆ(_) |> တိုး(_) |> နှစ်ဆ(_)
>> ရ ¶    // → ၂၂  (၅→၁၀→၁၁→၂၂)
```

## အမှားကိုင်တွယ်ခြင်း

```zymbol
!? {
    x = ၁၀ / ၀
} :! ##Div {
    >> "သုညဖြင့် စားခြင်း" ¶
```

## ဂဏန်းပုံစံများ

Zymbol သည် ဂဏန်းများကို **69 ယူနီကုဒ် ဂဏန်းအက္ခရာများဖြင့်** ပြသနိုင်သည် — ဒေဝနာဂရီ၊ အာရေဗျ-အိန္ဒိယ၊ ထိုင်း၊ Klingon pIqaD၊ သင်္ချာသားထူ၊ LCD အပိုင်းများ၊ နှင့် အခြားအများအပြား။ သက်ဝင်နေသော ပုံစံသည် `>>` အထွက်ကိုသာ သက်ရောက်မှုရှိသည်; အတွင်းပိုင်းဂဏန်းသင်္ချာသည် အမြဲတမ်း ဒွိစုံဖြစ်သည်။

### အက္ခရာတစ်ခုကို သက်ဝင်စေခြင်း

ပစ်မှတ်အက္ခရာ၏ `0` နှင့် `9` ဂဏန်းများကို `#…#` အတွင်းတွင် ရေးပါ:

```zymbol
#०९#    // ဒေဝနာဂရီ   (U+0966–U+096F)
#٠٩#    // အာရေဗျ-အိန္ဒိယ (U+0660–U+0669)
#๐๙#    // ထိုင်း         (U+0E50–U+0E59)
#09#    // ASCII သို့ ပြန်လည်သတ်မှတ်ရန်
```

### အထွက်နှင့် ဘူလီယန်များ

```zymbol
x = ၄၂
>> x ¶          // → ၄၂   (ASCII မူရင်း)

#०९#
>> x ¶          // → ४२
>> ၃.၁၄ ¶       // → ३.१४   (ဒသမအမှတ်သည် အမြဲတမ်း ASCII)
>> ၁ + ၂ ¶      // → ३

// ဘူလီယန်များ: # ရှေ့ဆက်သည် အမြဲတမ်း ASCII၊ ဂဏန်းသည် လိုက်လျောညီထွေဖြစ်သည်
>> #၁ ¶         // → #१   (ဒေဝနာဂရီတွင် မှန်သည်)
>> #၀ ¶         // → #०   (မှားသည် — ၀ ကိန်းပြည့်သုညနှင့် ကွဲပြားသည်)

x = ၂၈ > ၄
>> x ¶          // → #१   (နှိုင်းယှဉ်မှုရလဒ်သည် သက်ဝင်နေသောပုံစံကို လိုက်နာသည်)
```

### ရင်းမြစ်တွင် ဇာတိဂဏန်း စာသားအတိုင်းများ

မည်သည့်ထောက်ပံ့ထားသော အက္ခရာ၏ ဂဏန်းများသည် ခိုင်လုံသော စာသားအတိုင်းများဖြစ်သည် — အတိုင်းအသွားများ၊ အကြွင်းပိုင်း၊ နှိုင်းယှဉ်မှုများတွင်:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### မည်သည့်အက္ခရာတွင်မဆို ဘူလီယန် စာသားအတိုင်းများ

`#` + မည်သည့်ဘလောက်မှ ဂဏန်း `0` သို့မဟုတ် `1` သည် ခိုင်လုံသော ဘူလီယန် စာသားအတိုင်းဖြစ်သည်:

```zymbol
#٠٩#
သက်ဝင် = #١        // #၁ နှင့် တူညီသည်
>> သက်ဝင် ¶        // → #१
>> (#١ && #٠) ¶ // → #०
```

> `#` **အမြဲတမ်း ASCII** ဖြစ်သည်။ `#0` (မှားသည်) သည် အက္ခရာတိုင်းတွင် `0` (ကိန်းပြည့်သုည) မှ အမြဲတမ်း အမြင်အားဖြင့် ကွဲပြားသည်။

## ဒေတာအော်ပရေတာများ

```zymbol
// အမျိုးအစားပြောင်းလဲခြင်း ပုံသွန်းခြင်း
f = ##.၄၂         // → ၄၂.၀  (ရွေ့လျောနံပါတ်သို့)
i = ###၃.၇        // → ၄     (ကိန်းပြည့်သို့, ဝိုင်းရန်)
t = ##!၃.၇        // → ၃     (ကိန်းပြည့်သို့, ဖြတ်တောက်ရန်)

// စာကြောင်းကို ဂဏန်းအဖြစ် စာကြောင်းခွဲခြမ်းရန်
v၁ = #|"၄၂"|      // → ၄၂  (ကိန်းပြည့်)
v၂ = #|"၃.၁၄"|    // → ၃.၁၄  (ရွေ့လျောနံပါတ်)
v၃ = #|"abc"|     // → "abc"  (ဘေးကင်းသည်, အမှားမရှိ)

// ဝိုင်းရန် / ဖြတ်တောက်ရန်
π = ၃.၁၄၁၅၉၂၆၅
ဝိုင်း၂ = #.၂|π|      // → ၃.၁၄  (ဒသမနေရာ ၂ သို့ ဝိုင်းရန်)
ဝိုင်း၄ = #.၄|π|      // → ၃.၁၄၁၆
ဖြတ်၂ = #!၂|π|      // → ၃.၁၄  (ဖြတ်တောက်ရန်)

// ဂဏန်းဖော်မတ်ချခြင်း
ဖော်မတ် = #,|၁၂၃၄၅၆၇|  // → ၁,၂၃၄,၅၆၇  (ကော်မာ-ပိုင်းခြားထားသည်)
သိပ္ပံနည်းကျ = #^|၁၂၃၄၅.၆၇၈|    // → ၁.၂၃၄၅၆၇၈e၄  (သိပ္ပံနည်းကျ)

// အခြေခံ စာသားအတိုင်းများ
a = 0x41         // → 'A'  (ဆယ့်ခြောက်ဂဏန်း)
b = 0b01000001   // → 'A'  (ဒွိစုံ)
c = 0o101        // → 'A'  (ရှစ်ဂဏန်း)

// အခြေခံပြောင်းလဲခြင်း အထွက်
ဆယ့်ခြောက်ဂဏန်း = 0x|၂၅၅|    // → "0x00FF"
ဒွိစုံ = 0b|၆၅|     // → "0b1000001"
ရှစ်ဂဏန်း = 0o|၈|      // → "0o10"
ဒသမ = 0d|၂၅၅|    // → "0d0255"
```

## သင်္ကေတ ကိုးကားချက်

| သင်္ကေတ | လုပ်ဆောင်ချက် | သင်္ကေတ | လုပ်ဆောင်ချက် |
|--------|-----------|--------|-----------|
| `=` | ပြောင်းလဲတန်ဖိုး | `$#` | အလျား |
| `:=` | ကိန်းသေ | `$+` | ပေါင်းထည့်ရန် (ဆက်စပ်နိုင်သည်) |
| `>>` | အထွက် | `$+[i]` | ညွှန်းကိန်းတွင် ထည့်သွင်းရန် (1-အခြေခံ) |
| `<<` | အသွင်း | `$-` | တန်ဖိုးအရ ပထမတစ်ခုကို ဖယ်ရှားရန် |

## ထုတ်ပြန်မှု ပြောင်းလဲမှုမှတ်တမ်း

### v0.0.5 — TUI အခြေခံအပိုင်းများ, အဆင်သင့် အဓိပ္ပာယ်ဖွင့်ဆိုချက်နှင့် စာကြောင်း ထပ်ခါထပ်ခါလုပ်ခြင်း _(မေ 2026)_

- **ဖျက်လိုက်သော** ကိုက်ညီမှုလက်တံ ခွဲခြားသူ: `ပုံစံ : ရလဒ်` → `ပုံစံ => ရလဒ်`
- **ဖျက်လိုက်သော** သွင်းကုန် အမည်ပြောင်: `<# လမ်းကြောင်း <= အမည်ပြောင်` → `<# လမ်းကြောင်း => အမည်ပြောင်`
- **ဖျက်လိုက်သော** ထုတ်ကုန် အမည်ပြောင်းလဲခြင်း: `#> { fn <= အများသုံး }` → `#> { fn => အများသုံး }`
- **ထည့်သွင်းပြီး** TUI ဘလောက် `>>| { }` — အခြားမျက်နှာပြင် + ကုန်ကြမ်းမုဒ်; ထွက်သောအခါ ရှင်းလင်းသည်
- **ထည့်သွင်းပြီး** နေရာအလိုက် အထွက် `>>~ (အတန်း, ကော်လံ, BKS, fg, bg) > အရာများ` — ကြားနေနေရာများ၊ ANSI 256 အရောင်
- **ထည့်သွင်းပြီး** ကီးသွင်း `<<| ပြောင်းလဲတန်ဖိုး` (ပိတ်ဆို့ခြင်း) နှင့် `<<|? ပြောင်းလဲတန်ဖိုး` (ပိတ်ဆို့ခြင်းမရှိသော စစ်ဆေးခြင်း)
- **ထည့်သွင်းပြီး** `>>!` မျက်နှာပြင်ကို ရှင်းလင်းရန်, `>>?` ဂိတ်အရွယ်အစားကို မေးမြန်းရန်, `@~ N` N မီလီစက္ကန့် အိပ်စက်ရန်
- **ထည့်သွင်းပြီး** အဆင်သင့် အဓိပ္ပာယ်ဖွင့်ဆိုချက် `°x` / `x°` — ကွင်းဆက်များတွင် ပထမအသုံးပြုမှု၌ ပြောင်းလဲတန်ဖိုးကို အလိုအလျောက်စတင်ခြင်း
- **ထည့်သွင်းပြီး** စာကြောင်း ထပ်ခါထပ်ခါလုပ်ခြင်း `စာကြောင်း $* N` — စာကြောင်းတစ်ခုကို N ကြိမ် ထပ်ခါထပ်ခါလုပ်ရန်
- **VM** ညီမျှမှု: 436/436 စမ်းသပ်မှုများ အောင်မြင်

### v0.0.4 — 1-အခြေခံ ညွှန်းကိန်းသတ်မှတ်ခြင်း, ပထမအဆင့် လုပ်ဆောင်ချက်များနှင့် ဘလောက် မော်ဂျူးများ _(ဧပြီ 2026)_

- **ဖျက်လိုက်သော** ညွှန်းကိန်းသတ်မှတ်ခြင်းအားလုံးကို **1-အခြေခံ** သို့ ပြောင်းလဲခဲ့သည် — `arr[1]` သည် ပထမဒြပ်စင်ဖြစ်သည်; `arr[0]` သည် အလုပ်လုပ်ချိန်အမှားဖြစ်သည်
- **ထည့်သွင်းပြီး** အမည်ပေးထားသော လုပ်ဆောင်ချက်များသည် **ပထမအဆင့် တန်ဖိုးများ** ဖြစ်သည် — HOF သို့ တိုက်ရိုက်ပေးပို့ရန်: `ဂဏန်းများ$> နှစ်ဆ`
- **ထည့်သွင်းပြီး** မော်ဂျူးများအတွက် **ဘလောက် သဒ္ဒါ လိုအပ်သည်** — `# အမည် { ... }` — ညီညာသော သဒ္ဒါကို ဖယ်ရှားခဲ့သည်
- **ထည့်သွင်းပြီး** ဘက်စုံဒိုင်မင်းရှင် ညွှန်းကိန်းသတ်မှတ်ခြင်း: `arr[i>j>k]` (သွားလာမှု), `arr[p ; q]` (ညီညာသော နုတ်ယူခြင်း)
- **ထည့်သွင်းပြီး** အမျိုးအစားပြောင်းလဲခြင်း ပုံသွန်းခြင်း: `##.အသုံးအနှုန်း` (ရွေ့လျောနံပါတ်), `###အသုံးအနှုန်း` (ကိန်းပြည့်ဝိုင်းရန်), `##!အသုံးအနှုန်း` (ကိန်းပြည့်ဖြတ်တောက်ရန်)
- **ထည့်သွင်းပြီး** စာကြောင်းခွဲခြင်း: `စာကြောင်း$/ ပိုင်းခြားသင်္ကေတ` — `Array(စာကြောင်း)` ကို ပြန်ပေးသည်
- **ထည့်သွင်းပြီး** ပေါင်းစပ် တည်ဆောက်ခြင်း: `အခြေခံ$++ a b c` — အရာများစွာကို ပေါင်းထည့်သည်
- **ထည့်သွင်းပြီး** အကြိမ်အရေအတွက် ကွင်းဆက်: `@ N { }` — အတိအကျ N ကြိမ် ထပ်ခါထပ်ခါလုပ်ရန်
- **ထည့်သွင်းပြီး** အညွှန်းတပ်ထားသော ကွင်းဆက် သဒ္ဒါ: `@:အမည် { }`, `@:အမည်!`, `@:အမည်>` — `@ @အမည်` / `@! အမည်` ကို အစားထိုးသည်

- **ထည့်သွင်းပြီး** ခင်းကျင်းမှုများ, တပ်ပယ်များနှင့် အမည်ပေးထားသော တပ်ပယ်များအတွက် ဖွဲ့စည်းပုံဖျက်သိမ်းခြင်း သတ်မှတ်ခြင်း
- **ထည့်သွင်းပြီး** အနုတ်ညွှန်းကိန်းများ (`arr[-1]` = နောက်ဆုံးဒြပ်စင်)
- **ထည့်သွင်းပြီး** ဇာတိတပ်ဆင်သူများ — Linux (deb/rpm/pkg/musl), macOS (Intel + Apple Silicon), Windows (MSI, winget)

### v0.0.1 — ကနဦးအများပြည်သူထုတ်ပြန်ခြင်း _(မတ် 22, 2026)_
