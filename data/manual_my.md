# Zymbol-Lang အကျဉ်းချုပ် လမ်းညွှန်

**Zymbol-Lang** သည် သင်္ကေတအခြေပြု ပရိုဂရမ်းမင်းဘာသာစကားတစ်ခုဖြစ်သည်။ ၎င်းသည် keyword များကို အသုံးမပြုဘဲ — အရာအားလုံးသည် သင်္ကေတသာဖြစ်သည်။ လူသားဘာသာစကားမည်သည့်ခုတွင်မဆို တူညီစွာ အလုပ်လုပ်သည်။

- Keyword များမရှိ (`if`, `while`, `return` မတည်ရှိ — သင်္ကေတများ `?`, `@`, `<~` သာရှိသည်)
- Unicode ပြည့်စုံစွာ ပံ့ပိုးသည် — ဘာသာစကားမည်သည့်ခုဖြင့်မဆို သို့မဟုတ် Emoji 👋 ဖြင့် identifier သတ်မှတ်နိုင်သည်
- ဘာသာစကားမကွဲပြားသော ဒီဇိုင်း — ကုဒ်သည် ဘာသာစကားအားလုံးတွင် တူညီသည်

## ကိန်းရှင်များနှင့် ကိန်းသေများ

```zymbol
ပ = ၁၀           // ကိန်းရှင် (ပြောင်းလဲနိုင်သည်)
ပိုင် := ၃.၁၄၁၅၉    // ကိန်းသေ (ပြောင်းလဲမရ — ပြန်သတ်မှတ်ရင် error ဖြစ်သည်)
နာမည် = "အာနာ"
တက်ကြွ = #၁     // boolean true
👋 := "မင်္ဂလာပါ"
```

```zymbol
x = ၁၀
x += ၅    // ၁၅
x -= ၃    // ၁၂
x *= ၂    // ၂၄
x /= ၃    // ၈
x %= ၃    // ၂
x ^= ၂    // ၄
x++       // ၅
x--       // ၄
```

## ဒေတာအမျိုးအစားများ

| အမျိုးအစား       | ဥပမာ                | သင်္ကေတ `#?` | မှတ်ချက်                              |
|------------------|---------------------|--------------|---------------------------------------|
| ကိန်းပြည့်       | `၄၂`, `-၇`          | `###`        | 64-bit လက္ခဏာပါသော                   |
| ဒဿမကိန်း        | `၃.၁၄`, `၁.၅e၁၀`    | `##.`        | သိပ္ပံပညာဆိုင်ရာ notation ရနိုင်သည်  |
| စာကြောင်း        | `"မင်္ဂလာ"`         | `##"`        | Interpolation: `"မင်္ဂလာ {နာမည်}"`   |
| စာလုံးတစ်လုံး   | `'A'`               | `##'`        | Unicode စာလုံးတစ်လုံး                 |
| ဘူလီယံ          | `#၁`, `#၀`          | `##?`        | ကိန်း 1 နှင့် 0 မဟုတ်ပါ              |
| ခင်း             | `[၁, ၂, ၃]`         | `##]`        | တူညီသောအမျိုးအစားဒြပ်စင်များ          |
| တပ်ပုလ်          | `(a, b)`            | `##)`        | နေရာဆိုင်ရာ                           |
| နာမည်ပေး တပ်ပုလ်| `(x: ၁, y: ၂)`      | `##)`        | နာမည် သို့မဟုတ် index ဖြင့်ဝင်ရောက်  |

```zymbol
// အမျိုးအစားစစ်ဆေးခြင်း — (အမျိုးအစား, ဂဏန်း, တန်ဖိုး) ပြန်သည်
meta = ၄၂#?
>> meta ¶         // → (###, ၂, ၄၂)
t = meta[၀]
>> t ¶            // → ###
```

## အထွက်နှင့် အဝင်

```zymbol
>> "မင်္ဂလာပါ" ¶                   // ¶ သို့မဟုတ် \\ သည် တိကျသော newline ဖြစ်သည်
>> "က=" က " ခ=" ခ ¶                // juxtaposition ဖြင့် တန်ဖိုးများစွာ
>> (ခင်း$#) ¶                       // Postfix operator များသည် ကွင်းစကွင်းပိတ်လိုသည်

<< နာမည်                           // prompt မပါဘဲ — variable ထဲ ဖတ်သည်
<< "သင်၏နာမည်? " နာမည်            // prompt ပါသည်
```

> `¶` သို့မဟုတ် `\\` တို့သည် newline အဖြစ် ညီမျှသည်။

## အော်ပရေတာများ

```zymbol
// သင်္ချာ
က = ၁၀
ခ = ၃
ရ၁ = က + ခ    // ၁၃     ရ၂ = က - ခ    // ၇
ရ၃ = က * ခ    // ၃၀     ရ၄ = က / ခ    // ၃  (integer ဆ)
ရ၅ = က % ခ    // ၁      ရ၆ = က ^ ခ    // ၁၀၀၀  (ထပ်တင်)

// နှိုင်းယှဉ်
က == ခ    // #၀    က <> ခ    // #၁    က < ခ    // #၀
က <= ခ    // #၀   က > ခ     // #၁    က >= ခ   // #၁

// ယုတ္တိ
#၁ && #၀    // #၀
#၁ || #၀    // #၁
!#၁         // #၀
```

## စာကြောင်းများ

ပုံသဏ္ဍာန်သုံးခု တရားဝင်ဖြစ်သည် — တစ်ခုချင်းစီ context အလိုက်:

```zymbol
နာမည် = "အာနာ"
ည = ၂၅

// ၁. ကော်မာ — = သို့မဟုတ် := ဖြင့် သတ်မှတ်ချက်များတွင်
မက်ဆ = "မင်္ဂလာ ", နာမည်, "!"               // → မင်္ဂလာ အာနာ!

// ၂. Juxtaposition — >> output တွင်
>> "မင်္ဂလာ " နာမည် " သင်သည် " ည ¶        // → မင်္ဂလာ အာနာ သင်သည် ၂၅

// ၃. Interpolation — မည်သည့် context တွင်မဆို
ဖော်ပြချက် = "မင်္ဂလာ {နာမည်}, သင်သည် {ည}"  // → မင်္ဂလာ အာနာ, သင်သည် ၂၅
```

```zymbol
စ = "မင်္ဂလာပါ ကမ္ဘာ"
အရ = စ$#                  // အရှည်
ပိုင်း = စ$[၀..၅]           // slice
ပါ = စ$? "ကမ္ဘာ"             // #၁
ခွဲ = "က,ခ,ဂ,ဃ" / ','       // [က, ခ, ဂ, ဃ]
အစား = စ$~~["ပါ":"ပါ!"]       // အစားထိုး
အစား၁ = စ$~~["ပါ":"ပါ!":၁]   // N ပထမ
```

> `+` သည် ကိန်းဂဏန်းများအတွက်သာ။ စာကြောင်းများအတွက် `,`၊ juxtaposition သို့မဟုတ် interpolation သုံးပါ။

## စီးဆင်းမှုထိန်းချုပ်ခြင်း

```zymbol
ပ = ၇

? ပ > ၀ { >> "အပြုသဘောဆောင်သည်" ¶ }

? ပ > ၁၀၀ {
    >> "ကြီးသည်" ¶
} _? ပ > ၀ {
    >> "အပြုသဘောဆောင်သည်" ¶
} _? ပ == ၀ {
    >> "သုည" ¶
} _ {
    >> "အနုတ်သဘောဆောင်သည်" ¶
}
```

> Block `{ }` များသည် တစ်ကြောင်းတည်းဖြစ်သော်လည်း **မဖြစ်မနေလိုအပ်သည်**။

## Match

```zymbol
// range ဖြင့် match
ရမှတ် = ၈၅
အဆင့် = ?? ရမှတ် {
    ၉၀..၁၀၀ : 'A'
    ၈၀..၈၉  : 'B'
    ၇၀..၇၉  : 'C'
    _       : 'F'
}
>> အဆင့် ¶    // → B

// စာကြောင်း ဖြင့် match
အရောင် = "နီ"
ကုဒ် = ?? အရောင် {
    "နီ"    : "#FF၀၀၀၀"
    "စိမ်း" : "#၀၀FF၀၀"
    _       : "#၀၀၀၀၀၀"
}

// guard ဖြင့် match
အပူချိန် = -၅
အခြေအနေ = ?? အပူချိန် {
    _? အပူချိန် < ၀  : "ရေခဲ"
    _? အပူချိန် < ၂၀ : "အအေး"
    _? အပူချိန် < ၃၅ : "နွေးထွေး"
    _                : "ပူသည်"
}
>> အခြေအနေ ¶    // → ရေခဲ

// Statement ပုံစံ (block arms)
?? n {
    ၀       : { >> "သုည" ¶ }
    _? n < ၀: { >> "အနုတ်" ¶ }
    _       : { >> "အပြု" ¶ }
}
```

## ကွင်းဆက်များ

```zymbol
@ ည:၀..၄  { >> ည " " }        // inclusive range:  ၀ ၁ ၂ ၃ ၄
@ ည:၁..၉:၂ { >> ည " " }       // step ပါ:           ၁ ၃ ၅ ၇ ၉
@ ည:၅..၀:၁ { >> ည " " }       // ပြောင်းပြန်:       ၅ ၄ ၃ ၂ ၁ ၀

ည = ၁
@ ည <= ၆၄ { ည *= ၂ }
>> ည ¶                        // → ၁၂၈  (while)

သစ်သီး = ["သပ်ပယ်", "သရက်", "ဆလပ်"]
@ ဆ:သစ်သီး { >> ဆ ¶ }         // for-each array

@ မ:"မင်္ဂလာ" { >> မ "-" }
>> ¶                          // → မ-င်-္-ဂ-လ-ာ-  (for-each string)

@ ည:၁..၁၀ {
    ? ည % ၂ == ၀ { @> }        // @> ဆက်သွားသည်
    ? ည > ၇ { @! }              // @! ထွက်သည်
    >> ည " "
}
>> ¶                          // → ၁ ၃ ၅ ၇

// မကုန်နိုင်သော loop
i = ၀
@ {
    i++
    ? i >= ၅ { @! }
    >> i " "
}
>> ¶                          // → ၁ ၂ ၃ ၄

// Label ပါသော loop (nested break)
count = ၀
@ @အပြင် {
    count++
    ? count >= ၃ { @! အပြင် }
}
>> count ¶                    // → ၃
```

## လုပ်ဆောင်ချက်များ

```zymbol
ပေါင်းခြင်း(က, ခ) { <~ က + ခ }
>> ပေါင်းခြင်း(၃, ၄) ¶    // → ၇

factorial(ည) {
    ? ည <= ၁ { <~ ၁ }
    <~ ည * factorial(ည - ၁)
}
>> factorial(၅) ¶    // → ၁၂၀
```

လုပ်ဆောင်ချက်များတွင် **isolated scope** ရှိသည် — ပြင်ပ variable များကို ဖတ်မရ။ caller variable များကို ပြောင်းလဲရန် output parameter `<~` သုံးပါ:

```zymbol
လဲလှယ်(က<~, ခ<~) {
    tmp = က
    က = ခ
    ခ = tmp
}
x = ၁၀
y = ၂၀
လဲလှယ်(x, y)
>> "x=" x " y=" y ¶    // → x=၂၀ y=၁၀
```

> နာမည်ပေး function `နာမည်(params){ }` များသည် first-class တန်ဖိုးမဟုတ်ပါ။ argument အဖြစ် ပေးပို့ရန် wrap ပြုလုပ်ပါ: `x -> နာမည်(x)`။

## လမ်ဘ်ဒါ နှင့် ပိတ်မှတ်

```zymbol
နှစ်ဆ = ပ -> ပ * ၂
ပေါင်း = (က, ခ) -> က + ခ
>> နှစ်ဆ(၅) ¶    // → ၁၀
>> ပေါင်း(၃, ၇) ¶  // → ၁၀

// Block lambda
အမျိုးအစားသတ်မှတ် = ပ -> {
    ? ပ > ၀ { <~ "အပြုသဘောဆောင်သည်" }
    _? ပ < ၀ { <~ "အနုတ်သဘောဆောင်သည်" }
    <~ "သုည"
}

// ပိတ်မှတ် — ပြင်ပ variable ဖမ်းဆုပ်သည်
အခြေ = ၃
သုံးဆ = ပ -> ပ * အခြေ
>> သုံးဆ(၇) ¶    // → ၂၁

// function factory
ဖန်တည်(ည) { <~ ပ -> ပ + ည }
ဆယ်ပေါင် = ဖန်တည်(၁၀)
>> ဆယ်ပေါင်(၅) ¶    // → ၁၅

// array ထဲတွင်
ဆင့် = [ပ -> ပ+၁, ပ -> ပ*၂, ပ -> ပ*ပ]
>> ဆင့်[၂](၅) ¶    // → ၂၅
```

## ခင်းများ

ခင်းများသည် **ပြောင်းလဲနိုင်သော** ဖြစ်ပြီး **တူညီသောအမျိုးအစား** ဒြပ်စင်များ ကိုင်ဆောင်သည်။

```zymbol
စစ = [၁, ၂, ၃, ၄, ၅]

စစ[၀]          // ၁ — ဝင်ရောက်ကြည့်ရှုခြင်း (၀-based)
စစ[-၁]         // ၅ — အနုတ် index (နောက်ဆုံး)
စစ$#           // ၅ — အရေအတွက် (>> တွင် (စစ$#) သုံးပါ)

စစ = စစ$+ ၆            // ထည့်သည် → [၁,၂,၃,၄,၅,၆]
စစ၂ = စစ$+[၂] ၉၉       // index ၂ တွင် ထည့်သည်
စစ၃ = စစ$- ၃           // တန်ဖိုး၏ ပထမပေါ်ပေါက်မှုကို ဖယ်ရှားသည်
စစ၄ = စစ$-- ၃          // တန်ဖိုး၏ ပေါ်ပေါက်မှုအားလုံး ဖယ်ရှားသည်
စစ၅ = စစ$-[၀]          // index တွင် ဖယ်ရှားသည်
စစ၆ = စစ$-[၁..၃]       // range ဖယ်ရှားသည် (ကုန်ဆုံးဘက် exclusive)

ပါ = စစ$? ၃            // #၁ — ပါဝင်သည်
နေရာ = စစ$?? ၃         // [၂] — တန်ဖိုး၏ index များအားလုံး
sl = စစ$[၀..၃]          // [၁,၂,၃] — slice (ကုန်ဆုံးဘက် exclusive)
sl၂ = စစ$[၀:၃]          // [၁,၂,၃] — တူညီသည်၊ count-based syntax

မြင့် = စစ$^+           // ascending ခွဲခြား  (primitive သာ)
နိမ့် = စစ$^-           // descending ခွဲခြား (primitive သာ)

// Named/positional tuple array — comparator lambda နှင့် $^ သုံးပါ
db = [(နာမည်: "Carla", အသက်: ၂၈), (နာမည်: "Ana", အသက်: ၂၅), (နာမည်: "Bob", အသက်: ၃၀)]
အသက်အလိုက်  = db$^ (a, b -> a.အသက် < b.အသက်)    // ascending အသက်အလိုက် (<)
နာမည်အလိုက် = db$^ (a, b -> a.နာမည် > b.နာမည်)  // descending နာမည်အလိုက် (>)
>> အသက်အလိုက်[၀].နာမည် ¶     // → Ana
>> နာမည်အလိုက်[၀].နာမည် ¶    // → Carla

// ဒြပ်စင်တိုက်ရိုက်မွမ်းမံခြင်း (ခင်းများသာ)
စစ[၁] = ၉၉              // သတ်မှတ်သည်
စစ[၀] += ၅              // ပေါင်းစပ်: +=  -=  *=  /=  %=  ^=

// Functional မွမ်းမံခြင်း — ခင်းအသစ် ပြန်သည်; မူရင်းမပြောင်း
စစ၂ = စစ[၁]$~ ၉၉
```

> collection operator များအားလုံးသည် **array အသစ်** ပြန်သည်။ ပြန်သတ်မှတ်ပါ: `စစ = စစ$+ ၄`။
> Operator များကို chain မတွဲပါနှင့် — intermediate assignment များ သုံးပါ။
> `$^+` / `$^-` သည် **primitive array** (ကိန်းများ၊ string) ကို ခွဲခြားသည်။ tuple array အတွက် comparator lambda နှင့် `$^` သုံးပါ — လမ်းကြောင်းကို lambda (`<` = ascending, `>` = descending) ထဲ ကုဒ်ထည့်သည်။

**တန်ဖိုးအဓိပ္ပာယ်** — ခင်းကို ကိန်းရှင်တစ်ခုသို့ သတ်မှတ်ခြင်းသည် သီးခြားသောမိတ္တူ ဖန်တည်သည်:

```zymbol
a = [၁, ၂, ၃]
b = a
a[၀] = ၉၉
>> a ¶    // → [၉၉, ၂, ၃]
>> b ¶    // → [၁, ၂, ၃]   ← b မထိခိုက်
```

```zymbol
// Nested array
matrix = [[၁,၂,၃],[၄,၅,၆],[၇,၈,၉]]
>> matrix[၁][၂] ¶    // → ၆
```

## ဖြိုခွဲခြင်း

```zymbol
// Array
စစ = [၁၀, ၂၀, ၃၀, ၄၀, ၅၀]
[က, ခ, ဂ] = စစ              // က=၁၀  ခ=၂၀  ဂ=၃၀
[ပထမ, *ကျန်] = စစ             // ပထမ=၁၀  ကျန်=[၂၀,၃၀,၄၀,၅၀]
[x, _, z] = [၁, ၂, ၃]        // _ မှတ်မထားဘဲ

// Positional tuple
အမှတ် = (၁၀၀, ၂၀၀)
(px, py) = အမှတ်              // px=၁၀၀  py=၂၀၀

// Named tuple
လူ = (နာမည်: "Ana", အသက်: ၂၅, မြို: "နေပြည်တော်")
(နာမည်: နာ, အသက်: အ) = လူ    // နာ="Ana"  အ=၂၅
```

## တပ်ပုလ်

တပ်ပုလ်များသည် **မပြောင်းလဲနိုင်သော** ကိန်းအထားကိန်းနေ container များဖြစ်ပြီး **မတူညီသောအမျိုးအစား** တန်ဖိုးများ ကိုင်ဆောင်နိုင်သည်။ ခင်းများနှင့်မတူဘဲ ဖန်တည်ပြီးနောက် ဒြပ်စင်များ ပြောင်းလဲ၍မရ။

```zymbol
// Positional
အမှတ် = (၁၀, ၂၀)
>> အမှတ်[၀] ¶    // → ၁၀

ဒေတာ = (၄၂, "hello", #၁, ၃.၁၄)
>> ဒေတာ[၂] ¶     // → #၁

// နာမည်ပေး
လူတစ်ဦး = (နာမည်: "အဲလစ်", အသက်: ၂၅)
>> လူတစ်ဦး.နာမည် ¶    // → အဲလစ်
>> လူတစ်ဦး[၀] ¶       // → အဲလစ် (index လည်း အလုပ်လုပ်သည်)

// Nested
pos = (x: ၁၀, y: ၂၀)
p = (pos: pos, label: "ရင်းမြစ်")
>> p.pos.x ¶        // → ၁၀
```

**မပြောင်းလဲနိုင်မှု** — tuple ဒြပ်စင်ကို ပြောင်းလဲရန် မည်သည့်ကြိုးပမ်းမှုမဆို runtime error ဖြစ်သည်:

```zymbol
t = (၁၀, ၂၀, ၃၀)
// t[၀] = ၉၉    // ❌ runtime error: tuple များမပြောင်းလဲနိုင်
// t[၀] += ၅    // ❌ တူညီသောအမှား
```

ပြောင်းလဲသောတန်ဖိုး ထုတ်ယူရန် `$~` (functional မွမ်းမံခြင်း) သုံးပါ — **tuple အသစ်** ပြန်သည်:

```zymbol
t = (၁၀, ၂၀, ၃၀)
t2 = t[၁]$~ ၉၉၉
>> t ¶     // → (၁၀, ၂၀, ၃၀)   ← မူရင်းမပြောင်း
>> t2 ¶    // → (၁၀, ၉၉၉, ၃၀)

// နာမည်ပေး tuple — ပြန်လည်တည်ဆောက်ရန်
လူတစ်ဦး = (နာမည်: "အဲလစ်", အသက်: ၂၅)
အသက်ကြီးသူ = (နာမည်: လူတစ်ဦး.နာမည်, အသက်: ၂၆)
>> လူတစ်ဦး.အသက် ¶     // → ၂၅
>> အသက်ကြီးသူ.အသက် ¶  // → ၂၆
```

## အဆင့်မြင့်လုပ်ဆောင်ချက်များ

> HOF operator များသည် **အတွင်း lambda** လိုအပ်သည် — lambda variable ကို တိုက်ရိုက်မသုံးနိုင်ပါ။

```zymbol
ကိန်းများ = [၁, ၂, ၃, ၄, ၅, ၆, ၇, ၈, ၉, ၁၀]

နှစ်ဆများ    = ကိန်းများ$> (ပ -> ပ * ၂)                // map  → [၂,၄,၆…၂၀]
စုံကိန်းများ = ကိန်းများ$| (ပ -> ပ % ၂ == ၀)           // filter → [၂,၄,၆,၈,၁၀]
စုစုပေါင်း   = ကိန်းများ$< (၀, (စုစု, ပ) -> စုစု + ပ)  // reduce → ၅၅

// Intermediate မှတဆင့် chain
အဆင့်၁ = ကိန်းများ$| (ပ -> ပ > ၃)
အဆင့်၂ = အဆင့်၁$> (ပ -> ပ * ပ)
>> အဆင့်၂ ¶    // → [၁၆, ၂၅, ၃၆, ၄၉, ၆၄, ၈၁, ၁၀၀]

// HOF ထဲ နာမည်ပေး function — lambda ထဲ wrap ပြုလုပ်ပါ
နှစ်ဆဖြစ်အောင်(ပ) { <~ ပ * ၂ }
r = ကိန်းများ$> (ပ -> နှစ်ဆဖြစ်အောင်(ပ))    // ✅
```

## ပိုက်အော်ပရေတာ

ညာဘက်တွင် `_` placeholder အမြဲလိုအပ်သည်:

```zymbol
နှစ်ဆ = x -> x * ၂
ပေါင်း = (a, b) -> a + b
တိုး = x -> x + ၁

၅ |> နှစ်ဆ(_)        // → ၁၀
၁၀ |> ပေါင်း(_, ၅)   // → ၁၅
၅ |> ပေါင်း(၂, _)    // → ၇

// ချိတ်ဆက်
ရလဒ် = ၅ |> နှစ်ဆ(_) |> တိုး(_) |> နှစ်ဆ(_)
>> ရလဒ် ¶    // → ၂၂  (၅→၁၀→၁၁→၂၂)
```

## ကိန်းဂဏန်းစနစ်များ

Zymbol သည် **Unicode ကိန်းဂဏန်းဘလောက် ၆၉ ခုတွင်** ဂဏန်းများကို ပြသနိုင်သည် — မြန်မာ၊ ဒေဝနာဂရီ၊ အာရေဗျ-အင်ဒစ်ခ်၊ ထိုင်း၊ ကလင်ဂွန် pIqaD နှင့် အခြားများ။ အသုံးပြုနေသောမုဒ်သည် `>>` ထုတ်ပေးမှုကိုသာ သက်ရောက်မှုရှိသည်။ အတွင်းဂဏန်းသင်္ချာသည် အမြဲဒွိမဏ္ဍိုင်ဖြစ်သည်။

### စနစ်တစ်ခုကိုဖွင့်ရန်

ပစ်မှတ်ထားသောစာလုံးစနစ်၏ `၀` နှင့် `၉` ကိန်းဂဏန်းများကို `#…#` အကြားရေးပါ:

```zymbol
#၀၉#    // မြန်မာ     (U+1040–U+1049)
#٠٩#    // အာရေဗျ-အင်ဒစ်ခ် (U+0660–U+0669)
#09#    // ASCII ပြန်ညှိ
```

### ထုတ်ပေးမှုနှင့် Boolean

```zymbol
x = ၄၂
>> x ¶          // → ၄၂   (ASCII မူလ)

#၀၉#
>> x ¶          // → ၄၂
>> ၃.၁၄ ¶       // → ၃.၁၄   (ဒသမအမှတ်သည် ASCII အမြဲ)
>> ၁ + ၂ ¶      // → ၃

>> #၁ ¶         // → #၁   (မြန်မာဘာသာဖြင့် မှန်သည်)
>> #၀ ¶         // → #၀   (မှားသည် — ၀ သုညနှင့် မတူ)

x = ၂၈ > ၄
>> x ¶          // → #၁
```

### မူလစာလုံးများ

ပံ့ပိုးသောစနစ်မည်သည့်ခုကမဆို ကိန်းဂဏန်းများသည် range၊ modulo၊ နှိုင်းယှဉ်မှုများတွင် မှန်ကန်သောစာလုံးများဖြစ်သည်:

```zymbol
#၀၉#

@ ည:၁..၂၀ {
    ? ည % ၁၅ == ၀ { >> "FizzBuzz" ¶ }
    _? ည % ၃  == ၀ { >> "Fizz" ¶ }
    _? ည % ၅  == ၀ { >> "Buzz" ¶ }
    _ { >> ည ¶ }
}
```

### မည်သည့်စနစ်တွင်မဆို Boolean Literal များ

ပံ့ပိုးသောဘလောက်မည်သည့်ခုမဆိုမှ `#` + `၀` သို့မဟုတ် `၁` ကိန်းဂဏန်းသည် မှန်ကန်သော Boolean literal ဖြစ်သည်:

```zymbol
#٠٩#
نشط = #١        // #၁ နှင့်တူညီ
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` သည် **ASCII အမြဲ** ဖြစ်သည်။ `#၀` (မှားသည်) သည် မည်သည့်စနစ်တွင်မဆို `၀` (ဂဏန်းမဲ့) နှင့် မျက်မှန်းအပ်သောမတူညီ။

## သင်္ကေတကိုးကားမှု

| သင်္ကေတ   | လုပ်ဆောင်ချက်              | သင်္ကေတ      | လုပ်ဆောင်ချက်                    |
|-----------|---------------------------|--------------|----------------------------------|
| `=`       | ကိန်းရှင်                  | `$#`         | အရေအတွက်                         |
| `:=`      | ကိန်းသေ                    | `$+`         | ထည့်သည်                          |
