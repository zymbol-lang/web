## TUI პრიმიტივები

ტერმინალის UI ოპერატორები ინტერაქტიული პროგრამებისთვის. უმეტესობა `>>| { }` ბლოკს საჭიროებს (ალტ. ეკრანი + raw რეჟიმი).

```zymbol
>>| {
    >>!                              // ალტ. ეკრანის გასუფთავება
    >>~ (1, 1, 0, 10) > "მუშაობს"   // სტ. 1, სვ. 1, fg=10 (მწვანე)
    @~ 1000                          // 1 წამის პაუზა (1000 ms)
    >>~ (2, 1) > "დასრულდა."
}
// ტერმინალი ავტ. აღდგება გასვლისას
```

```zymbol
// კლავიშის დაჭერა და ტერმინალის ზომა
>>| {
    [სტრ, სვ] = >>?                  // ტერმინალის განზომილებების შეკითხვა
    >>~ (1, 1) > "ტერმინალი: " სტრ " x " სვ
    <<| კლ                           // დამბლოკავი კლავიშის წაკითხვა
    >>~ (2, 1) > "დაჭერილი: " კლ
}
```

> `>>!` ასუფთავებს ეკრანს. `>>?` აბრუნებს `[სტრ, სვ]`. `@~ N` ძინავს N ms.
> `<<|` კითხულობს ერთ კლავიშს (დამბლოკავი); `<<|?` კითხულობს დაბლოკვის გარეშე (აბრუნებს `'\0'`).
> პოზ. გამოტ. კოხტე: `(სტ, სვ, BKS, fg, bg)` — ნებ. ადგილი შეიძლება გამოტოვდეს (`>>~ (,,, 196) > "წი."`).
> BKS bitmask: `1`=მუქი, `2`=დახ., `4`=ხაზგ. ANSI 256-ფერი (`0`=ნაგ.).
> მხოლოდ ხე-გამვლელი (`>>!`, `>>?`, `@~`, `>>~` ასევე `--vm`-ში).

## მასივები

მასივები **ცვლადია** და ინახავს **ერთი ტიპის** ელემენტებს.

```zymbol
მას = [1, 2, 3, 4, 5]

x = მას[1]      // 1 — მიმართვა (1-ზე: პირველი ელ.)
x = მას[-1]     // 5 — უარყ. ინდექსი (ბოლო ელ.)
x = მას$#       // 5 — სიგრძე (>> -ში (მას$#))

მას = მას$+ 6            // დამატება → [1,2,3,4,5,6]
მას2 = მას$+[2] 99       // 2-ე პოზ.-ში ჩასმა (1-ზე)
მას3 = მას$- 3           // მნ.-ის პირველი შ.-ის წაშლა
მას4 = მას$-- 3          // ყველა შ.-ის წაშლა
მას5 = მას$-[1]          // 1-ლ ინდ.-ში წაშლა
მას6 = მას$-[2..3]       // დიაპ.-ის წაშლა (1-ზე, ბოლო ჩ.)

შ = მას$? 3              // #1 — შეიცავს
პ = მას$?? 3             // [3] — მნ.-ის ყველა ინდ. (1-ზე)
ნ = მას$[1..3]           // [1,2,3] — ნაჭერი (1-ზე, ბოლო ჩ.)
ნ2 = მას$[1:3]           // [1,2,3] — იგივე, რაოდ.-ზე

ზრდ = მას$^+             // ზრდ. სორტ. (მხ. პრიმ.)
კლ = მას$^-              // კლ. სორტ. (მხ. პრიმ.)

// სახ./პოზ. კოხტ.-მასივები — $^ გამ. ლამბდ.-ით
ბდ = [(სახ: "Carla", ას: 28), (სახ: "Ana", ას: 25), (სახ: "Bob", ას: 30)]
ას_მ  = ბდ$^ (ა, ბ -> ა.ას < ბ.ას)          // ას.-ით ზრდ. (<)
სახ_მ = ბდ$^ (ა, ბ -> ა.სახ > ბ.სახ)         // სახ.-ით კლ. (>)
>> ას_მ[1].სახ ¶     // → Ana
>> სახ_მ[1].სახ ¶    // → Carla

// პირდ. ელ.-ის განახ. (მხ. მას.)
მას[1] = 99              // მინ.
მას[2] += 5              // გაფ.: +=  -=  *=  /=  %=  ^=

// ფ. განახ. — ახ. მას. აბრ.; ორიგ. უცვ.
მას2 = მას[2]$~ 99
```

> ყველა კოლ. ოპ. **ახ. მასივს** აბრუნებს. უკან მიანიჭეთ: `მას = მას$+ 4`.
> `$+` შეიძლება ჯაჭვში: `მას = მას$+ 5$+ 6$+ 7`. სხვა ოპ. შუამ. მინ. გამ.
> **ინდ. 1-ზეა**: `მას[1]` პირველი ელ.-ია; `მას[0]` გაშ. შეც.-ია.
> `$^+` / `$^-` **პრიმ. მას.**-ს სრ. (რ., სტ.). კოხტ. მ.-ისთვის `$^` ლამ.-ით.

**მნიშვნ. სემ.** — მასივ. სხვ. ცვლ.-ში მინ. დამოუკ. ასლს ქმნის:

