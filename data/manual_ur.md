# Zymbol-Lang کی مختصر راہنما

**Zymbol-Lang** ایک علامتی پروگرامنگ زبان ہے۔ یہ کوئی مخصوص الفاظ (keywords) استعمال نہیں کرتی — سب کچھ علامت ہے۔ یہ ہر انسانی زبان میں یکساں طریقے سے کام کرتی ہے۔

- کوئی مخصوص الفاظ نہیں (`if`، `while`، `return` نہیں — صرف علامات `?`، `@`، `<~`)
- مکمل Unicode سپورٹ — کسی بھی زبان یا ایموجی سے شناخت کنندے 👋
- زبان سے غیر جانبدار — کوڈ تمام زبانوں میں یکساں ہے

---

## متغیرات اور ثوابت

```zymbol
م = ۱۰           // متغیر (قابلِ تبدیل)
پی := ۳.۱۴۱۵۹    // ثابت (ناقابلِ تبدیل — دوبارہ تفویض غلطی ہے)
نام = "آنا"
فعال = #۱        // Boolean سچ
👋 := "ہیلو"
```

### مرکب تفویض

```zymbol
م = ۱۰    // ۱۰
م += ۵    // ۱۵
م -= ۳    // ۱۲
م *= ۲    // ۲۴
م /= ۳    // ۸
م %= ۳    // ۲
م ^= ۲    // ۴
م++       // ۵
م--       // ۴
```

## آؤٹ پٹ اور ان پٹ

```zymbol
// آؤٹ پٹ — خودبخود نئی سطر نہیں آتی
>> "ہیلو، دنیا!" ¶              // ¶ یا \\ صریح نئی سطر دیتا ہے
>> "الف=" الف " ب=" ب ¶         // متعدد قدریں صرف ساتھ رکھ کر
>> "مجموع=" جمع(۲, ۳) ¶         // فنکشن کال کسی بھی جگہ
>> (صف$#) ¶                     // postfix آپریٹرز کو >> میں قوسین چاہیے

// ان پٹ
<< نام                           // بغیر اشارے — متغیر میں پڑھتا ہے
<< "آپ کا نام؟ " نام            // اشارے کے ساتھ
```

> `¶` اور `\\` نئی سطر کے لیے برابر ہیں۔

## آپریٹرز

```zymbol
// ریاضی
>> ۱۰ + ۳ ¶    // → ۱۳
>> ۱۰ - ۳ ¶    // → ۷
>> ۱۰ * ۳ ¶    // → ۳۰
>> ۱۰ / ۳ ¶    // → ۳.۳۳۳…
>> ۱۰ % ۳ ¶    // → ۱  (باقی)
>> ۲ ^ ۸  ¶    // → ۲۵۶  (قوت)

// موازنہ — #۱ یا #۰ دیتے ہیں
>> ۳ == ۳ ¶    // → #۱
>> ۳ != ۴ ¶    // → #۱
>> ۳ <  ۵ ¶    // → #۱
>> ۵ >= ۵ ¶    // → #۱

// منطق
>> #۱ && #۰ ¶  // → #۰  (اور)
>> #۱ || #۰ ¶  // → #۱  (یا)
>> !#۱     ¶   // → #۰  (نہیں)
```

## سٹرنگز

تین درست طریقے — ہر ایک اپنے سیاق کے لیے:

```zymbol
نام = "آنا"
ن = ۲۵

// ۱. کاما — = یا := تفویض میں
پیغام = "ہیلو ", نام, "!"              // → ہیلو آنا!
سرخی := "صارف: ", نام

// ۲. ساتھ رکھنا — >> آؤٹ پٹ میں
>> "ہیلو " نام " آپ " ن " سال کے ہیں" ¶  // → ہیلو آنا آپ ۲۵ سال کے ہیں

// ۳. درون سازی — کسی بھی سیاق میں
تعارف = "ہیلو {نام}، آپ {ن} سال کے ہیں"  // → ہیلو آنا، آپ ۲۵ سال کے ہیں
```

```zymbol
// سٹرنگ آپریٹرز
س = "دنیا"
>> (س$#) ¶                         // → ۴  (لمبائی — >> میں قوسین ضروری)
ٹکڑا = س$[۰..۳]                    // "دنی"  (ٹکڑا)
س۲ = س$~~["دنیا":"کائنات"]         // "کائنات"  (تبدیلی)
س۳ = س$~~["ن":۲]                   // پہلی ۲ بار "ن" بدلتا ہے
```

> **تنبیہ**: `+` صرف اعداد کے لیے ہے۔ سٹرنگز میں اسے استعمال کرنے پر انتباہ ملتا ہے۔

## بہاؤ کنٹرول

```zymbol
م = ۷

// سادہ اگر
? م > ۰ { >> "مثبت" ¶ }

// اگر / ورنہ اگر / ورنہ
? م > ۱۰۰ {
    >> "بڑا" ¶
} _? م > ۰ {
    >> "مثبت" ¶
} _? م == ۰ {
    >> "صفر" ¶
} _ {
    >> "منفی" ¶
}
```

`{ }` بلاکس **لازمی ہیں**، چاہے صرف ایک سطر ہو۔

## ملان (Match)

```zymbol
// عددی رینج سے ملان
نمبر = ۸۵
درجہ = ?? نمبر {
    ۹۰..۱۰۰ : 'A'
    ۸۰..۸۹  : 'B'
    ۷۰..۷۹  : 'C'
    _       : 'F'
}
>> درجہ ¶    // → B

// محافظوں کے ساتھ ملان
درجہ_حرارت = -۵
حالت = ?? درجہ_حرارت {
    _? درجہ_حرارت < ۰  : "برف"
    _? درجہ_حرارت < ۲۰ : "ٹھنڈا"
    _? درجہ_حرارت < ۳۵ : "گرم"
    _                   : "بہت گرم"
}
>> حالت ¶    // → برف

// سٹرنگز سے ملان
رنگ = "سرخ"
کوڈ = ?? رنگ {
    "سرخ"   : "#FF۰۰۰۰"
    "سبز"   : "#۰۰FF۰۰"
    _        : "#۰۰۰۰۰۰"
}
>> کوڈ ¶
```

## لوپس

```zymbol
// شامل رینج: ۰..۴ پر ۰,۱,۲,۳,۴ چلتا ہے
@ ن:۰..۴ { >> ن " " }
>> ¶    // → ۰ ۱ ۲ ۳ ۴

// قدم کے ساتھ رینج
@ ن:۱..۹:۲ { >> ن " " }
>> ¶    // → ۱ ۳ ۵ ۷ ۹

// الٹی رینج
@ ن:۵..۰:۱ { >> ن " " }
>> ¶    // → ۵ ۴ ۳ ۲ ۱ ۰

// جب تک (while)
ن = ۱
@ ن <= ۶۴ { ن *= ۲ }
>> ن ¶    // → ۱۲۸

// ہر عنصر پر (for-each)
پھل = ["سیب", "ناشپاتی", "انگور"]
@ پ:پھل { >> پ ¶ }

// سٹرنگ کے حروف پر
@ ح:"ہیلو" { >> ح "-" }
>> ¶    // → ہ-ی-ل-و-

// توقف اور جاری رکھنا
@ ن:۱..۱۰ {
    ? ن % ۲ == ۰ { @> }    // @> جاری رکھو
    ? ن > ۷ { @! }          // @! رکو
    >> ن " "
}
>> ¶    // → ۱ ۳ ۵ ۷
```

## فنکشنز

```zymbol
// تعریف اور کال
مجموع(الف، ب) { <~ الف + ب }
>> مجموع(۳, ۴) ¶    // → ۷

// بازگشت (recursion)
ضربی(ن) {
    ? ن <= ۱ { <~ ۱ }
    <~ ن * ضربی(ن - ۱)
}
>> ضربی(۵) ¶    // → ۱۲۰
```

فنکشنز کا **الگ scope ہے** — باہری متغیرات تک رسائی نہیں۔ کالر کے متغیرات بدلنے کے لیے آؤٹ پٹ پیرامیٹرز `<~` استعمال کریں:

```zymbol
تبادلہ(الف<~, ب<~) {
    عارضی = الف
    الف = ب
    ب = عارضی
}
م = ۱۰
ن = ۲۰
تبادلہ(م, ن)
>> "م=" م " ن=" ن ¶    // → م=۲۰ ن=۱۰
```

> نامی فنکشنز پہلے درجے کی قدریں نہیں ہیں۔ بطور دلیل بھیجنے کے لیے لپیٹیں: `x -> مجموع(x)`.

## لیمبڈا اور بندشیں

```zymbol
// سادہ لیمبڈا (بلا واپسی)
دگنا = م -> م * ۲
مجموع_ل = (الف، ب) -> الف + ب
>> دگنا(۵) ¶      // → ۱۰
>> مجموع_ل(۳, ۷) ¶   // → ۱۰

// بلاک لیمبڈا (صریح واپسی)
درجہ_قدر = م -> {
    ? م > ۰ { <~ "مثبت" }
    _? م < ۰ { <~ "منفی" }
    <~ "صفر"
}
>> درجہ_قدر(۵) ¶      // → مثبت
>> درجہ_قدر(۰) ¶      // → صفر
>> درجہ_قدر(-۵) ¶     // → منفی

// بندش — لیمبڈا باہری متغیرات پکڑتا ہے
عامل = ۳
تگنا = م -> م * عامل    // 'عامل' پکڑتا ہے
>> تگنا(۷) ¶    // → ۲۱

// فنکشن فیکٹری
آپریشن_بنائیں(ن) { <~ م -> م + ن }
جمع۱۰ = آپریشن_بنائیں(۱۰)
>> جمع۱۰(۵) ¶    // → ۱۵

// لیمبڈا بطور قدر: صف میں محفوظ
آپریشن = [م -> م+۱, م -> م*۲, م -> م*م]
>> آپریشن[۰](۵) ¶    // → ۶
>> آپریشن[۲](۵) ¶    // → ۲۵
```

## صفیں (Arrays)

صفیں **قابلِ تبدیل** ہیں اور **ایک ہی قسم** کے عناصر رکھتی ہیں۔

```zymbol
arr = [۱, ۲, ۳, ۴, ۵]

arr[۰]          // ۱ — رسائی (اشاریہ ۰ سے شروع)
arr[-۱]         // ۵ — منفی اشاریہ (آخری)
arr$#           // ۵ — طول (>> میں قوسین لازمی)

arr = arr$+ ۶            // اضافہ → [۱,۲,۳,۴,۵,۶]
arr۲ = arr$+[۲] ۹۹       // اشاریہ ۲ میں داخل
arr۳ = arr$- ۳           // قدر کا پہلا ظہور حذف
arr۴ = arr$-- ۳          // تمام ظہورات حذف
arr۵ = arr$-[۰]          // اشاریے پر حذف
arr۶ = arr$-[۱..۳]       // رینج حذف (آخر خارج)

موجود = arr$? ۳          // #۱ — موجودگی
مقامات = arr$?? ۳        // [۲] — قدر کے تمام اشاریے
ٹکڑا = arr$[۰..۳]        // [۱,۲,۳] — ٹکڑا (آخر خارج)
ٹکڑا۲ = arr$[۰:۳]        // [۱,۲,۳] — گنتی پر مبنی نحو

صعودی = arr$^+           // صعودی ترتیب (صرف بنیادی)
نزولی = arr$^-           // نزولی ترتیب (صرف بنیادی)

// ٹیپل صفیں — موازنہ لیمبڈا کے ساتھ $^ استعمال کریں
db = [(name: "کارلا", age: ۲۸), (name: "آنا", age: ۲۵), (name: "باب", age: ۳۰)]
عمر_کے_مطابق = db$^ (الف، ب -> الف.age < ب.age)
>> عمر_کے_مطابق[۰].name ¶    // → آنا

// براہِ راست عنصر اپ ڈیٹ (صرف صفیں)
arr[۱] = ۹۹              // تفویض
arr[۰] += ۵              // مرکب: +=  -=  *=  /=  %=  ^=

// فنکشنل اپ ڈیٹ — نئی صف دیتا ہے؛ اصل نہیں بدلتا
arr2 = arr[۱]$~ ۹۹
```

> تمام مجموعہ آپریٹرز **نئی صف** دیتے ہیں۔ دوبارہ تفویض کریں: `arr = arr$+ ۴`.
> زنجیر بندی نہیں: درمیانی تفویض استعمال کریں۔
> `$^+` / `$^-` **بنیادی صفیں** ترتیب دیتے ہیں۔ ٹیپل صفوں کے لیے `$^` موازنہ لیمبڈا کے ساتھ استعمال کریں۔

**قدر کی معنویات** — صف کو دوسرے متغیر میں تفویض کرنے سے ایک آزاد نقل بنتی ہے:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b متاثر نہیں ہوا
```

```zymbol
// گھونسلہ دار صفیں
میٹرکس = [[۱,۲,۳],[۴,۵,۶],[۷,۸,۹]]
>> میٹرکس[۱][۲] ¶    // → ۶
```

## تجزیہ (Destructuring)

```zymbol
// صف
arr = [۱۰, ۲۰, ۳۰, ۴۰, ۵۰]
[الف, ب, ج] = arr              // الف=۱۰  ب=۲۰  ج=۳۰
[پہلا, *باقی] = arr             // پہلا=۱۰  باقی=[۲۰,۳۰,۴۰,۵۰]
[م, _, ن] = [۱, ۲, ۳]           // _ نظرانداز

// پوزیشنی ٹیپل
نقطہ = (۱۰۰, ۲۰۰)
(نم, نن) = نقطہ                // نم=۱۰۰  نن=۲۰۰

// نامی ٹیپل
شخص = (name: "آنا", age: ۲۵, city: "کراچی")
(name: ن, age: ع) = شخص        // ن="آنا"  ع=۲۵
```

## ٹیوپلز (Tuples)

ٹیوپلز **ناقابلِ تبدیل** ترتیب یافتہ ظروف ہیں جو **مختلف اقسام** کی قدریں رکھ سکتے ہیں۔ صفوں کے برخلاف، بنانے کے بعد عناصر تبدیل نہیں کیے جا سکتے۔

```zymbol
// پوزیشنی
نقطہ = (۱۰, ۲۰)
>> نقطہ[۰] ¶    // → ۱۰

ڈیٹا = (42, "hello", #1, 3.14)
>> ڈیٹا[2] ¶     // → #1

// نامی
شخص = (نام: "ایلس"، عمر: ۲۵)
>> شخص.نام ¶     // → ایلس
>> شخص[۰] ¶     // → ایلس (اشاریہ بھی کام کرتا ہے)

// گھونسلہ دار
مقام = (م: ۱۰, ن: ۲۰)
ق = (مقام: مقام, لیبل: "آغاز")
>> ق.مقام.م ¶   // → ۱۰
```

**ناتبدیلیت** — ٹیوپل کے کسی عنصر کو تبدیل کرنے کی کوئی بھی کوشش رن ٹائم غلطی ہے:

```zymbol
t = (10, 20, 30)
// t[0] = 99    // ❌ رن ٹائم غلطی: ٹیوپلز ناقابلِ تبدیل ہیں
// t[0] += 5    // ❌ وہی غلطی
```

ترمیم شدہ قدر نکالنے کے لیے `$~` (فنکشنل اپ ڈیٹ) استعمال کریں — ایک **نیا** ٹیوپل دیتا ہے:

```zymbol
t = (10, 20, 30)
t2 = t[1]$~ 999
>> t ¶     // → (10, 20, 30)   ← اصل بغیر تبدیلی
>> t2 ¶    // → (10, 999, 30)

// نامی ٹیوپل — واضح طور پر دوبارہ بنائیں
شخص = (نام: "Alice", عمر: 25)
بڑا  = (نام: شخص.نام, عمر: 26)
>> شخص.عمر ¶    // → 25
>> بڑا.عمر ¶    // → 26
```

## اعلیٰ ترتیب فنکشنز (HOF)

> HOF آپریٹرز کو **inline لیمبڈا** چاہیے — لیمبڈا متغیر براہِ راست نہیں چلتا۔

```zymbol
اعداد = [۱, ۲, ۳, ۴, ۵, ۶, ۷, ۸, ۹, ۱۰]

// Map ($>)
دگنے = اعداد$> (م -> م * ۲)
>> دگنے ¶    // → [۲, ۴, ۶, ۸, ۱۰, ۱۲, ۱۴, ۱۶, ۱۸, ۲۰]

// Filter ($|)
جفت = اعداد$| (م -> م % ۲ == ۰)
>> جفت ¶    // → [۲, ۴, ۶, ۸, ۱۰]

// Reduce ($<) — (ابتدائی قدر، (جمع کنندہ، عنصر) -> جملہ)
کل_مجموع = اعداد$< (۰, (جمع, م) -> جمع + م)
>> کل_مجموع ¶    // → ۵۵

// درمیانی کے ذریعے زنجیر
قدم۱ = اعداد$| (م -> م > ۳)
قدم۲ = قدم۱$> (م -> م * م)
>> قدم۲ ¶    // → [۱۶, ۲۵, ۳۶, ۴۹, ۶۴, ۸۱, ۱۰۰]
```

## پائپ آپریٹر

دائیں طرف کو ہمیشہ `_` بطور جگہ رکھنے والا چاہیے:

```zymbol
دگنا = م -> م * ۲
جمع = (الف، ب) -> الف + ب
ایک_جوڑو = م -> م + ۱

۵ |> دگنا(_)        // → ۱۰
۱۰ |> جمع(_, ۵)     // → ۱۵
۵ |> جمع(۲, _)      // → ۷

// زنجیر
ن = ۵ |> دگنا(_) |> ایک_جوڑو(_) |> دگنا(_)
>> ن ¶    // → ۲۲  (۵→۱۰→۱۱→۲۲)
```

## عددی نظام

Zymbol **69 Unicode ہندسہ بلاکوں** میں اعداد ظاہر کر سکتا ہے — فارسی/اردو، عربی-ہندی، تھائی، کلنگون pIqaD، موٹا ریاضی، LCD ہندسے، اور مزید۔ فعال موڈ صرف `>>` آؤٹ پٹ کو متاثر کرتا ہے؛ اندرونی حساب ہمیشہ بائنری ہوتا ہے۔

### نظام فعال کریں

مطلوبہ نظام کے `۰` اور `۹` ہندسے `#…#` کے درمیان لکھیں:

```zymbol
#۰۹#    // فارسی/اردو (Extended Arabic-Indic) (U+06F0–U+06F9)
#٠٩#    // عربی-ہندی  (U+0660–U+0669)
#๐๙#    // تھائی      (U+0E50–U+0E59)
#09#    // ASCII دوبارہ ترتیب
```

### آؤٹ پٹ اور بولین

```zymbol
x = ۴۲
>> x ¶          // → ۴۲   (پیش طے ASCII)

#۰۹#
>> x ¶          // → ۴۲
>> ۳.۱۴ ¶       // → ۳.۱۴   (اعشاریہ نقطہ ہمیشہ ASCII)
>> ۱ + ۲ ¶      // → ۳

// بولین: # سابقہ ہمیشہ ASCII، ہندسہ موڈ کے مطابق
>> #۱ ¶         // → #۱   (اردو میں سچ)
>> #۰ ¶         // → #۰   (جھوٹ — ۰ صفر سے مختلف)

x = ۲۸ > ۴
>> x ¶          // → #۱   (موازنہ نتیجہ فعال موڈ کی پیروی کرتا ہے)
```

### ماخذ میں مقامی لٹرل

کسی بھی معاون نظام کے ہندسے رینج، ماڈیولو، موازنہ میں درست لٹرل ہیں:

```zymbol
#۰۹#

@ ن:۱..۱۵ {
    ? ن % ۱۵ == ۰ { >> "FizzBuzz" ¶ }
    _? ن % ۳  == ۰ { >> "Fizz" ¶ }
    _? ن % ۵  == ۰ { >> "Buzz" ¶ }
    _ { >> ن ¶ }
}
```

### کسی بھی نظام میں بولین لٹرل

کسی بھی معاون بلاک کا `#` + `۰` یا `۱` ہندسہ ایک درست بولین لٹرل ہے:

```zymbol
#٠٩#
نشط = #١        // #۱ جیسا
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **ہمیشہ ASCII** ہے۔ `#۰` (جھوٹ) کسی بھی نظام میں `۰` (صفر عدد صحیح) سے بصری طور پر مختلف ہے۔
