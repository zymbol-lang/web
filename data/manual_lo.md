## ພື້ນຖານ TUI

ຕົວດຳເນີນງານອິນເຕີເຟດຜູ້ໃຊ້ທີ່ລະບົບສຳລັບໂປຣແກຣມໂຕ້ຕອບ. ສ່ວນໃຫຍ່ຕ້ອງການບລັອກ `>>| { }` (ໜ້າຈໍສຳຮອງ + ໂໝດດິບ).

```zymbol
>>| {
    >>!                             // ລ້າງໜ້າຈໍສຳຮອງ
    >>~ (໑, ໑, ໐, ໑໐) > "ກຳລັງແລ່ນ"   // ແຖວ ໑, ຖັນ ໑, fg=໑໐ (ຂຽວ)
    @~ ໑໐໐໐                         // ຢຸດ ໑ ວິນາທີ (໑໐໐໐ ms)
    >>~ (໒, ໑) > "ສຳເລັດແລ້ວ."
}
// ລະບົບຖືກຟື້ນຟູໂດຍອັດຕະໂນມັດເມື່ອອອກ
```

```zymbol
// ການກົດປຸ່ມ ແລະ ຂະໜາດລະບົບ
>>| {
    [ແຖວ, ຖັນ] = >>?              // ຖາມຂະໜາດລະບົບ
    >>~ (໑, ໑) > "ລະບົບ: " ແຖວ " x " ຖັນ
    <<| ປຸ່ມ                         // ອ່ານການກົດປຸ່ມແບບສະກັດ
    >>~ (໒, ໑) > "ກົດ: " ປຸ່ມ
}
```

> `>>!` ລ້າງໜ້າຈໍ. `>>?` ສົ່ງຄືນ `[ແຖວ, ຖັນ]`. `@~ N` ນອນ N ມິນລິວິນາທີ.
> `<<|` ອ່ານການກົດປຸ່ມດຽວ (ສະກັດ); `<<|?` ກວດສອບໂດຍບໍ່ສະກັດ (ສົ່ງຄືນ `'\0'` ຖ້າບໍ່ມີ).
> ທູເປີ້ນຜົນຜະລິດຕາມຕຳແໜ່ງ: `(ແຖວ, ຖັນ, BKS, fg, bg)` — ຊ່ອງໃດກໍ່ຕາມສາມາດຖືກລະເວັ້ນໄດ້ດ້ວຍຈຸດ (`>>~ (,,, 196) > "ແດງ"`).
> BKS ບິດມາສ: `1`=ໜາ, `2`=ອຽງ, `4`=ຂີດກ້ອງ. ຈານສີ ANSI 256 (`0`=ຄ່າເລີ່ມຕົ້ນຂອງລະບົບ).
> ມີແຕ່ tree-walker (ຍົກເວັ້ນ `>>!`, `>>?`, `@~`, `>>~` ທີ່ເຮັດວຽກໃນ `--vm` ນຳ).

```zymbol
}
>> ¶                          // → ໑ ໓ ໕ ໗

// ວົງຈອນບໍ່ມີທີ່ສິ້ນສຸດ
i = ໐
@ {
    i++
    ? i >= ໕ { @! }
    >> i " "
}
>> ¶                          // → ໑ ໒ ໓ ໔

// ວົງຈອນທີ່ມີປ້າຍກຳກັບ (ການທຳລາຍແບບຊ້ອນກັນ)
ນັບ = ໐
@:ພາຍນອກ {
    ນັບ++
    ? ນັບ >= ໓ { @:ພາຍນອກ! }
}
>> ນັບ ¶                    // → ໓
```

## ຟັງຊັນ

```zymbol
ບວກ(a, b) { <~ a + b }
>> ບວກ(໓, ໔) ¶    // → ໗

ແຟັກທໍຣຽນ(n) {
    ? n <= ໑ { <~ ໑ }
    <~ n * ແຟັກທໍຣຽນ(n - ໑)
}
>> ແຟັກທໍຣຽນ(໕) ¶    // → ໑໒໐
```

ຟັງຊັນມີ **ຂອບເຂດແຍກຕ່າງຫາກ** — ພວກມັນບໍ່ສາມາດອ່ານຕົວແປພາຍນອກໄດ້. ໃຊ້ພາຣາມິເຕີຜົນຜະລິດ `<~>` ເພື່ອດັດແປງຕົວແປຂອງຜູ້ເອີ້ນໃຊ້:

```zymbol
ສະຫຼັບ(a<~, b<~) {
    ຊົ່ວຄາວ = a
    a = b
    b = ຊົ່ວຄາວ
}
x = ໑໐
y = ໒໐
ສະຫຼັບ(x, y)
>> "x=" x " y=" y ¶    // → x=໒໐ y=໑໐
```

> ຟັງຊັນທີ່ມີຊື່ແມ່ນ **ຄ່າຊັ້ນທຳອິດ** — ສົ່ງໂດຍກົງ: `ຈຳນວນ$> ສອງເທົ່າ`. ສຳລັບການຫໍ່: `x -> fn(x)` ກໍ່ຖືກຕ້ອງ.

## ແລມດາ ແລະ ການປິດ

```zymbol
ສອງເທົ່າ = x -> x * ໒
ບວກ = (a, b) -> a + b
>> ສອງເທົ່າ(໕) ¶    // → ໑໐
>> ບວກ(໓, ໗) ¶  // → ໑໐

// ແລມດາແບບບລັອກ
ຈັດປະເພດ = x -> {
    ? x > ໐ { <~ "ບວກ" }
    _? x < ໐ { <~ "ລົບ" }
    <~ "ສູນ"
}

// ການປິດ — ຈັບຂອບເຂດພາຍນອກ
ປັດໄຈ = ໓
ສາມເທົ່າ = x -> x * ປັດໄຈ
>> ສາມເທົ່າ(໗) ¶    // → ໒໑

// ໂຮງງານ
ຜູ້ສ້າງ_ບວກ(n) { <~ x -> x + n }
ບວກສິບ = ຜູ້ສ້າງ_ບວກ(໑໐)
>> ບວກສິບ(໕) ¶    // → ໑໕

// ໃນອາເຣ
ຕົວດຳເນີນງານ = [x -> x+໑, x -> x*໒, x -> x*x]
>> ຕົວດຳເນີນງານ[໓](໕) ¶    // → ໒໕
```

## ອາເຣ

ອາເຣ **ສາມາດປ່ຽນແປງໄດ້** ແລະ ມີອົງປະກອບ **ທີ່ມີປະເພດດຽວກັນ**.

```zymbol
arr = [໑, ໒, ໓, ໔, ໕]

x = arr[໑]      // ໑ — ການເຂົ້າເຖິງ (໑-ຖານ: ອົງປະກອບທຳອິດ)
x = arr[-໑]     // ໕ — ດັດຊະນີລົບ (ອົງປະກອບສຸດທ້າຍ)
x = arr$#       // ໕ — ຄວາມຍາວ (ໃຊ້ (arr$#) ໃນ >>)

arr = arr$+ ໖            // ເພີ່ມ → [໑,໒,໓,໔,໕,໖]
arr໒ = arr$+[໒] ໙໙       // ໃສ່ທີ່ຕຳແໜ່ງ ໒ (໑-ຖານ)
arr໓ = arr$- ໓           // ລຶບການເກີດຂຶ້ນຄັ້ງທຳອິດຂອງຄ່າ
arr໔ = arr$-- ໓          // ລຶບການເກີດຂຶ້ນທັງໝົດ
arr໕ = arr$-[໑]          // ລຶບທີ່ດັດຊະນີ ໑ (ອົງປະກອບທຳອິດ)
arr໖ = arr$-[໒..໓]       // ລຶບໄລຍະຫ່າງ (໑-ຖານ, ລວມທັງຕອນທ້າຍ)

ມີ = arr$? ໓            // #໑ — ມີ
ຕຳແໜ່ງ = arr$?? ໓           // [໓] — ດັດຊະນີທັງໝົດຂອງຄ່າ (໑-ຖານ)
ຕ່ອນ = arr$[໑..໓]          // [໑,໒,໓] — ຕ່ອນ (໑-ຖານ, ລວມທັງຕອນທ້າຍ)
ຕ່ອນ໒ = arr$[໑:໓]          // [໑,໒,໓] — ດຽວກັນ, ໄວຍະກອນທີ່ອີງໃສ່ຈຳນວນ

ຂຶ້ນ = arr$^+             // ຈັດລຽງຂຶ້ນ (ສະເພາະຄ່າດັ້ງເດີມ)
ລົງ = arr$^-            // ຈັດລຽງລົງ (ສະເພາະຄ່າດັ້ງເດີມ)

// ອາເຣທູເປີ້ນທີ່ມີຊື່/ຕາມຕຳແໜ່ງ — ໃຊ້ `$^` ກັບແລມດາປຽບທຽບ
ຖານຂໍ້ມູນ = [(ຊື່: "ຄາລາ", ອາຍຸ: ໒໘), (ຊື່: "ອານາ", ອາຍຸ: ໒໕), (ຊື່: "ບອບ", ອາຍຸ: ໓໐)]
ຕາມອາຍຸ  = ຖານຂໍ້ມູນ$^ (a, b -> a.ອາຍຸ < b.ອາຍຸ)    // ຕາມອາຍຸຂຶ້ນ (<)
ຕາມຊື່ = ຖານຂໍ້ມູນ$^ (a, b -> a.ຊື່ > b.ຊື່)   // ຕາມຊື່ລົງ (>)
>> ຕາມອາຍຸ[໑].ຊື່ ¶     // → ອານາ
>> ຕາມຊື່[໑].ຊື່ ¶    // → ຄາລາ

// ການອັບເດດອົງປະກອບໂດຍກົງ (ມີແຕ່ອາເຣ)
arr[໑] = ໙໙              // ກຳນົດ
arr[໒] += ໕              // ປະສົມ: +=  -=  *=  /=  %=  ^=

// ການອັບເດດແບບຟັງຊັນ — ສົ່ງຄືນອາເຣໃໝ່; ຕົ້ນສະບັບບໍ່ປ່ຽນແປງ
arr໒ = arr[໒]$~ ໙໙
```

> ຕົວດຳເນີນງານການລວບລວມທັງໝົດສົ່ງຄືນ **ອາເຣໃໝ່**. ກຳນົດຄ່າຄືນ: `arr = arr$+ 4`.
> `$+` ສາມາດເຊື່ອມໂສ້ໄດ້: `arr = arr$+ 5$+ 6$+ 7`. ຕົວດຳເນີນງານອື່ນໆໃຊ້ການກຳນົດຄ່າຂັ້ນກາງ.
> **ດັດຊະນີແມ່ນ 1-ຖານ**: `arr[1]` ແມ່ນອົງປະກອບທຳອິດ; `arr[0]` ແມ່ນຄວາມຜິດພາດໃນເວລາແລ່ນ.
> `$^+` / `$^-` ຈັດລຽງ **ອາເຣຄ່າດັ້ງເດີມ** (ຕົວເລກ, ສະຕຣິງ). ສຳລັບອາເຣທູເປີ້ນ, ໃຊ້ `$^` ກັບແລມດາປຽບທຽບ — ທິດທາງຖືກເຂົ້າລະຫັດໄວ້ໃນແລມດາ (`<` = ຂຶ້ນ, `>` = ລົງ).

**ຄວາມໝາຍຄ່າ** — ການກຳນົດອາເຣໃຫ້ກັບຕົວແປອື່ນສ້າງສຳເນົາທີ່ເປັນເອກະລາດ:

```zymbol
a = [໑, ໒, ໓]
b = a
a[໑] = ໙໙
>> a ¶    // → [໙໙, ໒, ໓]
>> b ¶    // → [໑, ໒, ໓]   ← b ບໍ່ໄດ້ຮັບຜົນກະທົບ
```

```zymbol
// ອາເຣທີ່ຊ້ອນກັນ (ດັດຊະນີ ໑-ຖານ)
ເມທຣິກ = [[໑,໒,໓],[໔,໕,໖],[໗,໘,໙]]
>> ເມທຣິກ[໒][໓] ¶    // → ໖  (ແຖວ ໒, ຖັນ ໓)
```

## ການແຍກໂຄງສ້າງ

```zymbol
// ອາເຣ
arr = [໑໐, ໒໐, ໓໐, ໔໐, ໕໐]
[a, b, c] = arr              // a=໑໐  b=໒໐  c=໓໐
[ທຳອິດ, *ທີ່ເຫຼືອ] = arr         // ທຳອິດ=໑໐  ທີ່ເຫຼືອ=[໒໐,໓໐,໔໐,໕໐]
[x, _, z] = [໑, ໒, ໓]        // _ ຖືກຍົກເລີກ

// ທູເປີ້ນຕາມຕຳແໜ່ງ
ຈຸດ = (໑໐໐, ໒໐໐)
(px, py) = ຈຸດ             // px=໑໐໐  py=໒໐໐

// ທູເປີ້ນທີ່ມີຊື່
ຄົນ = (ຊື່: "ອານາ", ອາຍຸ: ໒໕, ເມືອງ: "ມາດິດ")
(ຊື່: n, ອາຍຸ: ອ) = ຄົນ   // n="ອານາ"  ອ=໒໕
```

## ທູເປີ້ນ

ທູເປີ້ນແມ່ນຖັງທີ່ຖືກຈັດລຽງ **ທີ່ບໍ່ສາມາດປ່ຽນແປງໄດ້** ເຊິ່ງສາມາດຖືຄ່າ **ທີ່ມີປະເພດແຕກຕ່າງກັນ**.
ບໍ່ຄືກັບອາເຣ, ອົງປະກອບບໍ່ສາມາດປ່ຽນແປງໄດ້ຫຼັງຈາກການສ້າງ.

```zymbol
// ຕາມຕຳແໜ່ງ — ປະເພດປະສົມຖືກອະນຸຍາດ
ຈຸດ = (໑໐, ໒໐)
>> ຈຸດ[໑] ¶    // → ໑໐

ຂໍ້ມູນ = (໔໒, "ສະບາຍດີ", #໑, ໓.໑໔)
>> ຂໍ້ມູນ[໓] ¶     // → #໑

// ມີຊື່
ຄົນ = (ຊື່: "Alice", ອາຍຸ: ໒໕)
>> ຄົນ.ຊື່ ¶    // → Alice
>> ຄົນ[໑] ¶      // → Alice  (ດັດຊະນີກໍ່ເຮັດວຽກຄືກັນ, ໑-ຖານ)

// ຊ້ອນກັນ
ຕຳແໜ່ງ = (x: ໑໐, y: ໒໐)
p = (ຕຳແໜ່ງ: ຕຳແໜ່ງ, ປ້າຍ: "ຕົ້ນກຳເນີດ")
>> p.ຕຳແໜ່ງ.x ¶        // → ໑໐
```

**ການບໍ່ສາມາດປ່ຽນແປງໄດ້** — ຄວາມພະຍາຍາມໃດໆທີ່ຈະດັດແປງອົງປະກອບທູເປີ້ນແມ່ນຄວາມຜິດພາດໃນເວລາແລ່ນ:

```zymbol
t = (໑໐, ໒໐, ໓໐)
// t[໑] = ໙໙    // ❌ ຄວາມຜິດພາດໃນເວລາແລ່ນ: ທູເປີ້ນບໍ່ສາມາດປ່ຽນແປງໄດ້
// t[໑] += ໕    // ❌ ຄວາມຜິດພາດດຽວກັນ
```

ເພື່ອໃຫ້ໄດ້ຄ່າທີ່ຖືກດັດແປງ, ໃຊ້ `$~` (ການອັບເດດແບບຟັງຊັນ) — ສົ່ງຄືນ **ທູເປີ້ນໃໝ່**:

```zymbol
t = (໑໐, ໒໐, ໓໐)
t໒ = t[໒]$~ ໙໙໙
>> t ¶     // → (໑໐, ໒໐, ໓໐)   ← ຕົ້ນສະບັບບໍ່ປ່ຽນແປງ
>> t໒ ¶    // → (໑໐, ໙໙໙, ໓໐)

// ທູເປີ້ນທີ່ມີຊື່ — ສ້າງໃໝ່ຢ່າງຊັດເຈນ
ຄົນ = (ຊື່: "Alice", ອາຍຸ: ໒໕)
ເກົ່າກວ່າ  = (ຊື່: ຄົນ.ຊື່, ອາຍຸ: ໒໖)
>> ຄົນ.ອາຍຸ ¶    // → ໒໕
>> ເກົ່າກວ່າ.ອາຍຸ ¶     // → ໒໖
```

## ຟັງຊັນລຳດັບສູງ

```zymbol
ຈຳນວນ = [໑, ໒, ໓, ໔, ໕, ໖, ໗, ໘, ໙, ໑໐]

ສອງເທົ່າ  = ຈຳນວນ$> (x -> x * ໒)                  // map  → [໒,໔,໖…໒໐]
ຄູ່    = ຈຳນວນ$| (x -> x % ໒ == ໐)           // filter → [໒,໔,໖,໘,໑໐]
ທັງໝົດ    = ຈຳນວນ$< (໐, (ຜູ້ສະສົມ, x) -> ຜູ້ສະສົມ + x)     // reduce → ໕໕

// ເຊື່ອມໂສ້ຜ່ານຕົວກາງ
ຂັ້ນ໑ = ຈຳນວນ$| (x -> x > ໓)
ຂັ້ນ໒ = ຂັ້ນ໑$> (x -> x * x)
>> ຂັ້ນ໒ ¶    // → [໑໖, ໒໕, ໓໖, ໔໙, ໖໔, ໘໑, ໑໐໐]

// ຟັງຊັນທີ່ມີຊື່ສາມາດຖືກສົ່ງໂດຍກົງໄປຍັງ HOF
ສອງເທົ່າ(x) { <~ x * ໒ }
ໃຫຍ່(x) { <~ x > ໕ }
r = ຈຳນວນ$> ສອງເທົ່າ       // ✅ ການອ້າງອີງໂດຍກົງ
r = ຈຳນວນ$| ໃຫຍ່       // ✅ ການອ້າງອີງໂດຍກົງ
```

## ຕົວດຳເນີນງານທໍ່

ເບື້ອງຂວາສະເໝີຕ້ອງການ `_` ເປັນຕົວວາງແທນສຳລັບຄ່າທີ່ຖືກສົ່ງຜ່ານທໍ່:

```zymbol
ສອງເທົ່າ = x -> x * ໒
ບວກ = (a, b) -> a + b
ເພີ່ມ = x -> x + ໑

ຜ໑ = ໕ |> ສອງເທົ່າ(_)        // → ໑໐
ຜ໒ = ໑໐ |> ບວກ(_, ໕)       // → ໑໕
ຜ໓ = ໕ |> ບວກ(໒, _)        // → ໗

// ເຊື່ອມໂສ້
ຜ = ໕ |> ສອງເທົ່າ(_) |> ເພີ່ມ(_) |> ສອງເທົ່າ(_)
>> ຜ ¶    // → ໒໒  (໕→໑໐→໑໑→໒໒)
```

## ໂໝດຕົວເລກ

Zymbol ສາມາດສະແດງຕົວເລກໃນ **69 ຊຸດຕົວເລກ Unicode** — ເທວະນາຄີ, ອາຣັບ-ອິນເດຍ, ໄທ, Klingon pIqaD, ຄະນິດສາດໜາ, ສ່ວນ LCD, ແລະອື່ນໆອີກ. ໂໝດທີ່ເຄື່ອນໄຫວມີຜົນກະທົບຕໍ່ຜົນຜະລິດ `>>` ເທົ່ານັ້ນ; ເລກຄະນິດສາດພາຍໃນແມ່ນຖານສອງສະເໝີ.

### ການເປີດໃຊ້ຊຸດຕົວໜັງສື

ຂຽນຕົວເລກ `0` ແລະ `9` ຂອງຊຸດຕົວໜັງສືເປົ້າໝາຍພາຍໃນ `#…#`:

```zymbol
#०९#    // ເທວະນາຄີ   (U+0966–U+096F)
#٠٩#    // ອາຣັບ-ອິນເດຍ (U+0660–U+0669)
#๐๙#    // ໄທ         (U+0E50–U+0E59)
#09#    // ຣີເຊັດກັບ ASCII
```

### ຜົນຜະລິດ ແລະ boolean

```zymbol
x = ໔໒
>> x ¶          // → ໔໒   (ASCII ເລີ່ມຕົ້ນ)

#०९#
>> x ¶          // → ४२
>> ໓.໑໔ ¶       // → ३.१४   (ຈຸດທົດສະນິຍົມແມ່ນ ASCII ສະເໝີ)
>> ໑ + ໒ ¶      // → ३

// boolean: ຄຳນຳໜ້າ # ແມ່ນ ASCII ສະເໝີ, ຕົວເລກປັບຕົວເອງ
>> #໑ ¶         // → #१   (ຈິງໃນເທວະນາຄີ)
>> #໐ ¶         // → #०   (ຜິດ — ແຕກຕ່າງຈາກ ० ຈຳນວນເຕັມສູນ)

x = ໒໘ > ໔
>> x ¶          // → #१   (ຜົນການປຽບທຽບປະຕິບັດຕາມໂໝດທີ່ເຄື່ອນໄຫວ)
```

### ຕົວເລກຕາມຕົວໜັງສືພື້ນເມືອງໃນຊອດໂຄ້ດ

ຕົວເລກຂອງຊຸດຕົວໜັງສືທີ່ຮອງຮັບໃດກໍ່ຕາມແມ່ນຕົວໜັງສືທີ່ຖືກຕ້ອງ — ໃນໄລຍະຫ່າງ, modulo, ການປຽບທຽບ:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### ຕົວໜັງສື Boolean ໃນຊຸດຕົວໜັງສືໃດກໍ່ຕາມ

`#` + ຕົວເລກ `0` ຫຼື `1` ຈາກບລັອກໃດກໍ່ຕາມແມ່ນຕົວໜັງສື boolean ທີ່ຖືກຕ້ອງ:

```zymbol
#०९#
ເຄື່ອນໄຫວ = #१        // ຄືກັນກັບ #໑
>> ເຄື່ອນໄຫວ ¶        // → #१
>> (#१ && #०) ¶ // → #०
```

> `#` **ແມ່ນ ASCII ສະເໝີ**. `#0` (ຜິດ) ແມ່ນແຕກຕ່າງທາງສາຍຕາຈາກ `0` (ຈຳນວນເຕັມສູນ) ສະເໝີໃນທຸກຊຸດຕົວໜັງສື.

## ຕົວດຳເນີນງານຂໍ້ມູນ

```zymbol
// ການປ່ຽນແປງປະເພດ
f = ##.໔໒         // → ໔໒.໐  (ເປັນເລກທົດສະນິຍົມ)
i = ###໓.໗        // → ໔     (ເປັນຈຳນວນເຕັມ, ປັດ)
t = ##!໓.໗        // → ໓     (ເປັນຈຳນວນເຕັມ, ຕັດ)

// ວິເຄາະສະຕຣິງເປັນຕົວເລກ
v໑ = #|"໔໒"|      // → ໔໒  (ຈຳນວນເຕັມ)
v໒ = #|"໓.໑໔"|    // → ໓.໑໔  (ເລກທົດສະນິຍົມ)
v໓ = #|"abc"|     // → "abc"  (ປອດໄພ, ບໍ່ມີຄວາມຜິດພາດ)

// ປັດ / ຕັດ
π = ໓.໑໔໑໕໙໒໖໕
ປັດ໒ = #.໒|π|      // → ໓.໑໔  (ປັດໄປ ໒ ຕຳແໜ່ງທົດສະນິຍົມ)
ປັດ໔ = #.໔|π|      // → ໓.໑໔໑໖
ຕັດ໒ = #!໒|π|      // → ໓.໑໔  (ຕັດ)

// ການຈັດຮູບແບບຕົວເລກ
ຮູບແບບ = #,|໑໒໓໔໕໖໗|  // → ໑,໒໓໔,໕໖໗  (ຄັ້ນດ້ວຍຈຸດ)
ວິທະຍາສາດ = #^|໑໒໓໔໕.໖໗໘|    // → ໑.໒໓໔໕໖໗໘e໔  (ວິທະຍາສາດ)

// ຕົວໜັງສືຖານ
a = 0x41         // → 'A'  (ຖານສິບຫົກ)
b = 0b01000001   // → 'A'  (ຖານສອງ)
c = 0o101        // → 'A'  (ຖານແປດ)

// ຜົນຜະລິດການປ່ຽນຖານ
ຖານສິບຫົກ = 0x|໒໕໕|    // → "0x00FF"
ຖານສອງ = 0b|໖໕|     // → "0b1000001"
ຖານແປດ = 0o|໘|      // → "0o10"
ຖານສິບ = 0d|໒໕໕|    // → "0d0255"
```

## ເອກະສານອ້າງອີງສັນຍາລັກ

| `??` | ຈັບຄູ່ | `$??` | ຊອກຫາດັດຊະນີທັງໝົດ (1-ຖານ) |

| `@>` | ສືບຕໍ່ | `$<` | reduce |

### v0.0.5 — ພື້ນຖານ TUI, ຄຳນິຍາມຮ້ອນ & ການເຮັດສະຕຣິງຊ້ຳ _(ພຶດສະພາ 2026)_

- **ການທຳລາຍ** ຕົວຄັ້ນແຂນການຈັບຄູ່: `ຮູບແບບ : ຜົນໄດ້ຮັບ` → `ຮູບແບບ => ຜົນໄດ້ຮັບ`
- **ການທຳລາຍ** ຊື່ສົມມຸດການນຳເຂົ້າ: `<# ເສັ້ນທາງ <= ຊື່ສົມມຸດ` → `<# ເສັ້ນທາງ => ຊື່ສົມມຸດ`
- **ການທຳລາຍ** ການປ່ຽນຊື່ການສົ່ງອອກ: `#> { fn <= ສາທາລະນະ }` → `#> { fn => ສາທາລະນະ }`
- **ເພີ່ມແລ້ວ** ບລັອກ TUI `>>| { }` — ໜ້າຈໍສຳຮອງ + ໂໝດດິບ; ລ້າງເມື່ອອອກ
- **ເພີ່ມແລ້ວ** ຜົນຜະລິດຕາມຕຳແໜ່ງ `>>~ (ແຖວ, ຖັນ, BKS, fg, bg) > ລາຍການ` — ຊ່ອງຫ່າງໆ, ສີ ANSI 256
- **ເພີ່ມແລ້ວ** ການປ້ອນປຸ່ມ `<<| ຕົວແປ` (ສະກັດ) ແລະ `<<|? ຕົວແປ` (ກວດສອບແບບບໍ່ສະກັດ)
- **ເພີ່ມແລ້ວ** `>>!` ລ້າງໜ້າຈໍ, `>>?` ຖາມຂະໜາດລະບົບ, `@~ N` ນອນ N ມິນລິວິນາທີ
- **ເພີ່ມແລ້ວ** ຄຳນິຍາມຮ້ອນ `°x` / `x°` — ເລີ່ມຕົ້ນຕົວແປອັດຕະໂນມັດໃນການນຳໃຊ້ຄັ້ງທຳອິດໃນວົງຈອນ
- **ເພີ່ມແລ້ວ** ການເຮັດສະຕຣິງຊ້ຳ `ສະຕຣິງ $* N` — ເຮັດສະຕຣິງຊ້ຳ N ເທື່ອ
- **VM** ຄວາມເທົ່າທຽມ: 436/436 ການທົດສອບຜ່ານ

### v0.0.4 — ດັດຊະນີ 1-ຖານ, ຟັງຊັນຊັ້ນທຳອິດ & ໂມດູນບລັອກ _(ເມສາ 2026)_

- **ການທຳລາຍ** ດັດຊະນີທັງໝົດຖືກປ່ຽນເປັນ **1-ຖານ** — `arr[1]` ແມ່ນອົງປະກອບທຳອິດ; `arr[0]` ແມ່ນຄວາມຜິດພາດໃນເວລາແລ່ນ
- **ເພີ່ມແລ້ວ** ຟັງຊັນທີ່ມີຊື່ແມ່ນ **ຄ່າຊັ້ນທຳອິດ** — ສົ່ງໂດຍກົງໄປ HOF: `ຈຳນວນ$> ສອງເທົ່າ`
- **ເພີ່ມແລ້ວ** **ໄວຍະກອນບລັອກທີ່ຈຳເປັນ** ສຳລັບໂມດູນ: `# ຊື່ { ... }` — ໄວຍະກອນແບນຖືກລຶບອອກ
- **ເພີ່ມແລ້ວ** ດັດຊະນີຫຼາຍມິຕິ: `arr[i>j>k]` (ການນຳທາງ), `arr[p ; q]` (ການດຶງຂໍ້ມູນແບບແບນ)
- **ເພີ່ມແລ້ວ** ການປ່ຽນແປງປະເພດ: `##.ສຳນວນ` (ເລກທົດສະນິຍົມ), `###ສຳນວນ` (ຈຳນວນເຕັມປັດ), `##!ສຳນວນ` (ຈຳນວນເຕັມຕັດ)
- **ເພີ່ມແລ້ວ** ການແບ່ງສະຕຣິງ: `ສະຕຣິງ$/ ຕົວຄັ້ນ` — ສົ່ງຄືນ `Array(ສະຕຣິງ)`
- **ເພີ່ມແລ້ວ** ການສ້າງການຕໍ່ກັນ: `ຖານ$++ a b c` — ເພີ່ມຫຼາຍລາຍການ
- **ເພີ່ມແລ້ວ** ວົງຈອນເທື່ອ: `@ N { }` — ເຮັດຊ້ຳຢ່າງແນ່ນອນ N ເທື່ອ
- **ເພີ່ມແລ້ວ** ໄວຍະກອນວົງຈອນທີ່ມີປ້າຍກຳກັບ: `@:ຊື່ { }`, `@:ຊື່!`, `@:ຊື່>` — ແທນທີ່ `@ @ຊື່` / `@! ຊື່`
- **ເພີ່ມແລ້ວ** ກົດຂອບເຂດຕົວແປ: ຕົວແປ `_ຊື່` ມີຂອບເຂດບລັອກທີ່ແນ່ນອນ; `\ ຕົວແປ` ທຳລາຍໄວ
- **ເພີ່ມແລ້ວ** ຮູບແບບການປຽບທຽບການຈັບຄູ່: `< 0 =>`, `> 5 =>`, `== 42 =>`, ຯລຯ

- **ແກ້ໄຂແລ້ວ** `alias.CONST` ປັດຈຸບັນແກ້ໄຂໄດ້ຢ່າງຖືກຕ້ອງ; `#>` ສາມາດປາກົດຫຼັງຈາກຄຳນິຍາມຟັງຊັນ
