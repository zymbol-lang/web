# Zymbol-Lang — ᓀᐦᐃᔭᐍᐏᐣ ᑮᐢᑭᓇᐦᐊᒫᑫᐏᐣ (ᐊᓯᓂᑿᐣ / Syllabics)

**Zymbol-Lang** ᐯᔭᐠ ᑮᐢᑭᓇᐦᐊᒫᑫᐏᐣ ᐲᑭᐢᑵᐏᐣ ᒪᓯᓇᐦᐃᑲᓇᐢ ᐆᒪ — ᓇᒨᔭ ᑮᑿᔾ ᐋᐸᒋᐦᐊᐍᐏᐣ, ᒪᓯᓇᐦᐃᑲᓇᐢ ᐱᑯ. ᑳᐯᔭᑲᐣ ᐃᐢᐸᔨᐤ ᒥᒋ ᐆᒪ ᓀᐦᐃᔭᐍᐏᐣ.

- ᓇᒨᔭ ᐋᐸᒋᐦᐊᐍᐏᓇ (`if`, `while`, `return` ᓇᒨᔭ ᐋᐸᑎᓯᐤ — ᒪᓯᓇᐦᐃᑲᓇᐢ ᐱᑯ `?`, `@`, `<~`)
- ᑲᐦᑭᔭᐤ Unicode — ᐲᑭᐢᑵᐏᓇ ᑳᐃᓯᐲᑭᐢᑵᐟ ᐃᐢᐸᔨᐤ, ᒦᓇ emoji 👋
- ᐲᑭᐢᑵᐏᐣ-ᑳᑮᐅᓯᐦᑖᐟ — ᑮᑿᔾ ᒪᓯᓇᐦᐃᑲᓇᐢ ᑳᐯᔭᑲᐣ ᐃᐢᐸᔨᐤ ᑲᐦᑭᔭᐤ ᐲᑭᐢᑵᐏᐣᐦᐠ

## ᐲᑭᐢᑵᐏᓇ ᒦᓇ ᓇᒨᔭ ᐹᐢᑖᐦᐅᐘᐠ

```zymbol
ᑮ = 10              // ᐲᑭᐢᑵᐏᐣ (ᑳᐑᐦᑕᒫᐦᐠ)
ᐸᐃ := 3.14159       // ᓇᒨᔭ ᐹᐢᑖᐦᐆᐤ (ᐹᐢᑖᐦᐅᐍᐤ ᑳᒫᔨᓂᑫᐦᐠ)
ᐃᓯᔨᐦᑳᓱᐏᐣ = "ᓀᐦᐃᔭᐍᐏᐣ"
ᑭᐢᑌᔨ = #1          // ᑖᐻᐍᐏᐣ ᑭᐢᑌᔨᒨᐏᐣ
👋 := "ᑖᓂᓯ"
```

```zymbol
ᑮ = 10
ᑮ += 5    // 15
ᑮ -= 3    // 12
ᑮ *= 2    // 24
ᑮ /= 3    // 8
ᑮ %= 3    // 2
ᑮ ^= 2    // 4
ᑮ++       // 5
ᑮ--       // 4
```

## ᑮᑿᔾ ᐆᐦᐃ

| ᑮᑿᔾ               | ᑎᐸᐦᐃᑫᐏᐣ            | ᒪᓯᓇᐦᐃᑲᓇᐢ `#?`      | ᑭᐢᑫᔨᐦᑕᒧᐏᐣ                              |
|--------------------|---------------------|--------------------|-----------------------------------------|
| ᐊᑭᐦᑖᓱᓇ            | `42`, `-7`          | `###`              | 64-bit ᑳᑮᐃᐦᑕᑯᐠ                         |
| ᐲᑐᓵᔮᐏᐣ            | `3.14`, `1.5e10`    | `##.`              | ᑭᐢᑌᔨᒨᐏᐣ ᐲᑭᐢᑵᐏᐣ OK                     |
| ᒪᓯᓇᐦᐃᑲᓇᐢ          | `"ᑖᓂᓯ"`             | `##"`              | ᐑᐦᑕᒫᑫᐏᐣ: `"ᑖᓂᓯ {ᐃᓯᔨᐦᑳᓱᐏᐣ}"`           |
| ᐲᑭᐢᑵᐏᐣ            | `'A'`               | `##'`              | ᐯᔭᐠ Unicode ᐲᑭᐢᑵᐏᐣ                    |
| ᑖᐻᐍᐏᐣ             | `#1`, `#0`          | `##?`              | ᓇᒨᔭ ᐊᑭᐦᑖᓱᓇ 1 ᒦᓇ 0                    |
| ᒪᓯᓇᐦᐃᑲᓇ           | `[1, 2, 3]`         | `##]`              | ᑳᐯᔭᑲᐣ ᑮᑿᑾᔭ                            |
| ᑌᐸᓚ               | `(ᐊᐤ, ᐁᐠ)`          | `##)`              | ᑳᐃᓯᐃᑕᐦᐱᑌᐦᐠ                            |
| ᐑᐦᒋᑳᑌᐤ ᑌᐸᓚ       | `(ᑮ: 1, ᓃ: 2)`      | `##)`              | ᐑᐦᒋᑳᑌᐤ ᒦᓇ ᐊᑭᐦᑖᓱᓇ ᐋᐸᒋᐦᐊᐏᐣ           |

## ᓈᑕᐏᐦᐅᐦᐟ ᒦᓇ ᑖᐸᑫᐤ

```zymbol
>> "ᑖᓂᓯ" ¶                              // ¶ ᒦᓇ \\ ᐱᐦᑌᐏᐣ ᒪᓯᓇᐦᐃᑲᓇᐢ
>> "a=" ᑮ " b=" ᓃ ¶                      // ᒥᐦᒉᐟ ᑮᑿᑾᔭ ᒫᒪᐏ
>> (ᒪᓯᓇᐦᐃᑲᓇ$#) ¶                        // postfix ᐙᐏᐢ ( ) ᑭᐦᒋ

<< ᐃᓯᔨᐦᑳᓱᐏᐣ                             // ᓇᒨᔭ ᑲᑵᒉᐦᑫᒨᐏᐣ — ᐃᓯᓈᑿᐦᐠ ᐲᑭᐢᑵᐏᐣ
<< "ᑮᑭᓭᐹ ᑭᔭᐙᐤ? " ᐃᓯᔨᐦᑳᓱᐏᐣ             // ᑲᑵᒉᐦᑫᒨᐏᐣ ᑳᐑᐦᑕᒥᐦᐠ
```

> `¶` ᒦᓇ `\\` ᑳᐯᔭᑲᐣ — ᐱᐦᑌᐏᐣ ᒪᓯᓇᐦᐃᑲᓇᐢ.

## ᒥᓯᐧᔦᒑᑦᐧᐃᐧᓄᐧᐃᓯᐧᐊᐧᐃ

```zymbol
// ᐊᑭᐦᑖᓱᓇ
a = 10
b = 3
r1 = a + b    // 13     r2 = a - b    // 7
r3 = a * b    // 30     r4 = a / b    // 3  (ᐊᑭᐦᑖᓱᐣ ᓯᐱᐦᑫᐏᐣ)
r5 = a % b    // 1      r6 = a ^ b    // 1000  (ᑳᐃᐢᐸᔨᐠ)

// ᑳᓇᓇᑐᐦᑲᒥᐦᐠ
a == b    // #0    a <> b    // #1    a < b    // #0
a <= b    // #0   a > b     // #1    a >= b   // #1

// ᑖᐻᐍᐏᐣ
#1 && #0    // #0
#1 || #0    // #1
!#1         // #0
```

## ᒪᓯᓇᐦᐃᑲᓇ

```zymbol
// ᓂᐢᑕᒼ ᑳᐃᓯᐃᑑᑕᐦᐠ — ᑖᓂᓯ ᑳᐃᑑᑕᒥᐦᐠ
ᐃᓯᔨᐦᑳᓱᐏᐣ = "ᓀᐦᐃᔭᐍᐏᐣ"
ᐊᑭᐦᑖᓱᐣ = 25

ᐑᐦᑕᒨᐏᐣ = "ᑖᓂᓯ ", ᐃᓯᔨᐦᑳᓱᐏᐣ, "!"        // ᑳᑮᐢᑵᐸᔨᐠ — = ᒦᓇ :=
>> "ᑖᓂᓯ " ᐃᓯᔨᐦᑳᓱᐏᐣ " ᑭᔭᐙᐤ " ᐊᑭᐦᑖᓱᐣ ¶   // ᒫᒪᐏ >> ᑳᓈᑕᐏᐦᐅᐦᑕᐦᐠ
ᑭᐢᑭᓄᑕᐦᐙᐏᐣ = "ᑖᓂᓯ {ᐃᓯᔨᐦᑳᓱᐏᐣ}, ᑭᔭᐙᐤ {ᐊᑭᐦᑖᓱᐣ}"  // ᐑᐦᑕᒨᐏᐣ
```

```zymbol
s = "ᑖᓂᓯ ᓀᐦᐃᔭᐍᐏᐣ"
len = s$#                  // 10
sub = s$[0..5]             // "ᑖᓂᓯ "  (ᓇᒨᔭ ᑌᐦᒋ)
has = s$? "ᓀᐦᐃᔭᐍᐏᐣ"        // #1
parts = "a,b,c,d" / ','    // [a, b, c, d]
rep = s$~~["ᓂ":"ᓃ"]        // ᓃᐢᑌᔨ ᐑᐦᑕᒨᐏᐣ
rep1 = s$~~["ᓂ":"ᓃ":1]     // ᐯᔭᐠ ᐱᑯ
```

> **ᑭᐢᑫᔨᐦᑕᒧᐏᐣ**: `+` ᐊᑭᐦᑖᓱᓇ ᐱᑯ. ᒪᓯᓇᐦᐃᑲᓇᐢ ᑳᐑᐦᑕᒥᐦᐠ — ᒫᔨᐃᐢᐸᔨᐤ.

## ᑳᐃᓯᐃᐢᐸᔨᐠ

```zymbol
ᑮ = 7

? ᑮ > 0 { >> "ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ" ¶ }

? ᑮ > 100 {
    >> "ᒥᐢᑕᐦᐃ ᑭᐦᒋ" ¶
} _? ᑮ > 0 {
    >> "ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ" ¶
} _? ᑮ == 0 {
    >> "ᓲᐢᑿᐨ" ¶
} _ {
    >> "ᓇᒨᔭ ᑭᐦᒋ" ¶
}
```

ᒥᐦᑴᐦᐱᑌᐏᓇ `{ }` **ᑭᐦᒋ-ᐃᑑᑕᒥᐦᐠ** — ᒫᑲ ᐯᔭᐠ ᒪᓯᓇᐦᐃᑲᓇᐢ.

## ᑳᑮᐢᐱᐦᑕᐦᐠ

```zymbol
// ᑳᑮᐢᐱᐦᑕᐦᐠ ᑳᐃᓯᐊᑭᐦᑖᐦᐠ
ᑎᐸᐦᐃᑫᐏᐣ = 85
ᐑᐦᑕᒨᐏᐣ = ?? ᑎᐸᐦᐃᑫᐏᐣ {
    90..100 : 'A'
    80..89  : 'B'
    70..79  : 'C'
    _       : 'F'
}
>> ᐑᐦᑕᒨᐏᐣ ¶    // → B

// ᑳᑮᐢᐱᐦᑕᐦᐠ ᒪᓯᓇᐦᐃᑲᓇᐢ
ᐙᐱᑭᐘᐣ = "ᒥᐦᑯ"
ᑮᓯᑳᐤ = ?? ᐙᐱᑭᐘᐣ {
    "ᒥᐦᑯ"    : "#FF0000"
    "ᐊᐢᑭᐦᑕᐤ" : "#00FF00"
    _          : "#000000"
}

// ᑳᑮᐢᐱᐦᑕᐦᐠ ᑳᐃᓯᑳᑮᓯᒥᐦᐠ (guards)
ᑭᓯᑳᐤ = -5
ᐃᐢᐸᔨᐏᐣ = ?? ᑭᓯᑳᐤ {
    _? ᑭᓯᑳᐤ < 0  : "ᑰᓇ"
    _? ᑭᓯᑳᐤ < 20 : "ᑮᐢᐱᐣ"
    _? ᑭᓯᑳᐤ < 35 : "ᑮᓯᑳᐤ"
    _              : "ᑭᓯᑳᐤ-ᒥᐦᑵᐤ"
}
>> ᐃᐢᐸᔨᐏᐣ ¶    // → ᑰᓇ

// ᑳᑮᐢᐱᐦᑕᐦᐠ ᒥᐦᑴᐦᐱᑌᐏᐣ (block arms)
?? n {
    0       : { >> "ᓲᐢᑿᐨ" ¶ }
    _? n < 0: { >> "ᓇᒨᔭ ᑭᐦᒋ" ¶ }
    _       : { >> "ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ" ¶ }
}
```

## ᑳᐃᐢᐸᔨᐠ

```zymbol
@ ᐁ:0..4  { >> ᐁ " " }        // 0 1 2 3 4  — inclusive
@ ᐁ:1..9:2 { >> ᐁ " " }       // step:  1 3 5 7 9
@ ᐁ:5..0:1 { >> ᐁ " " }       // ᑳᑮᐍᐦᐠ: 5 4 3 2 1 0

ᓃ = 1
@ ᓃ <= 64 { ᓃ *= 2 }
>> ᓃ ¶                        // → 128  (while)

ᒦᓂᓴ = ["ᐸᐦᑵᓯᑲᐣ", "ᒪᐢᑫᑯᒥᓇᐠ", "ᓵᓵᑭᓯᐤ"]
@ ᒦ:ᒦᓂᓴ { >> ᒦ ¶ }

@ ᓭ:"ᑖᓂᓯ" { >> ᓭ "-" }
>> ¶                          // → ᑖ-ᓂ-ᓯ-

@ ᐁ:1..10 {
    ? ᐁ % 2 == 0 { @> }       // @> ᑳᑮᐍᐦᐠ
    ? ᐁ > 7 { @! }             // @! ᑳᓃᐹᐦᐊᐦᐠ
    >> ᐁ " "
}
>> ¶                          // → 1 3 5 7

// ᓇᒨᔭ ᑳᑮᐢᐱᐢᑕᐦᐠ
i = 0
@ {
    i++
    ? i >= 5 { @! }
    >> i " "
}
>> ¶                          // → 1 2 3 4

// Labeled loop
count = 0
@ @outer {
    count++
    ? count >= 3 { @! outer }
}
>> count ¶                    // → 3
```

## ᐋᐸᒋᐦᐊᐍᐏᓇ

```zymbol
ᒫᒪᐏᐦᐅᐦᐟ(ᑮ, ᓃ) { <~ ᑮ + ᓃ }
>> ᒫᒪᐏᐦᐅᐦᐟ(3, 4) ¶    // → 7

ᐊᑭᐦᑖᓱᐙᐸᒋᐦᐃᑫᐏᐣ(ᓃ) {
    ? ᓃ <= 1 { <~ 1 }
    <~ ᓃ * ᐊᑭᐦᑖᓱᐙᐸᒋᐦᐃᑫᐏᐣ(ᓃ - 1)
}
>> ᐊᑭᐦᑖᓱᐙᐸᒋᐦᐃᑫᐏᐣ(5) ¶    // → 120
```

ᐋᐸᒋᐦᐊᐍᐏᓇ **isolated scope** — ᓇᒨᔭ ᑮᑿᔾ ᐲᑐᐢ ᓈᑕᐏᐦᐅᐦᑌᐏᐣ. Output `<~`:

```zymbol
swap(a<~, b<~) {
    tmp = a
    a = b
    b = tmp
}
x = 10
y = 20
swap(x, y)
>> "x=" x " y=" y ¶    // → x=20 y=10
```

> **ᑭᐢᑌᔨᒨᐏᐣ**: ᐑᐦᒋᑳᑌᐤ ᐋᐸᒋᐦᐊᐍᐏᓇ `ᐃᓯᔨᐦᑳᓱᐏᐣ(ᐊᑭᐦᑖᓱᓇ){ }` ᓇᒨᔭ ᐯᔭᐠ ᑮᑿᔾ.
> ᑳᐱᒥᐸᔨᐦᒋᑫᐦᐠ: `ᑮ -> ᐃᓯᔨᐦᑳᓱᐏᐣ(ᑮ)`.

## ᓚᒻᑕ ᒦᓇ ᑳᑮᐍᐦᐋᐟ

```zymbol
ᐯᔭᑯᓯᐟ = ᑮ -> ᑮ * 2
ᒫᒪᐏ = (ᑮ, ᓃ) -> ᑮ + ᓃ
>> ᐯᔭᑯᓯᐟ(5) ¶    // → 10
>> ᒫᒪᐏ(3, 7) ¶    // → 10

// ᓚᒻᑕ ᒥᐦᑴᐦᐱᑌᐏᐣ
ᑭᐢᑌᔨᒨᐏᐣ = ᑮ -> {
    ? ᑮ > 0 { <~ "ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ" }
    _? ᑮ < 0 { <~ "ᓇᒨᔭ ᑭᐦᒋ" }
    <~ "ᓲᐢᑿᐨ"
}

// ᑳᑮᐍᐦᐋᑕᐠ — ᓚᒻᑕ ᑳᑮᐑᐦᑕᒥᐦᐠ ᐲᑐᐢ ᑮᑿᑾᔭ
ᑎᐸᐦᐃᑫᐏᐣ = 3
ᓂᐢᑐᒫᐸᒋᐦᐃᑫᐏᐣ = ᑮ -> ᑮ * ᑎᐸᐦᐃᑫᐏᐣ
>> ᓂᐢᑐᒫᐸᒋᐦᐃᑫᐏᐣ(7) ¶    // → 21

// ᐋᐸᒋᐦᐊᐍᐏᐣ ᑳᐅᓯᐦᑖᐦᐠ
ᐅᓯᐦᑖᑫ(ᓃ) { <~ ᑮ -> ᑮ + ᓃ }
ᒥᑕᐦᑌᐤᑮ = ᐅᓯᐦᑖᑫ(10)
>> ᒥᑕᐦᑌᐤᑮ(5) ¶    // → 15

// ᓚᒻᑕ ᑮᑿᑾᔭ: ᒪᓯᓇᐦᐃᑲᓇᐢᐦᐠ
ᐋᐸᒋᐦᐊᐍᐏᓇ = [ᑮ -> ᑮ+1, ᑮ -> ᑮ*2, ᑮ -> ᑮ*ᑮ]
>> ᐋᐸᒋᐦᐊᐍᐏᓇ[2](5) ¶    // → 25
```

## ᒪᓯᓇᐦᐃᑲᓇᐊ

```zymbol
ᒪᓯᓇᐦᐃᑲᓇ = [1, 2, 3, 4, 5]

ᒪᓯᓇᐦᐃᑲᓇ[0]          // 1 — ᐋᐸᒋᐦᐊᐏᐣ (ᓲᐢᑿᐨ ᐊᑭᐦᑖᓱᐣ)
ᒪᓯᓇᐦᐃᑲᓇ[-1]         // 5 — ᓇᒨᔭ ᐊᑭᐦᑖᓱᐣ (ᑌᐦᒋ)
ᒪᓯᓇᐦᐃᑲᓇ$#           // 5 — ᑎᐸᐦᐃᑫᐏᐣ

ᒪᓯᓇᐦᐃᑲᓇ = ᒪᓯᓇᐦᐃᑲᓇ$+ 6            // append
ᒪᓯᓇᐦᐃᑲᓇ2 = ᒪᓯᓇᐦᐃᑲᓇ$+[2] 99       // insert at index 2
ᒪᓯᓇᐦᐃᑲᓇ3 = ᒪᓯᓇᐦᐃᑲᓇ$- 3           // remove first value
ᒪᓯᓇᐦᐃᑲᓇ4 = ᒪᓯᓇᐦᐃᑲᓇ$-- 3          // remove all
ᒪᓯᓇᐦᐃᑲᓇ5 = ᒪᓯᓇᐦᐃᑲᓇ$-[0]          // remove at index
ᒪᓯᓇᐦᐃᑲᓇ6 = ᒪᓯᓇᐦᐃᑲᓇ$-[1..3]       // remove range (ᓇᒨᔭ ᑌᐦᒋ)

ᐊᔮᐏᐣ = ᒪᓯᓇᐦᐃᑲᓇ$? 3            // #1 — contains
pos = ᒪᓯᓇᐦᐃᑲᓇ$?? 3              // [2] — all indices
ᑮᐢᑭᐦᐏᐣ = ᒪᓯᓇᐦᐃᑲᓇ$[0..3]       // [1,2,3] — slice (ᓇᒨᔭ ᑌᐦᒋ)
sl2 = ᒪᓯᓇᐦᐃᑲᓇ$[0:3]             // [1,2,3] — count-based syntax

asc = ᒪᓯᓇᐦᐃᑲᓇ$^+               // sorted ascending  (primitives)
desc = ᒪᓯᓇᐦᐃᑲᓇ$^-              // sorted descending (primitives)

// ᐑᐦᒋᑳᑌᐤ ᑌᐸᓚ — $^ ᒦᓇ comparator ᓚᒻᑕ
db = [(ᐃᓯᔨᐦᑳᓱᐏᐣ: "Carla", ᐊᑭᐦᑖᓱᐣ: 28), (ᐃᓯᔨᐦᑳᓱᐏᐣ: "Ana", ᐊᑭᐦᑖᓱᐣ: 25)]
by_age = db$^ (a, b -> a.ᐊᑭᐦᑖᓱᐣ < b.ᐊᑭᐦᑖᓱᐣ)

## ᐸᐃᑊ ᒪᒥᐦᒋᒫᑎᓰᐧᐃᐧᐣ

double = ᑮ -> ᑮ * 2

```
