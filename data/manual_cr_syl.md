# Zymbol-Lang — ᓀᐦᐃᔭᐍᐏᐣ ᑮᐢᑭᓇᐦᐊᒫᑫᐏᐣ (ᐊᓯᓂᑿᐣ / Syllabics)

**Zymbol-Lang** ᐯᔭᐠ ᑮᐢᑭᓇᐦᐊᒫᑫᐏᐣ ᐲᑭᐢᑵᐏᐣ ᒪᓯᓇᐦᐃᑲᓇᐢ ᐆᒪ — ᓇᒨᔭ ᑮᑿᔾ ᐋᐸᒋᐦᐊᐍᐏᐣ, ᒪᓯᓇᐦᐃᑲᓇᐢ ᐱᑯ. ᑳᐯᔭᑲᐣ ᐃᐢᐸᔨᐤ ᒥᒋ ᐆᒪ ᓀᐦᐃᔭᐍᐏᐣ.

---

## ᑭᐢᑌᔨᒨᐏᐣ

- ᓇᒨᔭ ᐋᐸᒋᐦᐊᐍᐏᓇ (`if`, `while`, `return` ᓇᒨᔭ ᐋᐸᑎᓯᐤ — ᒪᓯᓇᐦᐃᑲᓇᐢ ᐱᑯ `?`, `@`, `<~`)
- ᑲᐦᑭᔭᐤ Unicode — ᐲᑭᐢᑵᐏᓇ ᑳᐃᓯᐲᑭᐢᑵᐟ ᐃᐢᐸᔨᐤ, ᒦᓇ emoji 👋
- ᐲᑭᐢᑵᐏᐣ-ᑳᑮᐅᓯᐦᑖᐟ — ᑮᑿᔾ ᒪᓯᓇᐦᐃᑲᓇᐢ ᑳᐯᔭᑲᐣ ᐃᐢᐸᔨᐤ ᑲᐦᑭᔭᐤ ᐲᑭᐢᑵᐏᐣᐦᐠ

---

## ᐲᑭᐢᑵᐏᓇ ᒦᓇ ᓇᒨᔭ ᐹᐢᑖᐦᐅᐘᐠ

```zymbol
ᑮ = 10              // ᐲᑭᐢᑵᐏᐣ (ᑳᐑᐦᑕᒫᐦᐠ)
ᐸᐃ := 3.14159       // ᓇᒨᔭ ᐹᐢᑖᐦᐆᐤ (ᐹᐢᑖᐦᐅᐍᐤ ᑳᒫᔨᓂᑫᐦᐠ)
ᐃᓯᔨᐦᑳᓱᐏᐣ = "ᓀᐦᐃᔭᐍᐏᐣ"
ᑭᐢᑌᔨ = #1          // ᑖᐻᐍᐏᐣ ᑭᐢᑌᔨᒨᐏᐣ
👋 := "ᑖᓂᓯ"
```

### ᑳᐑᐦᑕᒥᐦᐠ ᒫᒪᐏ

```zymbol
ᑮ = 10    // 10
ᑮ += 5    // 15
ᑮ -= 3    // 12
ᑮ *= 2    // 24
ᑮ /= 4    // 6
ᑮ %= 4    // 2
ᑮ++       // 3
ᑮ--       // 2
```

## ᑮᑿᔾ ᐆᐦᐃ

| ᑮᑿᔾ               | ᑎᐸᐦᐃᑫᐏᐣ            | ᒪᓯᓇᐦᐃᑲᓇᐢ `#?`      | ᑭᐢᑫᔨᐦᑕᒧᐏᐣ                              |
|--------------------|---------------------|--------------------|-----------------------------------------|
| ᐊᑭᐦᑖᓱᓇ            | `42`, `-7`          | `###`              | 64-bit ᑳᑮᐃᐦᑕᑯᐠ                         |
| ᐲᑐᓵᔮᐏᐣ            | `3.14`, `1.5e10`    | `##.`              | ᑭᐢᑌᔨᒨᐏᐣ ᐲᑭᐢᑵᐏᐣ OK                     |
| ᒪᓯᓇᐦᐃᑲᓇᐢ          | `"ᑖᓂᓯ"`             | `##"`              | ᐑᐦᑕᒫᑫᐏᐣ: `"ᑖᓂᓯ {ᐃᓯᔨᐦᑳᓱᐏᐣ}"`           |
| ᐲᑭᐢᑵᐏᐣ            | `'A'`               | `##'`              | ᐯᔭᐠ Unicode ᐲᑭᐢᑵᐏᐣ                    |
| ᑖᐻᐍᐏᐣ             | `#1`, `#0`          | `##?`              | ᓇᒨᔭ ᐊᑭᐦᑖᓱᓇ 1 ᒦᓇ 0                    |
| ᒪᓯᓇᐦᐃᑲᓇ           | `[1, 2, 3]`         | `##]`              | ᑳᐯᔭᑲᐣ ᑮᑿᑾᔭ                            |
| ᑌᐸᓚ               | `(ᐊᐤ, ᐁᐠ)`          | `##)`              | ᑳᐃᓯᐃᑕᐦᐱᑌᐦᐠ                            |
| ᐑᐦᒋᑳᑌᐤ ᑌᐸᓚ       | `(ᑮ: 1, ᓃ: 2)`      | `##)`              | ᐑᐦᒋᑳᑌᐤ ᒦᓇ ᐊᑭᐦᑖᓱᓇ ᐋᐸᒋᐦᐊᐏᐣ           |

## ᓈᑕᐏᐦᐅᐦᐟ ᒦᓇ ᑖᐸᑫᐤ

```zymbol
// ᓈᑕᐏᐦᐅᐦᐟ — ᓇᒨᔭ ᑮᑿᔾ ᐱᐦᑌᐏᐣ ᑮᐦᑐᐤ
>> "ᑖᓂᓯ" ¶                              // ¶ ᒦᓇ \ ᐱᐦᑌᐏᐣ ᒪᓯᓇᐦᐃᑲᓇᐢ
>> "a=" ᑮ " b=" ᓃ ¶                      // ᒥᐦᒉᐟ ᑮᑿᑾᔭ ᒫᒪᐏ
>> "ᒫᒪᐏ=" ᒫᒪᐏᐦᐅᐦᐟ(2, 3) ¶              // ᐋᐸᒋᐦᐊᐍᐏᓇ ᑖᓂᐢᐲᐦᐠ
>> (ᒪᓯᓇᐦᐃᑲᓇ$#) ¶                        // postfix ᐙᐏᐢ ( ) ᑭᐦᒋ

// ᑖᐸᑫᐤ
<< ᐃᓯᔨᐦᑳᓱᐏᐣ                             // ᓇᒨᔭ ᑲᑵᒉᐦᑫᒨᐏᐣ — ᐃᓯᓈᑿᐦᐠ ᐲᑭᐢᑵᐏᐣ
<< "ᑮᑭᓭᐹ ᑭᔭᐙᐤ? " ᐃᓯᔨᐦᑳᓱᐏᐣ             // ᑲᑵᒉᐦᑫᒨᐏᐣ ᑳᐑᐦᑕᒥᐦᐠ
```

> `¶` ᒦᓇ `\` ᑳᐯᔭᑲᐣ — ᐱᐦᑌᐏᐣ ᒪᓯᓇᐦᐃᑲᓇᐢ.

## ᒪᓯᓇᐦᐃᑲᓇᐢ ᐆᐦᒋ

ᓂᐢᑕᒼ ᑳᐃᓯᐃᑑᑕᐦᐠ — ᑖᓂᓯ ᑳᐃᑑᑕᒥᐦᐠ:

```zymbol
ᐃᓯᔨᐦᑳᓱᐏᐣ = "ᓀᐦᐃᔭᐍᐏᐣ"
ᐊᑭᐦᑖᓱᐣ = 25

// 1. ᑳᑮᐢᑵᐸᔨᐠ — ᐲᑭᐢᑵᐏᓇ ᑳᐑᐦᑕᒫᐦᐠ = ᒦᓇ :=
ᐑᐦᑕᒨᐏᐣ = "ᑖᓂᓯ ", ᐃᓯᔨᐦᑳᓱᐏᐣ, "!"        // → ᑖᓂᓯ ᓀᐦᐃᔭᐍᐏᐣ!
ᑮᐦᒋᐲᑭᐢᑵᐏᐣ := "ᑭᔭᐙᐤ: ", ᐃᓯᔨᐦᑳᓱᐏᐣ

// 2. ᒫᒪᐏ >> ᑳᓈᑕᐏᐦᐅᐦᑕᐦᐠ
>> "ᑖᓂᓯ " ᐃᓯᔨᐦᑳᓱᐏᐣ " ᑭᔭᐙᐤ " ᐊᑭᐦᑖᓱᐣ ¶   // → ᑖᓂᓯ ᓀᐦᐃᔭᐍᐏᐣ ᑭᔭᐙᐤ 25

// 3. ᐑᐦᑕᒨᐏᐣ — ᑖᓂᐢᐲᐦᐠ
ᑭᐢᑭᓄᑕᐦᐙᐏᐣ = "ᑖᓂᓯ {ᐃᓯᔨᐦᑳᓱᐏᐣ}, ᑭᔭᐙᐤ {ᐊᑭᐦᑖᓱᐣ}"
```

> **ᑭᐢᑫᔨᐦᑕᒧᐏᐣ**: `+` ᐊᑭᐦᑖᓱᓇ ᐱᑯ. ᒪᓯᓇᐦᐃᑲᓇᐢ ᑳᐑᐦᑕᒥᐦᐠ — ᒫᔨᐃᐢᐸᔨᐤ.

## ᑳᐃᓯᐃᐢᐸᔨᐠ

```zymbol
ᑮ = 7

// ᐯᔭᐠ ᑳᑖᐻᐍᐦᐠ
? ᑮ > 0 { >> "ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ" ¶ }

// ᑳᑖᐻᐍᐦᐠ / ᓇᒨᔭ / ᒫᑲ
? ᑮ > 100 {
    >> "ᒥᐢᑕᐦᐃ ᑭᐦᒋ" ¶
} _? ᑮ > 0 {
    >> "ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ" ¶
} _? ᑮ == 0 {
    >> "ᓲᐢᑿᐨ" ¶
} _ {
    >> "ᓇᒨᔭ ᑭᐦᒋ" ¶
}
```

ᒥᐦᑴᐦᐱᑌᐏᓇ `{ }` **ᑭᐦᒋ-ᐃᑑᑕᒥᐦᐠ** — ᒫᑲ ᐯᔭᐠ ᒪᓯᓇᐦᐃᑲᓇᐢ.

## ᑳᑮᐢᐱᐦᑕᐦᐠ

```zymbol
// ᑳᑮᐢᐱᐦᑕᐦᐠ ᑳᐃᓯᐊᑭᐦᑖᐦᐠ
ᑎᐸᐦᐃᑫᐏᐣ = 85
ᐑᐦᑕᒨᐏᐣ = ?? ᑎᐸᐦᐃᑫᐏᐣ {
    90..100 : 'A'
    80..89  : 'B'
    70..79  : 'C'
    _       : 'F'
}
>> ᐑᐦᑕᒨᐏᐣ ¶    // → B

// ᑳᑮᐢᐱᐦᑕᐦᐠ ᑳᐃᓯᑳᑮᓯᒥᐦᐠ (ᑲᐦᑭᔭᐤ ᑖᐻᐍᐏᓇ)
ᑭᓯᑳᐤ = -5
ᐃᐢᐸᔨᐏᐣ = ?? ᑭᓯᑳᐤ {
    _? ᑭᓯᑳᐤ < 0  : "ᑰᓇ"
    _? ᑭᓯᑳᐤ < 20 : "ᑮᐢᐱᐣ"
    _? ᑭᓯᑳᐤ < 35 : "ᑮᓯᑳᐤ"
    _              : "ᑭᓯᑳᐤ-ᒥᐦᑵᐤ"
}
>> ᐃᐢᐸᔨᐏᐣ ¶    // → ᑰᓇ

// ᑳᑮᐢᐱᐦᑕᐦᐠ ᒪᓯᓇᐦᐃᑲᓇᐢ
ᐙᐱᑭᐘᐣ = "ᒥᐦᑯ"
ᑮᓯᑳᐤ = ?? ᐙᐱᑭᐘᐣ {
    "ᒥᐦᑯ"    : "#FF0000"
    "ᐊᐢᑭᐦᑕᐤ" : "#00FF00"
    _          : "#000000"
}
>> ᑮᓯᑳᐤ ¶
```

## ᑳᐃᐢᐸᔨᐠ

```zymbol
// ᑳᐃᓯᐊᑭᐦᑖᐦᐠ ᒫᒪᐏ: 0..4 ᐃᓯᓈᑿᐣ 0,1,2,3,4
@ ᐁ:0..4 { >> ᐁ " " }
>> ¶    // → 0 1 2 3 4

// ᑳᐃᓯᐊᑭᐦᑖᐦᐠ ᑳᑫᔮᐱᐨ
@ ᐁ:1..9:2 { >> ᐁ " " }
>> ¶    // → 1 3 5 7 9

// ᑳᑮᐍᐦᐠ ᐊᑭᐦᑖᓱᓇ
@ ᐁ:5..0:1 { >> ᐁ " " }
>> ¶    // → 5 4 3 2 1 0

// ᑳᑖᐻᐍᐦᐠ (while)
ᓃ = 1
@ ᓃ <= 64 { ᓃ *= 2 }
>> ᓃ ¶    // → 128

// ᑲᐦᑭᔭᐤ ᑮᑿᑾᔭ
ᒦᓂᓴ = ["ᐸᐦᑵᓯᑲᐣ", "ᒪᐢᑫᑯᒥᓇᐠ", "ᓵᓵᑭᓯᐤ"]
@ ᒦ:ᒦᓂᓴ { >> ᒦ ¶ }

// ᒪᓯᓇᐦᐃᑲᓇᐢ ᑳᐃᓯᐲᑭᐢᑵᐟ
@ ᓭ:"ᑖᓂᓯ" { >> ᓭ "-" }
>> ¶    // → ᑖ-ᓂ-ᓯ-

// @! ᒦᓇ @>
@ ᐁ:1..10 {
    ? ᐁ % 2 == 0 { @> }    // @> ᑳᑮᐍᐦᐠ
    ? ᐁ > 7 { @! }           // @! ᑳᓃᐹᐦᐊᐦᐠ
    >> ᐁ " "
}
>> ¶    // → 1 3 5 7
```

## ᐋᐸᒋᐦᐊᐍᐏᓇ

```zymbol
// ᐑᐦᑕᒨᐏᐣ ᒦᓇ ᐋᐸᒋᐦᐊᐏᐣ
ᒫᒪᐏᐦᐅᐦᐟ(ᑮ, ᓃ) { <~ ᑮ + ᓃ }
>> ᒫᒪᐏᐦᐅᐦᐟ(3, 4) ¶    // → 7

// ᑮᐦᑐᐤ ᑳᓈᑕᐏᐦᐅᐦᐟ
ᐊᑭᐦᑖᓱᐙᐸᒋᐦᐃᑫᐏᐣ(ᓃ) {
    ? ᓃ <= 1 { <~ 1 }
    <~ ᓃ * ᐊᑭᐦᑖᓱᐙᐸᒋᐦᐃᑫᐏᐣ(ᓃ - 1)
}
>> ᐊᑭᐦᑖᓱᐙᐸᒋᐦᐃᑫᐏᐣ(5) ¶    // → 120

// ᐋᐸᒋᐦᐊᐍᐏᓇ ᐯᔭᑲᐣ ᑳᐃᑕᐦᐱᑌᐘᐠ — ᓇᒨᔭ ᑮᑿᔾ ᐲᑐᐢ ᓈᑕᐏᐦᐅᐦᑌᐏᐣ
ᑭᐦᒋ = 100
ᑮᐢᑭᓄᑕᐦᐙᐏᐣ() {
    ᑮ = 42    // ᐱᑯ ᐆᑕ
    <~ ᑮ
}
>> ᑮᐢᑭᓄᑕᐦᐙᐏᐣ() ¶    // → 42
```

> **ᑭᐢᑌᔨᒨᐏᐣ**: ᐑᐦᒋᑳᑌᐤ ᐋᐸᒋᐦᐊᐍᐏᓇ `ᐃᓯᔨᐦᑳᓱᐏᐣ(ᐊᑭᐦᑖᓱᓇ){ }` ᓇᒨᔭ ᐯᔭᐠ ᑮᑿᔾ.
> ᑳᐱᒥᐸᔨᐦᒋᑫᐦᐠ: `ᑮ -> ᐃᓯᔨᐦᑳᓱᐏᐣ(ᑮ)`.

## ᓚᒻᑕ ᒦᓇ ᑳᑮᐍᐦᐋᐟ

```zymbol
// ᐋᐸᒋᐦᐊᐏᐣ ᓚᒻᑕ (ᓇᒨᔭ ᐱᐦᑌᐏᐣ ᐑᐦᑕᒫᑫᐏᐣ)
ᐯᔭᑯᓯᐟ = ᑮ -> ᑮ * 2
ᒫᒪᐏ = (ᑮ, ᓃ) -> ᑮ + ᓃ
>> ᐯᔭᑯᓯᐟ(5) ¶    // → 10
>> ᒫᒪᐏ(3, 7) ¶    // → 10

// ᓚᒻᑕ ᒥᐦᑴᐦᐱᑌᐏᐣ (ᑭᐦᒋ-ᐑᐦᑕᒨᐏᐣ)
ᑭᐢᑌᔨᒨᐏᐣ = ᑮ -> {
    ? ᑮ > 0 { <~ "ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ" }
    _? ᑮ < 0 { <~ "ᓇᒨᔭ ᑭᐦᒋ" }
    <~ "ᓲᐢᑿᐨ"
}
>> ᑭᐢᑌᔨᒨᐏᐣ(5) ¶     // → ᑭᐦᒋ-ᐊᑭᐦᑖᓱᐣ
>> ᑭᐢᑌᔨᒨᐏᐣ(0) ¶     // → ᓲᐢᑿᐨ
>> ᑭᐢᑌᔨᒨᐏᐣ(-5) ¶    // → ᓇᒨᔭ ᑭᐦᒋ

// ᑳᑮᐍᐦᐋᑕᐠ — ᓚᒻᑕ ᑳᑮᐑᐦᑕᒥᐦᐠ ᐲᑐᐢ ᑮᑿᑾᔭ
ᑎᐸᐦᐃᑫᐏᐣ = 3
ᓂᐢᑐᒫᐸᒋᐦᐃᑫᐏᐣ = ᑮ -> ᑮ * ᑎᐸᐦᐃᑫᐏᐣ    // ᑳᑮᐑᐦᑕᒥᐦᐠ 'ᑎᐸᐦᐃᑫᐏᐣ'
>> ᓂᐢᑐᒫᐸᒋᐦᐃᑫᐏᐣ(7) ¶    // → 21

// ᐋᐸᒋᐦᐊᐍᐏᐣ ᑳᐅᓯᐦᑖᐦᐠ
ᐅᓯᐦᑖᑫ(ᓃ) { <~ ᑮ -> ᑮ + ᓃ }
ᒥᑕᐦᑌᐤᑮ = ᐅᓯᐦᑖᑫ(10)
>> ᒥᑕᐦᑌᐤᑮ(5) ¶    // → 15

// ᓚᒻᑕ ᑮᑿᑾᔭ: ᒪᓯᓇᐦᐃᑲᓇᐢᐦᐠ
ᐋᐸᒋᐦᐊᐍᐏᓇ = [ᑮ -> ᑮ+1, ᑮ -> ᑮ*2, ᑮ -> ᑮ*ᑮ]
>> ᐋᐸᒋᐦᐊᐍᐏᓇ[0](5) ¶    // → 6
>> ᐋᐸᒋᐦᐊᐍᐏᓇ[2](5) ¶    // → 25
```

## ᒪᓯᓇᐦᐃᑲᓇᐊ

```zymbol
ᒪᓯᓇᐦᐃᑲᓇ = [10, 20, 30, 40, 50]

// ᐋᐸᒋᐦᐊᐏᐣ (ᓲᐢᑿᐨ ᐊᑭᐦᑖᓱᐣ)
>> ᒪᓯᓇᐦᐃᑲᓇ[0] ¶    // → 10

// ᑎᐸᐦᐃᑫᐏᐣ (ᐙᐏᐢ ( ) ᑳᓈᑕᐏᐦᐅᐦᑕᐦᐠ >>)
ᓃ = ᒪᓯᓇᐦᐃᑲᓇ$#
>> (ᒪᓯᓇᐦᐃᑲᓇ$#) ¶    // → 5

// $+ ᐑᐦᑕᒨᐏᐣ, $- ᑳᐙᐦᐸᒥᐦᐠ, $? ᑳᐃᑐᐦᑕᑕᐦᐠ, $[..] ᑳᑮᐢᑭᐦᐠ
ᒪᓯᓇᐦᐃᑲᓇ = ᒪᓯᓇᐦᐃᑲᓇ$+ 60               // $+ ᐑᐦᑕᒨᐏᐣ
ᒪᓯᓇᐦᐃᑲᓇ = ᒪᓯᓇᐦᐃᑲᓇ$- 0                // ᐊᑭᐦᑖᓱᐣ 0 ᑳᐑᐦᑕᒥᐦᐠ
ᐊᔮᐏᐣ = ᒪᓯᓇᐦᐃᑲᓇ$? 30           // → #1
```
