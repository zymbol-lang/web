# Zymbol-Lang ᎧᏃᎮᎵ ᎦᏅᎯᏓ

**Zymbol-Lang** ᏣᎳᎩ ᎦᏬᏂᎯᏍᏗ ᏄᎾᎵᏍᏔᏅ ᏚᎾᏓᏢ. ᎯᏍᏗᏉ ᎠᏂᏃᎮᎵᏍᎨ — ᏄᎾᎵᏍᏔᏅ ᏣᎳᎩ ᎦᏬᏂᎯᏍᏗ. ᏣᎳᎩ ᎦᏬᏂᎯᏍᏗ ᏗᏟᎶᏍᏔᏅ.

- ᎠᏂᏃᎮᎵᏍᎨ ᎠᏂᎯᎸᏍᎩ ᎥᏝ (`if`, `while`, `return` ᎥᏝ ᎠᏂᏃᎮᎵᏍᎨ — ᏄᎾᎵᏍᏔᏅ `?`, `@`, `<~`)
- ᎤᎵᏍᎪᏂᏗᏍᎩ Unicode — ᎯᏃᎮᎵ ᎤᏃᎮᎵᏍᏗ ᎦᏬᏂᎯᏍᏗ ᎠᎴ emoji 👋
- ᎦᏬᏂᎯᏍᏗ ᏗᏟᎶᏍᏔᏅ — ᎠᏓᏅᏖᏍᎬ ᎦᏬᏂᎯᏍᏗ ᎤᎾᏓᏢᏗ ᎤᎾᏤᎵᏱ

---

## ᏧᏂᎸᏫᏍᏓᏁᏗ ᏧᏓᎦᏅᎢ

```zymbol
ᎤᎾᏓᏢ = 10           // ᏧᏂᎸᏫᏍᏓᏁᏗ (ᎠᏓᏅᏖᏍᎬ ᎠᏍᏆᏂᎪᏍᏓᏁᏗ)
ᏓᎵᏍᏗ := 3.14159     // ᏧᏓᎦᏅᎢ (ᎠᏓᏅᏖᏍᎬ ᎠᏍᏆᏂᎪᏍᏓᏁᏗ ᎥᏝ — ᎠᎾᏙᎴᎰᏒ)
ᎯᏃᎮᎵ = "ᎠᏂᎩᏍᎩ"
ᎠᎾᏓᏟ = #1           // ᎤᎾᏤᎵ ᏅᏩᏙᏗ
👋 := "ꭣꏏᏲ"
```

```zymbol
ᎤᎾᏓᏢ = 10
ᎤᎾᏓᏢ += 5    // 15
ᎤᎾᏓᏢ -= 3    // 12
ᎤᎾᏓᏢ *= 2    // 24
ᎤᎾᏓᏢ /= 3    // 8
ᎤᎾᏓᏢ %= 3    // 2
ᎤᎾᏓᏢ ^= 2    // 4
ᎤᎾᏓᏢ++       // 5
ᎤᎾᏓᏢ--       // 4
```

## ᎠᏂᏃᎮᎵᏍᎨ ᎠᏂᎯᎸᏍᎩ

| ᎠᏂᎯᎸᏍᎩ         | ᏂᎨᏒᎾ               | ᎠᏓᏂᎸᏗ `#?` | ᎠᏂᏃᎮᎵᏍᎨ                        |
|-----------------|---------------------|--------------|----------------------------------|
| ᎠᏍᏆᏂᏗᏍᏙᏗ      | `42`, `-7`          | `###`        | 64-bit ᎠᏂᎯᎸᏍᎩ                  |
| ᎤᏃᎮᎵᏍᏗ        | `3.14`, `1.5e10`    | `##.`        | ᎤᏃᎮᎵᏍᏗ ᏂᎨᏒᎾ OK               |
| ᎦᏬᏂᎯᏍᏗ        | `"ꭣꏏᏲ"`            | `##"`        | ᎤᏂᎳᏫᏍᏔᏅ: `"ꭣꏏᏲ {ᎯᏃᎮᎵ}"`     |
| ᎠᏍᏓᏩᏛᏍᏗ       | `'Ꭰ'`               | `##'`        | ᎤᏃᎮᎵᏍᏗ Unicode ᎠᏍᏓᏩᏛᏍᏗ      |
| ᏅᏩᏙᏗ           | `#1`, `#0`          | `##?`        | ᎥᏝ ᎠᏍᏆᏂᏗᏍᏙᏗ 1 ᎠᎴ 0          |
| ᏧᏂᎸᏫᏍᏓᏁᏗ      | `[1, 2, 3]`         | `##]`        | ᎤᎾᏤᎵ ᎠᏂᎯᎸᏍᎩ                  |
| ᎤᎾᏓᏢ          | `(a, b)`            | `##)`        | ᎤᎾᏓᏢ ᎯᏃᎮᎵ                     |
| ᎯᏃᎮᎵ ᎤᎾᏓᏢ    | `(x: 1, y: 2)`      | `##)`        | ᎠᏩᏛᏗ ᎯᏃᎮᎵ ᎠᎴ ᎤᎾᏓᏢ ᎯᏃᎮᎵ    |

## ᎠᏩᏛᏗ ᏧᎾᏁᎶᏗ

```zymbol
>> "ꭣꏏᏲ" ¶                        // ¶ ᎠᎴ \\ ᎤᏂᎳᏫᏍᏔᏅ ᎤᎦᏅᎯ
>> "a=" ᎤᎾᏓᏢ " b=" ᎤᏃᎮᎵᏍᏗ ¶      // ᎤᏂᎳᏫᏍᏔᏅ ᎠᏂᏃᎮᎵᏍᎨ ᎤᎾᏤᎵᏱ
>> (ᏧᏂᎸᏫᏍᏓᏁᏗ$#) ¶                // ᏧᏂᎳᏆᏍᏗ ᎠᏓᏅᏖᏍᎬ ᎯᎾᏍᏗ ᎯᎾᏍᏗ

<< ᎯᏃᎮᎵ                           // ᎥᏝ ᎠᏩᏛᏗ — ᎤᏂᎳᏫᏍᏔᏅ ᏧᏂᎸᏫᏍᏓᏁᏗ
<< "ᎢᎬᏔᏯᏍᏗ? " ᎯᏃᎮᎵ              // ᎠᏩᏛᏗ ᎠᏍᏓᏩᏛᏍᏗ
```

> `¶` ᎠᎴ `\\` ᎤᎾᏤᎵ ᎤᏂᎳᏫᏍᏔᏅ ᎤᎦᏅᎯ.

## ᎠᏍᏓᏯᏛᎯᏍᏗ

```zymbol
// ᎠᏍᏆᏂᏗᏍᏙᏗ
a = 10
b = 3
r1 = a + b    // 13     r2 = a - b    // 7
r3 = a * b    // 30     r4 = a / b    // 3  (ᎠᏍᏆᏂᏗᏍᏙᏗ ᎠᏂᏃᎮᎵᏍᎨ)
r5 = a % b    // 1      r6 = a ^ b    // 1000  (ᎤᏂᎳᏫᏍᏔᏅ)

// ᎦᏬᏂᎯᏍᏗ ᎠᏂᏃᎮᎵᏍᎨ
a == b    // #0    a <> b    // #1    a < b    // #0
a <= b    // #0   a > b     // #1    a >= b   // #1

// ᏅᏩᏙᏗ
#1 && #0    // #0
#1 || #0    // #1
!#1         // #0
```

## ᎠᏁᏍᏓᏍᏗᎭ

```zymbol
// ᏄᎾᎦᎸᏗ ᎠᏍᏓᏩᏛᏍᏗ — ᎤᎾᏤᎵᏱ ᎤᏂᎳᏫᏍᏔᏅ
ᎯᏃᎮᎵ = "ᎠᏂᎩᏍᎩ"
ᎤᎾᏓᏢ = 25

ᎠᏩᏛᏗ = "ꭣꏏᏲ ", ᎯᏃᎮᎵ, "!"               // ᎤᏂᎳᏫᏍᏔᏅ — = ᎠᎴ :=
>> "ꭣꏏᏲ " ᎯᏃᎮᎵ " ᎯᏂᎦᏛ " ᎤᎾᏓᏢ ¶       // ᎤᎾᏤᎵᏱ — >> ᎠᏩᏛᏗ
ᎠᏍᎦᏂᎨᏍᏗ = "ꭣꏏᏲ {ᎯᏃᎮᎵ}, ᎯᏂᎦᏛ {ᎤᎾᏓᏢ}"  // ᎤᏂᎳᏫᏍᏔᏅ — ᎤᎾᏤᎵᏱ ᎠᏍᎦᏂᎨᏍᏗ
```

```zymbol
s = "ꭣꏏᏲ ᎠᏂᎩᏍᎩ"
len = s$#                  // 8
sub = s$[0..4]             // "ꭣꏏᏲ "  (ᎥᏝ ᎤᏂᎳᏫᏍᏔᏅ)
has = s$? "ᎠᏂᎩᏍᎩ"          // #1
parts = "a,b,c,d" / ','    // [a, b, c, d]
rep = s$~~["Ꭰ":"Ꭱ"]        // ᎤᏂᎳᏫᏍᏔᏅ ᎠᏓᏅᏖᏍᎬ
rep1 = s$~~["Ꭰ":"Ꭱ":1]     // ᎯᏁᎸᏗ ᎢᏤ
```

> **ᎠᏂᏃᎮᎵᏍᎨ**: `+` ᎠᏍᏆᏂᏗᏍᏙᏗ ᎢᏤ. ᎦᏬᏂᎯᏍᏗ ᎯᏁᎸᏗ ᎠᏂᏃᎮᎵᏍᎨ.

## ᎠᎵᏍᏙᏗ

```zymbol
ᎤᎾᏓᏢ = 7

? ᎤᎾᏓᏢ > 0 { >> "ᎠᏂᏃᎮᎵᏍᎨ" ¶ }

? ᎤᎾᏓᏢ > 100 {
    >> "ᎤᏂᎳᏫᏍᏔᏅ" ¶
} _? ᎤᎾᏓᏢ > 0 {
    >> "ᎠᏂᏃᎮᎵᏍᎨ" ¶
} _? ᎤᎾᏓᏢ == 0 {
    >> "ᎤᏍᏗᏍᏗ" ¶
} _ {
    >> "ᎠᏂᏃᎮᎵᏍᎨ ᎥᏝ" ¶
}
```

ᎠᏍᎦᏂᎨᏍᏗ `{ }` **ᎯᏁᎸᏗ**, ᎯᏃᎮᎵ ᎤᎾᏤᎵᏱ ᎠᏍᎦᏂᎨᏍᏗ.

## Match

```zymbol
// Match ᎤᎾᏓᏢ ᎯᏃᎮᎵ
ᎯᏃᎮᎵ = 85
ᎠᏓᏅᏖᏍᎬ = ?? ᎯᏃᎮᎵ {
    90..100 : 'Ꭰ'
    80..89  : 'Ꭱ'
    70..79  : 'Ꭲ'
    _       : 'Ꮲ'
}
>> ᎠᏓᏅᏖᏍᎬ ¶    // → Ꭱ

// Match ᎦᏬᏂᎯᏍᏗ
ᎤᎾᏓᏢᏗ = "ᎠᏓᏢᏓ"
ᎠᏓᏂᎸᏗ = ?? ᎤᎾᏓᏢᏗ {
    "ᎠᏓᏢᏓ"  : "#FF0000"
    "ᎤᏂᎯᏍᏗ" : "#00FF00"
    _         : "#000000"
}

// Match ᎯᏃᎮᎵ (ᎠᏂᏃᎮᎵᏍᎨ ᎠᎵᏍᏙᏗ)
ᎤᏂᎳᏫᏍᏔᏅ = -5
ᎤᏍᏗᏍᏗ = ?? ᎤᏂᎳᏫᏍᏔᏅ {
    _? ᎤᏂᎳᏫᏍᏔᏅ < 0  : "ᎤᏍᏗᏍᏗ ᎠᏓᏢᏓ"
    _? ᎤᏂᎳᏫᏍᏔᏅ < 20 : "ᎤᏂᎳᏫᏍᏔᏅ"
    _? ᎤᏂᎳᏫᏍᏔᏅ < 35 : "ᎤᏂᎯᏍᏗ"
    _                  : "ᎤᏂᎳᏫᏍᏔᏅ ᎤᏂᎯᏍᏗ"
}
>> ᎤᏍᏗᏍᏗ ¶    // → ᎤᏍᏗᏍᏗ ᎠᏓᏢᏓ

// Match ᎠᏍᎦᏂᎨᏍᏗ (block arms)
?? n {
    0       : { >> "ᎤᏍᏗᏍᏗ" ¶ }
    _? n < 0: { >> "ᎠᏂᏃᎮᎵᏍᎨ ᎥᏝ" ¶ }
    _       : { >> "ᎠᏂᏃᎮᎵᏍᎨ" ¶ }
}
```

## ᎠᏂᎩᏍᎩ

```zymbol
@ ᎤᎾᏓᏢ:0..4  { >> ᎤᎾᏓᏢ " " }        // 0 1 2 3 4  — inclusive
@ ᎤᎾᏓᏢ:1..9:2 { >> ᎤᎾᏓᏢ " " }       // step:  1 3 5 7 9
@ ᎤᎾᏓᏢ:5..0:1 { >> ᎤᎾᏓᏢ " " }       // ᎤᏂᎳᏫᏍᏔᏅ ᎯᏃᎮᎵ: 5 4 3 2 1 0

ᎤᎾᏓᏢ = 1
@ ᎤᎾᏓᏢ <= 64 { ᎤᎾᏓᏢ *= 2 }
>> ᎤᎾᏓᏢ ¶    // → 128  (while)

ᏧᏂᎸᏫᏍᏓᏁᏗ = ["ᎠᏂᎩᏍᎩ", "ᎠᏂᏃᎮᎵᏍᎨ", "ᎤᏂᎳᏫᏍᏔᏅ"]
@ ᎯᏃᎮᎵ:ᏧᏂᎸᏫᏍᏓᏁᏗ { >> ᎯᏃᎮᎵ ¶ }

@ ᎠᏍᏓᏩᏛᏍᏗ:"ꭣꏏᏲ" { >> ᎠᏍᏓᏩᏛᏍᏗ "-" }
>> ¶

@ ᎤᎾᏓᏢ:1..10 {
    ? ᎤᎾᏓᏢ % 2 == 0 { @> }    // @> ᎤᏂᎳᏫᏍᏔᏅ
    ? ᎤᎾᏓᏢ > 7 { @! }          // @! ᎠᏍᏆᏂᏗᏍᏙᏗ
    >> ᎤᎾᏓᏢ " "
}
>> ¶    // → 1 3 5 7

// ᎠᏂᎩᏍᎩ ᎤᎩᏣᏓ
i = 0
@ {
    i++
    ? i >= 5 { @! }
    >> i " "
}
>> ¶    // → 1 2 3 4

// Labeled loop
count = 0
@ @outer {
    count++
    ? count >= 3 { @! outer }
}
>> count ¶    // → 3
```

## ᎠᏓᏅᏖᏍᎬ

```zymbol
ᎦᏁᎸᏗ(ᎯᏃᎮᎵ, ᎠᏓᏂᎸᏗ) { <~ ᎯᏃᎮᎵ + ᎠᏓᏂᎸᏗ }
>> ᎦᏁᎸᏗ(3, 4) ¶    // → 7

ᎤᏂᎳᏫᏍᏔᏅ(ᎤᎾᏓᏢ) {
    ? ᎤᎾᏓᏢ <= 1 { <~ 1 }
    <~ ᎤᎾᏓᏢ * ᎤᏂᎳᏫᏍᏔᏅ(ᎤᎾᏓᏢ - 1)
}
>> ᎤᏂᎳᏫᏍᏔᏅ(5) ¶    // → 120
```

ᎠᏓᏅᏖᏍᎬ **isolated scope** — ᎥᏝ ᎤᏂᎳᏫᏍᏔᏅ ᎠᏓᏅᏖᏍᎬ. Output `<~`:

```zymbol
swap(a<~, b<~) {
    tmp = a
    a = b
    b = tmp
}
x = 10
y = 20
swap(x, y)
>> "x=" x " y=" y ¶    // → x=20 y=10
```

> **ᎠᏂᏃᎮᎵᏍᎨ**: ᎯᏃᎮᎵ ᎠᏓᏅᏖᏍᎬ `ᎯᏃᎮᎵ(ᎯᏃᎮᎵ){ }` ᎥᏝ ᎠᏂᏃᎮᎵᏍᎨ ᎠᏂᎯᎸᏍᎩ.
> ᎤᏂᎳᏫᏍᏔᏅ ᎠᏓᏅᏖᏍᎬ: `ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ(ᎯᏃᎮᎵ)`.

## ᎳᎹᏓ ᏚᏫᏪᏒ

```zymbol
ᎤᏂᎦᎾᎸ = ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ * 2
ᏄᎾᏍᏛ = (ᎯᏃᎮᎵ, ᎠᏓᏂᎸᏗ) -> ᎯᏃᎮᎵ + ᎠᏓᏂᎸᏗ
>> ᎤᏂᎦᎾᎸ(5) ¶    // → 10
>> ᏄᎾᏍᏛ(3, 7) ¶   // → 10

// ᎳᎹᏓ ᎠᏍᎦᏂᎨᏍᏗ
ᎠᏓᏅᏖᏍᎬ = ᎯᏃᎮᎵ -> {
    ? ᎯᏃᎮᎵ > 0 { <~ "ᎠᏂᏃᎮᎵᏍᎨ" }
    _? ᎯᏃᎮᎵ < 0 { <~ "ᎠᏂᏃᎮᎵᏍᎨ ᎥᏝ" }
    <~ "ᎤᏍᏗᏍᏗ"
}

// ᏚᏫᏪᏒ — ᎳᎹᏓ ᎠᏂᏃᎮᎵᏍᎨ ᎠᏓᏅᏖᏍᎬ ᎯᏃᎮᎵ
ᎤᏂᎳᏫᏍᏔᏅ = 3
ᏄᎾᎾᏛ = ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ * ᎤᏂᎳᏫᏍᏔᏅ
>> ᏄᎾᎾᏛ(7) ¶    // → 21

// ᎠᏓᏅᏖᏍᎬ ᏗᎦᎸᏗ
ᎦᏁᎸᏗ(ᎤᎾᏓᏢ) { <~ ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ + ᎤᎾᏓᏢ }
ᎤᏂᎳᏫᏍᏔᏅ10 = ᎦᏁᎸᏗ(10)
>> ᎤᏂᎳᏫᏍᏔᏅ10(5) ¶    // → 15

// ᎳᎹᏓ ᏧᏂᎸᏫᏍᏓᏁᏗ
ᎠᏓᏅᏖᏍᎬᏍ = [ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ+1, ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ*2, ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ*ᎯᏃᎮᎵ]
>> ᎠᏓᏅᏖᏍᎬᏍ[2](5) ¶    // → 25
```

## ᏧᏂᎸᏫᏍᏓᏁᏗ

ᏧᏂᎸᏫᏍᏓᏁᏗ **ᎠᏓᏅᏖᏍᎬ ᎠᏍᏆᏂᎪᏍᏓᏁᏗ** ᎠᎴ **ᎤᎾᏤᎵᏱ ᎠᏂᎯᎸᏍᎩ** ᎤᏂᎳᏫᏍᏔᏅ ᎠᏍᏗᏉ ᎥᏝ.

```zymbol
arr = [1, 2, 3, 4, 5]

arr[0]          // 1 — ᎠᏩᏛᏗ (0-ᎤᎾᏓᏢ ᎯᏃᎮᎵ)
arr[-1]         // 5 — ᎥᏝ ᎤᎾᏓᏢ (ᎯᏁᎸᏗ)
arr$#           // 5 — ᎤᏂᎳᏫᏍᏔᏅ

arr = arr$+ 6            // append → [1,2,3,4,5,6]
arr2 = arr$+[2] 99       // insert at index 2
arr3 = arr$- 3           // remove first value
arr4 = arr$-- 3          // remove all
arr5 = arr$-[0]          // remove at index
arr6 = arr$-[1..3]       // remove range (ᎥᏝ ᎤᏂᎳᏫᏍᏔᏅ)

has = arr$? 3            // #1 — contains
pos = arr$?? 3           // [2] — all indices
sl = arr$[0..3]          // [1,2,3] — slice (ᎥᏝ ᎤᏂᎳᏫᏍᏔᏅ)
sl2 = arr$[0:3]          // [1,2,3] — count-based syntax

asc = arr$^+             // sorted ascending  (primitives)
desc = arr$^-            // sorted descending (primitives)

// ᎯᏃᎮᎵ ᎤᎾᏓᏢ — $^ ᎠᎴ comparator ᎳᎹᏓ
db = [(ᎯᏃᎮᎵ: "Carla", ᎤᎾᏓᏢ: 28), (ᎯᏃᎮᎵ: "Ana", ᎤᎾᏓᏢ: 25), (ᎯᏃᎮᎵ: "Bob", ᎤᎾᏓᏢ: 30)]
by_age  = db$^ (a, b -> a.ᎤᎾᏓᏢ < b.ᎤᎾᏓᏢ)
by_name = db$^ (a, b -> a.ᎯᏃᎮᎵ > b.ᎯᏃᎮᎵ)
>> by_age[0].ᎯᏃᎮᎵ ¶     // → Ana
>> by_name[0].ᎯᏃᎮᎵ ¶    // → Carla

// ᎠᏍᎦᏂᎨᏍᏗ ᎤᏂᎳᏫᏍᏔᏅ ᏧᏂᎸᏫᏍᏓᏁᏗ (ᏧᏂᎸᏫᏍᏓᏁᏗ ᎢᏤ)
arr[1] = 99              // ᎤᏂᎳᏫᏍᏔᏅ
arr[0] += 5              // ᎠᎴ: +=  -=  *=  /=  %=  ^=

// ᎠᏓᏅᏖᏴᏍᏙᏗ ᎤᏂᎳᏫᏍᏔᏅ — ᏄᎾᎵᏍᏔᏅ ᏧᏂᎸᏫᏍᏓᏁᏗ ᎤᏂᎳᏫᏍᏔᏅ; ᎤᎾᏤᎵᏱ ᎥᏝ ᎠᏓᏅᏖᏴᏍᏙᏗ
arr2 = arr[1]$~ 99
```

> ᎤᎾᏤᎵᏱ **ᏄᎾᎵᏍᏔᏅ ᏧᏂᎸᏫᏍᏓᏁᏗ** ᎤᏂᎳᏫᏍᏔᏅ — ᏧᏂᎸᏫᏍᏓᏁᏗ = ᏧᏂᎸᏫᏍᏓᏁᏗ$+ 4.
> ᎥᏝ ᎤᏂᎦᎾᎸ: ᎯᏁᎸᏗ ᏧᏂᎸᏫᏍᏓᏁᏗ ᎤᏂᎳᏫᏍᏔᏅ ᎠᏓᏅᏖᏍᎬ.
> `$^+` / `$^-` — primitives. ᎯᏃᎮᎵ ᎤᎾᏓᏢ — `$^` ᎠᎴ ᎳᎹᏓ.

**ᏄᎵᏍᏔᏅ ᎠᏓᏅᏖᏍᎬ** — ᏧᏂᎸᏫᏍᏓᏁᏗ ᎯᏃᎮᎵ ᏄᎾᎵᏍᏔᏅ ᏧᏂᎸᏫᏍᏓᏁᏗ ᎠᎴ ᎠᏂᎩᏍᎩ ᎠᏓᏅᏖᏍᎬ ᎩᎵ ᎠᏂᏃᎮᎵᏍᎨ ᎠᎴ ᎢᏤ:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b ᎥᏝ ᎠᏓᏅᏖᏴᏍᏙᏗ
```

```zymbol
// Nested ᏧᏂᎸᏫᏍᏓᏁᏗ
matrix = [[1,2,3],[4,5,6],[7,8,9]]
>> matrix[1][2] ¶    // → 6
```

ᎤᎾᏓᏢ **ᎥᏝ ᎠᏓᏅᏖᏴᏍᏙᏗ** ᎤᎾᏓᏢ ᏚᏫᏪᏒ ᎯᏃᎮᎵ **ᎠᏂᎯᎸᏍᎩ ᎠᏍᏆᏂᏗᏍᏙᏗ** ᎤᏂᎳᏫᏍᏔᏅ ᏂᎨᏒᎾ ᎠᏂᎯᎸᏍᎩ. ᏧᏂᎸᏫᏍᏓᏁᏗ ᎥᏝ, ᎠᏂᎯᎸᏍᎩ ᎥᏝ ᎠᏓᏅᏖᏴᏍᏙᏗ ᎯᏃᎮᎵ ᎤᎾᏓᏢ ᎠᎴ.

>> ᎠᎴᏗ[2] ¶     // → #1

// ᎯᏃᎮᎵ ᎤᎾᏓᏢ
ᎠᏂᎩᏍᎩ = (ᎯᏃᎮᎵ: "Alice", ᎤᎾᏓᏢ: 25)

**ᎯᏛᎦᏅ** — ᎤᎾᏓᏢ ᎠᏂᎯᎸᏍᎩ ᎠᎴ ᎯᏁᎸᏗ ᎤᏂᎳᏫᏍᏔᏅ ᎠᎾᏙᎴᎰᏒ runtime ᎯᏃᎮᎵ:

```zymbol

// t[0] = 99    // ❌ runtime ᎠᎾᏙᎴᎰᏒ: ᎤᎾᏓᏢ ᎥᏝ ᎠᏓᏅᏖᏴᏍᏙᏗ

ᎠᏓᏅᏖᏍᎬ ᏂᎨᏒᎾ `$~` (ᎠᏓᏅᏖᏴᏍᏙᏗ ᎤᏂᎳᏫᏍᏔᏅ) ᎤᏂᎳᏫᏍᏔᏅ — **ᏄᎾᎵᏍᏔᏅ** ᎤᎾᏓᏢ ᎤᏂᎳᏫᏍᏔᏅ:

t2 = t[1]$~ 999

>> t2 ¶    // → (10, 999, 30)

// ᎯᏃᎮᎵ ᎤᎾᏓᏢ — ᎤᎾᏓᏢ ᎠᏓᏅᏖᏍᎬ ᎤᏂᎳᏫᏍᏔᏅ

ᎠᏂᎩᏍᎩ2 = (ᎯᏃᎮᎵ: ᎠᏂᎩᏍᎩ.ᎯᏃᎮᎵ, ᎤᎾᏓᏢ: 26)

```

## ᎠᏂᏱᎨᏍᏙᏗ ᎢᏳᏍᏗ ᎦᎸᏉᏗ

Zymbol **Unicode ᎢᏳᏍᏗ 69 ᏗᏓᎴᎲᏍᎬ ᎠᏂᏤᎵ** ᏂᎨᏒᎾ ᎢᏳᏍᏗ ᏨᎵᏍᎬᎢ — Devanagari, ᎠᎴᎹᏂ-ᎢᏅᏗᏎᏍᏙᏗ, Thai, Klingon pIqaD ᏂᎨᏒᎾ ᎠᏑᏰᏒᎢ. `>>` ᎠᏏᏴᏒ ᎤᏩᏒᎢ ᏂᎨᏒᎾ ᎤᎩᎵ ᏄᎵᏍᏔᏅᎢ.

### ᎠᏑᏰᏍᎩ ᎦᏁᎶᏗ

```zymbol
#٠٩#    // Arabic-Indic (U+0660–U+0669)
#०९#    // Devanagari   (U+0966–U+096F)
#09#    // ASCII ᎤᎵᏍᏓᏅᏗ
```

### ᎠᏏᏴᏒ ᏄᎵᏍᏔᏅ ᏂᎨᏒᎾ Boolean

```zymbol
x = 42
>> x ¶          // → 42   (ASCII ᎯᎠ)

#٠٩#
>> x ¶          // → ٤٢
>> 1 + 2 ¶      // → ٣

>> #1 ¶         // → #١
>> #0 ¶         // → #٠
```

### ᎠᏑᏰᏍᎩ ᎢᏳᏍᏗ ᏰᎵᏊ ᏗᏚᎴᎰᏍᎬᎢ

```zymbol
#٠٩#

@ i:١..١٥ {
    ? i % ١٥ == ٠ { >> "FizzBuzz" ¶ }
    _? i % ٣  == ٠ { >> "Fizz" ¶ }
    _? i % ٥  == ٠ { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

> `#` **ASCII ᎠᎴ ᏂᎨᏒᎾ**. `#0` ᏂᎨᏒᎾ `0` ᎤᎵᏍᏐᏗᏍᏗ ᎤᎪᏗᏳᏒ.
