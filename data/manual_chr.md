# Zymbol-Lang ᎧᏃᎮᎵ ᎦᏅᎯᏓ

**Zymbol-Lang** ᏣᎳᎩ ᎦᏬᏂᎯᏍᏗ ᏄᎾᎵᏍᏔᏅ ᏚᎾᏓᏢ. ᎯᏍᏗᏉ ᎠᏂᏃᎮᎵᏍᎨ — ᏄᎾᎵᏍᏔᏅ ᏣᎳᎩ ᎦᏬᏂᎯᏍᏗ. ᏣᎳᎩ ᎦᏬᏂᎯᏍᏗ ᏗᏟᎶᏍᏔᏅ.

---

## ᎢᎬᏔᏯᏍᏗ

- ᎠᏂᏃᎮᎵᏍᎨ ᎠᏂᎯᎸᏍᎩ ᎥᏝ (`if`, `while`, `return` ᎥᏝ ᎠᏂᏃᎮᎵᏍᎨ — ᏄᎾᎵᏍᏔᏅ `?`, `@`, `<~`)
- ᎤᎵᏍᎪᏂᏗᏍᎩ Unicode — ᎯᏃᎮᎵ ᎤᏃᎮᎵᏍᏗ ᎦᏬᏂᎯᏍᏗ ᎠᎴ emoji 👋
- ᎦᏬᏂᎯᏍᏗ ᏗᏟᎶᏍᏔᏅ — ᎠᏓᏅᏖᏍᎬ ᎦᏬᏂᎯᏍᏗ ᎤᎾᏓᏢᏗ ᎤᎾᏤᎵᎦ

---

## ᏧᏂᎸᏫᏍᏓᏁᏗ ᏧᏓᎦᏅᎢ

```zymbol
ᎤᎾᏓᏢ = 10           // ᏧᏂᎸᏫᏍᏓᏁᏗ (ᎠᏓᏅᏖᏍᎬ ᎠᏍᏆᏂᎪᏍᏓᏁᏗ)
ᏓᎵᏍᏗ := 3.14159     // ᏧᏓᎦᏅᎢ (ᎠᏓᏅᏖᏍᎬ ᎠᏍᏆᏂᎪᏍᏓᏁᏗ ᎥᏝ — ᎠᎾᏙᎴᎰᏒ)
ᎯᏃᎮᎵ = "ᎠᏂᎩᏍᎩ"
ᎠᎾᏓᏟ = #1           // ᎤᎾᏤᎵ ᏅᏩᏙᏗ
👋 := "ꭣꏏᏲ"
```

### ᎤᏂᎳᏫᏍᏔᏅ ᏧᏂᎸᏫᏍᏓᏁᏗ

```zymbol
ᎤᎾᏓᏢ = 10    // 10
ᎤᎾᏓᏢ += 5    // 15
ᎤᎾᏓᏢ -= 3    // 12
ᎤᎾᏓᏢ *= 2    // 24
ᎤᎾᏓᏢ /= 4    // 6
ᎤᎾᏓᏢ %= 4    // 2
ᎤᎾᏓᏢ++       // 3
ᎤᎾᏓᏢ--       // 2
```

## ᎠᏂᏃᎮᎵᏍᎨ ᎠᏂᎯᎸᏍᎩ

| ᎠᏂᎯᎸᏍᎩ         | ᏂᎨᏒᎾ               | ᎠᏓᏂᎸᏗ `#?` | ᎠᏂᏃᎮᎵᏍᎨ                        |
|-----------------|---------------------|--------------|----------------------------------|
| ᎠᏍᏆᏂᏗᏍᏙᏗ      | `42`, `-7`          | `###`        | 64-bit ᎠᏂᎯᎸᏍᎩ                  |
| ᎤᏃᎮᎵᏍᏗ        | `3.14`, `1.5e10`    | `##.`        | ᎤᏃᎮᎵᏍᏗ ᏂᎨᏒᎾ OK               |
| ᎦᏬᏂᎯᏍᏗ        | `"ꭣꏏᏲ"`            | `##"`        | ᎤᏂᎳᏫᏍᏔᏅ: `"ꭣꏏᏲ {ᎯᏃᎮᎵ}"`     |
| ᎠᏍᏓᏩᏛᏍᏗ       | `'Ꭰ'`               | `##'`        | ᎤᏃᎮᎵᏍᏗ Unicode ᎠᏍᏓᏩᏛᏍᏗ      |
| ᏅᏩᏙᏗ           | `#1`, `#0`          | `##?`        | ᎥᏝ ᎠᏍᏆᏂᏗᏍᏙᏗ 1 ᎠᎴ 0          |
| ᏧᏂᎸᏫᏍᏓᏁᏗ      | `[1, 2, 3]`         | `##]`        | ᎤᎾᏤᎵ ᎠᏂᎯᎸᏍᎩ                  |
| ᎤᎾᏓᏢ          | `(a, b)`            | `##)`        | ᎤᎾᏓᏢ ᎯᏃᎮᎵ                     |
| ᎯᏃᎮᎵ ᎤᎾᏓᏢ    | `(x: 1, y: 2)`      | `##)`        | ᎠᏩᏛᏗ ᎯᏃᎮᎵ ᎠᎴ ᎤᎾᏓᏢ ᎯᏃᎮᎵ    |

## ᎠᏩᏛᏗ ᏧᎾᏁᎶᏗ

```zymbol
// ᎠᏩᏛᏗ — ᎥᏝ ᎠᎦᏛᎢ ᎤᎦᏅᎯ ᎤᏂᎳᏫᏍᏔᏅ
>> "ꭣꏏᏲ" ¶                        // ¶ ᎠᎴ \\ ᎤᏂᎳᏫᏍᏔᏅ ᎤᎦᏅᎯ
>> "a=" ᎤᎾᏓᏢ " b=" ᎤᏃᎮᎵᏍᏗ ¶      // ᎤᏂᎳᏫᏍᏔᏅ ᎠᏂᏃᎮᎵᏍᎨ ᎤᎾᏤᎵᏱ
>> "ᎤᏂᎳᏫᏍᏔᏅ=" ᎦᏁᎸᏗ(2, 3) ¶      // ᎠᏓᏅᏖᏍᎬ ᎯᏃᎮᎵ ᎤᎾᏤᎵᏱ
>> (ᏧᏂᎸᏫᏍᏓᏁᏗ$#) ¶                // ᏧᏂᎳᏆᏍᏗ ᎠᏓᏅᏖᏍᎬ ᎯᎾᏍᏗ ᎯᎾᏍᏗ

// ᏧᎾᏁᎶᏗ
<< ᎯᏃᎮᎵ                           // ᎥᏝ ᎠᏩᏛᏗ — ᎤᏂᎳᏫᏍᏔᏅ ᏧᏂᎸᏫᏍᏓᏁᏗ
<< "ᎢᎬᏔᏯᏍᏗ? " ᎯᏃᎮᎵ              // ᎠᏩᏛᏗ ᎠᏍᏓᏩᏛᏍᏗ
```

> `¶` ᎠᎴ `\\` ᎤᎾᏤᎵ ᎤᏂᎳᏫᏍᏔᏅ ᎤᎦᏅᎯ.

---

## ᎠᏂᏃᎮᎵᏍᎨ ᎦᎸᏗ

ᏄᎾᎦᎸᏗ ᎠᏍᏓᏩᏛᏍᏗ — ᎤᎾᏤᎵᏱ ᎤᏂᎳᏫᏍᏔᏅ:

```zymbol
ᎯᏃᎮᎵ = "ᎠᏂᎩᏍᎩ"
ᎤᎾᏓᏢ = 25

// 1. ᎤᏂᎳᏫᏍᏔᏅ — ᏧᏂᎸᏫᏍᏓᏁᏗ = ᎠᎴ :=
ᎠᏩᏛᏗ = "ꭣꏏᏲ ", ᎯᏃᎮᎵ, "!"               // → ꭣꏏᏲ ᎠᏂᎩᏍᎩ!
ᎧᏃᎮᎵ := "ᎠᏂᎩᏍᎩ: ", ᎯᏃᎮᎵ

// 2. ᎤᎾᏤᎵᏱ — >> ᎠᏩᏛᏗ
>> "ꭣꏏᏲ " ᎯᏃᎮᎵ " ᎯᏂᎦᏛ " ᎤᎾᏓᏢ ¶       // → ꭣꏏᏲ ᎠᏂᎩᏍᎩ ᎯᏂᎦᏛ 25

// 3. ᎤᏂᎳᏫᏍᏔᏅ — ᎤᎾᏤᎵᏱ ᎠᏍᎦᏂᎨᏍᏗ
ᎠᏍᎦᏂᎨᏍᏗ = "ꭣꏏᏲ {ᎯᏃᎮᎵ}, ᎯᏂᎦᏛ {ᎤᎾᏓᏢ}"  // → ꭣꏏᏲ ᎠᏂᎩᏍᎩ, ᎯᏂᎦᏛ 25
```

> **ᎠᏂᏃᎮᎵᏍᎨ**: `+` ᎠᏍᏆᏂᏗᏍᏙᏗ ᎢᏤ. ᎦᏬᏂᎯᏍᏗ ᎯᏁᎸᏗ ᎠᏂᏃᎮᎵᏍᎨ.

## ᎠᎵᏍᏙᏗ

```zymbol
ᎤᎾᏓᏢ = 7

// ᎯᏁᎸᏗ ᏗᎦᎸᏗ
? ᎤᎾᏓᏢ > 0 { >> "ᎠᏂᏃᎮᎵᏍᎨ" ¶ }

// ᎯᏁᎸᏗ / ᎯᏁᎸᏗ ᎯᏃᎮᎵ / ᎯᏁᎸᏗ ᎥᏝ
? ᎤᎾᏓᏢ > 100 {
    >> "ᎤᏂᎳᏫᏍᏔᏅ" ¶
} _? ᎤᎾᏓᏢ > 0 {
    >> "ᎠᏂᏃᎮᎵᏍᎨ" ¶
} _? ᎤᎾᏓᏢ == 0 {
    >> "ᎤᏍᏗᏍᏗ" ¶
} _ {
    >> "ᎠᏂᏃᎮᎵᏍᎨ ᎥᏝ" ¶
}
```

ᎠᏍᎦᏂᎨᏍᏗ `{ }` **ᎯᏁᎸᏗ**, ᎯᏃᎮᎵ ᎤᎾᏤᎵᏱ ᎠᏍᎦᏂᎨᏍᏗ.

## Match

```zymbol
// Match ᎤᎾᏓᏢ ᎯᏃᎮᎵ
ᎯᏃᎮᎵ = 85
ᎠᏓᏅᏖᏍᎬ = ?? ᎯᏃᎮᎵ {
    90..100 : 'Ꭰ'
    80..89  : 'Ꭱ'
    70..79  : 'Ꭲ'
    _       : 'Ꮲ'
}
>> ᎠᏓᏅᏖᏍᎬ ¶    // → Ꭱ

// Match ᎯᏃᎮᎵ (ᎠᏂᏃᎮᎵᏍᎨ ᎠᎵᏍᏙᏗ)
ᎤᏂᎳᏫᏍᏔᏅ = -5
ᎤᏍᏗᏍᏗ = ?? ᎤᏂᎳᏫᏍᏔᏅ {
    _? ᎤᏂᎳᏫᏍᏔᏅ < 0  : "ᎤᏍᏗᏍᏗ ᎠᏓᏢᏓ"
    _? ᎤᏂᎳᏫᏍᏔᏅ < 20 : "ᎤᏂᎳᏫᏍᏔᏅ"
    _? ᎤᏂᎳᏫᏍᏔᏅ < 35 : "ᎤᏂᎯᏍᏗ"
    _                  : "ᎤᏂᎳᏫᏍᏔᏅ ᎤᏂᎯᏍᏗ"
}
>> ᎤᏍᏗᏍᏗ ¶    // → ᎤᏍᏗᏍᏗ ᎠᏓᏢᏓ

// Match ᎦᏬᏂᎯᏍᏗ
ᎤᎾᏓᏢᏗ = "ᎠᏓᏢᏓ"
ᎠᏓᏂᎸᏗ = ?? ᎤᎾᏓᏢᏗ {
    "ᎠᏓᏢᏓ"  : "#FF0000"
    "ᎤᏂᎯᏍᏗ" : "#00FF00"
    _         : "#000000"
}
>> ᎠᏓᏂᎸᏗ ¶
```

## ᎠᏂᎩᏍᎩ

```zymbol
// ᎤᏂᎳᏫᏍᏔᏅ ᎯᏃᎮᎵ: 0..4 ᎤᏂᎳᏫᏍᏔᏅ 0,1,2,3,4
@ ᎤᎾᏓᏢ:0..4 { >> ᎤᎾᏓᏢ " " }
>> ¶    // → 0 1 2 3 4

// ᎯᏃᎮᎵ ᎤᏂᎳᏫᏍᏔᏅ ᎠᏍᏆᏂᏗᏍᏙᏗ
@ ᎤᎾᏓᏢ:1..9:2 { >> ᎤᎾᏓᏢ " " }
>> ¶    // → 1 3 5 7 9

// ᎤᏂᎳᏫᏍᏔᏅ ᎯᏃᎮᎵ ᎤᏍᏗᏍᏗ
@ ᎤᎾᏓᏢ:5..0:1 { >> ᎤᎾᏓᏢ " " }
>> ¶    // → 5 4 3 2 1 0

// ᎠᏂᎩᏍᎩ (while)
ᎤᎾᏓᏢ = 1
@ ᎤᎾᏓᏢ <= 64 { ᎤᎾᏓᏢ *= 2 }
>> ᎤᎾᏓᏢ ¶    // → 128

// ᎤᎾᏤᎵᏱ ᎠᏂᎯᎸᏍᎩ
ᏧᏂᎸᏫᏍᏓᏁᏗ = ["ᎠᏂᎩᏍᎩ", "ᎠᏂᏃᎮᎵᏍᎨ", "ᎤᏂᎳᏫᏍᏔᏅ"]
@ ᎯᏃᎮᎵ:ᏧᏂᎸᏫᏍᏓᏁᏗ { >> ᎯᏃᎮᎵ ¶ }

// ᎦᏬᏂᎯᏍᏗ ᎠᏍᏓᏩᏛᏍᏗ ᏚᏂᎸᏗ
@ ᎠᏍᏓᏩᏛᏍᏗ:"ꭣꏏᏲ" { >> ᎠᏍᏓᏩᏛᏍᏗ "-" }
>> ¶    // → ꭣ-Ꭷ-Ꮝ-Ꮒ-Ꮿ-

// ᎠᏍᏆᏂᏗᏍᏙᏗ ᎠᎴ ᎤᏂᎳᏫᏍᏔᏅ
@ ᎤᎾᏓᏢ:1..10 {
    ? ᎤᎾᏓᏢ % 2 == 0 { @> }    // @> ᎤᏂᎳᏫᏍᏔᏅ
    ? ᎤᎾᏓᏢ > 7 { @! }          // @! ᎠᏍᏆᏂᏗᏍᏙᏗ
    >> ᎤᎾᏓᏢ " "
}
>> ¶    // → 1 3 5 7
```

## ᎠᏓᏅᏖᏍᎬ

```zymbol
// ᎤᏂᎳᏫᏍᏔᏅ ᎠᎴ ᎠᏓᏅᏖᏍᎬ
ᎦᏁᎸᏗ(ᎯᏃᎮᎵ, ᎠᏓᏂᎸᏗ) { <~ ᎯᏃᎮᎵ + ᎠᏓᏂᎸᏗ }
>> ᎦᏁᎸᏗ(3, 4) ¶    // → 7

// ᎤᏂᎳᏫᏍᏔᏅ ᎤᏂᎳᏫᏍᏔᏅ
ᎤᏂᎳᏫᏍᏔᏅ(ᎤᎾᏓᏢ) {
    ? ᎤᎾᏓᏢ <= 1 { <~ 1 }
    <~ ᎤᎾᏓᏢ * ᎤᏂᎳᏫᏍᏔᏅ(ᎤᎾᏓᏢ - 1)
}
>> ᎤᏂᎳᏫᏍᏔᏅ(5) ¶    // → 120

// ᎠᏓᏅᏖᏍᎬ ᎤᏃᎮᎵᏍᏗ ᎤᏂᎳᏫᏍᏔᏅ — ᎥᏝ ᎤᏂᎳᏫᏍᏔᏅ ᎠᏓᏅᏖᏍᎬ
ᎤᎾᏓᏢᏗ = 100
ᎯᏃᎮᎵ() {
    ᎤᎾᏓᏢ = 42    // ᎤᏃᎮᎵᏍᏗ ᎢᏤ
    <~ ᎤᎾᏓᏢ
}
>> ᎯᏃᎮᎵ() ¶    // → 42
```

> **ᎠᏂᏃᎮᎵᏍᎨ**: ᎯᏃᎮᎵ ᎠᏓᏅᏖᏍᎬ `ᎯᏃᎮᎵ(ᎯᏃᎮᎵ){ }` ᎥᏝ ᎠᏂᏃᎮᎵᏍᎨ ᎠᏂᎯᎸᏍᎩ.
> ᎤᏂᎳᏫᏍᏔᏅ ᎠᏓᏅᏖᏍᎬ: `ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ(ᎯᏃᎮᎵ)`.

## ᎳᎹᏓ ᏚᏫᏪᏒ

```zymbol
// ᎯᏁᎸᏗ ᎳᎹᏓ (ᎤᏂᎳᏫᏍᏔᏅ <~)
ᎤᏂᎦᎾᎸ = ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ * 2
ᏄᎾᏍᏛ = (ᎯᏃᎮᎵ, ᎠᏓᏂᎸᏗ) -> ᎯᏃᎮᎵ + ᎠᏓᏂᎸᏗ
>> ᎤᏂᎦᎾᎸ(5) ¶    // → 10
>> ᏄᎾᏍᏛ(3, 7) ¶   // → 10

// ᎳᎹᏓ ᎠᏍᎦᏂᎨᏍᏗ (ᎤᏂᎳᏫᏍᏔᏅ <~)
ᎠᏓᏅᏖᏍᎬ = ᎯᏃᎮᎵ -> {
    ? ᎯᏃᎮᎵ > 0 { <~ "ᎠᏂᏃᎮᎵᏍᎨ" }
    _? ᎯᏃᎮᎵ < 0 { <~ "ᎠᏂᏃᎮᎵᏍᎨ ᎥᏝ" }
    <~ "ᎤᏍᏗᏍᏗ"
}
>> ᎠᏓᏅᏖᏍᎬ(5) ¶     // → ᎠᏂᏃᎮᎵᏍᎨ
>> ᎠᏓᏅᏖᏍᎬ(0) ¶     // → ᎤᏍᏗᏍᏗ
>> ᎠᏓᏅᏖᏍᎬ(-5) ¶    // → ᎠᏂᏃᎮᎵᏍᎨ ᎥᏝ

// ᏚᏫᏪᏒs — ᎳᎹᏓ ᎠᏂᏃᎮᎵᏍᎨ ᎠᏓᏅᏖᏍᎬ ᎯᏃᎮᎵ
ᎤᏂᎳᏫᏍᏔᏅ = 3
ᏄᎾᎾᏛ = ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ * ᎤᏂᎳᏫᏍᏔᏅ    // ᎠᏂᏃᎮᎵᏍᎨ 'ᎤᏂᎳᏫᏍᏔᏅ'
>> ᏄᎾᎾᏛ(7) ¶    // → 21

// ᎠᏓᏅᏖᏍᎬ ᏗᎦᎸᏗ
ᎦᏁᎸᏗ(ᎤᎾᏓᏢ) { <~ ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ + ᎤᎾᏓᏢ }
ᎤᏂᎳᏫᏍᏔᏅ10 = ᎦᏁᎸᏗ(10)
>> ᎤᏂᎳᏫᏍᏔᏅ10(5) ¶    // → 15

// ᎳᎹᏓ ᎠᏍᏓᏩᏛᏍᏗ ᏧᏂᎸᏫᏍᏓᏁᏗ
ᎠᏓᏅᏖᏍᎬᏍ = [ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ+1, ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ*2, ᎯᏃᎮᎵ -> ᎯᏃᎮᎵ*ᎯᏃᎮᎵ]
>> ᎠᏓᏅᏖᏍᎬᏍ[0](5) ¶    // → 6
>> ᎠᏓᏅᏖᏍᎬᏍ[2](5) ¶    // → 25
```

## ᏧᏂᎸᏫᏍᏓᏁᏗ

```zymbol
ᏧᏂᎸᏫᏍᏓᏁᏗ = [10, 20, 30, 40, 50]

// ᎠᏩᏛᏗ (0-ᎤᎾᏓᏢ ᎯᏃᎮᎵ)
>> ᏧᏂᎸᏫᏍᏓᏁᏗ[0] ¶    // → 10

// ᎤᏂᎳᏫᏍᏔᏅ (ᎯᎾᏍᏗ >> ᎯᎾᏍᏗ)
ᎤᎾᏓᏢ = ᏧᏂᎸᏫᏍᏓᏁᏗ$#
>> (ᏧᏂᎸᏫᏍᏓᏁᏗ$#) ¶    // → 5
```
