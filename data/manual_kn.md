# Zymbol-Lang ನ ಸಂಕ್ಷಿಪ್ತ ಮಾರ್ಗದರ್ಶಿ

**Zymbol-Lang** ಒಂದು ಸಾಂಕೇತಿಕ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷೆ. ಇದರಲ್ಲಿ ಯಾವುದೇ ಕೀವರ್ಡ್‌ಗಳಿಲ್ಲ — ಎಲ್ಲವೂ ಒಂದು ಚಿಹ್ನೆ. ಇದು ಯಾವುದೇ ಮಾನವ ಭಾಷೆಯಲ್ಲಿ ಒಂದೇ ರೀತಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

- ಯಾವುದೇ ಕೀವರ್ಡ್‌ಗಳಿಲ್ಲ (`if`, `while`, `return` ಇಲ್ಲ — ಕೇವಲ ಚಿಹ್ನೆಗಳು `?`, `@`, `<~`)
- ಸಂಪೂರ್ಣ Unicode — ಯಾವುದೇ ಭಾಷೆ ಅಥವಾ ಇಮೋಜಿಯಲ್ಲಿ ಗುರುತಿಸಿ 👋
- ಮಾನವ-ಭಾಷೆ ತಟಸ್ಥ — ಕೋಡ್ ಎಲ್ಲ ಭಾಷೆಗಳಲ್ಲಿ ಒಂದೇ ಆಗಿರುತ್ತದೆ

## ಅಸ್ಥಿರಗಳು ಮತ್ತು ಸ್ಥಿರಾಂಕಗಳು

```zymbol
ಚ = ೧೦              // ಅಸ್ಥಿರ (ಬದಲಾಯಿಸಬಹುದು)
ಪೈ := ೩.೧೪೧೫೯       // ಸ್ಥಿರಾಂಕ — ಮತ್ತೆ ನಿಯೋಜಿಸಿದರೆ ರನ್‌ಟೈಮ್ ದೋಷ
ಹೆಸರು = "ಅನು"
ಸಕ್ರಿಯ = #೧        // ಬೂಲಿಯನ್ ಸತ್ಯ
👋 := "ನಮಸ್ಕಾರ"
```

```zymbol
ಚ = ೧೦
ಚ += ೫    // ೧೫
ಚ -= ೩    // ೧೨
ಚ *= ೨    // ೨೪
ಚ /= ೩    // ೮
ಚ %= ೩    // ೨
ಚ ^= ೨    // ೪
ಚ++       // ೫
ಚ--       // ೪
```

## ಡೇಟಾ ಪ್ರಕಾರಗಳು

| ಪ್ರಕಾರ | ಉದಾಹರಣೆ | `#?` ಚಿಹ್ನೆ | ಟಿಪ್ಪಣಿ |
|--------|---------|-------------|---------|
| ಪೂರ್ಣಾಂಕ (Int) | `೪೨`, `-೭` | `###` | 64-bit ಸಹಿ ಮಾಡಿದ |
| ದಶಮಾಂಶ (Float) | `೩.೧೪`, `೧.೫e೧೦` | `##.` | ವೈಜ್ಞಾನಿಕ ಸಂಕೇತನ ಬೆಂಬಲಿತ |
| ಸ್ಟ್ರಿಂಗ್ (String) | `"ನಮಸ್ಕಾರ"` | `##"` | ಇಂಟರ್‌ಪೋಲೇಶನ್: `"ನಮಸ್ಕಾರ {ಹೆಸರು}"` |
| ಅಕ್ಷರ (Char) | `'ಕ'` | `##'` | ಒಂದು Unicode ಅಕ್ಷರ |
| ಬೂಲಿಯನ್ (Bool) | `#೧`, `#೦` | `##?` | ಸಂಖ್ಯಾ ಅಲ್ಲ — `#೧ ≠ ೧` |
| ಪಟ್ಟಿ (Array) | `[೧, ೨, ೩]` | `##]` | ಸಜಾತೀಯ ಅಂಶಗಳು |
| ಟಪಲ್ (Tuple) | `(ಅ, ಬ)` | `##)` | ಸ್ಥಾನ ಆಧಾರಿತ |
| ಹೆಸರಿನ ಟಪಲ್ | `(x: ೧, y: ೨)` | `##)` | ಹೆಸರಿನ ಕ್ಷೇತ್ರಗಳು |

```zymbol
// ಪ್ರಕಾರ ಅಂತರ್ದರ್ಶನ — (ಪ್ರಕಾರ, ಅಂಕೆಗಳು, ಮೌಲ್ಯ) ಹಿಂತಿರುಗಿಸುತ್ತದೆ
meta = ೪೨#?
>> meta ¶         // → (###, ೨, ೪೨)
t = meta[೦]
>> t ¶            // → ###
```

## ಔಟ್‌ಪುಟ್ ಮತ್ತು ಇನ್‌ಪುಟ್

```zymbol
>> "ನಮಸ್ಕಾರ" ¶                     // ¶ ಅಥವಾ \\ ಸ್ಪಷ್ಟ ಹೊಸ ಸಾಲು
>> "ಅ=" ಅ " ಬ=" ಬ ¶               // juxtaposition — ಹಲವು ಮೌಲ್ಯಗಳು
>> (ಪಟ್ಟಿ$#) ¶                    // ಪೋಸ್ಟ್‌ಫಿಕ್ಸ್ ಆಪರೇಟರ್‌ಗೆ >> ನಲ್ಲಿ ( ) ಬೇಕು

<< ಹೆಸರು                           // ಪ್ರಾಂಪ್ಟ್ ಇಲ್ಲದೆ ಓದುತ್ತದೆ
<< "ನಿಮ್ಮ ಹೆಸರು? " ಹೆಸರು           // ಪ್ರಾಂಪ್ಟ್ ಜೊತೆ
```

> `¶` (ಸ್ಪ್ಯಾನಿಷ್ ಕೀಬೋರ್ಡ್‌ನಲ್ಲಿ AltGr+R) ಮತ್ತು `\\` ಸಮಾನ ಹೊಸ ಸಾಲುಗಳು.

## ಸ್ಟ್ರಿಂಗ್‌ಗಳು

```zymbol
// ಮೂರು ಅನುಸಂಧಾನ ರೂಪಗಳು
ಹೆಸರು = "ಅನು"
ನ = ೨೫

ಸಂದೇಶ = "ನಮಸ್ಕಾರ ", ಹೆಸರು, "!"          // ಅಲ್ಪವಿರಾಮ — ನಿಯೋಜನೆಯಲ್ಲಿ
>> "ನಮಸ್ಕಾರ " ಹೆಸರು " ನಿಮ್ಮ ವಯಸ್ಸು " ನ ¶  // juxtaposition — >> ನಲ್ಲಿ
ವಿವರಣೆ = "ನಮಸ್ಕಾರ {ಹೆಸರು}, ವಯಸ್ಸು {ನ}"  // ಇಂಟರ್‌ಪೋಲೇಶನ್ — ಎಲ್ಲೆಡೆ
```

```zymbol
ಸ = "ನಮಸ್ಕಾರ ಜಗತ್ತು"
ಉ = ಸ$#                  // length
ಉ೨ = ಸ$[೦..೫]            // slice
ಇ = ಸ$? "ಜಗ"             // #೧
ಭಾ = "ಅ,ಆ,ಇ,ಈ" / ','     // [ಅ, ಆ, ಇ, ಈ]
ಬ = ಸ$~~["ಕ":"ಕ್"]         // replace
ಬ೧ = ಸ$~~["ಕ":"ಕ್":೧]     // first N
```

> `+` ಕೇವಲ ಸಂಖ್ಯೆಗಳಿಗೆ. ಸ್ಟ್ರಿಂಗ್‌ಗಳಿಗೆ `,`, juxtaposition ಅಥವಾ interpolation ಬಳಸಿ.

## ಹರಿವು ನಿಯಂತ್ರಣ

```zymbol
ಚ = ೭

? ಚ > ೦ { >> "ಧನಾತ್ಮಕ" ¶ }

? ಚ > ೧೦೦ {
    >> "ದೊಡ್ಡದು" ¶
} _? ಚ > ೦ {
    >> "ಧನಾತ್ಮಕ" ¶
} _? ಚ == ೦ {
    >> "ಶೂನ್ಯ" ¶
} _ {
    >> "ಋಣಾತ್ಮಕ" ¶
}
```

> `{ }` ಬ್ಲಾಕ್‌ಗಳು **ಕಡ್ಡಾಯ**, ಒಂದು ಸಾಲಿಗೂ ಸಹ.

## ಲೂಪ್‌ಗಳು

```zymbol
@ ಇ:೦..೪  { >> ಇ " " }        // ಶ್ರೇಣಿ ಸಮ್ಮಿಳಿತ:  ೦ ೧ ೨ ೩ ೪
@ ಇ:೧..೯:೨ { >> ಇ " " }       // ಹಂತದೊಂದಿಗೆ:       ೧ ೩ ೫ ೭ ೯
@ ಇ:೫..೦:೧ { >> ಇ " " }       // ವಿಲೋಮ:            ೫ ೪ ೩ ೨ ೧ ೦

ನ = ೧
@ ನ <= ೬೪ { ನ *= ೨ }
>> ನ ¶                        // → ೧೨೮  (while)

ಹಣ್ಣುಗಳು = ["ಮಾವು", "ಬಾಳೆ", "ದ್ರಾಕ್ಷಿ"]
@ ಹ:ಹಣ್ಣುಗಳು { >> ಹ ¶ }       // ಪಟ್ಟಿಯ ಮೇಲೆ for-each

@ ಅ:"hello" { >> ಅ "-" }
>> ¶                          // → h-e-l-l-o-  (ಸ್ಟ್ರಿಂಗ್ ಅಕ್ಷರಗಳ ಮೇಲೆ)

@ ಇ:೧..೧೦ {
    ? ಇ % ೨ == ೦ { @> }       // @> ಮುಂದುವರಿಸು
    ? ಇ > ೭ { @! }             // @! ನಿಲ್ಲಿಸು
    >> ಇ " "
}
>> ¶                          // → ೧ ೩ ೫ ೭

// ಅನಂತ ಲೂಪ್
ಇ = ೦
@ {
    ಇ++
    ? ಇ >= ೫ { @! }
    >> ಇ " "
}
>> ¶                          // → ೧ ೨ ೩ ೪

// ಲೇಬಲ್ ಮಾಡಿದ ಲೂಪ್ (ನೆಸ್ಟೆಡ್ ಬ್ರೇಕ್)
count = ೦
@ @outer {
    count++
    ? count >= ೩ { @! outer }
}
>> count ¶                    // → ೩
```

## ಕಾರ್ಯಗಳು

```zymbol
ಮೊತ್ತ(ಅ, ಬ) { <~ ಅ + ಬ }
>> ಮೊತ್ತ(೩, ೪) ¶    // → ೭

ಅಪವರ್ತ್ಯ(ನ) {
    ? ನ <= ೧ { <~ ೧ }
    <~ ನ * ಅಪವರ್ತ್ಯ(ನ - ೧)
}
>> ಅಪವರ್ತ್ಯ(೫) ¶    // → ೧೨೦
```

ಕಾರ್ಯಗಳಿಗೆ **ಪ್ರತ್ಯೇಕ scope** ಇದೆ — ಹೊರಗಿನ ಅಸ್ಥಿರಗಳನ್ನು ಓದಲು ಸಾಧ್ಯವಿಲ್ಲ. ಔಟ್‌ಪುಟ್ ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು `<~` ಮೂಲಕ ಕಾಲರ್ ಅಸ್ಥಿರಗಳನ್ನು ಮಾರ್ಪಡಿಸಬಹುದು:

```zymbol
swap(a<~, b<~) {
    tmp = a
    a = b
    b = tmp
}
x = ೧೦
y = ೨೦
swap(x, y)
>> "x=" x " y=" y ¶    // → x=೨೦ y=೧೦
```

> ಹೆಸರಿನ ಕಾರ್ಯಗಳು ಪ್ರಥಮ-ದರ್ಜೆ ಮೌಲ್ಯಗಳಲ್ಲ. ತರ್ಕವಾಗಿ ರವಾನಿಸಲು ಸುತ್ತಿಕೊಳ್ಳಿ: `x -> ಮೊತ್ತ(x)`.

## ಲ್ಯಾಂಬ್ಡಾ ಮತ್ತು ಕ್ಲೋಜರ್‌ಗಳು

```zymbol
ದ್ವಿಗುಣ = ಚ -> ಚ * ೨
ಮೊತ್ತ = (ಅ, ಬ) -> ಅ + ಬ
>> ದ್ವಿಗುಣ(೫) ¶    // → ೧೦
>> ಮೊತ್ತ(೩, ೭) ¶   // → ೧೦

// ಬ್ಲಾಕ್ ಲ್ಯಾಂಬ್ಡಾ
ವರ್ಗೀಕರಿಸು = ಚ -> {
    ? ಚ > ೦ { <~ "ಧನಾತ್ಮಕ" }
    _? ಚ < ೦ { <~ "ಋಣಾತ್ಮಕ" }
    <~ "ಶೂನ್ಯ"
}

// ಕ್ಲೋಜರ್ — ಹೊರಗಿನ scope ಅನ್ನು ಸೆರೆಹಿಡಿಯುತ್ತದೆ
ಗುಣಕ = ೩
ತ್ರಿಗುಣ = ಚ -> ಚ * ಗುಣಕ
>> ತ್ರಿಗುಣ(೭) ¶    // → ೨೧

// ಕಾರ್ಖಾನೆ
ಕೂಡಿಸು_ತಯಾರಿ(ನ) { <~ ಚ -> ಚ + ನ }
ಹತ್ತು_ಕೂಡಿಸು = ಕೂಡಿಸು_ತಯಾರಿ(೧೦)
>> ಹತ್ತು_ಕೂಡಿಸು(೫) ¶    // → ೧೫

// ಪಟ್ಟಿಯಲ್ಲಿ
ಕ್ರಿಯೆಗಳು = [ಚ -> ಚ+೧, ಚ -> ಚ*೨, ಚ -> ಚ*ಚ]
>> ಕ್ರಿಯೆಗಳು[೨](೫) ¶    // → ೨೫
```

## ಪಟ್ಟಿ

Arrays are **mutable** and hold elements of the **same type**.

```zymbol
ಪಟ್ಟಿ = [೧, ೨, ೩, ೪, ೫]

ಪಟ್ಟಿ[೦]          // ೧ — ಪ್ರವೇಶ (೦-ಸೂಚಿ)
ಪಟ್ಟಿ[-೧]         // ೫ — ಋಣಾತ್ಮಕ ಸೂಚಿ (ಕೊನೆಯದು)
ಪಟ್ಟಿ$#           // ೫ — ಉದ್ದ

ಪಟ್ಟಿ = ಪಟ್ಟಿ$+ ೬            // ಸೇರಿಸು → [೧,೨,೩,೪,೫,೬]
ಪಟ್ಟಿ೨ = ಪಟ್ಟಿ$+[೨] ೯೯       // ಸೂಚಿ ೨ ರಲ್ಲಿ ಸೇರಿಸು
ಪಟ್ಟಿ೩ = ಪಟ್ಟಿ$- ೩           // ಮೊದಲ ಸಂಭವ ತೆಗೆ
ಪಟ್ಟಿ೪ = ಪಟ್ಟಿ$-- ೩          // ಎಲ್ಲ ಸಂಭವಗಳು ತೆಗೆ
ಪಟ್ಟಿ೫ = ಪಟ್ಟಿ$-[೦]          // ಸೂಚಿಯಿಂದ ತೆಗೆ
ಪಟ್ಟಿ೬ = ಪಟ್ಟಿ$-[೧..೩]       // ಶ್ರೇಣಿ ತೆಗೆ (ಕೊನೆ ಹೊರತು)

ಇದೆ = ಪಟ್ಟಿ$? ೩            // #೧
ಸ್ಥಾನಗಳು = ಪಟ್ಟಿ$?? ೩       // [೨] — ಎಲ್ಲ ಸೂಚಿಗಳು
ತುಂಡು = ಪಟ್ಟಿ$[೦..೩]        // [೧,೨,೩]
ತುಂಡು೨ = ಪಟ್ಟಿ$[೦:೩]        // [೧,೨,೩] — ಎಣಿಕೆ ಸಿಂಟ್ಯಾಕ್ಸ್

ಏರು = ಪಟ್ಟಿ$^+              // ಆರೋಹಣ ಕ್ರಮ (primitives)
ಇಳಿ = ಪಟ್ಟಿ$^-              // ಅವರೋಹಣ ಕ್ರಮ (primitives)

// Named tuple arrays
ಡೇಟಾ = [(ಹೆಸರು: "Carla", ವ: ೨೮), (ಹೆಸರು: "Ana", ವ: ೨೫), (ಹೆಸರು: "Bob", ವ: ೩೦)]
ವ_ಕ್ರಮ  = ಡೇಟಾ$^ (ಅ, ಬ -> ಅ.ವ < ಬ.ವ)
ಹ_ಕ್ರಮ = ಡೇಟಾ$^ (ಅ, ಬ -> ಅ.ಹೆಸರು > ಬ.ಹೆಸರು)
>> ವ_ಕ್ರಮ[೦].ಹೆಸರು ¶     // → Ana
>> ಹ_ಕ್ರಮ[೦].ಹೆಸರು ¶    // → Carla

// Direct element update (arrays only)
ಪಟ್ಟಿ[೧] = ೯೯              // ನಿಯೋಜಿಸು
ಪಟ್ಟಿ[೦] += ೫              // ಸಂಯುಕ್ತ: +=  -=  *=  /=  %=  ^=

// Functional update — returns a new array; original unchanged
ಪಟ್ಟಿ೨ = ಪಟ್ಟಿ[೧]$~ ೯೯
```

> ಎಲ್ಲ collection operators **ಹೊಸ ಪಟ್ಟಿಯನ್ನು** ಹಿಂತಿರುಗಿಸುತ್ತವೆ.
> Chaining ಇಲ್ಲ: ಪ್ರತ್ಯೇಕ ನಿಯೋಜನೆಗಳನ್ನು ಬಳಸಿ.
> `$^+` / `$^-` **primitive arrays** ಅನ್ನು ವರ್ಗೀಕರಿಸುತ್ತವೆ. Tuple arrays ಗೆ comparator lambda ಜೊತೆ `$^` ಬಳಸಿ.

**Value semantics** — assigning an array to another variable creates an independent copy:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b is unaffected
```

```zymbol
// ನೆಸ್ಟೆಡ್ ಪಟ್ಟಿಗಳು
matrix = [[೧,೨,೩],[೪,೫,೬],[೭,೮,೯]]
>> matrix[೧][೨] ¶    // → ೬
```

## ಡಿಸ್ಟ್ರಕ್ಚರಿಂಗ್

```zymbol
// ಶ್ರೇಣಿ
ಪಟ್ಟಿ = [೧೦, ೨೦, ೩೦, ೪೦, ೫೦]
[ಅ, ಆ, ಇ] = ಪಟ್ಟಿ              // ಅ=೧೦  ಆ=೨೦  ಇ=೩೦
[ಮೊ, *ಉ] = ಪಟ್ಟಿ               // ಮೊ=೧೦  ಉ=[೨೦,೩೦,೪೦,೫೦]
[ಚ, _, ಟ] = [೧, ೨, ೩]          // _ ಬಿಟ್ಟುಬಿಡುತ್ತದೆ

// Positional tuple
ಬಿಂದು = (೧೦೦, ೨೦೦)
(px, py) = ಬಿಂದು               // px=೧೦೦  py=೨೦೦

// Named tuple
ವ್ಯಕ್ತಿ = (ಹೆಸರು: "Ana", ವಯಸ್ಸು: ೨೫, ನಗರ: "Bengaluru")
(ಹೆಸರು: ಹ, ವಯಸ್ಸು: ವ) = ವ್ಯಕ್ತಿ  // ಹ="Ana"  ವ=೨೫
```

## ಟಪಲ್

Tuples are **immutable** ordered containers that can hold values of **different types**. Unlike arrays, elements cannot be changed after creation.

```zymbol
// ಸ್ಥಾನ
point = (೧೦, ೨೦)
>> point[೦] ¶    // → ೧೦

data = (42, "hello", #1, 3.14)
>> data[2] ¶     // → #1

// ಹೆಸರಿನ
ವ್ಯಕ್ತಿ = (ಹೆಸರು: "ಆಲಿಸ್", ವಯಸ್ಸು: ೨೫)
>> ವ್ಯಕ್ತಿ.ಹೆಸರು ¶    // → ಆಲಿಸ್
>> ವ್ಯಕ್ತಿ[೦] ¶        // → ಆಲಿಸ್  (ಸೂಚಿಯೂ ಕೆಲಸ ಮಾಡುತ್ತದೆ)

// ನೆಸ್ಟೆಡ್
pos = (x: ೧೦, y: ೨೦)
p = (pos: pos, label: "origin")
>> p.pos.x ¶        // → ೧೦
```

**Immutability** — any attempt to modify a tuple element is a runtime error:

```zymbol
t = (10, 20, 30)
// t[0] = 99    // ❌ runtime error: tuples are immutable
// t[0] += 5    // ❌ same error
```

To derive a modified value use `$~` (functional update) — returns a **new** tuple:

```zymbol
t = (10, 20, 30)
t2 = t[1]$~ 999
>> t ¶     // → (10, 20, 30)   ← original unchanged
>> t2 ¶    // → (10, 999, 30)

// Named tuple — rebuild explicitly
ವ್ಯಕ್ತಿ = (ಹೆಸರು: "Alice", ವಯಸ್ಸು: 25)
ವಯಸ್ಕ = (ಹೆಸರು: ವ್ಯಕ್ತಿ.ಹೆಸರು, ವಯಸ್ಸು: 26)
>> ವ್ಯಕ್ತಿ.ವಯಸ್ಸು ¶    // → 25
>> ವಯಸ್ಕ.ವಯಸ್ಸು ¶      // → 26
```

## ಉನ್ನತ-ಕ್ರಮ ಕಾರ್ಯಗಳು

> HOF ಆಪರೇಟರ್‌ಗಳಿಗೆ **ಇನ್‌ಲೈನ್ ಲ್ಯಾಂಬ್ಡಾ** ಬೇಕು — ಲ್ಯಾಂಬ್ಡಾ ಅಸ್ಥಿರಗಳನ್ನು ನೇರವಾಗಿ ರವಾನಿಸಲು ಸಾಧ್ಯವಿಲ್ಲ.

```zymbol
ಸಂಖ್ಯೆಗಳು = [೧, ೨, ೩, ೪, ೫, ೬, ೭, ೮, ೯, ೧೦]

ದ್ವಿಗುಣಗಳು  = ಸಂಖ್ಯೆಗಳು$> (ಚ -> ಚ * ೨)                // map  → [೨,೪,೬…೨೦]
ಜೋಡಿ        = ಸಂಖ್ಯೆಗಳು$| (ಚ -> ಚ % ೨ == ೦)           // filter → [೨,೪,೬,೮,೧೦]
ಒಟ್ಟು        = ಸಂಖ್ಯೆಗಳು$< (೦, (ಸಂ, ಚ) -> ಸಂ + ಚ)     // reduce → ೫೫

// ಮಧ್ಯಂತರಗಳ ಮೂಲಕ ಸರಪಳಿ
step೧ = ಸಂಖ್ಯೆಗಳು$| (ಚ -> ಚ > ೩)
step೨ = step೧$> (ಚ -> ಚ * ಚ)
>> step೨ ¶    // → [೧೬, ೨೫, ೩೬, ೪೯, ೬೪, ೮೧, ೧೦೦]

// HOF ನಲ್ಲಿ ಹೆಸರಿನ ಕಾರ್ಯಗಳು — ಲ್ಯಾಂಬ್ಡಾದಲ್ಲಿ ಸುತ್ತಿಕೊಳ್ಳಿ
ದ್ವಿಗುಣ(ಚ) { <~ ಚ * ೨ }
r = ಸಂಖ್ಯೆಗಳು$> (ಚ -> ದ್ವಿಗುಣ(ಚ))    // ✅
```

## ಸಂಖ್ಯಾ ಪದ್ಧತಿಗಳು

Zymbol **69 Unicode ಅಂಕ ನಿಲಯಗಳಲ್ಲಿ** ಸಂಖ್ಯೆಗಳನ್ನು ತೋರಿಸಬಹುದು — ಕನ್ನಡ, ದೇವನಾಗರಿ, ಅರೇಬಿಕ್-ಇಂಡಿಕ್, ಥಾಯ್, ಕ್ಲಿಂಗಾನ್ pIqaD ಮತ್ತು ಇನ್ನೂ ಹೆಚ್ಚು. ಸಕ್ರಿಯ ಮೋಡ್ `>>` ಔಟ್‌ಪುಟ್ ಮೇಲೆ ಮಾತ್ರ ಪರಿಣಾಮ ಬೀರುತ್ತದೆ; ಆಂತರಿಕ ಗಣನೆ ಯಾವಾಗಲೂ ದ್ವಿಮಾನ.

### ಒಂದು ವ್ಯವಸ್ಥೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ

ಬಯಸಿದ ಲಿಪಿಯ `೦` ಮತ್ತು `೯` ಅಂಕಿಗಳನ್ನು `#…#` ನಡುವೆ ಬರೆಯಿರಿ:

```zymbol
#೦೯#    // KN script (೦..೯)
#٠٩#    // Arabic-Indic  (U+0660–U+0669)
#09#    // reset to ASCII
```

### ಔಟ್‌ಪುಟ್ ಮತ್ತು ಬೂಲಿಯನ್

```zymbol
x = ೪೨
>> x ¶          // → ೪೨   (ASCII ಡೀಫಾಲ್ಟ್)

#೦೯#
>> x ¶          // → ೪೨
>> ೩.೧೪ ¶       // → ೩.೧೨೩೪   (ದಶಮಾಂಶ ಬಿಂದು ಯಾವಾಗಲೂ ASCII)
>> ೧ + ೨ ¶      // → ೩

>> #೧ ¶         // → #೧   (ಸತ್ಯ)
>> #೦ ¶         // → #೦   (ಸುಳ್ಳು — ೦ ನಿಂದ ಭಿನ್ನ)

x = ೨೮ > ೪
>> x ¶          // → #೧   (ಹೋಲಿಕೆ ಫಲಿತಾಂಶ)
```

### ಮೂಲದಲ್ಲಿ ಸ್ಥಳೀಯ ಅಕ್ಷರಾರ್ಥಗಳು

ಯಾವುದೇ ಬೆಂಬಲಿತ ವ್ಯವಸ್ಥೆಯ ಅಂಕಿಗಳು ಶ್ರೇಣಿಗಳು, ಮಾಡ್ಯುಲೋ, ಹೋಲಿಕೆಗಳಲ್ಲಿ ಮಾನ್ಯ ಅಕ್ಷರಾರ್ಥಗಳು:

```zymbol
#೦೯#

@ ಇ:೧..೧೫ {
    ? ಇ % ೧೫ == ೦ { >> "FizzBuzz" ¶ }
    _? ಇ % ೩  == ೦ { >> "Fizz" ¶ }
    _? ಇ % ೫  == ೦ { >> "Buzz" ¶ }
    _ { >> ಇ ¶ }
}
```

### ಯಾವುದೇ ವ್ಯವಸ್ಥೆಯಲ್ಲಿ ಬೂಲಿಯನ್ ಅಕ್ಷರಾರ್ಥಗಳು

ಯಾವುದೇ ಬೆಂಬಲಿತ ಬ್ಲಾಕ್‌ನ `#` + `೦` ಅಥವಾ `೧` ಅಂಕಿ ಮಾನ್ಯ ಬೂಲಿಯನ್ ಅಕ್ಷರಾರ್ಥ:

```zymbol
#٠٩#
نشط = #١        // same as #೧
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **ಯಾವಾಗಲೂ ASCII**. `#೦` (ಸುಳ್ಳು) ಯಾವುದೇ ವ್ಯವಸ್ಥೆಯಲ್ಲಿ `೦` (ಶೂನ್ಯ ಪೂರ್ಣಾಂಕ) ನಿಂದ ದೃಶ್ಯಾತ್ಮಕವಾಗಿ ಭಿನ್ನ.

| `_` | else / wildcard | `$?` | ಒಳಗಿದೆ |

| `@>` | ಮುಂದುವರಿಸು | `$\|` | filter |
| `->` | ಲ್ಯಾಂಬ್ಡಾ | `$<` | reduce |

| `<~` | return | `!?` | try |
| `\|>` | pipe | `:!` | catch |
| `#೧` | ಸತ್ಯ | `:>` | finally |

- **ಸೇರಿಸಲಾಗಿದೆ** REPL ಎಕೋ ಮತ್ತು ವೇರಿಯಬಲ್ ಡಿಸ್‌ಪ್ಲೇಯಲ್ಲಿ ಸಕ್ರಿಯ ಅಂಕ ಮೋಡ್ ಅನ್ನು ಗೌರವಿಸುತ್ತದೆ
- **ಬದಲಾಯಿಸಲಾಗಿದೆ** ಬೂಲಿಯನ್ `>>` ಔಟ್‌ಪುಟ್‌ನಲ್ಲಿ ಎಲ್ಲಾ ಮೋಡ್‌ಗಳಲ್ಲಿ `#` ಉಪಸರ್ಗ (`#೦` / `#೧`) ಸೇರ್ಪಡೆ
