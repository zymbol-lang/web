# Zymbol-Lang-এর সংক্ষিপ্ত নির্দেশিকা

**Zymbol-Lang** একটি প্রতীকী প্রোগ্রামিং ভাষা। এটি কোনো নির্দিষ্ট কীওয়ার্ড ব্যবহার করে না — সবকিছুই প্রতীক। এটি প্রতিটি মানবভাষায় একইভাবে কাজ করে।

- কোনো কীওয়ার্ড নেই (`if`, `while`, `return` নেই — শুধু প্রতীক `?`, `@`, `<~`)
- সম্পূর্ণ Unicode সমর্থন — যেকোনো ভাষা বা ইমোজি দিয়ে শনাক্তকারী 👋
- ভাষা-নিরপেক্ষ — কোড সব ভাষায় একই

## ডেটা ধরন

| ধরন | উদাহরণ | `#?` প্রতীক | নোট |
|-----|--------|-------------|-----|
| পূর্ণসংখ্যা (Int) | `৪২`, `-৭` | `###` | ৬৪-বিট চিহ্নযুক্ত |
| দশমিক (Float) | `৩.১৪`, `১.৫e১০` | `##.` | বৈজ্ঞানিক স্বরলিপি গ্রহণযোগ্য |
| স্ট্রিং (String) | `"হ্যালো"` | `##"` | আন্তঃসংযোজন: `"হ্যালো {নাম}"` |
| অক্ষর (Char) | `'A'` | `##'` | একটি Unicode অক্ষর |
| বুলিয়ান (Bool) | `#১`, `#০` | `##?` | সংখ্যাগত নয় — `#১ ≠ ১` |
| তালিকা (Array) | `[১, ২, ৩]` | `##]` | সমজাতীয় উপাদান |
| টাপল (Tuple) | `(ক, খ)` | `##)` | অবস্থানগত |
| নামযুক্ত টাপল | `(x: ১, y: ২)` | `##)` | নামযুক্ত ক্ষেত্র |

```zymbol
// ধরন পরীক্ষা — (ধরন, অঙ্ক, মান) ফেরত দেয়
meta = ৪২#?
>> meta ¶         // → (###, ২, ৪২)
t = meta[০]
>> t ¶            // → ###
```

## আউটপুট এবং ইনপুট

```zymbol
>> "হ্যালো" ¶                      // ¶ বা \\ স্পষ্ট নতুন লাইন
>> "ক=" চ " খ=" ন ¶                // পাশাপাশি — একাধিক মান
>> (তালিকা$#) ¶                    // >> তে postfix অপারেটরের জন্য বন্ধনী দরকার

<< নাম                              // প্রম্পট ছাড়া — চলে পড়ে
<< "আপনার নাম? " নাম               // প্রম্পটসহ
```

> `¶` (স্প্যানিশ কীবোর্ডে AltGr+R) এবং `\\` সমতুল্য নতুন লাইন।

## ফাংশন

```zymbol
যোগ(ক, খ) { <~ ক + খ }
>> যোগ(৩, ৪) ¶    // → ৭

গুণনীয়ক(ন) {
    ? ন <= ১ { <~ ১ }
    <~ ন * গুণনীয়ক(ন - ১)
}
>> গুণনীয়ক(৫) ¶    // → ১২০
```

ফাংশনের **পৃথক scope** আছে — বাইরের চল পড়তে পারে না। আউটপুট প্যারামিটার `<~` দিয়ে কলার চল পরিবর্তন করুন:

```zymbol
swap(ক<~, খ<~) {
    tmp = ক
    ক = খ
    খ = tmp
}
চ = ১০
য = ২০
swap(চ, য)
>> "চ=" চ " য=" য ¶    // → চ=২০ য=১০
```

> নামযুক্ত ফাংশন প্রথম-শ্রেণির নয়। আর্গুমেন্ট হিসেবে পাঠাতে মোড়ান: `x -> যোগ(x)`।

## তালিকা

তালিকাগুলি **পরিবর্তনযোগ্য** এবং **একই ধরনের** উপাদান রাখে।

```zymbol
তালিকা = [১, ২, ৩, ৪, ৫]

তালিকা[০]          // ১ — অ্যাক্সেস (০ থেকে সূচকযুক্ত)
তালিকা[-১]         // ৫ — ঋণাত্মক সূচক (শেষ)
তালিকা$#           // ৫ — দৈর্ঘ্য (>> তে (তালিকা$#) ব্যবহার করুন)

তালিকা = তালিকা$+ ৬            // যোগ → [১,২,৩,৪,৫,৬]
arr২ = তালিকা$+[২] ৯৯           // সূচক ২ তে ঢোকান
arr৩ = তালিকা$- ৩               // মানের প্রথম ঘটনা মুছুন
arr৪ = তালিকা$-- ৩              // সব ঘটনা মুছুন
arr৫ = তালিকা$-[০]              // সূচকে মুছুন
arr৬ = তালিকা$-[১..৩]           // রেঞ্জ মুছুন (শেষ বাদ)

has = তালিকা$? ৩                // #১ — আছে কিনা
pos = তালিকা$?? ৩               // [২] — মানের সব সূচক
sl = তালিকা$[০..৩]              // [১,২,৩] — অংশ (শেষ বাদ)
sl২ = তালিকা$[০:৩]              // [১,২,৩] — গণনা-ভিত্তিক বাক্যরীতি

asc = তালিকা$^+                 // আরোহী ক্রম (শুধু primitive)
desc = তালিকা$^-                // অবরোহী ক্রম (শুধু primitive)

// নামযুক্ত/অবস্থানগত টাপল তালিকা — $^ lambda দিয়ে
db = [(name: "কার্লা", age: ২৮), (name: "আনা", age: ২৫), (name: "বব", age: ৩০)]
by_age  = db$^ (ক, খ -> ক.age < খ.age)    // বয়স অনুযায়ী আরোহী (<)
by_name = db$^ (ক, খ -> ক.name > খ.name)  // নাম অনুযায়ী অবরোহী (>)
>> by_age[০].name ¶     // → আনা
>> by_name[০].name ¶    // → কার্লা

// সরাসরি উপাদান আপডেট (শুধু তালিকা)
তালিকা[১] = ৯৯              // নিয়োগ
তালিকা[০] += ৫              // যৌগিক: +=  -=  *=  /=  %=  ^=

// ফাংশনাল আপডেট — নতুন তালিকা ফেরত দেয়; মূল অপরিবর্তিত থাকে
তালিকা২ = তালিকা[১]$~ ৯৯
```

> সব সংগ্রহ অপারেটর **নতুন তালিকা** ফেরত দেয়। পুনর্নিয়োগ করুন: `তালিকা = তালিকা$+ ৪`।
> অপারেটর শৃঙ্খলা করা যায় না — মধ্যবর্তী নিয়োগ ব্যবহার করুন।
> `$^+` / `$^-` **primitive তালিকা** (সংখ্যা, স্ট্রিং) সাজায়। টাপল তালিকার জন্য `$^` এবং lambda ব্যবহার করুন — দিকনির্দেশ lambda তে এনকোড (`<` = আরোহী, `>` = অবরোহী)।

**মান শব্দার্থ** — তালিকা অন্য চলে নিয়োগ করলে একটি স্বাধীন প্রতিলিপি তৈরি হয়:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b অপ্রভাবিত থাকে
```

```zymbol
// নেস্টেড তালিকা
matrix = [[১,২,৩],[৪,৫,৬],[৭,৮,৯]]
>> matrix[১][২] ¶    // → ৬
```

## টাপল

টাপলগুলি **অপরিবর্তনীয়** ক্রমবদ্ধ ধারক যা **বিভিন্ন ধরনের** মান রাখতে পারে। তালিকার বিপরীতে, তৈরির পর উপাদান পরিবর্তন করা যায় না।

```zymbol
// অবস্থানগত
point = (১০, ২০)
>> point[০] ¶    // → ১০

তথ্য = (৪২, "হ্যালো", #১, ৩.১৪)
>> তথ্য[২] ¶     // → #১

// নামযুক্ত
ব্যক্তি = (name: "অনু", age: ২৫)
>> ব্যক্তি.name ¶    // → অনু
>> ব্যক্তি[০] ¶      // → অনু  (সূচকও কাজ করে)

// নেস্টেড
pos = (x: ১০, y: ২০)

>> p.pos.x ¶        // → ১০
```

**অপরিবর্তনীয়তা** — টাপল উপাদান পরিবর্তনের যেকোনো প্রচেষ্টা রানটাইম ত্রুটি:

```zymbol
t = (১০, ২০, ৩০)
// t[০] = ৯৯    // ❌ রানটাইম ত্রুটি: টাপল অপরিবর্তনীয়
// t[০] += ৫    // ❌ একই ত্রুটি
```

পরিবর্তিত মান পেতে `$~` (ফাংশনাল আপডেট) ব্যবহার করুন — **নতুন** টাপল ফেরত দেয়:

```zymbol
t = (১০, ২০, ৩০)
t২ = t[১]$~ ৯৯৯
>> t ¶     // → (১০, ২০, ৩০)   ← মূল অপরিবর্তিত
>> t২ ¶    // → (১০, ৯৯৯, ৩০)

// নামযুক্ত টাপল — স্পষ্টভাবে পুনর্নির্মাণ করুন
ব্যক্তি = (name: "অনু", age: ২৫)
বড়  = (name: ব্যক্তি.name, age: ২৬)
>> ব্যক্তি.age ¶    // → ২৫
>> বড়.age ¶         // → ২৬
```

## সংখ্যা পদ্ধতি

Zymbol **69টি Unicode অঙ্ক ব্লকে** সংখ্যা প্রদর্শন করতে পারে — বাংলা, দেবনাগরী, আরবি-ভারতীয়, থাই, ক্লিঙ্গন pIqaD এবং আরও। সক্রিয় মোড শুধুমাত্র `>>` আউটপুটকে প্রভাবিত করে; অভ্যন্তরীণ গণনা সর্বদা বাইনারি।

### একটি সিস্টেম সক্রিয় করুন

লক্ষ্য স্ক্রিপ্টের `০` এবং `৯` অঙ্কটি `#…#` এর মধ্যে লিখুন:

```zymbol
#০৯#    // BN script (০..৯)
#٠٩#    // Arabic-Indic  (U+0660–U+0669)
#09#    // reset to ASCII
```

### আউটপুট এবং বুলিয়ান

```zymbol
x = ৪২
>> x ¶          // → ৪২   (ASCII ডিফল্ট)

#০৯#
>> x ¶          // → ৪২
>> ৩.১৪ ¶       // → ৩.১২৩৪   (দশমলব বিন্দু সর্বদা ASCII)
>> ১ + ২ ¶      // → ৩

>> #১ ¶         // → #১   (যেকোনো মোডে সত্য)
>> #০ ¶         // → #০   (মিথ্যা — ০ থেকে আলাদা)

x = ২৮ > ৪
>> x ¶          // → #১   (তুলনার ফলাফল সক্রিয় মোড অনুসরণ করে)
```

### উৎসে নেটিভ লিটারেল

যেকোনো সমর্থিত সিস্টেমের অঙ্ক পরিসর, মডুলো, তুলনায় বৈধ লিটারেল:

```zymbol
#০৯#

@ ন:১..১৫ {
    ? ন % ১৫ == ০ { >> "FizzBuzz" ¶ }
    _? ন % ৩  == ০ { >> "Fizz" ¶ }
    _? ন % ৫  == ০ { >> "Buzz" ¶ }
    _ { >> ন ¶ }
}
```

### যেকোনো সিস্টেমে বুলিয়ান লিটারেল

`#` + যেকোনো সমর্থিত ব্লকের `০` বা `১` অঙ্ক একটি বৈধ বুলিয়ান লিটারেল:

```zymbol
#٠٩#
نشط = #١        // same as #১
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **সবসময় ASCII**। `#০` (মিথ্যা) যেকোনো সিস্টেমে `০` (শূন্য পূর্ণসংখ্যা) থেকে দৃশ্যত আলাদা।

| প্রতীক | কাজ | প্রতীক | কাজ |
|--------|-----|--------|-----|
| `=` | চল | `$#` | দৈর্ঘ্য |
| `:=` | ধ্রুবক | `$+` | যোগ |
| `>>` | আউটপুট | `$+[i]` | সূচকে ঢোকান |
| `<<` | ইনপুট | `$--` | মানের সব ঘটনা মুছুন |
| `¶` / `\` | নতুন লাইন | `$-[i]` | সূচকে মুছুন |
| `?` | if | `$-[i..j]` | রেঞ্জ মুছুন |
| `_?` | else-if | `$?` | আছে কিনা |
| `_` | else / wildcard | `$??` | সব সূচক খুঁজুন |
| `??` | মিলান | `$[s..e]` | অংশ |
| `@` | লুপ | `$>` | map |
| `@!` | বিরতি | `$\|` | filter |
| `@>` | চালিয়ে যাও | | |
| `->` | lambda | `$<` | reduce |
| `arr[i] = val` | উপাদান আপডেট (শুধু তালিকা) | `arr[i] += val` | যৌগিক আপডেট |
| `arr[i]$~` | ফাংশনাল আপডেট (নতুন প্রতিলিপি) | `$^+` | আরোহী ক্রম |

| `\|>` | pipe | `<~` | return |

| `!?` | try | `#` | মডিউল ঘোষণা |

| `#^\|..\|` | বৈজ্ঞানিক | `<\ ..\>` | শেল কমান্ড |
