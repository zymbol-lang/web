## টিইউআই প্রিমিটিভ

ইন্টারঅ্যাকটিভ প্রোগ্রামের জন্য টার্মিনাল ইউজার ইন্টারফেস অপারেটর। অধিকাংশের জন্য `>>| { }` ব্লক প্রয়োজন (বিকল্প স্ক্রিন + র মোড)।

```zymbol
>>| {
    >>!                             // বিকল্প স্ক্রিন পরিষ্কার করুন
    >>~ (১, ১, ০, ১০) > "চলছে"    // সারি ১, কলাম ১, fg=১০ (সবুজ)
    @~ ১০০০                         // ১ সেকেন্ড বিরতি (১০০০ ms)
    >>~ (২, ১) > "সম্পন্ন।"
}
// প্রস্থান করার সময় টার্মিনাল স্বয়ংক্রিয়ভাবে পুনরুদ্ধার হয়
```

```zymbol
// কী প্রেস ও টার্মিনাল সাইজ
>>| {
    [সারি, কলাম] = >>?              // টার্মিনালের মাত্রা জিজ্ঞাসা করুন
    >>~ (১, ১) > "টার্মিনাল: " সারি " x " কলাম
    <<| কী                         // ব্লকিং কী প্রেস পড়ুন
    >>~ (২, ১) > "চাপলেন: " কী
}
```

> `>>!` স্ক্রিন পরিষ্কার করে। `>>?` `[সারি, কলাম]` রিটার্ন করে। `@~ N` N মিলিসেকেন্ড ঘুমায়।
> `<<|` একটি কী প্রেস পড়ে (ব্লকিং); `<<|?` ব্লক না করে পোল করে (কোনোটি না থাকলে `'\0'` রিটার্ন করে)।
> অবস্থানগত আউটপুট টাপল: `(সারি, কলাম, BKS, fg, bg)` — যেকোনো স্লট কমা দিয়ে বাদ দেওয়া যেতে পারে (`>>~ (,,, 196) > "লাল"`)।
> BKS বিটমাস্ক: `1`=বোল্ড, `2`=ইটালিক, `4`=আন্ডারলাইন। ANSI 256 রঙের প্যালেট (`0`=টার্মিনাল ডিফল্ট)।
> শুধু tree-walker (`>>!`, `>>?`, `@~`, `>>~` ব্যতীত যেগুলি `--vm`-এও চলে)।

```zymbol
arr = [১, ২, ৩, ৪, ৫]

x = arr[১]      // ১ — অ্যাক্সেস (১-ভিত্তিক: প্রথম উপাদান)
x = arr[-১]     // ৫ — নেতিবাচক সূচক (শেষ উপাদান)
x = arr$#       // ৫ — দৈর্ঘ্য (>> এ (arr$#) ব্যবহার করুন)

arr = arr$+ ৬            // যুক্ত করুন → [১,২,৩,৪,৫,৬]
arr২ = arr$+[২] ৯৯       // অবস্থান ২ এ সন্নিবেশ করুন (১-ভিত্তিক)
arr৩ = arr$- ৩           // মানের প্রথম উপস্থিতি সরান
arr৪ = arr$-- ৩          // সমস্ত উপস্থিতি সরান
arr৫ = arr$-[১]          // সূচক ১ এ সরান (প্রথম উপাদান)
arr৬ = arr$-[২..৩]       // রেঞ্জ সরান (১-ভিত্তিক, শেষ অন্তর্ভুক্ত)

আছে = arr$? ৩            // #১ — ধারণ করে
অবস্থান = arr$?? ৩           // [৩] — মানের সমস্ত সূচক (১-ভিত্তিক)
স্লাইস = arr$[১..৩]          // [১,২,৩] — স্লাইস (১-ভিত্তিক, শেষ অন্তর্ভুক্ত)
স্লাইস২ = arr$[১:৩]          // [১,২,৩] — একই, গণনা-ভিত্তিক সিনট্যাক্স

ঊর্ধ্ব = arr$^+             // ঊর্ধ্বক্রমে সাজানো (শুধু আদিম)
নিম্ন = arr$^-            // নিম্নক্রমে সাজানো (শুধু আদিম)

// নামকৃত/অবস্থানগত টাপল অ্যারে — তুলনাকারী ল্যাম্বডা সহ $^ ব্যবহার করুন
ডিবি = [(নাম: "কারলা", বয়স: ২৮), (নাম: "আনা", বয়স: ২৫), (নাম: "বব", বয়স: ৩০)]
বয়স_অনুযায়ী  = ডিবি$^ (a, b -> a.বয়স < b.বয়স)    // বয়স অনুযায়ী ঊর্ধ্ব (<)
নাম_অনুযায়ী = ডিবি$^ (a, b -> a.নাম > b.নাম)   // নাম অনুযায়ী নিম্ন (>)
>> বয়স_অনুযায়ী[১].নাম ¶     // → আনা
>> নাম_অনুযায়ী[১].নাম ¶    // → কারলা

// সরাসরি উপাদান আপডেট (শুধু অ্যারে)
arr[১] = ৯৯              // অ্যাসাইন
arr[২] += ৫              // যৌগিক: +=  -=  *=  /=  %=  ^=

// ফাংশনাল আপডেট — একটি নতুন অ্যারে রিটার্ন করে; মূল অপরিবর্তিত
arr২ = arr[২]$~ ৯৯
```

> সমস্ত কালেকশন অপারেটর একটি **নতুন অ্যারে** রিটার্ন করে। পিছনে অ্যাসাইন করুন: `arr = arr$+ 4`।
> `$+` চেইন করা যেতে পারে: `arr = arr$+ 5$+ 6$+ 7`। অন্যান্য অপারেটর মধ্যবর্তী অ্যাসাইনমেন্ট ব্যবহার করে।
> **সূচকীকরণ 1-ভিত্তিক**: `arr[1]` প্রথম উপাদান; `arr[0]` একটি রানটাইম ত্রুটি।
> `$^+` / `$^-` **আদিম অ্যারে** (সংখ্যা, স্ট্রিং) সাজায়। টাপল অ্যারের জন্য তুলনাকারী ল্যাম্বডা সহ `$^` ব্যবহার করুন — দিক ল্যাম্বডাতে এনকোড করা আছে (`<` = ঊর্ধ্ব, `>` = নিম্ন)।

**মান সিম্যান্টিক্স** — একটি অ্যারেকে অন্য ভেরিয়েবলে অ্যাসাইন করলে একটি স্বাধীন কপি তৈরি হয়:

```zymbol
a = [১, ২, ৩]
b = a
a[১] = ৯৯
>> a ¶    // → [৯৯, ২, ৩]
>> b ¶    // → [১, ২, ৩]   ← b অপ্রভাবিত
```

```zymbol
// নেস্টেড অ্যারে (১-ভিত্তিক সূচকীকরণ)
ম্যাট্রিক্স = [[১,২,৩],[৪,৫,৬],[৭,৮,৯]]
>> ম্যাট্রিক্স[২][৩] ¶    // → ৬  (সারি ২, কলাম ৩)
```

## টাপল

টাপল হল **অপরিবর্তনীয়** ক্রমবদ্ধ ধারক যা **বিভিন্ন টাইপের** মান ধারণ করতে পারে।
অ্যারের বিপরীতে, তৈরি করার পর উপাদান পরিবর্তন করা যায় না।

```zymbol
// অবস্থানগত — মিশ্র টাইপ অনুমোদিত
বিন্দু = (১০, ২০)
>> বিন্দু[১] ¶    // → ১০

ডেটা = (৪২, "হ্যালো", #১, ৩.১৪)
>> ডেটা[৩] ¶     // → #১

// নামকৃত
ব্যক্তি = (নাম: "আলিস", বয়স: ২৫)
>> ব্যক্তি.নাম ¶    // → আলিস
>> ব্যক্তি[১] ¶      // → আলিস  (সূচক ও কাজ করে, ১-ভিত্তিক)

// নেস্টেড
অব = (x: ১০, y: ২০)
p = (অব: অব, লেবেল: "মূল")
>> p.অব.x ¶        // → ১০
```

**অপরিবর্তনীয়তা** — টাপল উপাদান পরিবর্তনের যেকোনো প্রচেষ্টা একটি রানটাইম ত্রুটি:

```zymbol
t = (১০, ২০, ৩০)
// t[১] = ৯৯    // ❌ রানটাইম ত্রুটি: টাপল অপরিবর্তনীয়
// t[১] += ৫    // ❌ একই ত্রুটি
```

পরিবর্তিত মান বের করতে `$~` (ফাংশনাল আপডেট) ব্যবহার করুন — একটি **নতুন** টাপল রিটার্ন করে:

```zymbol
t = (১০, ২০, ৩০)
t২ = t[২]$~ ৯৯৯
>> t ¶     // → (১০, ২০, ৩০)   ← মূল অপরিবর্তিত
>> t২ ¶    // → (১০, ৯৯৯, ৩০)

// নামকৃত টাপল — স্পষ্টভাবে পুনর্নির্মাণ
ব্যক্তি = (নাম: "আলিস", বয়স: ২৫)
বয়স্ক  = (নাম: ব্যক্তি.নাম, বয়স: ২৬)
>> ব্যক্তি.বয়স ¶    // → ২৫
>> বয়স্ক.বয়স ¶     // → ২৬
```

```zymbol
সংখ্যা = [১, ২, ৩, ৪, ৫, ৬, ৭, ৮, ৯, ১০]

দ্বিগুণিত  = সংখ্যা$> (x -> x * ২)                  // map  → [২,৪,৬…২০]
জোড়    = সংখ্যা$| (x -> x % ২ == ০)           // filter → [২,৪,৬,৮,১০]
মোট    = সংখ্যা$< (০, (সঞ্চয়ক, x) -> সঞ্চয়ক + x)     // reduce → ৫৫

// মধ্যবর্তীদের মাধ্যমে চেইন
ধাপ১ = সংখ্যা$| (x -> x > ৩)
ধাপ২ = ধাপ১$> (x -> x * x)
>> ধাপ২ ¶    // → [১৬, ২৫, ৩৬, ৪৯, ৬৪, ৮১, ১০০]

// নামকৃত ফাংশন সরাসরি HOF-এ পাস করা যেতে পারে
দ্বিগুণ(x) { <~ x * ২ }
বড়(x) { <~ x > ৫ }
r = সংখ্যা$> দ্বিগুণ       // ✅ সরাসরি রেফারেন্স
r = সংখ্যা$| বড়       // ✅ সরাসরি রেফারেন্স
```

## পাইপ অপারেটর

ডান দিকে সবসময় পাইপকৃত মানের জন্য `_` প্লেসহোল্ডার প্রয়োজন:

```zymbol
দ্বিগুণ = x -> x * ২
যোগ = (a, b) -> a + b
বৃদ্ধি = x -> x + ১

ফ১ = ৫ |> দ্বিগুণ(_)        // → ১০
ফ২ = ১০ |> যোগ(_, ৫)       // → ১৫
ফ৩ = ৫ |> যোগ(২, _)        // → ৭

// চেইনড
ফ = ৫ |> দ্বিগুণ(_) |> বৃদ্ধি(_) |> দ্বিগুণ(_)
>> ফ ¶    // → ২২  (৫→১০→১১→২২)
```

## সংখ্যাসূচক মোড

Zymbol সংখ্যা **69টি ইউনিকোড অঙ্ক লিপিতে** প্রদর্শন করতে পারে — দেবনাগরী, আরবি-ভারতীয়, থাই, ক্লিঙ্গন পিআইকিউএডি, গাণিতিক বোল্ড, এলসিডি সেগমেন্ট, এবং আরও অনেক কিছু। সক্রিয় মোড শুধুমাত্র `>>` আউটপুটকে প্রভাবিত করে; অভ্যন্তরীণ পাটিগণিত সবসময় বাইনারি।

### একটি লিপি সক্রিয় করা

লক্ষ্য লিপির `0` এবং `9` অঙ্ক `#…#` এর ভিতরে লিখুন:

```zymbol
#०९#    // দেবনাগরী   (U+0966–U+096F)
#٠٩#    // আরবি-ভারতীয় (U+0660–U+0669)
#๐๙#    // থাই         (U+0E50–U+0E59)
#09#    // এএসসিআইআই-এ রিসেট করুন
```

### আউটপুট ও বুলিয়ান

```zymbol
x = ৪২
>> x ¶          // → ৪২   (এএসসিআইআই ডিফল্ট)

#०९#
>> x ¶          // → ४२
>> ৩.১৪ ¶       // → ३.१४   (দশমিক বিন্দু সবসময় এএসসিআইআই)
>> ১ + ২ ¶      // → ३

// বুলিয়ান: # উপসর্গ সবসময় এএসসিআইআই, অঙ্ক অভিযোজিত হয়
>> #১ ¶         // → #१   (দেবনাগরীতে সত্য)
>> #০ ¶         // → #०   (মিথ্যা — ০ পূর্ণসংখ্যা শূন্য থেকে আলাদা)

x = ২৮ > ৪
>> x ¶          // → #१   (তুলনা ফলাফল সক্রিয় মোড অনুসরণ করে)
```

### উৎসে স্থানীয় অঙ্ক লিটারেল

যেকোনো সমর্থিত লিপির অঙ্ক বৈধ লিটারেল — রেঞ্জ, মডুলো, তুলনায়:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### যেকোনো লিপিতে বুলিয়ান লিটারেল

`#` + যেকোনো ব্লক থেকে অঙ্ক `0` বা `1` একটি বৈধ বুলিয়ান লিটারেল:

```zymbol
#٠٩#
সক্রিয় = #١        // #১ এর সমান
>> সক্রিয় ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **সবসময় এএসসিআইআই**। `#0` (মিথ্যা) সবসময় প্রতিটি লিপিতে `0` (পূর্ণসংখ্যা শূন্য) থেকে দৃশ্যত আলাদা।

## ডেটা অপারেটর

```zymbol
// টাইপ রূপান্তর কাস্ট
f = ##.৪২         // → ৪২.০  (ভাসমানে)
i = ###৩.৭        // → ৪     (পূর্ণসংখ্যায়, রাউন্ড)
t = ##!৩.৭        // → ৩     (পূর্ণসংখ্যায়, ট্রাঙ্কেট)

// স্ট্রিং থেকে সংখ্যায় পার্স করুন
v১ = #|"৪২"|      // → ৪২  (পূর্ণসংখ্যা)
v২ = #|"৩.১৪"|    // → ৩.১৪  (ভাসমান)
v৩ = #|"abc"|     // → "abc"  (নিরাপদ, কোনো ত্রুটি নেই)

// রাউন্ড / ট্রাঙ্কেট
π = ৩.১৪১৫৯২৬৫
রাউন্ড২ = #.২|π|      // → ৩.১৪  (২ দশমিক স্থানে রাউন্ড)
রাউন্ড৪ = #.৪|π|      // → ৩.১৪১৬
ট্রাঙ্ক২ = #!২|π|      // → ৩.১৪  (ট্রাঙ্কেট)

// সংখ্যা ফরম্যাটিং
ফরম্যাট = #,|১২৩৪৫৬৭|  // → ১,২৩৪,৫৬৭  (কমা-বিচ্ছিন্ন)
বৈজ্ঞানিক = #^|১২৩৪৫.৬৭৮|    // → ১.২৩৪৫৬৭৮e৪  (বৈজ্ঞানিক)

// বেস লিটারেল
a = 0x41         // → 'A'  (হেক্স)
b = 0b01000001   // → 'A'  (বাইনারি)
c = 0o101        // → 'A'  (অক্টাল)

// বেস রূপান্তর আউটপুট
হেক্স = 0x|২৫৫|    // → "0x00FF"
বাইনারি = 0b|৬৫|     // → "0b1000001"
অক্টাল = 0o|৮|      // → "0o10"
দশমিক = 0d|২৫৫|    // → "0d0255"
```

## শেল ইন্টিগ্রেশন

```zymbol
তারিখ = <\ date +%Y-%m-%d \>     // stdout ক্যাপচার করে (শেষে \n অন্তর্ভুক্ত)
>> "আজ: " তারিখ

ফাইল = "data.txt"
বিষয়বস্তু = <\ cat {ফাইল} \>      // কমান্ডে ইন্টারপোলেশন

আউটপুট = </"./subscript.zy"/>   // অন্য Zymbol স্ক্রিপ্ট 실행 করুন, আউটপুট ক্যাপচার করুন
>> আউটপুট
```

> `><` CLI আর্গুমেন্ট স্ট্রিং অ্যারে হিসেবে ক্যাপচার করে (শুধু tree-walker)।

## রিলিজ চেঞ্জলগ

### v0.0.5 — টিইউআই প্রিমিটিভ, হট ডেফিনিশন ও স্ট্রিং রিপিট _(মে ২০২৬)_

- **ব্রেকিং** ম্যাচ আর্ম সেপারেটর: `প্যাটার্ন : ফলাফল` → `প্যাটার্ন => ফলাফল`
- **ব্রেকিং** ইম্পোর্ট এলিয়াস: `<# পাথ <= এলিয়াস` → `<# পাথ => এলিয়াস`
- **ব্রেকিং** এক্সপোর্ট রিনেম: `#> { fn <= পাব }` → `#> { fn => পাব }`
- **যোগ করা হয়েছে** টিইউআই ব্লক `>>| { }` — বিকল্প স্ক্রিন + র মোড; প্রস্থানে পরিষ্কার করে
- **যোগ করা হয়েছে** অবস্থানগত আউটপুট `>>~ (সারি, কলাম, BKS, fg, bg) > আইটেম` — বিচ্ছুরিত স্লট, ANSI 256 রঙ
- **যোগ করা হয়েছে** কী ইনপুট `<<| var` (ব্লকিং) ও `<<|? var` (নন-ব্লকিং পোল)
- **যোগ করা হয়েছে** `>>!` স্ক্রিন পরিষ্কার করুন, `>>?` টার্মিনাল সাইজ জিজ্ঞাসা করুন, `@~ N` N মিলিসেকেন্ড ঘুমান
- **যোগ করা হয়েছে** হট ডেফিনিশন `°x` / `x°` — লুপে প্রথম ব্যবহারে ভেরিয়েবল স্বয়ংক্রিয় আরম্ভ
- **যোগ করা হয়েছে** স্ট্রিং রিপিট `স্ট্রিং $* N` — একটি স্ট্রিং N বার পুনরাবৃত্তি করুন
- **ভিএম** সমতা: 436/436 টেস্ট পাস

### v0.0.4 — 1-ভিত্তিক সূচকীকরণ, প্রথম-শ্রেণীর ফাংশন ও ব্লক মডিউল _(এপ্রিল ২০২৬)_

- **ব্রেকিং** সমস্ত সূচকীকরণ **1-ভিত্তিকে** পরিবর্তিত হয়েছে — `arr[1]` প্রথম উপাদান; `arr[0]` একটি রানটাইম ত্রুটি
- **যোগ করা হয়েছে** নামকৃত ফাংশনগুলি **প্রথম-শ্রেণীর মান** — সরাসরি এইচওএফ-এ পাস করুন: `সংখ্যা$> দ্বিগুণ`
- **যোগ করা হয়েছে** মডিউলের জন্য **ব্লক সিনট্যাক্স প্রয়োজন** — `# নাম { ... }` — ফ্ল্যাট সিনট্যাক্স সরানো হয়েছে
- **যোগ করা হয়েছে** বহু-মাত্রিক সূচকীকরণ: `arr[i>j>k]` (নেভিগেশন), `arr[p ; q]` (ফ্ল্যাট নিষ্কাশন)
- **যোগ করা হয়েছে** টাইপ রূপান্তর কাস্ট: `##.এক্সপ্রেশন` (ভাসমান), `###এক্সপ্রেশন` (পূর্ণসংখ্যা রাউন্ড), `##!এক্সপ্রেশন` (পূর্ণসংখ্যা ট্রাঙ্কেট)
- **যোগ করা হয়েছে** স্ট্রিং বিভক্ত: `স্ট্রিং$/ সীমানা` — `Array(স্ট্রিং)` রিটার্ন করে
- **যোগ করা হয়েছে** কনক্যাট বিল্ড: `বেস$++ a b c` — একাধিক আইটেম যুক্ত করে
- **যোগ করা হয়েছে** টাইমস লুপ: `@ N { }` — ঠিক N বার পুনরাবৃত্তি করুন
- **যোগ করা হয়েছে** লেবেলযুক্ত লুপ সিনট্যাক্স: `@:নাম { }`, `@:নাম!`, `@:নাম>` — `@ @নাম` / `@! নাম` প্রতিস্থাপন করে
- **যোগ করা হয়েছে** ভেরিয়েবল স্কোপ নিয়ম: `_নাম` ভেরিয়েবলের সুনির্দিষ্ট ব্লক স্কোপ আছে; `\ var` তাড়াতাড়ি ধ্বংস করে
- **যোগ করা হয়েছে** ম্যাচ তুলনা প্যাটার্ন: `< 0 =>`, `> 5 =>`, `== 42 =>` ইত্যাদি
- **যোগ করা হয়েছে** মডিউল E013 ত্রুটি: মডিউল বডিতে কার্যকর স্টেটমেন্ট নিষিদ্ধ
- **ফিক্স করা হয়েছে** `alias.CONST` এখন সঠিকভাবে সমাধান করে; `#>` ফাংশন সংজ্ঞার পরে আসতে পারে
- **ভিএম** পূর্ণ সমতা: 393/393 টেস্ট পাস
