| قسم | لفظی | `#?` ٹیگ | نوٹس |
|------|---------|----------|---------|
| عدد صحیح | `42`، `-7` | `###` | 64 بٹ دستخط شدہ |
| اعشاری | `3.14`، `1.5e10` | `##.` | سائنسی علامت درست |
| سٹرنگ | `"متن"` | `##"` | درون کاری: `"ہیلو {نام}"` |
| حرف | `'الف'` | `##'` | ایک یونیکوڈ حرف |
| بولین | `#1`، `#0` | `##?` | عددی نہیں — `#1 ≠ 1` |
| صف | `[1, 2, 3]` | `##]` | یکساں عناصر |
| ٹپل | `(الف، ب)` | `##)` | مقامی |
| نام والا ٹپل | `(س: 1، ص: 2)` | `##)` | نام والے فیلڈز |
| فنکشن | نام والے فنکشن کا حوالہ | `##()` | فرسٹ کلاس؛ ظاہری شکل `<funct/N>` |

## TUI پرائمیٹوز

انٹرایکٹو پروگراموں کے لیے ٹرمینل یوزر انٹرفیس آپریٹرز۔ زیادہ تر کو `>>| { }` بلاک کی ضرورت ہوتی ہے (متبادل اسکرین + خام موڈ)۔

```zymbol
>>| {
    >>!                             // متبادل اسکرین صاف کریں
    >>~ (۱, ۱, ۰, ۱۰) > "چل رہا ہے"  // قطار ۱, کالم ۱, fg=۱۰ (سبز)
    @~ ۱۰۰۰                         // ۱ سیکنڈ توقف (۱۰۰۰ ms)
    >>~ (۲, ۱) > "مکمل ہوا۔"
}
// باہر نکلنے پر ٹرمینل خود بخود بحال ہو جاتا ہے
```

```zymbol
// کی پریس اور ٹرمینل کا سائز
>>| {
    [قطاریں, کالم] = >>?              // ٹرمینل کے طول و عرض پوچھیں
    >>~ (۱, ۱) > "ٹرمینل: " قطاریں " x " کالم
    <<| کلید                         // مسدود کرنے والی کی پریس پڑھیں
    >>~ (۲, ۱) > "دبایا: " کلید
}
```

> `>>!` اسکرین صاف کرتا ہے۔ `>>?` `[قطاریں, کالم]` واپس کرتا ہے۔ `@~ ن` ن ملی سیکنڈ سوتا ہے۔
> `<<|` ایک کی پریس پڑھتا ہے (مسدود کرنے والا); `<<|?` بغیر مسدود کیے پول کرتا ہے (اگر کوئی نہ ہو تو `'\0'` واپس کرتا ہے)۔
> پوزیشن والا آؤٹ پٹ ٹپل: `(قطار, کالم, BKS, fg, bg)` — کوئی بھی سلاٹ کاما کے ساتھ چھوڑا جا سکتا ہے (`>>~ (,,, 196) > "سرخ"`)۔
> BKS بٹ ماسک: `1`=بولڈ، `2`=ترچھا، `4`=انڈرلائن۔ ANSI 256 رنگ پیلیٹ (`0`=ٹرمینل ڈیفالٹ)۔
> صرف tree-walker (سوائے `>>!`، `>>?`، `@~`، `>>~` جو `--vm` میں بھی چلتے ہیں)۔

## صفیں

صفیں **قابل تغیر** ہوتی ہیں اور **ایک ہی قسم** کے عناصر رکھتی ہیں۔

```zymbol
صف = [۱, ۲, ۳, ۴, ۵]

س = صف[۱]      // ۱ — رسائی (۱-based: پہلا عنصر)
س = صف[-۱]     // ۵ — منفی اشاریہ (آخری عنصر)
س = صف$#       // ۵ — لمبائی (>> میں (صف$#) استعمال کریں)

صف = صف$+ ۶            // شامل کریں → [۱,۲,۳,۴,۵,۶]
صف۲ = صف$+[۲] ۹۹       // پوزیشن ۲ پر داخل کریں (۱-based)
صف۳ = صف$- ۳           // قدر کی پہلی موجودگی کو ہٹائیں
صف۴ = صف$-- ۳          // تمام موجودگیاں ہٹائیں
صف۵ = صف$-[۱]          // اشاریہ ۱ پر ہٹائیں (پہلا عنصر)
صف۶ = صف$-[۲..۳]       // حد ہٹائیں (۱-based, آخر شامل)

ہے = صف$? ۳            // #۱ — شامل ہے
مقامات = صف$?? ۳           // [۳] — قدر کے تمام اشاریے (۱-based)
سلائس = صف$[۱..۳]          // [۱,۲,۳] — سلائس (۱-based, آخر شامل)
سلائس۲ = صف$[۱:۳]          // [۱,۲,۳] — وہی, شمار پر مبنی نحو

صعودی = صف$^+             // صعودی ترتیب دیا گیا (صرف قدیم اقسام)
نزولی = صف$^-            // نزولی ترتیب دیا گیا (صرف قدیم اقسام)

// نام والے/مقامی ٹپل صفیں — موازنہ کرنے والے لیمبڈا کے ساتھ $^ استعمال کریں
ڈیٹا = [(نام: "کارلا", عمر: ۲۸), (نام: "انا", عمر: ۲۵), (نام: "بوب", عمر: ۳۰)]
عمر_کے_حسب = ڈیٹا$^ (الف, ب -> الف.عمر < ب.عمر)    // عمر کے لحاظ سے صعودی (<)
نام_کے_حسب = ڈیٹا$^ (الف, ب -> الف.نام > ب.نام)   // نام کے لحاظ سے نزولی (>)
>> عمر_کے_حسب[۱].نام ¶     // → انا
>> نام_کے_حسب[۱].نام ¶    // → کارلا

// براہ راست عنصر کی تبدیلی (صرف صفیں)
صف[۱] = ۹۹              // تفویض
صف[۲] += ۵              // مرکب: +=  -=  *=  /=  %=  ^=

// فنکشنل تبدیلی — ایک نئی صف واپس کرتا ہے؛ اصل میں کوئی تبدیلی نہیں
صف۲ = صف[۲]$~ ۹۹
```

> تمام کلیکشن آپریٹرز ایک **نئی صف** واپس کرتے ہیں۔ واپس تفویض کریں: `صف = صف$+ 4`۔
> `$+` کو زنجیر کیا جا سکتا ہے: `صف = صف$+ 5$+ 6$+ 7`۔ دیگر آپریٹرز درمیانی تفویض استعمال کرتے ہیں۔
> **اشاریہ سازی 1-based**: `صف[1]` پہلا عنصر ہے؛ `صف[0]` رن ٹائم ایرر ہے۔
> `$^+` / `$^-` **قدیم اقسام کی صفوں** (اعداد، سٹرنگز) کو ترتیب دیتے ہیں۔ ٹپل صفوں کے لیے موازنہ کرنے والے لیمبڈا کے ساتھ `$^` استعمال کریں — سمت لیمبڈا میں انکوڈ شدہ ہے (`<` = صعودی، `>` = نزولی)۔

**قدر کی سیمنٹکس** — صف کو دوسرے متغیر میں تفویض کرنے سے ایک آزاد کاپی بنتی ہے:

```zymbol
الف = [۱, ۲, ۳]
ب = الف
الف[۱] = ۹۹
>> الف ¶    // → [۹۹, ۲, ۳]
>> ب ¶    // → [۱, ۲, ۳]   ← ب متاثر نہیں ہوتا
```

```zymbol
// نیسٹڈ صفیں (۱-based اشاریہ سازی)
میٹرکس = [[۱,۲,۳],[۴,۵,۶],[۷,۸,۹]]
>> میٹرکس[۲][۳] ¶    // → ۶  (قطار ۲, کالم ۳)
```

```zymbol
[س, _, ض] = [۱, ۲, ۳]        // _ خارج کرتا ہے

// مقامی ٹپل
نقطہ = (۱۰۰, ۲۰۰)
(پکس, پوائے) = نقطہ             // پکس=۱۰۰  پوائے=۲۰۰

// نام والا ٹپل
شخص = (نام: "انا", عمر: ۲۵, شہر: "مادرید")
(نام: ن, عمر: ع) = شخص   // ن="انا"  ع=۲۵
```

## ٹپل

ٹپل **ناقابل تغیر** ترتیب شدہ کنٹینر ہوتے ہیں جو **مختلف اقسام** کی قدریں رکھ سکتے ہیں۔
صفوں کے برعکس، تخلیق کے بعد عناصر کو تبدیل نہیں کیا جا سکتا۔

```zymbol
// مقامی — مخلوط اقسام کی اجازت ہے
نقطہ = (۱۰, ۲۰)
>> نقطہ[۱] ¶    // → ۱۰

ڈیٹا = (۴۲, "ہیلو", #۱, ۳.۱۴)
>> ڈیٹا[۳] ¶     // → #۱

// نام والا
شخص = (نام: "ایلس", عمر: ۲۵)
>> شخص.نام ¶    // → ایلس
>> شخص[۱] ¶      // → ایلس  (اشاریہ بھی کام کرتا ہے, ۱-based)

// نیسٹڈ
پوزیشن = (س: ۱۰, ص: ۲۰)
پ = (پوزیشن: پوزیشن, لیبل: "اصل")
>> پ.پوزیشن.س ¶        // → ۱۰
```

**ناقابل تغیر** — ٹپل عنصر کو تبدیل کرنے کی کوئی بھی کوشش رن ٹائم ایرر ہے:

```zymbol
ٹ = (۱۰, ۲۰, ۳۰)
// ٹ[۱] = ۹۹    // ❌ رن ٹائم ایرر: ٹپل ناقابل تغیر ہیں
// ٹ[۱] += ۵    // ❌ وہی ایرر
```

تبدیل شدہ قدر حاصل کرنے کے لیے `$~` استعمال کریں (فنکشنل تبدیلی) — ایک **نیا** ٹپل واپس کرتا ہے:

```zymbol
ٹ = (۱۰, ۲۰, ۳۰)
ٹ۲ = ٹ[۲]$~ ۹۹۹
>> ٹ ¶     // → (۱۰, ۲۰, ۳۰)   ← اصلی میں کوئی تبدیلی نہیں
>> ٹ۲ ¶    // → (۱۰, ۹۹۹, ۳۰)

// نام والا ٹپل — واضح طور پر دوبارہ تعمیر کریں
شخص = (نام: "ایلس", عمر: ۲۵)
بڑا  = (نام: شخص.نام, عمر: ۲۶)
>> شخص.عمر ¶    // → ۲۵
>> بڑا.عمر ¶     // → ۲۶
```

## اعلیٰ درجے کے فنکشنز

```zymbol
اعداد = [۱, ۲, ۳, ۴, ۵, ۶, ۷, ۸, ۹, ۱۰]

دوگنا  = اعداد$> (س -> س * ۲)                  // map  → [۲,۴,۶…۲۰]
جفت    = اعداد$| (س -> س % ۲ == ۰)           // filter → [۲,۴,۶,۸,۱۰]
کل    = اعداد$< (۰, (اکیومولیٹر, س) -> اکیومولیٹر + س)     // reduce → ۵۵

// انٹرمیڈیٹس کے ذریعے زنجیر
مرحلہ۱ = اعداد$| (س -> س > ۳)
مرحلہ۲ = مرحلہ۱$> (س -> س * س)
>> مرحلہ۲ ¶    // → [۱۶, ۲۵, ۳۶, ۴۹, ۶۴, ۸۱, ۱۰۰]

// نام والے فنکشنز کو براہ راست HOF میں پاس کیا جا سکتا ہے
دوگنا(س) { <~ س * ۲ }
بڑا_ہے(س) { <~ س > ۵ }
ر = اعداد$> دوگنا       // ✅ براہ راست حوالہ
ر = اعداد$| بڑا_ہے       // ✅ براہ راست حوالہ
```

## پائپ آپریٹر

دائیں جانب کو ہمیشہ پائپ شدہ قدر کے لیے `_` بطور پلیس ہولڈر درکار ہوتا ہے:

```zymbol
دوگنا = س -> س * ۲
جمع = (الف, ب) -> الف + ب
ایک_اضافہ = س -> س + ۱

ن۱ = ۵ |> دوگنا(_)        // → ۱۰
ن۲ = ۱۰ |> جمع(_, ۵)       // → ۱۵
ن۳ = ۵ |> جمع(۲, _)        // → ۷

// زنجیر
ن = ۵ |> دوگنا(_) |> ایک_اضافہ(_) |> دوگنا(_)
>> ن ¶    // → ۲۲  (۵→۱۰→۱۱→۲۲)
```

## عددی موڈز

Zymbol اعداد کو **69 یونیکوڈ ہندسوں کے نظاموں** میں ظاہر کر سکتا ہے — دیوناگری، عربی-ہندی، تھائی، کلنگون pIqaD، ریاضیاتی بولڈ، LCD حصے، اور مزید۔ فعال موڈ صرف `>>` آؤٹ پٹ کو متاثر کرتا ہے؛ اندرونی ریاضی ہمیشہ بائنری ہوتی ہے۔

### اسکرپٹ کو فعال کرنا

ہدف اسکرپٹ کے `0` اور `9` ہندسے `#…#` کے اندر لکھیں:

```zymbol
#०९#    // دیوناگری   (U+0966–U+096F)
#٠٩#    // عربی-ہندی (U+0660–U+0669)
#๐๙#    // تھائی     (U+0E50–U+0E59)
#09#    // ASCII پر دوبارہ ترتیب دیں
```

### آؤٹ پٹ اور بولینز

```zymbol
س = ۴۲
>> س ¶          // → ۴۲   (ASCII ڈیفالٹ)

#०९#
>> س ¶          // → ४२
>> ۳.۱۴ ¶       // → ३.१४   (اعشاری نقطہ ہمیشہ ASCII)
>> ۱ + ۲ ¶      // → ३

// بولینز: # سابقہ ہمیشہ ASCII, ہندسہ ڈھل جاتا ہے
>> #۱ ¶         // → #१   (دیوناگری میں سچ)
>> #۰ ¶         // → #०   (غلط — ० عدد صحیح صفر سے ممتاز)

س = ۲۸ > ۴
>> س ¶          // → #१   (موازنہ کا نتیجہ فعال موڈ کی پیروی کرتا ہے)
```

### ماخذ میں مقامی ہندسوں کے لفظی

کوئی بھی تعاون یافتہ اسکرپٹ کے ہندسے درست لفظی ہیں — حدود، ماڈیولو، موازنوں میں:

```zymbol
#०९#

@ ی:१..१५ {
    ? ی % १५ == ० { >> "FizzBuzz" ¶ }
    _? ی % ३  == ० { >> "Fizz" ¶ }
    _? ی % ५  == ० { >> "Buzz" ¶ }
    _ { >> ی ¶ }
}
```

### کسی بھی اسکرپٹ میں بولین لفظی

`#` + کسی بھی بلاک سے ہندسہ `0` یا `1` ایک درست بولین لفظی ہے:

```zymbol
#٠٩#
فعال = #١        // #۱ کے برابر
>> فعال ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **ہمیشہ ASCII** ہے۔ `#0` (غلط) ہمیشہ ہر اسکرپٹ میں `0` (عدد صحیح صفر) سے بصری طور پر ممتاز ہوتا ہے۔

## ریلیز چینج لاگ

### v0.0.5 — TUI پرائمیٹوز، گرم تعریف اور سٹرنگ دہرائیں _(مئی 2026)_

- **توڑنے والا** میچ آر بازو جداکار: `پیٹرن : نتیجہ` → `پیٹرن => نتیجہ`
- **توڑنے والا** درآمد عرف: `<# راستہ <= عرف` → `<# راستہ => عرف`
- **توڑنے والا** برآمد نام تبدیل: `#> { فنکشن <= عوامی }` → `#> { فنکشن => عوامی }`
- **شامل کیا گیا** TUI بلاک `>>| { }` — متبادل اسکرین + خام موڈ؛ باہر نکلنے پر صاف کرتا ہے
- **شامل کیا گیا** پوزیشن والا آؤٹ پٹ `>>~ (قطار, کالم, BKS, fg, bg) > اشیاء` — ویران سلاٹس، ANSI 256 رنگ
- **شامل کیا گیا** کلید ان پٹ `<<| متغیر` (مسدود کرنے والا) اور `<<|? متغیر` (غیر مسدود کرنے والا پول)
- **شامل کیا گیا** `>>!` اسکرین صاف کریں، `>>?` ٹرمینل سائز پوچھیں، `@~ ن` ن ملی سیکنڈ سوئے
- **شامل کیا گیا** گرم تعریف `°س` / `س°` — لوپس میں پہلے استعمال پر متغیر خودکار آغاز
- **شامل کیا گیا** سٹرنگ دہرائیں `سٹر $* ن` — سٹرنگ کو ن بار دہرائیں
- **VM** برابری: 436/436 ٹیسٹ پاس

### v0.0.4 — 1-based اشاریہ سازی، فرسٹ کلاس فنکشنز اور بلاک ماڈیولز _(اپریل 2026)_

- **توڑنے والا** تمام اشاریہ سازی **1-based** میں تبدیل — `صف[1]` پہلا عنصر ہے؛ `صف[0]` رن ٹائم ایرر ہے
