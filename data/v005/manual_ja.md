# Zymbol-Lang マニュアル

> **v0.0.5 改訂版 — 2026-05-12**

**Zymbol-Lang** はシンボリックプログラミング言語です。キーワードなし — すべてがシンボルです。あらゆる人間の言語で同一に動作します。

- `if`、`while`、`return` なし — `?`、`@`、`<~` のみ
- 完全なUnicode — あらゆる言語や絵文字の識別子
- 人間の言語に依存しない — コードはどこでも同じ

**インタープリターバージョン**：v0.0.5 | **テストカバレッジ**：436/436（TW ↔ VMパリティ）

## 変数と定数

```zymbol
x = 10              // 変更可能な変数
PI := 3.14159       // 定数 — 再代入はランタイムエラー
名前 = "アリス"
有効 = #1           // 真偽値 真
👋 := "こんにちは"
```

```zymbol
x = 10    // 10
x += 5    // 15
x -= 3    // 12
x *= 2    // 24
x /= 3    // 8
x %= 3    // 2
x ^= 2    // 4
x++        // 5
x--        // 4
```

`°`（度記号、U+00B0）は最初の使用時に変数をその中立値に自動初期化します：

```zymbol
数値 = [3, 1, 4, 1, 5]
@ n:数値 {
    °合計 += n    // ループより上で0に自動初期化；@後も存続
}
>> 合計 ¶         // → 14
```

> `°x`（前置）はループの上に固定 — 結果は `@` の後もアクセス可能。
> `x°`（後置）はループの内側に固定 — ループが終わると消滅。
> ツリーウォーカーのみ。

## データ型

| 型 | リテラル | `#?` タグ | 備考 |
|-------|---------|----------|------|
| Int | `42`, `-7` | `###` | 64ビット符号付き |
| Float | `3.14`, `1.5e10` | `##.` | 科学的表記OK |
| String | `"テキスト"` | `##"` | 補間：`"こんにちは{名前}"` |

## TUIプリミティブ

インタラクティブプログラム用の端末UIオペレーター。ほとんどは `>>| { }` ブロック（代替画面 + rawモード）が必要です。

```zymbol
>>| {
    >>!                               // 代替画面をクリア
    >>~ (1, 1, 0, 10) > "実行中"      // 行1、列1、fg=10（緑）
    @~ 1000                           // 1秒一時停止（1000ミリ秒）
    >>~ (2, 1) > "完了。"
}
// 終了時に端末が自動的に復元される
```

```zymbol
// キー入力と端末サイズ
>>| {
    [行, 列] = >>?                    // 端末のサイズを取得
    >>~ (1, 1) > "ターミナル：" 行 " x " 列
    <<| キー                          // ブロッキングキー入力読み込み
    >>~ (2, 1) > "入力：" キー
}
```

> `>>!` 画面消去。`>>?` は `[行, 列]` を返す。`@~ N` はNミリ秒スリープ。
> `<<|` は1キー入力を読む（ブロッキング）；`<<|?` はブロックなしでポーリング（なければ `'\0'` を返す）。
> 位置出力タプル：`(行, 列, BKS, fg, bg)` — どのスロットもカンマで省略可（`>>~ (,,, 196) > "赤"`）。
> BKSビットマスク：`1`=太字、`2`=斜体、`4`=下線。ANSI256色パレット（`0`=端末デフォルト）。
> ツリーウォーカーのみ（ただし `>>!`、`>>?`、`@~`、`>>~` は `--vm` でも動作）。

## ループ

```zymbol
@ i:0..4  { >> i " " }        // 範囲（含む）：  0 1 2 3 4
@ i:1..9:2 { >> i " " }       // ステップ付き：  1 3 5 7 9
@ i:5..0:1 { >> i " " }       // 逆順：          5 4 3 2 1 0

n = 1
@ n <= 64 { n *= 2 }
>> n ¶                        // → 128  (while)

果物 = ["りんご", "なし", "ぶどう"]
@ f:果物 { >> f ¶ }           // 配列をfor-each

@ c:"こんにちは" { >> c "-" }
>> ¶                          // → こ-ん-に-ち-は-  (文字列をfor-each)

@ i:1..10 {
    ? i % 2 == 0 { @> }       // @> 継続
    ? i > 7 { @! }             // @! 中断
    >> i " "
}
>> ¶                          // → 1 3 5 7

// 無限ループ
i = 0
@ {
    i++
    ? i >= 5 { @! }
    >> i " "
}
>> ¶                          // → 1 2 3 4

// ラベル付きループ（入れ子の中断）
数 = 0
@:外 {
    数++
    ? 数 >= 3 { @:外! }
}
>> 数 ¶                    // → 3
```

## 関数

```zymbol
加算(a, b) { <~ a + b }
>> 加算(3, 4) ¶    // → 7

階乗(n) {
    ? n <= 1 { <~ 1 }
    <~ n * 階乗(n - 1)
}
>> 階乗(5) ¶    // → 120
```

関数は**独立したスコープ**を持ちます — 外部変数を読み取れません。出力パラメーター `<~` を使って呼び出し元の変数を変更します：

```zymbol
交換(a<~, b<~) {
    tmp = a
    a = b
    b = tmp
}
x = 10
y = 20
交換(x, y)
>> "x=" x " y=" y ¶    // → x=20 y=10
```

> 名前付き関数は**第一級の値**です — 直接渡せます：`数値$> 二倍`。ラップする場合：`x -> 関数(x)` も有効。

## 配列

配列は**変更可能**で、**同じ型**の要素を保持します。

```zymbol
arr = [1, 2, 3, 4, 5]

x = arr[1]      // 1 — アクセス（1始まり：最初の要素）
x = arr[-1]     // 5 — 負のインデックス（最後の要素）
x = arr$#       // 5 — 長さ（>> では (arr$#) を使用）

arr = arr$+ 6            // 追加 → [1,2,3,4,5,6]
arr2 = arr$+[2] 99       // 位置2に挿入（1始まり）
arr3 = arr$- 3           // 値の最初の出現を削除
arr4 = arr$-- 3          // すべての出現を削除
arr5 = arr$-[1]          // インデックス1を削除（最初の要素）
arr6 = arr$-[2..3]       // 範囲を削除（1始まり、終端含む）

含む = arr$? 3            // #1 — 含む
位置 = arr$?? 3           // [3] — 値のすべてのインデックス（1始まり）
切片 = arr$[1..3]         // [1,2,3] — スライス（1始まり、終端含む）
切片2 = arr$[1:3]         // [1,2,3] — 同じ、個数ベース構文

昇順 = arr$^+             // 昇順ソート  （プリミティブのみ）
降順 = arr$^-             // 降順ソート  （プリミティブのみ）

// 名前付き/位置タプル配列 — 比較ラムダで $^ を使用
db = [(名前: "カルラ", 年齢: 28), (名前: "アナ", 年齢: 25), (名前: "ボブ", 年齢: 30)]
年齢順  = db$^ (a, b -> a.年齢 < b.年齢)    // 年齢の昇順（<）
名前順  = db$^ (a, b -> a.名前 > b.名前)    // 名前の降順（>）
>> 年齢順[1].名前 ¶     // → アナ
>> 名前順[1].名前 ¶     // → カルラ

// 直接要素更新（配列のみ）
arr[1] = 99              // 代入
arr[2] += 5              // 複合：+=  -=  *=  /=  %=  ^=

// 関数的更新 — 新しい配列を返す；元は不変
arr2 = arr[2]$~ 99
```

> すべてのコレクション演算子は**新しい配列**を返します。代入し直す：`arr = arr$+ 4`。
> `$+` はチェーン可：`arr = arr$+ 5$+ 6$+ 7`。他の演算子は中間代入を使用。
> **インデックスは1始まり**：`arr[1]` が最初の要素；`arr[0]` はランタイムエラー。
> `$^+` / `$^-` は**プリミティブ配列**（数値、文字列）をソート。タプル配列は比較ラムダで `$^` を使用 — 方向はラムダに記述（`<` = 昇順、`>` = 降順）。

**値セマンティクス** — 配列を別の変数に代入すると独立したコピーが作られます：

```zymbol
a = [1, 2, 3]
b = a
a[1] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b は影響を受けない
```

```zymbol
// ネスト配列（1始まりのインデックス）
行列 = [[1,2,3],[4,5,6],[7,8,9]]
>> 行列[2][3] ¶    // → 6  （行2、列3）
```

## タプル

タプルは**変更不可**で、**異なる型**の値を保持できる順序付きコンテナです。
配列と異なり、作成後に要素を変更できません。

```zymbol
// 位置 — 混合型可
点 = (10, 20)
>> 点[1] ¶    // → 10

data = (42, "こんにちは", #1, 3.14)
>> data[3] ¶     // → #1

// 名前付き
人物 = (名前: "アリス", 年齢: 25)
>> 人物.名前 ¶    // → アリス
>> 人物[1] ¶      // → アリス  （インデックスも使用可能、1始まり）

// ネスト
位置 = (x: 10, y: 20)
p = (位置: 位置, ラベル: "原点")
>> p.位置.x ¶        // → 10
```

**変更不可性** — タプル要素の変更を試みるとランタイムエラーになります：

```zymbol
t = (10, 20, 30)
// t[1] = 99    // ❌ ランタイムエラー：タプルは変更不可
// t[1] += 5    // ❌ 同じエラー
```

変更された値を得るには `$~`（関数的更新）を使用 — **新しい**タプルを返します：

```zymbol
t = (10, 20, 30)
t2 = t[2]$~ 999
>> t ¶     // → (10, 20, 30)   ← 元は変更されない
>> t2 ¶    // → (10, 999, 30)

// 名前付きタプル — 明示的に再構築
人物 = (名前: "アリス", 年齢: 25)
年上  = (名前: 人物.名前, 年齢: 26)
>> 人物.年齢 ¶    // → 25
>> 年上.年齢 ¶    // → 26
```

## 数字モード

Zymbolは**69のUnicode数字スクリプト**で数値を表示できます — デーヴァナーガリー、アラビア・インド文字、タイ文字、クリンゴンpIqaD、数学太字、LCD文字など。アクティブなモードは `>>` 出力にのみ影響します；内部演算は常に2進数です。

### スクリプトの有効化

対象スクリプトの `0` と `9` の数字を `#…#` で囲んで記述します：

```zymbol
#०९#    // デーヴァナーガリー   (U+0966–U+096F)
#٠٩#    // アラビア・インド文字 (U+0660–U+0669)
#๐๙#    // タイ文字             (U+0E50–U+0E59)
#09#    // ASCIIにリセット
```

### 出力と真偽値

```zymbol
x = 42
>> x ¶          // → 42   （ASCIIデフォルト）

#०९#
>> x ¶          // → ४२
>> 3.14 ¶       // → ३.१४   （小数点は常にASCII）
>> 1 + 2 ¶      // → ३

// 真偽値：# プレフィックスは常にASCII、数字は適応
>> #1 ¶         // → #१   （デーヴァナーガリーの真）
>> #0 ¶         // → #०   （偽 — ०整数ゼロとは別）

x = 28 > 4
>> x ¶          // → #१   （比較結果はアクティブなモードに従う）
```

### ソースでのネイティブ数字リテラル

サポートされているスクリプトの数字は有効なリテラルです — 範囲、剰余、比較に使用できます：

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### 任意のスクリプトの真偽値リテラル

任意のブロックの `#` + 数字 `0` または `1` は有効な真偽値リテラルです：

```zymbol
#٠٩#
نشط = #١        // #1 と同じ
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` は**常にASCII**です。`#0`（偽）はすべてのスクリプトで `0`（整数ゼロ）と視覚的に異なります。

## データ演算子

```zymbol
// 型変換キャスト
f = ##.42         // → 42.0  （Floatへ）
i = ###3.7        // → 4     （Intへ、丸め）
t = ##!3.7        // → 3     （Intへ、切り捨て）

// 文字列を数値に変換
v1 = #|"42"|      // → 42  (Int)
v2 = #|"3.14"|    // → 3.14  (Float)
v3 = #|"abc"|     // → "abc"  （フェイルセーフ、エラーなし）

// 丸め / 切り捨て
pi = 3.14159265
r2 = #.2|pi|      // → 3.14  （小数2桁に丸め）
r4 = #.4|pi|      // → 3.1416
t2 = #!2|pi|      // → 3.14  （切り捨て）

// 数値フォーマット
fmt = #,|1234567|      // → 1,234,567  （カンマ区切り）
sci = #^|12345.678|    // → 1.2345678e4  （科学的表記）

// 基数リテラル
a = 0x41         // → 'A'  （16進数）
b = 0b01000001   // → 'A'  （2進数）
c = 0o101        // → 'A'  （8進数）

// 基数変換出力
hex = 0x|255|    // → "0x00FF"
bin = 0b|65|     // → "0b1000001"
oct = 0o|8|      // → "0o10"
dec = 0d|255|    // → "0d0255"
```

## シェル統合

```zymbol
日付 = <\ date +%Y-%m-%d \>     // stdoutをキャプチャ（末尾の\nを含む）
>> "今日：" 日付

ファイル = "data.txt"
内容 = <\ cat {ファイル} \>     // コマンド内での補間

出力 = </"./サブスクリプト.zy"/>   // 別のZymbolスクリプトを実行、出力をキャプチャ
>> 出力
```

> `><` はCLI引数を文字列配列としてキャプチャします（ツリーウォーカーのみ）。

## 完全な例：FizzBuzz

```zymbol
判定(数値) {
    ? 数値 % 15 == 0 { <~ "FizzBuzz" }
    _? 数値 % 3  == 0 { <~ "Fizz" }
    _? 数値 % 5  == 0 { <~ "Buzz" }
    _ { <~ 数値 }
}

@ i:1..20 { >> 判定(i) ¶ }
```
