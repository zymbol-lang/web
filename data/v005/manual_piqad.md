## ␣␣␣␣␣

␣␣␣␣␣ **␣␣␣␣␣␣** ␣␣␣␣␣␣␣ ␣␣␣␣␣ **␣␣␣␣ ␣␣␣␣**.

```zymbol
arr = [1, 2, 3, 4, 5]

x = arr[1]      // 1 — ␣␣␣␣␣ (1-␣␣␣␣: ␣␣␣␣␣␣)
x = arr[-1]     // 5 — ␣␣␣␣␣␣ ␣␣␣␣␣ (␣␣␣␣␣␣)
x = arr$#       // 5 — ␣␣␣␣ (␣␣␣␣␣ (arr$#) ␣␣ >>)

arr = arr$+ 6            // ␣␣␣␣ → [1,2,3,4,5,6]
arr2 = arr$+[2] 99       // ␣␣␣␣␣␣ 2 (1-␣␣␣␣)
arr3 = arr$- 3           // ␣␣␣␣ ␣␣␣␣␣␣
arr4 = arr$-- 3          // ␣␣␣␣ ␣␣␣␣␣
arr5 = arr$-[1]          // ␣␣␣␣ ␣␣␣␣␣ 1 (␣␣␣␣␣␣)
arr6 = arr$-[2..3]       // ␣␣␣␣ ␣␣␣␣␣␣ (1-␣␣␣␣, ␣␣␣␣␣␣)

␣␣␣ = arr$? 3            // #1 — ␣␣␣␣␣
␣␣␣ = arr$?? 3           // [3] — ␣␣␣␣␣ ␣␣␣␣␣ (1-␣␣␣␣)
␣␣␣ = arr$[1..3]          // [1,2,3] — ␣␣␣␣ (1-␣␣␣␣, ␣␣␣␣␣␣)
␣␣␣2 = arr$[1:3]          // [1,2,3] — ␣␣␣␣␣␣␣

␣␣␣␣ = arr$^+             // ␣␣␣␣␣␣␣␣ (␣␣␣␣␣␣)
␣␣␣␣ = arr$^-            // ␣␣␣␣␣␣␣␣ (␣␣␣␣␣␣)

// ␣␣␣␣␣␣/␣␣␣␣␣␣␣␣ — ␣␣␣␣␣ $^ ␣␣␣␣␣␣
␣␣␣ = [(␣␣␣: "␣␣␣␣", ␣␣␣: 28), (␣␣␣: "␣␣", ␣␣␣: 25), (␣␣␣: "␣␣␣", ␣␣␣: 30)]
␣␣␣  = ␣␣␣$^ (a, b -> a.␣␣␣ < b.␣␣␣)    // ␣␣␣␣␣␣␣␣ (<)
␣␣␣ = ␣␣␣$^ (a, b -> a.␣␣␣ > b.␣␣␣)   // ␣␣␣␣␣␣␣␣ (>)
>> ␣␣␣[1].␣␣␣ ¶     // → ␣␣
>> ␣␣␣[1].␣␣␣ ¶    // → ␣␣␣␣

// ␣␣␣␣␣␣␣␣␣ (␣␣␣␣␣)
arr[1] = 99              // ␣␣␣␣␣
arr[2] += 5              // ␣␣␣␣␣␣: +=  -=  *=  /=  %=  ^=

// ␣␣␣␣␣␣␣␣␣ — ␣␣␣␣␣␣; ␣␣␣␣␣␣␣
arr2 = arr[2]$~ 99
```

> ␣␣␣␣␣␣␣␣␣␣␣␣. ␣␣␣␣␣: `arr = arr$+ 4`.
> `$+` ␣␣␣␣␣␣: `arr = arr$+ 5$+ 6$+ 7`. ␣␣␣␣␣␣␣␣ ␣␣␣␣␣␣␣␣␣␣.
