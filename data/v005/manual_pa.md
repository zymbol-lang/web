## ਟੀਯੂਆਈ ਪ੍ਰਿਮਿਟਿਵ

ਇੰਟਰਐਕਟਿਵ ਪ੍ਰੋਗਰਾਮਾਂ ਲਈ ਟਰਮੀਨਲ ਯੂਜ਼ਰ ਇੰਟਰਫੇਸ ਓਪਰੇਟਰ। ਜ਼ਿਆਦਾਤਰ ਨੂੰ `>>| { }` ਬਲਾਕ ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ (ਬਦਲਵੀਂ ਸਕ੍ਰੀਨ + ਕੱਚਾ ਮੋਡ)।

```zymbol
>>| {
    >>!                             // ਬਦਲਵੀਂ ਸਕ੍ਰੀਨ ਸਾਫ਼ ਕਰੋ
    >>~ (੧, ੧, ੦, ੧੦) > "ਚੱਲ ਰਿਹਾ ਹੈ"   // ਕਤਾਰ ੧, ਕਾਲਮ ੧, fg=੧੦ (ਹਰਾ)
    @~ ੧੦੦੦                         // ੧ ਸਕਿੰਟ ਰੁਕੋ (੧੦੦੦ ms)
    >>~ (੨, ੧) > "ਮੁਕੰਮਲ ਹੋ ਗਿਆ।"
}
// ਬਾਹਰ ਨਿਕਲਣ 'ਤੇ ਟਰਮੀਨਲ ਆਪਣੇ ਆਪ ਰੀਸਟੋਰ ਹੋ ਜਾਂਦਾ ਹੈ
```

```zymbol
// ਕੀ ਪ੍ਰੈੱਸ ਅਤੇ ਟਰਮੀਨਲ ਆਕਾਰ
>>| {
    [ਕਤਾਰਾਂ, ਕਾਲਮ] = >>?              // ਟਰਮੀਨਲ ਦੇ ਅਯਾਮ ਪੁੱਛੋ
    >>~ (੧, ੧) > "ਟਰਮੀਨਲ: " ਕਤਾਰਾਂ " x " ਕਾਲਮ
    <<| ਕੁੰਜੀ                         // ਬਲਾਕਿੰਗ ਕੀ ਪ੍ਰੈੱਸ ਪੜ੍ਹੋ
    >>~ (੨, ੧) > "ਦਬਾਇਆ: " ਕੁੰਜੀ
}
```

> `>>!` ਸਕ੍ਰੀਨ ਸਾਫ਼ ਕਰਦਾ ਹੈ। `>>?` `[ਕਤਾਰਾਂ, ਕਾਲਮ]` ਵਾਪਸ ਕਰਦਾ ਹੈ। `@~ N` N ਮਿਲੀਸਕਿੰਟ ਸੌਂਦਾ ਹੈ।
> `<<|` ਇੱਕ ਕੀ ਪ੍ਰੈੱਸ ਪੜ੍ਹਦਾ ਹੈ (ਬਲਾਕਿੰਗ); `<<|?` ਬਿਨਾਂ ਬਲਾਕ ਕੀਤੇ ਪੋਲ ਕਰਦਾ ਹੈ (ਜੇ ਕੋਈ ਨਹੀਂ ਤਾਂ `'\0'` ਵਾਪਸ ਕਰਦਾ ਹੈ)।
> ਸਥਾਨਿਕ ਆਉਟਪੁਟ ਟਪਲ: `(ਕਤਾਰ, ਕਾਲਮ, BKS, fg, bg)` — ਕੋਈ ਵੀ ਸਲਾਟ ਕਾਮੇ ਨਾਲ ਛੱਡਿਆ ਜਾ ਸਕਦਾ ਹੈ (`>>~ (,,, 196) > "ਲਾਲ"`)।
> BKS ਬਿਟਮਾਸਕ: `1`=ਬੋਲਡ, `2`=ਤਿਰਛਾ, `4`=ਅੰਡਰਲਾਈਨ। ANSI 256 ਰੰਗ ਪੈਲੇਟ (`0`=ਟਰਮੀਨਲ ਡਿਫੌਲਟ)।
> ਸਿਰਫ tree-walker (`>>!`, `>>?`, `@~`, `>>~` ਤੋਂ ਇਲਾਵਾ ਜੋ `--vm` ਵਿੱਚ ਵੀ ਚਲਦੇ ਹਨ)।

ਫੰਕਸ਼ਨਾਂ ਦਾ **ਵੱਖਰਾ ਸਕੋਪ** ਹੁੰਦਾ ਹੈ — ਉਹ ਬਾਹਰੀ ਵੇਰੀਏਬਲ ਨਹੀਂ ਪੜ੍ਹ ਸਕਦੇ। ਕਾਲਰ ਵੇਰੀਏਬਲ ਬਦਲਣ ਲਈ ਆਉਟਪੁਟ ਪੈਰਾਮੀਟਰ `<~` ਵਰਤੋ:

```zymbol

    ਅਸਥਾਈ = a
    a = b
    b = ਅਸਥਾਈ
}
x = ੧੦
y = ੨੦
ਅਦਲਾ_ਬਦਲੀ(x, y)
>> "x=" x " y=" y ¶    // → x=੨੦ y=੧੦
```

> ਨਾਮਿਤ ਫੰਕਸ਼ਨ **ਪਹਿਲੀ-ਸ਼੍ਰੇਣੀ ਮੁੱਲ** ਹਨ — ਸਿੱਧਾ ਪਾਸ ਕਰੋ: `ਸੰਖਿਆਵਾਂ$> ਦੁੱਗਣਾ`। ਲਪੇਟਣ ਲਈ: `x -> fn(x)` ਵੀ ਮਾਨਯ ਹੈ।

## ਲੈਂਬਡਾ ਅਤੇ ਕਲੋਜਰ

```zymbol
ਦੁੱਗਣਾ = x -> x * ੨
ਜੋੜ = (a, b) -> a + b
>> ਦੁੱਗਣਾ(੫) ¶    // → ੧੦
>> ਜੋੜ(੩, ੭) ¶  // → ੧੦

// ਬਲਾਕ ਲੈਂਬਡਾ
ਵਰਗੀਕਰਨ = x -> {
    ? x > ੦ { <~ "ਸਕਾਰਾਤਮਕ" }
    _? x < ੦ { <~ "ਨਕਾਰਾਤਮਕ" }
    <~ "ਜ਼ੀਰੋ"
}

// ਕਲੋਜਰ — ਬਾਹਰੀ ਸਕੋਪ ਕੈਪਚਰ ਕਰਦਾ ਹੈ
ਗੁਣਕ = ੩
ਤਿਗੁਣਾ = x -> x * ਗੁਣਕ
>> ਤਿਗੁਣਾ(੭) ¶    // → ੨੧

// ਫੈਕਟਰੀ
ਜੋੜ_ਬਣਾਉਣ_ਵਾਲਾ(n) { <~ x -> x + n }
ਦਸ_ਜੋੜ = ਜੋੜ_ਬਣਾਉਣ_ਵਾਲਾ(੧੦)
>> ਦਸ_ਜੋੜ(੫) ¶    // → ੧੫

// ਐਰੇ ਵਿੱਚ
ਓਪਰੇਟਰ = [x -> x+੧, x -> x*੨, x -> x*x]
>> ਓਪਰੇਟਰ[੩](੫) ¶    // → ੨੫
```

## ਐਰੇ

ਐਰੇ **ਬਦਲਣਯੋਗ** ਹੁੰਦੇ ਹਨ ਅਤੇ **ਇੱਕੋ ਕਿਸਮ ਦੇ** ਤੱਤ ਰੱਖਦੇ ਹਨ।

```zymbol
arr = [੧, ੨, ੩, ੪, ੫]

x = arr[੧]      // ੧ — ਪਹੁੰਚ (੧-ਅਧਾਰਿਤ: ਪਹਿਲਾ ਤੱਤ)
x = arr[-੧]     // ੫ — ਨਕਾਰਾਤਮਕ ਸੂਚਕਾਂਕ (ਅੰਤਿਮ ਤੱਤ)
x = arr$#       // ੫ — ਲੰਬਾਈ (>> ਵਿੱਚ (arr$#) ਵਰਤੋ)

arr = arr$+ ੬            // ਜੋੜੋ → [੧,੨,੩,੪,੫,੬]
arr੨ = arr$+[੨] ੯੯       // ਸਥਿਤੀ ੨ ਵਿੱਚ ਪਾਓ (੧-ਅਧਾਰਿਤ)
arr੩ = arr$- ੩           // ਮੁੱਲ ਦੀ ਪਹਿਲੀ ਵਾਪਰੀ ਹਟਾਓ
arr੪ = arr$-- ੩          // ਸਾਰੀਆਂ ਵਾਪਰੀਆਂ ਹਟਾਓ
arr੫ = arr$-[੧]          // ਸੂਚਕਾਂਕ ੧ ਤੇ ਹਟਾਓ (ਪਹਿਲਾ ਤੱਤ)
arr੬ = arr$-[੨..੩]       // ਰੇਂਜ ਹਟਾਓ (੧-ਅਧਾਰਿਤ, ਅੰਤ ਸ਼ਾਮਲ)

ਹੈ = arr$? ੩            // #੧ — ਸ਼ਾਮਲ ਹੈ
ਸਥਾਨ = arr$?? ੩           // [੩] — ਮੁੱਲ ਦੇ ਸਾਰੇ ਸੂਚਕਾਂਕ (੧-ਅਧਾਰਿਤ)
ਸਲਾਈਸ = arr$[੧..੩]          // [੧,੨,੩] — ਸਲਾਈਸ (੧-ਅਧਾਰਿਤ, ਅੰਤ ਸ਼ਾਮਲ)
ਸਲਾਈਸ੨ = arr$[੧:੩]          // [੧,੨,੩] — ਇੱਕੋ, ਗਿਣਤੀ-ਅਧਾਰਿਤ ਸੰਟੈਕਸ

ਚੜ੍ਹਦਾ = arr$^+             // ਚੜ੍ਹਦੇ ਕ੍ਰਮ ਵਿੱਚ ਕ੍ਰਮਬੱਧ ਕਰੋ (ਸਿਰਫ ਆਦਿਮ)
ਲਹਿੰਦਾ = arr$^-            // ਲਹਿੰਦੇ ਕ੍ਰਮ ਵਿੱਚ ਕ੍ਰਮਬੱਧ ਕਰੋ (ਸਿਰਫ ਆਦਿਮ)

// ਨਾਮਿਤ/ਸਥਾਨਿਕ ਟਪਲ ਐਰੇ — ਤੁਲਨਾਕਾਰੀ ਲੈਂਬਡਾ ਨਾਲ $^ ਵਰਤੋ
ਡਾਟਾਬੇਸ = [(ਨਾਮ: "ਕਾਰਲਾ", ਉਮਰ: ੨੮), (ਨਾਮ: "ਅਨਾ", ਉਮਰ: ੨੫), (ਨਾਮ: "ਬੌਬ", ਉਮਰ: ੩੦)]
ਉਮਰ_ਅਨੁਸਾਰ  = ਡਾਟਾਬੇਸ$^ (a, b -> a.ਉਮਰ < b.ਉਮਰ)    // ਉਮਰ ਅਨੁਸਾਰ ਚੜ੍ਹਦਾ (<)
ਨਾਮ_ਅਨੁਸਾਰ = ਡਾਟਾਬੇਸ$^ (a, b -> a.ਨਾਮ > b.ਨਾਮ)   // ਨਾਮ ਅਨੁਸਾਰ ਲਹਿੰਦਾ (>)
>> ਉਮਰ_ਅਨੁਸਾਰ[੧].ਨਾਮ ¶     // → ਅਨਾ
>> ਨਾਮ_ਅਨੁਸਾਰ[੧].ਨਾਮ ¶    // → ਕਾਰਲਾ

// ਸਿੱਧਾ ਤੱਤ ਅਪਡੇਟ (ਸਿਰਫ ਐਰੇ)
arr[੧] = ੯੯              // ਅਸਾਈਨ
arr[੨] += ੫              // ਸੰਯੁਕਤ: +=  -=  *=  /=  %=  ^=

// ਫੰਕਸ਼ਨਲ ਅਪਡੇਟ — ਇੱਕ ਨਵਾਂ ਐਰੇ ਵਾਪਸ ਕਰਦਾ ਹੈ; ਮੂਲ ਅਪਰਿਵਰਤਿਤ
arr੨ = arr[੨]$~ ੯੯
```

> ਸਾਰੇ ਕਲੈਕਸ਼ਨ ਓਪਰੇਟਰ ਇੱਕ **ਨਵਾਂ ਐਰੇ** ਵਾਪਸ ਕਰਦੇ ਹਨ। ਵਾਪਸ ਅਸਾਈਨ ਕਰੋ: `arr = arr$+ 4`।
> `$+` ਨੂੰ ਲੜੀਬੱਧ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ: `arr = arr$+ 5$+ 6$+ 7`। ਹੋਰ ਓਪਰੇਟਰ ਵਿਚਕਾਰਲੀਆਂ ਅਸਾਈਨਮੈਂਟਾਂ ਵਰਤਦੇ ਹਨ।
> **ਸੂਚਕਾਂਕਨ 1-ਅਧਾਰਿਤ**: `arr[1]` ਪਹਿਲਾ ਤੱਤ ਹੈ; `arr[0]` ਇੱਕ ਰਨਟਾਈਮ ਗਲਤੀ ਹੈ।
> `$^+` / `$^-` **ਆਦਿਮ ਐਰੇ** (ਸੰਖਿਆਵਾਂ, ਸਤਰਾਂ) ਨੂੰ ਕ੍ਰਮਬੱਧ ਕਰਦੇ ਹਨ। ਟਪਲ ਐਰੇ ਲਈ ਤੁਲਨਾਕਾਰੀ ਲੈਂਬਡਾ ਨਾਲ `$^` ਵਰਤੋ — ਦਿਸ਼ਾ ਲੈਂਬਡਾ ਵਿੱਚ ਇਨਕੋਡ ਹੁੰਦੀ ਹੈ (`<` = ਚੜ੍ਹਦਾ, `>` = ਲਹਿੰਦਾ)।

**ਮੁੱਲ ਸਿਮੈਂਟਿਕਸ** — ਐਰੇ ਨੂੰ ਕਿਸੇ ਹੋਰ ਵੇਰੀਏਬਲ ਵਿੱਚ ਅਸਾਈਨ ਕਰਨ ਨਾਲ ਇੱਕ ਸੁਤੰਤਰ ਕਾਪੀ ਬਣਦੀ ਹੈ:

```zymbol
a = [੧, ੨, ੩]
b = a
a[੧] = ੯੯
>> a ¶    // → [੯੯, ੨, ੩]
>> b ¶    // → [੧, ੨, ੩]   ← b ਪ੍ਰਭਾਵਿਤ ਨਹੀਂ ਹੁੰਦਾ
```

```zymbol
// ਨੇਸਟਡ ਐਰੇ (੧-ਅਧਾਰਿਤ ਸੂਚਕਾਂਕਨ)
ਮੈਟ੍ਰਿਕਸ = [[੧,੨,੩],[੪,੫,੬],[੭,੮,੯]]
>> ਮੈਟ੍ਰਿਕਸ[੨][੩] ¶    // → ੬  (ਕਤਾਰ ੨, ਕਾਲਮ ੩)
```

## ਟਪਲ

ਟਪਲ **ਅਪਰਿਵਰਤਨੀ** ਕ੍ਰਮਬੱਧ ਕੰਟੇਨਰ ਹਨ ਜੋ **ਵੱਖ-ਵੱਖ ਕਿਸਮਾਂ ਦੇ** ਮੁੱਲ ਰੱਖ ਸਕਦੇ ਹਨ।
ਐਰੇ ਦੇ ਉਲਟ, ਸਿਰਜਣਾ ਤੋਂ ਬਾਅਦ ਤੱਤ ਬਦਲੇ ਨਹੀਂ ਜਾ ਸਕਦੇ।

```zymbol
// ਸਥਾਨਿਕ — ਮਿਸ਼ਰਤ ਕਿਸਮਾਂ ਦੀ ਆਗਿਆ ਹੈ
ਬਿੰਦੂ = (੧੦, ੨੦)
>> ਬਿੰਦੂ[੧] ¶    // → ੧੦

ਡਾਟਾ = (੪੨, "ਹੈਲੋ", #੧, ੩.੧੪)
>> ਡਾਟਾ[੩] ¶     // → #੧

// ਨਾਮਿਤ
ਵਿਅਕਤੀ = (ਨਾਮ: "ਐਲਿਸ", ਉਮਰ: ੨੫)
>> ਵਿਅਕਤੀ.ਨਾਮ ¶    // → ਐਲਿਸ
>> ਵਿਅਕਤੀ[੧] ¶      // → ਐਲਿਸ  (ਸੂਚਕਾਂਕ ਵੀ ਕੰਮ ਕਰਦਾ ਹੈ, ੧-ਅਧਾਰਿਤ)

// ਨੇਸਟਡ
ਸਥਿਤੀ = (x: ੧੦, y: ੨੦)
p = (ਸਥਿਤੀ: ਸਥਿਤੀ, ਲੇਬਲ: "ਮੂਲ")
>> p.ਸਥਿਤੀ.x ¶        // → ੧੦
```

**ਅਪਰਿਵਰਤਨਸ਼ੀਲਤਾ** — ਟਪਲ ਤੱਤ ਨੂੰ ਬਦਲਣ ਦੀ ਕੋਈ ਵੀ ਕੋਸ਼ਿਸ਼ ਰਨਟਾਈਮ ਗਲਤੀ ਹੈ:

```zymbol
t = (੧੦, ੨੦, ੩੦)
// t[੧] = ੯੯    // ❌ ਰਨਟਾਈਮ ਗਲਤੀ: ਟਪਲ ਅਪਰਿਵਰਤਨੀ ਹਨ
// t[੧] += ੫    // ❌ ਇਹੋ ਗਲਤੀ
```

ਸੋਧਿਆ ਮੁੱਲ ਪ੍ਰਾਪਤ ਕਰਨ ਲਈ `$~` (ਫੰਕਸ਼ਨਲ ਅਪਡੇਟ) ਵਰਤੋ — ਇੱਕ **ਨਵਾਂ** ਟਪਲ ਵਾਪਸ ਕਰਦਾ ਹੈ:

```zymbol
t = (੧੦, ੨੦, ੩੦)
t੨ = t[੨]$~ ੯੯੯
>> t ¶     // → (੧੦, ੨੦, ੩੦)   ← ਮੂਲ ਅਪਰਿਵਰਤਿਤ
>> t੨ ¶    // → (੧੦, ੯੯੯, ੩੦)

// ਨਾਮਿਤ ਟਪਲ — ਸਪਸ਼ਟ ਤੌਰ 'ਤੇ ਮੁੜ ਉਸਾਰੋ
ਵਿਅਕਤੀ = (ਨਾਮ: "ਐਲਿਸ", ਉਮਰ: ੨੫)
ਵੱਡਾ  = (ਨਾਮ: ਵਿਅਕਤੀ.ਨਾਮ, ਉਮਰ: ੨੬)
>> ਵਿਅਕਤੀ.ਉਮਰ ¶    // → ੨੫
>> ਵੱਡਾ.ਉਮਰ ¶     // → ੨੬
```

## ਉੱਚ-ਕ੍ਰਮ ਫੰਕਸ਼ਨ

```zymbol
ਸੰਖਿਆਵਾਂ = [੧, ੨, ੩, ੪, ੫, ੬, ੭, ੮, ੯, ੧੦]

ਦੁੱਗਣੇ_ਕੀਤੇ  = ਸੰਖਿਆਵਾਂ$> (x -> x * ੨)                  // map  → [੨,੪,੬…੨੦]
ਬਰਾਬਰ    = ਸੰਖਿਆਵਾਂ$| (x -> x % ੨ == ੦)           // filter → [੨,੪,੬,੮,੧੦]
ਕੁੱਲ    = ਸੰਖਿਆਵਾਂ$< (੦, (ਸੰਚਯਕ, x) -> ਸੰਚਯਕ + x)     // reduce → ੫੫

// ਵਿਚਕਾਰਲਿਆਂ ਰਾਹੀਂ ਲੜੀ
ਕਦਮ੧ = ਸੰਖਿਆਵਾਂ$| (x -> x > ੩)
ਕਦਮ੨ = ਕਦਮ੧$> (x -> x * x)
>> ਕਦਮ੨ ¶    // → [੧੬, ੨੫, ੩੬, ੪੯, ੬੪, ੮੧, ੧੦੦]

// ਨਾਮਿਤ ਫੰਕਸ਼ਨ ਸਿੱਧੇ HOF ਵਿੱਚ ਪਾਸ ਕੀਤੇ ਜਾ ਸਕਦੇ ਹਨ
ਦੁੱਗਣਾ(x) { <~ x * ੨ }
ਵੱਡਾ_ਹੈ(x) { <~ x > ੫ }
r = ਸੰਖਿਆਵਾਂ$> ਦੁੱਗਣਾ       // ✅ ਸਿੱਧਾ ਰੈਫਰੰਸ
r = ਸੰਖਿਆਵਾਂ$| ਵੱਡਾ_ਹੈ       // ✅ ਸਿੱਧਾ ਰੈਫਰੰਸ
```

## ਪਾਈਪ ਓਪਰੇਟਰ

ਸੱਜੇ ਪਾਸੇ ਨੂੰ ਹਮੇਸ਼ਾ ਪਾਈਪ ਕੀਤੇ ਮੁੱਲ ਲਈ `_` ਪਲੇਸਹੋਲਡਰ ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ:

ਪ੨ = ੧੦ |> ਜੋੜ(_, ੫)       // → ੧੫
ਪ੩ = ੫ |> ਜੋੜ(੨, _)        // → ੭

ਪ = ੫ |> ਦੁੱਗਣਾ(_) |> ਵਾਧਾ(_) |> ਦੁੱਗਣਾ(_)
>> ਪ ¶    // → ੨੨  (੫→੧੦→੧੧→੨੨)
```

>> ਐਮ::ਜੋੜ(੩, ੪) ¶    // → ੭  (ਅੰਦਰੂਨੀ ਨਾਮ _ਅੰਦਰੂਨੀ_ਜੋੜ ਲੁਕਿਆ ਹੋਇਆ ਹੈ)
```

> **ਮੋਡੀਊਲ ਨਿਯਮ**: `# ਨਾਮ { }` ਦੇ ਅੰਦਰ ਸਿਰਫ `#>`, ਫੰਕਸ਼ਨ ਪਰਿਭਾਸ਼ਾਵਾਂ, ਅਤੇ ਸ਼ਾਬਦਿਕ ਵੇਰੀਏਬਲ/ਸਥਿਰਾਂਕ ਸ਼ੁਰੂਕਰਤਾਵਾਂ ਦੀ ਇਜਾਜ਼ਤ ਹੈ। ਚੱਲਣਯੋਗ ਸਟੇਟਮੈਂਟਾਂ (`>>`, `<<`, ਲੂਪ, ਆਦਿ) ਗਲਤੀ E013 ਚੁੱਕਦੀਆਂ ਹਨ।

## ਅੰਕੀ ਮੋਡ

Zymbol ਸੰਖਿਆਵਾਂ ਨੂੰ **69 ਯੂਨੀਕੋਡ ਅੰਕ ਲਿਪੀਆਂ** ਵਿੱਚ ਪ੍ਰਦਰਸ਼ਿਤ ਕਰ ਸਕਦਾ ਹੈ — ਦੇਵਨਾਗਰੀ, ਅਰਬੀ-ਭਾਰਤੀ, ਥਾਈ, ਕਲਿੰਗੋਨ pIqaD, ਗਣਿਤਿਕ ਬੋਲਡ, ਐਲਸੀਡੀ ਸੈਗਮੈਂਟ, ਅਤੇ ਹੋਰ। ਸਰਗਰਮ ਮੋਡ ਸਿਰਫ `>>` ਆਉਟਪੁਟ ਨੂੰ ਪ੍ਰਭਾਵਿਤ ਕਰਦਾ ਹੈ; ਅੰਦਰੂਨੀ ਅੰਕਗਣਿਤ ਹਮੇਸ਼ਾ ਬਾਈਨਰੀ ਹੁੰਦਾ ਹੈ।

### ਲਿਪੀ ਨੂੰ ਸਰਗਰਮ ਕਰਨਾ

ਟਾਰਗੇਟ ਲਿਪੀ ਦੇ `0` ਅਤੇ `9` ਅੰਕ `#…#` ਦੇ ਅੰਦਰ ਲਿਖੋ:

```zymbol
#०९#    // ਦੇਵਨਾਗਰੀ   (U+0966–U+096F)
#٠٩#    // ਅਰਬੀ-ਭਾਰਤੀ (U+0660–U+0669)
#๐๙#    // ਥਾਈ         (U+0E50–U+0E59)
#09#    // ASCII ਉੱਤੇ ਮੁੜ ਸੈੱਟ ਕਰੋ
```

### ਆਉਟਪੁਟ ਅਤੇ ਬੂਲੀਅਨ

```zymbol
x = ੪੨
>> x ¶          // → ੪੨   (ASCII ਮੂਲ)

#०९#
>> x ¶          // → ४२
>> ੩.੧੪ ¶       // → ३.१४   (ਦਸ਼ਮਲਵ ਬਿੰਦੂ ਹਮੇਸ਼ਾ ASCII)
>> ੧ + ੨ ¶      // → ३

// ਬੂਲੀਅਨ: # ਉਪਸਰਗ ਹਮੇਸ਼ਾ ASCII, ਅੰਕ ਅਨੁਕੂਲ ਹੁੰਦਾ ਹੈ
>> #੧ ¶         // → #१   (ਦੇਵਨਾਗਰੀ ਵਿੱਚ ਸੱਚ)
>> #੦ ¶         // → #०   (ਝੂਠ — ੦ ਪੂਰਨ ਅੰਕ ਜ਼ੀਰੋ ਤੋਂ ਵੱਖਰਾ)

x = ੨੮ > ੪
>> x ¶          // → #१   (ਤੁਲਨਾ ਨਤੀਜਾ ਸਰਗਰਮ ਮੋਡ ਦੀ ਪਾਲਣਾ ਕਰਦਾ ਹੈ)
```

### ਸਰੋਤ ਵਿੱਚ ਮੂਲ ਅੰਕ ਲਿਟਰਲ

ਕੋਈ ਵੀ ਸਮਰਥਿਤ ਲਿਪੀ ਦੇ ਅੰਕ ਮਾਨਯ ਲਿਟਰਲ ਹਨ — ਰੇਂਜਾਂ, ਮੋਡਿਊਲੋ, ਤੁਲਨਾਵਾਂ ਵਿੱਚ:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### ਕਿਸੇ ਵੀ ਲਿਪੀ ਵਿੱਚ ਬੂਲੀਅਨ ਲਿਟਰਲ

`#` + ਕਿਸੇ ਵੀ ਬਲਾਕ ਤੋਂ ਅੰਕ `0` ਜਾਂ `1` ਇੱਕ ਮਾਨਯ ਬੂਲੀਅਨ ਲਿਟਰਲ ਹੈ:

```zymbol
#٠٩#
ਸਰਗਰਮ = #١        // #੧ ਦੇ ਬਰਾਬਰ
>> ਸਰਗਰਮ ¶        // → #१
>> (#١ && #٠) ¶ // → #०
```

> `#` **ਹਮੇਸ਼ਾ ASCII** ਹੈ। `#0` (ਝੂਠ) ਹਮੇਸ਼ਾ ਹਰ ਲਿਪੀ ਵਿੱਚ `0` (ਪੂਰਨ ਅੰਕ ਜ਼ੀਰੋ) ਤੋਂ ਦ੍ਰਿਸ਼ਟੀਗਤ ਤੌਰ 'ਤੇ ਵੱਖਰਾ ਹੁੰਦਾ ਹੈ।

```zymbol
// ਕਿਸਮ ਪਰਿਵਰਤਨ ਕਾਸਟ

v੨ = #|"੩.੧੪"|    // → ੩.੧੪  (ਫਲੋਟਿੰਗ)
v੩ = #|"abc"|     // → "abc"  (ਸੁਰੱਖਿਅਤ, ਕੋਈ ਗਲਤੀ ਨਹੀਂ)

ਗੋਲ੨ = #.੨|π|      // → ੩.੧੪  (੨ ਦਸ਼ਮਲਵ ਸਥਾਨਾਂ 'ਤੇ ਗੋਲ)
ਗੋਲ੪ = #.੪|π|      // → ੩.੧੪੧੬
ਟ੍ਰੰਕ੨ = #!੨|π|      // → ੩.੧੪  (ਟ੍ਰੰਕੇਟ)

// ਸੰਖਿਆ ਫਾਰਮੈਟਿੰਗ
ਫਾਰਮੈਟ = #,|੧੨੩੪੫੬੭|  // → ੧,੨੩੪,੫੬੭  (ਕਾਮੇ-ਵੱਖ)
ਵਿਗਿਆਨਕ = #^|੧੨੩੪੫.੬੭੮|    // → ੧.੨੩੪੫੬੭੮e੪  (ਵਿਗਿਆਨਕ)

// ਬੇਸ ਲਿਟਰਲ
a = 0x41         // → 'A'  (ਹੈਕਸ)

// ਬੇਸ ਪਰਿਵਰਤਨ ਆਉਟਪੁਟ
ਹੈਕਸ = 0x|੨੫੫|    // → "0x00FF"
ਬਾਈਨਰੀ = 0b|੬੫|     // → "0b1000001"
ਆਕਟਲ = 0o|੮|      // → "0o10"
ਦਸ਼ਮਲਵ = 0d|੨੫੫|    // → "0d0255"
```

## ਰਿਲੀਜ਼ ਚੇਂਜਲੌਗ

### v0.0.5 — ਟੀਯੂਆਈ ਪ੍ਰਿਮਿਟਿਵ, ਹੌਟ ਪਰਿਭਾਸ਼ਾ ਅਤੇ ਸਤਰ ਦੁਹਰਾਉ _(ਮਈ 2026)_

- **ਬ੍ਰੇਕਿੰਗ** ਮੈਚ ਆਰਮ ਵੱਖਰਾ ਕਰਨ ਵਾਲਾ: `ਪੈਟਰਨ : ਨਤੀਜਾ` → `ਪੈਟਰਨ => ਨਤੀਜਾ`
- **ਬ੍ਰੇਕਿੰਗ** ਇੰਪੋਰਟ ਉਪਨਾਮ: `<# ਪਾਥ <= ਉਪਨਾਮ` → `<# ਪਾਥ => ਉਪਨਾਮ`
- **ਬ੍ਰੇਕਿੰਗ** ਐਕਸਪੋਰਟ ਦੁਬਾਰਾ ਨਾਮ: `#> { fn <= ਪਬ }` → `#> { fn => ਪਬ }`
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਟੀਯੂਆਈ ਬਲਾਕ `>>| { }` — ਬਦਲਵੀਂ ਸਕ੍ਰੀਨ + ਕੱਚਾ ਮੋਡ; ਬਾਹਰ ਨਿਕਲਣ 'ਤੇ ਸਾਫ਼ ਕਰਦਾ ਹੈ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਸਥਾਨਿਕ ਆਉਟਪੁਟ `>>~ (ਕਤਾਰ, ਕਾਲਮ, BKS, fg, bg) > ਆਈਟਮਾਂ` — ਵਿਰਲ ਸਲਾਟ, ANSI 256 ਰੰਗ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਕੀ ਇਨਪੁਟ `<<| ਵੇਰੀਏਬਲ` (ਬਲਾਕਿੰਗ) ਅਤੇ `<<|? ਵੇਰੀਏਬਲ` (ਨਾਨ-ਬਲਾਕਿੰਗ ਪੋਲ)
- **ਸ਼ਾਮਲ ਕੀਤਾ** `>>!` ਸਕ੍ਰੀਨ ਸਾਫ਼ ਕਰੋ, `>>?` ਟਰਮੀਨਲ ਆਕਾਰ ਪੁੱਛੋ, `@~ N` N ਮਿਲੀਸਕਿੰਟ ਸੌਂ ਜਾਓ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਹੌਟ ਪਰਿਭਾਸ਼ਾ `°x` / `x°` — ਲੂਪਾਂ ਵਿੱਚ ਪਹਿਲੀ ਵਰਤੋਂ 'ਤੇ ਵੇਰੀਏਬਲ ਆਟੋਮੈਟਿਕ ਸ਼ੁਰੂ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਸਤਰ ਦੁਹਰਾਉ `ਸਤਰ $* N` — ਇੱਕ ਸਤਰ ਨੂੰ N ਵਾਰ ਦੁਹਰਾਓ
- **ਵੀਐਮ** ਸਮਾਨਤਾ: 436/436 ਟੈਸਟ ਪਾਸ

### v0.0.4 — 1-ਅਧਾਰਿਤ ਸੂਚਕਾਂਕਨ, ਪਹਿਲੀ-ਸ਼੍ਰੇਣੀ ਫੰਕਸ਼ਨ ਅਤੇ ਬਲਾਕ ਮੋਡੀਊਲ _(ਅਪ੍ਰੈਲ 2026)_

- **ਬ੍ਰੇਕਿੰਗ** ਸਾਰੇ ਸੂਚਕਾਂਕਨ **1-ਅਧਾਰਿਤ** ਵਿੱਚ ਬਦਲੇ — `arr[1]` ਪਹਿਲਾ ਤੱਤ ਹੈ; `arr[0]` ਇੱਕ ਰਨਟਾਈਮ ਗਲਤੀ ਹੈ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਨਾਮਿਤ ਫੰਕਸ਼ਨ **ਪਹਿਲੀ-ਸ਼੍ਰੇਣੀ ਮੁੱਲ** ਹਨ — ਸਿੱਧੇ HOF ਵਿੱਚ ਪਾਸ ਕਰੋ: `ਸੰਖਿਆਵਾਂ$> ਦੁੱਗਣਾ`
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਮੋਡੀਊਲਾਂ ਲਈ **ਬਲਾਕ ਸੰਟੈਕਸ ਲੋੜੀਂਦਾ** — `# ਨਾਮ { ... }` — ਫਲੈਟ ਸੰਟੈਕਸ ਹਟਾਇਆ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਬਹੁ-ਅਯਾਮੀ ਸੂਚਕਾਂਕਨ: `arr[i>j>k]` (ਨੇਵੀਗੇਸ਼ਨ), `arr[p ; q]` (ਫਲੈਟ ਕੱਢਣਾ)
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਕਿਸਮ ਪਰਿਵਰਤਨ ਕਾਸਟ: `##.ਐਕਸਪ੍ਰੈਸ਼ਨ` (ਫਲੋਟਿੰਗ), `###ਐਕਸਪ੍ਰੈਸ਼ਨ` (ਪੂਰਨ ਅੰਕ ਗੋਲ), `##!ਐਕਸਪ੍ਰੈਸ਼ਨ` (ਪੂਰਨ ਅੰਕ ਟ੍ਰੰਕੇਟ)
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਸਤਰ ਵੰਡ: `ਸਤਰ$/ ਵੰਡਕਾਰੀ` — `Array(ਸਤਰ)` ਵਾਪਸ ਕਰਦਾ ਹੈ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਕੌਨਕੈਟ ਬਿਲਡ: `ਅਧਾਰ$++ a b c` — ਕਈ ਆਈਟਮਾਂ ਜੋੜਦਾ ਹੈ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਟਾਈਮਜ਼ ਲੂਪ: `@ N { }` — ਬਿਲਕੁਲ N ਵਾਰ ਦੁਹਰਾਓ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਲੇਬਲ ਵਾਲਾ ਲੂਪ ਸੰਟੈਕਸ: `@:ਨਾਮ { }`, `@:ਨਾਮ!`, `@:ਨਾਮ>` — `@ @ਨਾਮ` / `@! ਨਾਮ` ਦੀ ਥਾਂ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਵੇਰੀਏਬਲ ਸਕੋਪ ਨਿਯਮ: `_ਨਾਮ` ਵੇਰੀਏਬਲਾਂ ਦਾ ਸਹੀ ਬਲਾਕ ਸਕੋਪ ਹੁੰਦਾ ਹੈ; `\ ਵੇਰੀਏਬਲ` ਜਲਦੀ ਨਸ਼ਟ ਕਰਦਾ ਹੈ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਮੈਚ ਤੁਲਨਾ ਪੈਟਰਨ: `< 0 =>`, `> 5 =>`, `== 42 =>` ਆਦਿ
- **ਸ਼ਾਮਲ ਕੀਤਾ** ਮੋਡੀਊਲ E013 ਗਲਤੀ: ਮੋਡੀਊਲ ਬਾਡੀ ਵਿੱਚ ਚੱਲਣਯੋਗ ਸਟੇਟਮੈਂਟਾਂ ਵਰਜਿਤ ਹਨ
- **ਠੀਕ ਕੀਤਾ** `alias.CONST` ਹੁਣ ਸਹੀ ਢੰਗ ਨਾਲ ਹੱਲ ਕਰਦਾ ਹੈ; `#>` ਫੰਕਸ਼ਨ ਪਰਿਭਾਸ਼ਾਵਾਂ ਤੋਂ ਬਾਅਦ ਆ ਸਕਦਾ ਹੈ
