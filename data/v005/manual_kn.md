## ಅಸ್ಥರಗಳು ಮತ್ತು ಸ್ಥಿರಾಂಕಗಳು

```zymbol
x = ೧೦              // ಬದಲಾಯಿಸಬಹುದಾದ ಅಸ್ಥರ
π := ೩.೧೪೧೫೯        // ಸ್ಥಿರಾಂಕ — ಪುನಃ ನಿಯೋಜನೆ ರನ್ಟೈಮ್ ದೋಷವಾಗಿದೆ
ಹೆಸರು = "ಅಲಿಸ್"
ಸಕ್ರಿಯ = #೧         // ಬೂಲಿಯನ್ ಸತ್ಯ
👋 := "ಹಲೋ"
```

```zymbol
x = ೧೦    // ೧೦
x += ೫    // ೧೫
x -= ೩    // ೧೨
x *= ೨    // ೨೪
x /= ೩    // ೮
x %= ೩    // ೨
x ^= ೨    // ೪
x++        // ೫
x--        // ೪
```

`°` (ಡಿಗ್ರಿ ಚಿಹ್ನೆ, U+00B0) ಮೊದಲ ಬಳಕೆಯಲ್ಲಿ ಅಸ್ಥರವನ್ನು ಅದರ ತಟಸ್ಥ ಮೌಲ್ಯಕ್ಕೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಆರಂಭಿಸುತ್ತದೆ:

```zymbol
ಸಂಖ್ಯೆಗಳು = [೩, ೧, ೪, ೧, ೫]
@ n:ಸಂಖ್ಯೆಗಳು {
    °ಒಟ್ಟು += n    // ಲೂಪ್ನ ಮೇಲೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ೦ ಗೆ ಆರಂಭ; @ ನಂತರ ಉಳಿಯುತ್ತದೆ
}
>> ಒಟ್ಟು ¶         // → ೧೪
```

> `°x` (ಪೂರ್ವಪ್ರತ್ಯಯ) ಲೂಪ್ನ ಮೇಲೆ ನಂಗುರು ಹಾಕುತ್ತದೆ — ಫಲಿತಾಂಶ `@` ನಂತರ ಪ್ರವೇಶಿಸಬಹುದಾಗಿದೆ.
> `x°` (ಪ್ರತ್ಯಯ) ಲೂಪ್ನ ಒಳಗೆ ನಂಗುರು ಹಾಕುತ್ತದೆ — ಲೂಪ್ ಮುಗಿದಾಗ ಸಾಯುತ್ತದೆ.

|------|---------|----------|---------|
| ಪೂರ್ಣಾಂಕ | `42`, `-7` | `###` | 64-ಬಿಟ್ ಸೈನ್ ಸಹಿತ |
| ತೇಲುವ ಬಿಂದು | `3.14`, `1.5e10` | `##.` | ವೈಜ್ಞಾನಿಕ ಸಂಕೇತ ಅನುಮತಿಸಲಾಗಿದೆ |
| ಸ್ಟ್ರಿಂಗ್ | `"ಪಠ್ಯ"` | `##"` | ಇಂಟರ್ಪೋಲೇಶನ್: `"ಹಲೋ {ಹೆಸರು}"` |
| ಅಕ್ಷರ | `'ಅ'` | `##'` | ಒಂದೇ ಯುನಿಕೋಡ್ ಅಕ್ಷರ |
| ಬೂಲಿಯನ್ | `#1`, `#0` | `##?` | ಸಂಖ್ಯಾತ್ಮಕವಲ್ಲ — `#1 ≠ 1` |
| ಆರೇ | `[1, 2, 3]` | `##]` | ಏಕರೂಪದ ಅಂಶಗಳು |

## ಟಿಯುಐ ಪ್ರಾಥಮಿಕಗಳು

ಸಂವಾದಾತ್ಮಕ ಪ್ರೋಗ್ರಾಂಗಳಿಗಾಗಿ ಟರ್ಮಿನಲ್ ಯೂಸರ್ ಇಂಟರ್ಫೇಸ್ ಆಪರೇಟರ್ಗಳು. ಹೆಚ್ಚಿನವುಗಳಿಗೆ `>>| { }` ಬ್ಲಾಕ್ ಅಗತ್ಯವಿದೆ (ಪರ್ಯಾಯ ಸ್ಕ್ರೀನ್ + ಕಚ್ಚಾ ಮೋಡ್).

```zymbol
>>| {
    >>!                             // ಪರ್ಯಾಯ ಸ್ಕ್ರೀನ್ ಅಳಿಸಿ
    >>~ (೧, ೧, ೦, ೧೦) > "ಚಾಲನೆಯಲ್ಲಿದೆ"  // ಸಾಲು ೧, ಕಾಲಮ್ ೧, fg=೧೦ (ಹಸಿರು)
    @~ ೧೦೦೦                         // ೧ ಸೆಕೆಂಡ್ ವಿರಾಮ (೧೦೦೦ ms)
    >>~ (೨, ೧) > "ಪೂರ್ಣಗೊಂಡಿತು."
}
// ನಿರ್ಗಮಿಸಿದಾಗ ಟರ್ಮಿನಲ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪುನಃಸ್ಥಾಪಿಸಲ್ಪಡುತ್ತದೆ
```

```zymbol
// ಕೀ ಪ್ರೆಸ್ ಮತ್ತು ಟರ್ಮಿನಲ್ ಗಾತ್ರ
>>| {
    [ಸಾಲುಗಳು, ಕಾಲಮ್ಗಳು] = >>?              // ಟರ್ಮಿನಲ್ನ ಆಯಾಮಗಳನ್ನು ಕೇಳಿ
    >>~ (೧, ೧) > "ಟರ್ಮಿನಲ್: " ಸಾಲುಗಳು " x " ಕಾಲಮ್ಗಳು
    <<| ಕೀ                         // ಬ್ಲಾಕಿಂಗ್ ಕೀ ಪ್ರೆಸ್ ಓದಿ
    >>~ (೨, ೧) > "ಒತ್ತಿದಿರಿ: " ಕೀ
}
```

> `>>!` ಸ್ಕ್ರೀನ್ ಅಳಿಸುತ್ತದೆ. `>>?` `[ಸಾಲುಗಳು, ಕಾಲಮ್ಗಳು]` ಹಿಂತಿರುಗಿಸುತ್ತದೆ. `@~ N` N ಮಿಲಿಸೆಕೆಂಡುಗಳು ನಿದ್ರಿಸುತ್ತದೆ.
> `<<|` ಒಂದು ಕೀ ಪ್ರೆಸ್ ಅನ್ನು ಓದುತ್ತದೆ (ಬ್ಲಾಕಿಂಗ್); `<<|?` ಬ್ಲಾಕ್ ಮಾಡದೆಯೇ ಪೋಲ್ ಮಾಡುತ್ತದೆ (ಯಾವುದೂ ಇಲ್ಲದಿದ್ದರೆ `'\0'` ಹಿಂತಿರುಗಿಸುತ್ತದೆ).
> ಸ್ಥಾನಿಕ ಔಟ್ಪುಟ್ ಟಪಲ್: `(ಸಾಲು, ಕಾಲಮ್, BKS, fg, bg)` — ಯಾವುದೇ ಸ್ಲಾಟ್ ಅನ್ನು ಅಲ್ಪವಿರಾಮದಿಂದ ಬಿಟ್ಟುಬಿಡಬಹುದು (`>>~ (,,, 196) > "ಕೆಂಪು"`).
> BKS ಬಿಟ್ಮಾಸ್ಕ್: `1`=ದಪ್ಪ, `2`=ಓರೆ, `4`=ಅಂಡರ್ಲೈನ್. ANSI 256 ಬಣ್ಣದ ಪ್ಯಾಲೆಟ್ (`0`=ಟರ್ಮಿನಲ್ ಡೀಫಾಲ್ಟ್).
> ಕೇವಲ tree-walker (`>>!`, `>>?`, `@~`, `>>~` ಹೊರತುಪಡಿಸಿ ಇವು `--vm` ನಲ್ಲೂ ಚಲಿಸುತ್ತವೆ).

## ಲೂಪ್ಗಳು

```zymbol
@ i:೦..೪  { >> i " " }        // ವ್ಯಾಪ್ತಿ ಸೇರಿದೆ:  ೦ ೧ ೨ ೩ ೪
@ i:೧..೯:೨ { >> i " " }       // ಹೆಜ್ಜೆಯೊಂದಿಗೆ:         ೧ ೩ ೫ ೭ ೯
@ i:೫..೦:೧ { >> i " " }       // ವಿರುದ್ಧ:           ೫ ೪ ೩ ೨ ೧ ೦

n = ೧
@ n <= ೬೪ { n *= ೨ }
>> n ¶                        // → ೧೨೮  (while)

ಹಣ್ಣುಗಳು = ["ಸೇಬು", "ಪೇರು", "ದ್ರಾಕ್ಷಿ"]
@ ಹ:ಹಣ್ಣುಗಳು { >> ಹ ¶ }         // ಆರೆಯ ಪ್ರತಿಯೊಂದು ಅಂಶಕ್ಕೆ

@ ಅ:"ಹಲೋ" { >> ಅ "-" }
>> ¶                          // → ಹ-ಲ-ೋ-  (ಸ್ಟ್ರಿಂಗ್ನ ಪ್ರತಿಯೊಂದು ಅಕ್ಷರಕ್ಕೆ)

@ i:೧..೧೦ {
    ? i % ೨ == ೦ { @> }       // @> ಮುಂದುವರಿಸಿ
    ? i > ೭ { @! }             // @! ಮುರಿಯಿರಿ
    >> i " "
}
>> ¶                          // → ೧ ೩ ೫ ೭

// ಅನಂತ ಲೂಪ್
i = ೦
@ {
    i++
    ? i >= ೫ { @! }
    >> i " "
}
>> ¶                          // → ೧ ೨ ೩ ೪

// ಲೇಬಲ್ ಮಾಡಿದ ಲೂಪ್ (ನೆಸ್ಟೆಡ್ ಮುರಿತ)
ಎಣಿಕೆ = ೦
@:ಬಾಹ್ಯ {
    ಎಣಿಕೆ++
    ? ಎಣಿಕೆ >= ೩ { @:ಬಾಹ್ಯ! }
}
>> ಎಣಿಕೆ ¶                    // → ೩
```

## ಫಂಕ್ಷನ್ಗಳು

```zymbol
ಮೊತ್ತ(a, b) { <~ a + b }
>> ಮೊತ್ತ(೩, ೪) ¶    // → ೭

ಅಪವರ್ತನೀಯ(n) {
    ? n <= ೧ { <~ ೧ }
    <~ n * ಅಪವರ್ತನೀಯ(n - ೧)
}
>> ಅಪವರ್ತನೀಯ(೫) ¶    // → ೧೨೦
```

ಫಂಕ್ಷನ್ಗಳು **ಪ್ರತ್ಯೇಕ ಸ್ಕೋಪ್** ಅನ್ನು ಹೊಂದಿವೆ — ಅವು ಬಾಹ್ಯ ಅಸ್ಥರಗಳನ್ನು ಓದಲು ಸಾಧ್ಯವಿಲ್ಲ. ಕರ್ತೃ ಅಸ್ಥರಗಳನ್ನು ಬದಲಾಯಿಸಲು ಔಟ್ಪುಟ್ ನಿಯತಾಂಕಗಳು `<~` ಬಳಸಿ:

```zymbol
ವಿನಿಮಯ(a<~, b<~) {
    ತಾತ್ಕಾಲಿಕ = a
    a = b
    b = ತಾತ್ಕಾಲಿಕ
}
x = ೧೦
y = ೨೦
ವಿನಿಮಯ(x, y)
>> "x=" x " y=" y ¶    // → x=೨೦ y=೧೦
```

> ಹೆಸರಿಸಿದ ಫಂಕ್ಷನ್ಗಳು **ಪ್ರಥಮ-ಶ್ರೇಣಿಯ ಮೌಲ್ಯಗಳಾಗಿವೆ** — ನೇರವಾಗಿ ರವಾನಿಸಿ: `ಸಂಖ್ಯೆಗಳು$> ದುಪ್ಪಟ್ಟು`. ಸುತ್ತಲು: `x -> fn(x)` ಸಹ ಮಾನ್ಯವಾಗಿದೆ.

## ಲ್ಯಾಂಬ್ಡಾಗಳು ಮತ್ತು ಕ್ಲೋಷರ್ಗಳು

```zymbol
ದುಪ್ಪಟ್ಟು = x -> x * ೨
ಮೊತ್ತ = (a, b) -> a + b
>> ದುಪ್ಪಟ್ಟು(೫) ¶    // → ೧೦
>> ಮೊತ್ತ(೩, ೭) ¶  // → ೧೦

// ಬ್ಲಾಕ್ ಲ್ಯಾಂಬ್ಡಾ
ವರ್ಗೀಕರಿಸು = x -> {
    ? x > ೦ { <~ "ಧನಾತ್ಮಕ" }
    _? x < ೦ { <~ "ಋಣಾತ್ಮಕ" }
    <~ "ಶೂನ್ಯ"
}

// ಕ್ಲೋಷರ್ — ಬಾಹ್ಯ ಸ್ಕೋಪ್ ಅನ್ನು ಸೆರೆಹಿಡಿಯುತ್ತದೆ
ಅಂಶ = ೩
ಮುಮ್ಮಡಿ = x -> x * ಅಂಶ
>> ಮುಮ್ಮಡಿ(೭) ¶    // → ೨೧

// ಫ್ಯಾಕ್ಟರಿ
ಮೊತ್ತ_ನಿರ್ಮಾಪಕ(n) { <~ x -> x + n }
ಹತ್ತರ_ಮೊತ್ತ = ಮೊತ್ತ_ನಿರ್ಮಾಪಕ(೧೦)
>> ಹತ್ತರ_ಮೊತ್ತ(೫) ¶    // → ೧೫

// ಆರೆಯಲ್ಲಿ
ಆಪರೇಟರ್ಗಳು = [x -> x+೧, x -> x*೨, x -> x*x]
>> ಆಪರೇಟರ್ಗಳು[೩](೫) ¶    // → ೨೫
```

## ಆರೇ

ಆರೇಗಳು **ಬದಲಾಯಿಸಬಹುದಾದ** ಮತ್ತು **ಒಂದೇ ಪ್ರಕಾರದ** ಅಂಶಗಳನ್ನು ಹೊಂದಿರುತ್ತವೆ.

```zymbol
arr = [೧, ೨, ೩, ೪, ೫]

x = arr[೧]      // ೧ — ಪ್ರವೇಶ (೧-ಆಧಾರಿತ: ಮೊದಲ ಅಂಶ)
x = arr[-೧]     // ೫ — ಋಣಾತ್ಮಕ ಸೂಚ್ಯಂಕ (ಕೊನೆಯ ಅಂಶ)
x = arr$#       // ೫ — ಉದ್ದ (>> ನಲ್ಲಿ (arr$#) ಬಳಸಿ)

arr = arr$+ ೬            // ಸೇರಿಸಿ → [೧,೨,೩,೪,೫,೬]
arr೨ = arr$+[೨] ೯೯       // ಸ್ಥಾನ ೨ ರಲ್ಲಿ ಸೇರಿಸಿ (೧-ಆಧಾರಿತ)
arr೩ = arr$- ೩           // ಮೌಲ್ಯದ ಮೊದಲ ಸಂಭವವನ್ನು ತೆಗೆದುಹಾಕಿ
arr೪ = arr$-- ೩          // ಎಲ್ಲಾ ಸಂಭವಗಳನ್ನು ತೆಗೆದುಹಾಕಿ
arr೫ = arr$-[೧]          // ಸೂಚ್ಯಂಕ ೧ ರಲ್ಲಿ ತೆಗೆದುಹಾಕಿ (ಮೊದಲ ಅಂಶ)
arr೬ = arr$-[೨..೩]       // ವ್ಯಾಪ್ತಿಯನ್ನು ತೆಗೆದುಹಾಕಿ (೧-ಆಧಾರಿತ, ಕೊನೆ ಸೇರಿದೆ)

ಇದೆ = arr$? ೩            // #೧ — ಹೊಂದಿದೆ
ಸ್ಥಾನಗಳು = arr$?? ೩           // [೩] — ಮೌಲ್ಯದ ಎಲ್ಲಾ ಸೂಚ್ಯಂಕಗಳು (೧-ಆಧಾರಿತ)
ಸ್ಲೈಸ್ = arr$[೧..೩]          // [೧,೨,೩] — ಸ್ಲೈಸ್ (೧-ಆಧಾರಿತ, ಕೊನೆ ಸೇರಿದೆ)
ಸ್ಲೈಸ್೨ = arr$[೧:೩]          // [೧,೨,೩] — ಒಂದೇ, ಎಣಿಕೆ-ಆಧಾರಿತ ವಾಕ್ಯರಚನೆ

ಆರೋಹಣ = arr$^+             // ಆರೋಹಣ ಕ್ರಮದಲ್ಲಿ ವಿಂಗಡಿಸಿ (ಕೇವಲ ಆದಿಮ)
ಅವರೋಹಣ = arr$^-            // ಅವರೋಹಣ ಕ್ರಮದಲ್ಲಿ ವಿಂಗಡಿಸಿ (ಕೇವಲ ಆದಿಮ)

// ಹೆಸರಿಸಿದ/ಸ್ಥಾನಿಕ ಟಪಲ್ ಆರೇ — ಹೋಲಿಕೆದಾರ ಲ್ಯಾಂಬ್ಡಾದೊಂದಿಗೆ $^ ಬಳಸಿ
ಡೇಟಾಬೇಸ್ = [(ಹೆಸರು: "ಕಾರ್ಲಾ", ವಯಸ್ಸು: ೨೮), (ಹೆಸರು: "ಅನಾ", ವಯಸ್ಸು: ೨೫), (ಹೆಸರು: "ಬಾಬ್", ವಯಸ್ಸು: ೩೦)]
ವಯಸ್ಸಿನ_ಪ್ರಕಾರ  = ಡೇಟಾಬೇಸ್$^ (a, b -> a.ವಯಸ್ಸು < b.ವಯಸ್ಸು)    // ವಯಸ್ಸಿನ ಪ್ರಕಾರ ಆರೋಹಣ (<)
ಹೆಸರಿನ_ಪ್ರಕಾರ = ಡೇಟಾಬೇಸ್$^ (a, b -> a.ಹೆಸರು > b.ಹೆಸರು)   // ಹೆಸರಿನ ಪ್ರಕಾರ ಅವರೋಹಣ (>)
>> ವಯಸ್ಸಿನ_ಪ್ರಕಾರ[೧].ಹೆಸರು ¶     // → ಅನಾ
>> ಹೆಸರಿನ_ಪ್ರಕಾರ[೧].ಹೆಸರು ¶    // → ಕಾರ್ಲಾ

// ನೇರ ಅಂಶ ಅಪ್ಡೇಟ್ (ಕೇವಲ ಆರೇ)
arr[೧] = ೯೯              // ನಿಯೋಜಿಸಿ
arr[೨] += ೫              // ಸಂಯುಕ್ತ: +=  -=  *=  /=  %=  ^=

// ಕಾರ್ಯಾತ್ಮಕ ಅಪ್ಡೇಟ್ — ಹೊಸ ಆರೇ ಅನ್ನು ಹಿಂತಿರುಗಿಸುತ್ತದೆ; ಮೂಲ ಬದಲಾಗದೆ ಉಳಿಯುತ್ತದೆ
arr೨ = arr[೨]$~ ೯೯
```

> ಎಲ್ಲಾ ಸಂಗ್ರಹಣಾ ಆಪರೇಟರ್ಗಳು **ಹೊಸ ಆರೇ** ಅನ್ನು ಹಿಂತಿರುಗಿಸುತ್ತವೆ. ಹಿಂತಿರುಗಿ ನಿಯೋಜಿಸಿ: `arr = arr$+ 4`.
> `$+` ಅನ್ನು ಸರಪಳಿ ಮಾಡಬಹುದು: `arr = arr$+ 5$+ 6$+ 7`. ಇತರ ಆಪರೇಟರ್ಗಳು ಮಧ್ಯಂತರ ನಿಯೋಜನೆಗಳನ್ನು ಬಳಸುತ್ತವೆ.
> **ಸೂಚ್ಯಂಕೀಕರಣ 1-ಆಧಾರಿತ**: `arr[1]` ಮೊದಲ ಅಂಶವಾಗಿದೆ; `arr[0]` ರನ್ಟೈಮ್ ದೋಷವಾಗಿದೆ.
> `$^+` / `$^-` **ಆದಿಮ ಆರೇಗಳನ್ನು** (ಸಂಖ್ಯೆಗಳು, ಸ್ಟ್ರಿಂಗ್ಗಳು) ವಿಂಗಡಿಸುತ್ತದೆ. ಟಪಲ್ ಆರೇಗಳಿಗೆ ಹೋಲಿಕೆದಾರ ಲ್ಯಾಂಬ್ಡಾದೊಂದಿಗೆ `$^` ಬಳಸಿ — ದಿಕ್ಕನ್ನು ಲ್ಯಾಂಬ್ಡಾದಲ್ಲಿ ಎನ್ಕೋಡ್ ಮಾಡಲಾಗಿದೆ (`<` = ಆರೋಹಣ, `>` = ಅವರೋಹಣ).

**ಮೌಲ್ಯ ಸೆಮ್ಯಾಂಟಿಕ್ಸ್** — ಆರೇ ಅನ್ನು ಮತ್ತೊಂದು ಅಸ್ಥರಕ್ಕೆ ನಿಯೋಜಿಸುವುದು ಸ್ವತಂತ್ರ ಪ್ರತಿಯನ್ನು ಸೃಷ್ಟಿಸುತ್ತದೆ:

```zymbol
a = [೧, ೨, ೩]
b = a
a[೧] = ೯೯
>> a ¶    // → [೯೯, ೨, ೩]
>> b ¶    // → [೧, ೨, ೩]   ← b ಪ್ರಭಾವಿತವಾಗಿಲ್ಲ
```

```zymbol
// ನೆಸ್ಟೆಡ್ ಆರೇ (೧-ಆಧಾರಿತ ಸೂಚ್ಯಂಕೀಕರಣ)
ಮ್ಯಾಟ್ರಿಕ್ಸ್ = [[೧,೨,೩],[೪,೫,೬],[೭,೮,೯]]
>> ಮ್ಯಾಟ್ರಿಕ್ಸ್[೨][೩] ¶    // → ೬  (ಸಾಲು ೨, ಕಾಲಮ್ ೩)
```

## ಡಿಸ್ಟ್ರಕ್ಚರಿಂಗ್

```zymbol
// ಆರೇ
arr = [೧೦, ೨೦, ೩೦, ೪೦, ೫೦]
[a, b, c] = arr              // a=೧೦  b=೨೦  c=೩೦
[ಮೊದಲನೆಯ, *ಉಳಿದ] = arr         // ಮೊದಲನೆಯ=೧೦  ಉಳಿದ=[೨೦,೩೦,೪೦,೫೦]
[x, _, z] = [೧, ೨, ೩]        // _ ತಿರಸ್ಕರಿಸುತ್ತದೆ

// ಸ್ಥಾನಿಕ ಟಪಲ್
ಬಿಂದು = (೧೦೦, ೨೦೦)
(px, py) = ಬಿಂದು             // px=೧೦೦  py=೨೦೦

// ಹೆಸರಿಸಿದ ಟಪಲ್
ವ್ಯಕ್ತಿ = (ಹೆಸರು: "ಅನಾ", ವಯಸ್ಸು: ೨೫, ನಗರ: "ಮ್ಯಾಡ್ರಿಡ್")
(ಹೆಸರು: n, ವಯಸ್ಸು: v) = ವ್ಯಕ್ತಿ   // n="ಅನಾ"  v=೨೫
```

## ಟಪಲ್

ಟಪಲ್ಗಳು **ಬದಲಾಯಿಸಲಾಗದ** ಕ್ರಮಬದ್ಧ ಕಂಟೇನರ್ಗಳಾಗಿವೆ, ಅವು **ವಿವಿಧ ಪ್ರಕಾರದ** ಮೌಲ್ಯಗಳನ್ನು ಹೊಂದಿರಬಹುದು.
ಆರೇಗಳಿಗಿಂತ ಭಿನ್ನವಾಗಿ, ರಚನೆಯ ನಂತರ ಅಂಶಗಳನ್ನು ಬದಲಾಯಿಸಲು ಸಾಧ್ಯವಿಲ್ಲ.

```zymbol
// ಸ್ಥಾನಿಕ — ಮಿಶ್ರ ಪ್ರಕಾರಗಳಿಗೆ ಅನುಮತಿ ಇದೆ
ಬಿಂದು = (೧೦, ೨೦)
>> ಬಿಂದು[೧] ¶    // → ೧೦

ಡೇಟಾ = (೪೨, "ಹಲೋ", #೧, ೩.೧೪)
>> ಡೇಟಾ[೩] ¶     // → #೧

// ಹೆಸರಿಸಿದ
ವ್ಯಕ್ತಿ = (ಹೆಸರು: "ಅಲಿಸ್", ವಯಸ್ಸು: ೨೫)
>> ವ್ಯಕ್ತಿ.ಹೆಸರು ¶    // → ಅಲಿಸ್
>> ವ್ಯಕ್ತಿ[೧] ¶      // → ಅಲಿಸ್  (ಸೂಚ್ಯಂಕವೂ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ, ೧-ಆಧಾರಿತ)

// ನೆಸ್ಟೆಡ್
ಸ್ಥಾನ = (x: ೧೦, y: ೨೦)
p = (ಸ್ಥಾನ: ಸ್ಥಾನ, ಲೇಬಲ್: "ಮೂಲ")
>> p.ಸ್ಥಾನ.x ¶        // → ೧೦
```

**ಬದಲಾಯಿಸಲಾಗದಿರುವಿಕೆ** — ಟಪಲ್ ಅಂಶವನ್ನು ಬದಲಾಯಿಸುವ ಯಾವುದೇ ಪ್ರಯತ್ನ ರನ್ಟೈಮ್ ದೋಷವಾಗಿದೆ:

```zymbol
t = (೧೦, ೨೦, ೩೦)
// t[೧] = ೯೯    // ❌ ರನ್ಟೈಮ್ ದೋಷ: ಟಪಲ್ಗಳು ಬದಲಾಯಿಸಲಾಗದವು
// t[೧] += ೫    // ❌ ಅದೇ ದೋಷ
```

ಮಾರ್ಪಡಿಸಿದ ಮೌಲ್ಯವನ್ನು ಪಡೆಯಲು `$~` (ಕಾರ್ಯಾತ್ಮಕ ಅಪ್ಡೇಟ್) ಬಳಸಿ — **ಹೊಸ** ಟಪಲ್ ಅನ್ನು ಹಿಂತಿರುಗಿಸುತ್ತದೆ:

```zymbol
t = (೧೦, ೨೦, ೩೦)
t೨ = t[೨]$~ ೯೯೯
>> t ¶     // → (೧೦, ೨೦, ೩೦)   ← ಮೂಲ ಬದಲಾಗಿಲ್ಲ
>> t೨ ¶    // → (೧೦, ೯೯೯, ೩೦)

// ಹೆಸರಿಸಿದ ಟಪಲ್ — ಸ್ಪಷ್ಟವಾಗಿ ಪುನರ್ನಿರ್ಮಿಸಿ
ವ್ಯಕ್ತಿ = (ಹೆಸರು: "ಅಲಿಸ್", ವಯಸ್ಸು: ೨೫)
ದೊಡ್ಡದು  = (ಹೆಸರು: ವ್ಯಕ್ತಿ.ಹೆಸರು, ವಯಸ್ಸು: ೨೬)
>> ವ್ಯಕ್ತಿ.ವಯಸ್ಸು ¶    // → ೨೫
>> ದೊಡ್ಡದು.ವಯಸ್ಸು ¶     // → ೨೬
```

## ಉನ್ನತ-ಕ್ರಮಾಂಕದ ಫಂಕ್ಷನ್ಗಳು

```zymbol
ಸಂಖ್ಯೆಗಳು = [೧, ೨, ೩, ೪, ೫, ೬, ೭, ೮, ೯, ೧೦]

ದುಪ್ಪಟ್ಟಾದವು  = ಸಂಖ್ಯೆಗಳು$> (x -> x * ೨)                  // map  → [೨,೪,೬…೨೦]
ಸಮ    = ಸಂಖ್ಯೆಗಳು$| (x -> x % ೨ == ೦)           // filter → [೨,೪,೬,೮,೧೦]
ಒಟ್ಟು    = ಸಂಖ್ಯೆಗಳು$< (೦, (ಸಂಚಯಕ, x) -> ಸಂಚಯಕ + x)     // reduce → ೫೫

// ಮಧ್ಯಂತರಗಳ ಮೂಲಕ ಸರಪಳಿ
ಹೆಜ್ಜೆ೧ = ಸಂಖ್ಯೆಗಳು$| (x -> x > ೩)
ಹೆಜ್ಜೆ೨ = ಹೆಜ್ಜೆ೧$> (x -> x * x)
>> ಹೆಜ್ಜೆ೨ ¶    // → [೧೬, ೨೫, ೩೬, ೪೯, ೬೪, ೮೧, ೧೦೦]

// ಹೆಸರಿಸಿದ ಫಂಕ್ಷನ್ಗಳನ್ನು ನೇರವಾಗಿ HOF ಗೆ ರವಾನಿಸಬಹುದು
ದುಪ್ಪಟ್ಟು(x) { <~ x * ೨ }
ದೊಡ್ಡದಾಗಿದೆ(x) { <~ x > ೫ }
r = ಸಂಖ್ಯೆಗಳು$> ದುಪ್ಪಟ್ಟು       // ✅ ನೇರ ಉಲ್ಲೇಖ
r = ಸಂಖ್ಯೆಗಳು$| ದೊಡ್ಡದಾಗಿದೆ       // ✅ ನೇರ ಉಲ್ಲೇಖ
```

## ಪೈಪ್ ಆಪರೇಟರ್

ಬಲಭಾಗವು ಯಾವಾಗಲೂ ಪೈಪ್ ಮಾಡಲಾದ ಮೌಲ್ಯಕ್ಕಾಗಿ `_` ಪ್ಲೇಸ್ಹೋಲ್ಡರ್ ಅನ್ನು ಅಗತ್ಯಪಡಿಸುತ್ತದೆ:

```zymbol
ದುಪ್ಪಟ್ಟು = x -> x * ೨
ಮೊತ್ತ = (a, b) -> a + b
ಹೆಚ್ಚಳ = x -> x + ೧

ಫ೧ = ೫ |> ದುಪ್ಪಟ್ಟು(_)        // → ೧೦
ಫ೨ = ೧೦ |> ಮೊತ್ತ(_, ೫)       // → ೧೫
ಫ೩ = ೫ |> ಮೊತ್ತ(೨, _)        // → ೭

// ಸರಪಳಿಯಾದ
ಫ = ೫ |> ದುಪ್ಪಟ್ಟು(_) |> ಹೆಚ್ಚಳ(_) |> ದುಪ್ಪಟ್ಟು(_)
>> ಫ ¶    // → ೨೨  (೫→೧೦→೧೧→೨೨)
```

## ದೋಷ ನಿರ್ವಹಣೆ

```zymbol
!? {
    x = ೧೦ / ೦
} :! ##Div {
    >> "ಶೂನ್ಯದಿಂದ ಭಾಗಾಕಾರ" ¶
} :! {
    >> "ಇತರೆ: " _err ¶    // _err ದೋಷ ಸಂದೇಶವನ್ನು ಹೊಂದಿರುತ್ತದೆ
} :> {
    >> "ಯಾವಾಗಲೂ ಚಲಿಸುತ್ತದೆ" ¶
}
```

| ಪ್ರಕಾರ | ಯಾವಾಗ |
|------|------|
| `##Div` | ಶೂನ್ಯದಿಂದ ಭಾಗಾಕಾರ |
| `##IO` | ಕಡತ / ವ್ಯವಸ್ಥೆ |
| `##Index` | ವ್ಯಾಪ್ತಿಯ ಹೊರಗಿನ ಸೂಚ್ಯಂಕ |
| `##Type` | ಪ್ರಕಾರದ ಹೊಂದಾಣಿಕೆಯಾಗದಿರುವುದು |
| `##Parse` | ಡೇಟಾ ಪಾರ್ಸಿಂಗ್ |
| `##Network` | ಜಾಲಬಂಧ ದೋಷಗಳು |
| `##_` | ಯಾವುದೇ ದೋಷ (ಎಲ್ಲವನ್ನು-ಸೆರೆಹಿಡಿಯುವ) |

## ಮಾಡ್ಯೂಲ್ಗಳು

```zymbol
// lib/calc.zy — ಮಾಡ್ಯೂಲ್ ದೇಹವು ಬ್ರೇಸ್ಗಳಲ್ಲಿ ಸುತ್ತುವರಿದಿದೆ
# calc {
    #> { ಮೊತ್ತ, get_PI }

    _π := ೩.೧೪೧೫೯
    ಮೊತ್ತ(a, b) { <~ a + b }
    get_PI() { <~ _π }
}
```

```zymbol
// main.zy
<# ./lib/calc => ಸಿ    // ಅಲಿಯಾಸ್ ಅಗತ್ಯವಿದೆ

>> ಸಿ::ಮೊತ್ತ(೫, ೩) ¶     // → ೮
π = ಸಿ::get_PI()
>> π ¶               // → ೩.೧೪೧೫೯
```

```zymbol
// ಬೇರೆ ಸಾರ್ವಜನಿಕ ಹೆಸರಿನೊಂದಿಗೆ ರಫ್ತು ಮಾಡಿ
# mylib {
    #> { _ಆಂತರಿಕ_ಮೊತ್ತ => ಮೊತ್ತ }

    _ಆಂತರಿಕ_ಮೊತ್ತ(a, b) { <~ a + b }
}
```

```zymbol
<# ./mylib => ಎಂ

>> ಎಂ::ಮೊತ್ತ(೩, ೪) ¶    // → ೭  (ಆಂತರಿಕ ಹೆಸರು _ಆಂತರಿಕ_ಮೊತ್ತ ಮರೆಯಾಗಿದೆ)
```

> **ಮಾಡ್ಯೂಲ್ ನಿಯಮಗಳು**: `# ಹೆಸರು { }` ಒಳಗೆ ಕೇವಲ `#>`, ಫಂಕ್ಷನ್ ವ್ಯಾಖ್ಯಾನಗಳು, ಮತ್ತು ಅಕ್ಷರಶಃ ಅಸ್ಥರ/ಸ್ಥಿರಾಂಕ ಆರಂಭಕಗಳನ್ನು ಮಾತ್ರ ಅನುಮತಿಸಲಾಗಿದೆ. ಕಾರ್ಯಗತಗೊಳಿಸಬಲ್ಲ ಹೇಳಿಕೆಗಳು (`>>`, `<<`, ಲೂಪ್ಗಳು, ಇತ್ಯಾದಿ) ದೋಷ E013 ಅನ್ನು ಎತ್ತುತ್ತವೆ.

## ಸಂಖ್ಯಾ ವಿಧಾನಗಳು

Zymbol ಸಂಖ್ಯೆಗಳನ್ನು **69 ಯುನಿಕೋಡ್ ಅಂಕೆಗಳ ಲಿಪಿಗಳಲ್ಲಿ** ಪ್ರದರ್ಶಿಸಬಹುದು — ದೇವನಾಗರಿ, ಅರಬಿ-ಭಾರತೀಯ, ಥಾಯ್, ಕ್ಲಿಂಗನ್ pIqaD, ಗಣಿತದ ದಪ್ಪ, ಎಲ್ಸಿಡಿ ವಿಭಾಗಗಳು, ಮತ್ತು ಹೆಚ್ಚಿನವು. ಸಕ್ರಿಯ ವಿಧಾನವು ಕೇವಲ `>>` ಔಟ್ಪುಟ್ ಅನ್ನು ಮಾತ್ರ ಪರಿಣಾಮಿಸುತ್ತದೆ; ಆಂತರಿಕ ಅಂಕಗಣಿತವು ಯಾವಾಗಲೂ ದ್ವಿಮಾನವಾಗಿರುತ್ತದೆ.

### ಲಿಪಿಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುವುದು

ಗುರಿ ಲಿಪಿಯ `0` ಮತ್ತು `9` ಅಂಕೆಗಳನ್ನು `#…#` ಒಳಗೆ ಬರೆಯಿರಿ:

```zymbol
#०९#    // ದೇವನಾಗರಿ   (U+0966–U+096F)
#٠٩#    // ಅರಬಿ-ಭಾರತೀಯ (U+0660–U+0669)
#๐๙#    // ಥಾಯ್         (U+0E50–U+0E59)
#09#    // ASCII ಗೆ ಮರುಹೊಂದಿಸಿ
```

### ಔಟ್ಪುಟ್ ಮತ್ತು ಬೂಲಿಯನ್ಗಳು

```zymbol
x = ೪೨
>> x ¶          // → ೪೨   (ASCII ಡೀಫಾಲ್ಟ್)

#०९#
>> x ¶          // → ४२
>> ೩.೧೪ ¶       // → ३.१४   (ದಶಮಾಂಶ ಬಿಂದು ಯಾವಾಗಲೂ ASCII)
>> ೧ + ೨ ¶      // → ३

// ಬೂಲಿಯನ್ಗಳು: # ಪೂರ್ವಪ್ರತ್ಯಯ ಯಾವಾಗಲೂ ASCII, ಅಂಕೆ ಅಳವಡಿಸಿಕೊಳ್ಳುತ್ತದೆ
>> #೧ ¶         // → #१   (ದೇವನಾಗರಿಯಲ್ಲಿ ಸತ್ಯ)
>> #೦ ¶         // → #०   (ಸುಳ್ಳು — ೦ ಪೂರ್ಣಾಂಕ ಶೂನ್ಯದಿಂದ ಭಿನ್ನವಾಗಿದೆ)

x = ೨೮ > ೪
>> x ¶          // → #१   (ಹೋಲಿಕೆ ಫಲಿತಾಂಶವು ಸಕ್ರಿಯ ವಿಧಾನವನ್ನು ಅನುಸರಿಸುತ್ತದೆ)
```

### ಮೂಲದಲ್ಲಿ ಸ್ಥಳೀಯ ಅಂಕೆಗಳ ಲಿಟರಲ್ಗಳು

ಯಾವುದೇ ಬೆಂಬಲಿತ ಲಿಪಿಯ ಅಂಕೆಗಳು ಮಾನ್ಯ ಲಿಟರಲ್ಗಳಾಗಿವೆ — ವ್ಯಾಪ್ತಿಗಳಲ್ಲಿ, ಮಾಡುಲೋದಲ್ಲಿ, ಹೋಲಿಕೆಗಳಲ್ಲಿ:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### ಯಾವುದೇ ಲಿಪಿಯಲ್ಲಿ ಬೂಲಿಯನ್ ಲಿಟರಲ್ಗಳು

`#` + ಯಾವುದೇ ಬ್ಲಾಕ್ನಿಂದ ಅಂಕೆ `0` ಅಥವಾ `1` ಮಾನ್ಯ ಬೂಲಿಯನ್ ಲಿಟರಲ್ ಆಗಿದೆ:

```zymbol
#٠٩#
ಸಕ್ರಿಯ = #١        // #೧ ನಂತೆಯೇ
>> ಸಕ್ರಿಯ ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **ಯಾವಾಗಲೂ ASCII** ಆಗಿದೆ. `#0` (ಸುಳ್ಳು) ಯಾವಾಗಲೂ ಪ್ರತಿ ಲಿಪಿಯಲ್ಲಿ `0` (ಪೂರ್ಣಾಂಕ ಶೂನ್ಯ) ದಿಂದ ದೃಷ್ಟಿಗೋಚರವಾಗಿ ಭಿನ್ನವಾಗಿರುತ್ತದೆ.

## ಡೇಟಾ ಆಪರೇಟರ್ಗಳು

```zymbol
// ಪ್ರಕಾರ ಪರಿವರ್ತನೆ ಕಾಸ್ಟ್ಗಳು
f = ##.೪೨         // → ೪೨.೦  (ತೇಲುವ ಬಿಂದುವಿಗೆ)
i = ###೩.೭        // → ೪     (ಪೂರ್ಣಾಂಕಕ್ಕೆ, ಸುತ್ತಿಕೊಳ್ಳಿ)
t = ##!೩.೭        // → ೩     (ಪೂರ್ಣಾಂಕಕ್ಕೆ, ಕತ್ತರಿಸಿ)

// ಸ್ಟ್ರಿಂಗ್ ಅನ್ನು ಸಂಖ್ಯೆಗೆ ಪಾರ್ಸ್ ಮಾಡಿ
v೧ = #|"೪೨"|      // → ೪೨  (ಪೂರ್ಣಾಂಕ)
v೨ = #|"೩.೧೪"|    // → ೩.೧೪  (ತೇಲುವ ಬಿಂದು)
v೩ = #|"abc"|     // → "abc"  (ಸುರಕ್ಷಿತ, ಯಾವುದೇ ದೋಷವಿಲ್ಲ)

// ಸುತ್ತಿಕೊಳ್ಳಿ / ಕತ್ತರಿಸಿ
π = ೩.೧೪೧೫೯೨೬೫
ಸುತ್ತು೨ = #.೨|π|      // → ೩.೧೪  (೨ ದಶಮಾಂಶ ಸ್ಥಾನಗಳಿಗೆ ಸುತ್ತಿಕೊಳ್ಳಿ)
ಸುತ್ತು೪ = #.೪|π|      // → ೩.೧೪೧೬
ಕತ್ತರಿಸು೨ = #!೨|π|      // → ೩.೧೪  (ಕತ್ತರಿಸಿ)

// ಸಂಖ್ಯೆ ಫಾರ್ಮ್ಯಾಟಿಂಗ್
ಫಾರ್ಮ್ಯಾಟ್ = #,|೧೨೩೪೫೬೭|  // → ೧,೨೩೪,೫೬೭  (ಅಲ್ಪವಿರಾಮ-ಪ್ರತ್ಯೇಕಿಸಲಾಗಿದೆ)
ವೈಜ್ಞಾನಿಕ = #^|೧೨೩೪೫.೬೭೮|    // → ೧.೨೩೪೫೬೭೮e೪  (ವೈಜ್ಞಾನಿಕ)

// ಆಧಾರ ಲಿಟರಲ್ಗಳು
a = 0x41         // → 'A'  (ಹೆಕ್ಸ್)
b = 0b01000001   // → 'A'  (ಬೈನರಿ)
c = 0o101        // → 'A'  (ಆಕ್ಟಲ್)

// ಆಧಾರ ಪರಿವರ್ತನೆ ಔಟ್ಪುಟ್
ಹೆಕ್ಸ್ = 0x|೨೫೫|    // → "0x00FF"
ಬೈನರಿ = 0b|೬೫|     // → "0b1000001"
ಆಕ್ಟಲ್ = 0o|೮|      // → "0o10"
ದಶಮಾಂಶ = 0d|೨೫೫|    // → "0d0255"
```

## ಶೆಲ್ ಸಂಯೋಜನೆ

```zymbol
ದಿನಾಂಕ = <\ date +%Y-%m-%d \>     // stdout ಅನ್ನು ಸೆರೆಹಿಡಿಯುತ್ತದೆ (ಕೊನೆಯಲ್ಲಿ \n ಸಹಿತ)
>> "ಇಂದು: " ದಿನಾಂಕ

ಕಡತ = "data.txt"
ವಿಷಯ = <\ cat {ಕಡತ} \>      // ಆಜ್ಞೆಗಳಲ್ಲಿ ಇಂಟರ್ಪೋಲೇಶನ್

ಔಟ್ಪುಟ್ = </"./subscript.zy"/>   // ಮತ್ತೊಂದು Zymbol ಸ್ಕ್ರಿಪ್ಟ್ ಅನ್ನು ಕಾರ್ಯಗತಗೊಳಿಸಿ, ಔಟ್ಪುಟ್ ಅನ್ನು ಸೆರೆಹಿಡಿಯಿರಿ

```

> `><` CLI ಆರ್ಗ್ಯುಮೆಂಟ್ಗಳನ್ನು ಸ್ಟ್ರಿಂಗ್ ಆರೇ ಆಗಿ ಸೆರೆಹಿಡಿಯುತ್ತದೆ (ಕೇವಲ tree-walker).

## ಸಂಕೇತಗಳ ಉಲ್ಲೇಖ

| ಸಂಕೇತ | ಕಾರ್ಯಾಚರಣೆ | ಸಂಕೇತ | ಕಾರ್ಯಾಚರಣೆ |
|--------|-----------|--------|-----------|
| `=` | ಅಸ್ಥರ | `$#` | ಉದ್ದ |
| `:=` | ಸ್ಥಿರಾಂಕ | `$+` | ಸೇರಿಸಿ (ಸರಪಳಿ ಮಾಡಬಹುದಾದ) |
| `>>` | ಔಟ್ಪುಟ್ | `$+[i]` | ಸೂಚ್ಯಂಕದಲ್ಲಿ ಸೇರಿಸಿ (1-ಆಧಾರಿತ) |
| `<<` | ಇನ್ಪುಟ್ | `$-` | ಮೌಲ್ಯದಿಂದ ಮೊದಲನೆಯದನ್ನು ತೆಗೆದುಹಾಕಿ |
| `¶` / `\\` | ಹೊಸ ಸಾಲು | `$--` | ಮೌಲ್ಯದಿಂದ ಎಲ್ಲವನ್ನು ತೆಗೆದುಹಾಕಿ |
| `?` | ಆಗಿದ್ದರೆ | `$-[i]` | ಸೂಚ್ಯಂಕದಲ್ಲಿ ತೆಗೆದುಹಾಕಿ (1-ಆಧಾರಿತ) |
| `_?` | ಇಲ್ಲದಿದ್ದರೆ-ಆಗಿದ್ದರೆ | `$-[i..j]` | ವ್ಯಾಪ್ತಿಯನ್ನು ತೆಗೆದುಹಾಕಿ (1-ಆಧಾರಿತ) |
| `_` | ಇಲ್ಲದಿದ್ದರೆ / ವೈಲ್ಡ್ಕಾರ್ಡ್ | `$?` | ಹೊಂದಿದೆ |
| `??` | ಹೊಂದಾಣಿಕೆ | `$??` | ಎಲ್ಲಾ ಸೂಚ್ಯಂಕಗಳನ್ನು ಹುಡುಕಿ (1-ಆಧಾರಿತ) |
| `@` | ಲೂಪ್ | `$[s..e]` | ಸ್ಲೈಸ್ (1-ಆಧಾರಿತ) |
| `@ N { }` | N ಬಾರಿ ಲೂಪ್ | `$>` | map |
| `@!` | ಮುರಿಯಿರಿ | `$\|` | filter |
| `@>` | ಮುಂದುವರಿಸಿ | `$<` | reduce |
| `@:ಹೆಸರು { }` | ಲೇಬಲ್ ಮಾಡಿದ ಲೂಪ್ | `$/ ಗಡಿ` | ಸ್ಟ್ರಿಂಗ್ ಅನ್ನು ವಿಭಾಗಿಸಿ |
| `@:ಹೆಸರು!` | ಲೇಬಲ್ ಅನ್ನು ಮುರಿಯಿರಿ | `$++ a b c` | ಕಾನ್ಕ್ಯಾಟ್ ನಿರ್ಮಿಸಿ |
| `@:ಹೆಸರು>` | ಲೇಬಲ್ ಅನ್ನು ಮುಂದುವರಿಸಿ | `arr[i>j>k]` | ನ್ಯಾವಿಗೇಶನ್ ಸೂಚ್ಯಂಕ |
| `->` | ಲ್ಯಾಂಬ್ಡಾ | `arr[i] = ಮೌಲ್ಯ` | ಅಂಶ ಅಪ್ಡೇಟ್ ಮಾಡಿ (ಕೇವಲ ಆರೇ) |
| `arr[i] += ಮೌಲ್ಯ` | ಸಂಯುಕ್ತ ಅಪ್ಡೇಟ್ | `arr[i]$~` | ಕಾರ್ಯಾತ್ಮಕ ಅಪ್ಡೇಟ್ (ಹೊಸ ಪ್ರತಿ) |
| `$^+` | ಆರೋಹಣ ವಿಂಗಡಿಸಿ (ಆದಿಮ) | `$^-` | ಅವರೋಹಣ ವಿಂಗಡಿಸಿ (ಆದಿಮ) |
| `$^` | ಹೋಲಿಕೆದಾರನೊಂದಿಗೆ ವಿಂಗಡಿಸಿ (ಟಪಲ್) | `<~` | ಹಿಂತಿರುಗಿಸಿ |
| `\|>` | ಪೈಪ್ | `!?` | ಪ್ರಯತ್ನಿಸಿ |
| `:!` | ಸೆರೆಹಿಡಿಯಿರಿ | `:>` | ಅಂತಿಮವಾಗಿ |
| `#1` | ಸತ್ಯ | `#0` | ಸುಳ್ಳು |
| `$!` | ದೋಷವೇ | `$!!` | ದೋಷವನ್ನು ಪ್ರಸಾರ ಮಾಡಿ |
| `<#` | ಆಮದು ಮಾಡಿ | `#>` | ರಫ್ತು ಮಾಡಿ |
| `#` | ಮಾಡ್ಯೂಲ್ ಘೋಷಿಸಿ | `::` | ಮಾಡ್ಯೂಲ್ ಅನ್ನು ಕರೆಯಿರಿ |
| `.` | ಕ್ಷೇತ್ರ ಪ್ರವೇಶ | `#?` | ಪ್ರಕಾರ ಮೆಟಾಡೇಟಾ |
| `#\|..\|` | ಸಂಖ್ಯೆಯನ್ನು ಪಾರ್ಸ್ ಮಾಡಿ | `##.` | ತೇಲುವ ಬಿಂದುವಿಗೆ ಪರಿವರ್ತಿಸಿ |
| `###` | ಪೂರ್ಣಾಂಕಕ್ಕೆ ಪರಿವರ್ತಿಸಿ (ಸುತ್ತಿಕೊಳ್ಳಿ) | `##!` | ಪೂರ್ಣಾಂಕಕ್ಕೆ ಪರಿವರ್ತಿಸಿ (ಕತ್ತರಿಸಿ) |
| `#.N\|..\|` | ಸುತ್ತಿಕೊಳ್ಳಿ | `#!N\|..\|` | ಕತ್ತರಿಸಿ |
| `#,\|..\|` | ಅಲ್ಪವಿರಾಮ ಫಾರ್ಮ್ಯಾಟ್ | `#^\|..\|` | ವೈಜ್ಞಾನಿಕ |
| `#d0d9#` | ಸಂಖ್ಯಾ ವಿಧಾನವನ್ನು ಬದಲಾಯಿಸಿ | `#09#` | ASCII ಗೆ ಮರುಹೊಂದಿಸಿ |
| `<\ ..\>` | ಶೆಲ್ ಕಾರ್ಯಗತಗೊಳಿಸಿ | `>\<` | CLI ಆರ್ಗ್ಯುಮೆಂಟ್ಗಳು |
| `\ ಅಸ್ಥರ` | ಸ್ಪಷ್ಟವಾಗಿ ಅಸ್ಥರವನ್ನು ನಾಶಪಡಿಸಿ | `°x` / `x°` | ಹಾಟ್ ವ್ಯಾಖ್ಯಾನ (ಸ್ವಯಂ-ಆರಂಭ) |
| `>>|` | ಟಿಯುಐ ಬ್ಲಾಕ್ (ಪರ್ಯಾಯ ಸ್ಕ್ರೀನ್) | `>>~` | ಸ್ಥಾನಿಕ ಔಟ್ಪುಟ್ |
| `>>!` | ಸ್ಕ್ರೀನ್ ಅಳಿಸಿ | `>>?` | ಟರ್ಮಿನಲ್ ಗಾತ್ರವನ್ನು ಕೇಳಿ |
| `<<\|` | ಬ್ಲಾಕಿಂಗ್ ಕೀ ಪ್ರೆಸ್ | `<<\|?` | ನಾನ್-ಬ್ಲಾಕಿಂಗ್ ಕೀ ಪ್ರೆಸ್ ಪೋಲ್ |
| `@~ N` | N ಮಿಲಿಸೆಕೆಂಡುಗಳು ನಿದ್ರಿಸಿ | `$*` | ಸ್ಟ್ರಿಂಗ್ ಅನ್ನು N ಬಾರಿ ಪುನರಾವರ್ತಿಸಿ |

### v0.0.5 — ಟಿಯುಐ ಪ್ರಾಥಮಿಕಗಳು, ಹಾಟ್ ವ್ಯಾಖ್ಯಾನ ಮತ್ತು ಸ್ಟ್ರಿಂಗ್ ಪುನರಾವರ್ತನೆ _(ಮೇ ೨೦೨೬)_

- **ಬ್ರೇಕಿಂಗ್** ಮ್ಯಾಚ್ ಆರ್ಮ್ ಸೆಪರೇಟರ್: `ಮಾದರಿ : ಫಲಿತಾಂಶ` → `ಮಾದರಿ => ಫಲಿತಾಂಶ`
- **ಬ್ರೇಕಿಂಗ್** ಆಮದು ಅಲಿಯಾಸ್: `<# ಮಾರ್ಗ <= ಅಲಿಯಾಸ್` → `<# ಮಾರ್ಗ => ಅಲಿಯಾಸ್`
- **ಬ್ರೇಕಿಂಗ್** ರಫ್ತು ಮರುಹೆಸರಿಸು: `#> { fn <= ಪಬ್ }` → `#> { fn => ಪಬ್ }`
- **ಸೇರಿಸಲಾಗಿದೆ** ಟಿಯುಐ ಬ್ಲಾಕ್ `>>| { }` — ಪರ್ಯಾಯ ಸ್ಕ್ರೀನ್ + ಕಚ್ಚಾ ಮೋಡ್; ನಿರ್ಗಮಿಸಿದಾಗ ಸ್ವಚ್ಛಗೊಳಿಸುತ್ತದೆ
- **ಸೇರಿಸಲಾಗಿದೆ** ಸ್ಥಾನಿಕ ಔಟ್ಪುಟ್ `>>~ (ಸಾಲು, ಕಾಲಮ್, BKS, fg, bg) > ಐಟಂಗಳು` — ಅಪರೂಪದ ಸ್ಲಾಟ್ಗಳು, ANSI 256 ಬಣ್ಣ
- **ಸೇರಿಸಲಾಗಿದೆ** ಕೀ ಇನ್ಪುಟ್ `<<| ಅಸ್ಥರ` (ಬ್ಲಾಕಿಂಗ್) ಮತ್ತು `<<|? ಅಸ್ಥರ` (ನಾನ್-ಬ್ಲಾಕಿಂಗ್ ಪೋಲ್)
- **ಸೇರಿಸಲಾಗಿದೆ** `>>!` ಸ್ಕ್ರೀನ್ ಅಳಿಸಿ, `>>?` ಟರ್ಮಿನಲ್ ಗಾತ್ರವನ್ನು ಕೇಳಿ, `@~ N` N ಮಿಲಿಸೆಕೆಂಡುಗಳು ನಿದ್ರಿಸಿ
- **ಸೇರಿಸಲಾಗಿದೆ** ಹಾಟ್ ವ್ಯಾಖ್ಯಾನ `°x` / `x°` — ಲೂಪ್ಗಳಲ್ಲಿ ಮೊದಲ ಬಳಕೆಯಲ್ಲಿ ಅಸ್ಥರ ಸ್ವಯಂಚಾಲಿತ ಆರಂಭ
- **ಸೇರಿಸಲಾಗಿದೆ** ಸ್ಟ್ರಿಂಗ್ ಪುನರಾವರ್ತನೆ `ಸ್ಟ್ರಿಂಗ್ $* N` — ಸ್ಟ್ರಿಂಗ್ ಅನ್ನು N ಬಾರಿ ಪುನರಾವರ್ತಿಸಿ
- **ವಿಎಂ** ಸಮಾನತೆ: 436/436 ಪರೀಕ್ಷೆಗಳು ಉತ್ತೀರ್ಣ

### v0.0.4 — 1-ಆಧಾರಿತ ಸೂಚ್ಯಂಕೀಕರಣ, ಪ್ರಥಮ-ಶ್ರೇಣಿಯ ಫಂಕ್ಷನ್ಗಳು ಮತ್ತು ಬ್ಲಾಕ್ ಮಾಡ್ಯೂಲ್ಗಳು _(ಏಪ್ರಿಲ್ ೨೦೨೬)_

- **ಬ್ರೇಕಿಂಗ್** ಎಲ್ಲಾ ಸೂಚ್ಯಂಕೀಕರಣವನ್ನು **1-ಆಧಾರಿತ** ಗೆ ಬದಲಾಯಿಸಲಾಗಿದೆ — `arr[1]` ಮೊದಲ ಅಂಶವಾಗಿದೆ; `arr[0]` ರನ್ಟೈಮ್ ದೋಷವಾಗಿದೆ
- **ಸೇರಿಸಲಾಗಿದೆ** ಹೆಸರಿಸಿದ ಫಂಕ್ಷನ್ಗಳು **ಪ್ರಥಮ-ಶ್ರೇಣಿಯ ಮೌಲ್ಯಗಳಾಗಿವೆ** — ನೇರವಾಗಿ HOF ಗೆ ರವಾನಿಸಿ: `ಸಂಖ್ಯೆಗಳು$> ದುಪ್ಪಟ್ಟು`
- **ಸೇರಿಸಲಾಗಿದೆ** ಮಾಡ್ಯೂಲ್ಗಳಿಗೆ **ಬ್ಲಾಕ್ ವಾಕ್ಯರಚನೆ ಅಗತ್ಯ** — `# ಹೆಸರು { ... }` — ಫ್ಲಾಟ್ ವಾಕ್ಯರಚನೆಯನ್ನು ತೆಗೆದುಹಾಕಲಾಗಿದೆ
- **ಸೇರಿಸಲಾಗಿದೆ** ಬಹು-ಆಯಾಮದ ಸೂಚ್ಯಂಕೀಕರಣ: `arr[i>j>k]` (ನ್ಯಾವಿಗೇಶನ್), `arr[p ; q]` (ಫ್ಲಾಟ್ ಹೊರತೆಗೆಯುವಿಕೆ)
- **ಸೇರಿಸಲಾಗಿದೆ** ಪ್ರಕಾರ ಪರಿವರ್ತನೆ ಕಾಸ್ಟ್ಗಳು: `##.ಎಕ್ಸ್ಪ್ರೆಶನ್` (ತೇಲುವ ಬಿಂದು), `###ಎಕ್ಸ್ಪ್ರೆಶನ್` (ಪೂರ್ಣಾಂಕ ಸುತ್ತಿಕೊಳ್ಳಿ), `##!ಎಕ್ಸ್ಪ್ರೆಶನ್` (ಪೂರ್ಣಾಂಕ ಕತ್ತರಿಸಿ)
- **ಸೇರಿಸಲಾಗಿದೆ** ಸ್ಟ್ರಿಂಗ್ ವಿಭಾಗ: `ಸ್ಟ್ರಿಂಗ್$/ ಗಡಿ` — `Array(ಸ್ಟ್ರಿಂಗ್)` ಹಿಂತಿರುಗಿಸುತ್ತದೆ
- **ಸೇರಿಸಲಾಗಿದೆ** ಕಾನ್ಕ್ಯಾಟ್ ನಿರ್ಮಿಸಿ: `ಆಧಾರ$++ a b c` — ಬಹು ಐಟಂಗಳನ್ನು ಸೇರಿಸುತ್ತದೆ
- **ಸೇರಿಸಲಾಗಿದೆ** ಟೈಮ್ಸ್ ಲೂಪ್: `@ N { }` — ನಿಖರವಾಗಿ N ಬಾರಿ ಪುನರಾವರ್ತಿಸಿ
- **ಸೇರಿಸಲಾಗಿದೆ** ಲೇಬಲ್ ಮಾಡಿದ ಲೂಪ್ ವಾಕ್ಯರಚನೆ: `@:ಹೆಸರು { }`, `@:ಹೆಸರು!`, `@:ಹೆಸರು>` — `@ @ಹೆಸರು` / `@! ಹೆಸರು` ಅನ್ನು ಬದಲಾಯಿಸುತ್ತದೆ
- **ಸೇರಿಸಲಾಗಿದೆ** ಅಸ್ಥರ ಸ್ಕೋಪ್ ನಿಯಮಗಳು: `_ಹೆಸರು` ಅಸ್ಥರಗಳು ನಿಖರವಾದ ಬ್ಲಾಕ್ ಸ್ಕೋಪ್ ಅನ್ನು ಹೊಂದಿವೆ; `\ ಅಸ್ಥರ` ಬೇಗನೆ ನಾಶಪಡಿಸುತ್ತದೆ
- **ಸೇರಿಸಲಾಗಿದೆ** ಮ್ಯಾಚ್ ಹೋಲಿಕೆ ಮಾದರಿಗಳು: `< 0 =>`, `> 5 =>`, `== 42 =>` ಇತ್ಯಾದಿ
- **ಸೇರಿಸಲಾಗಿದೆ** ಮಾಡ್ಯೂಲ್ E013 ದೋಷ: ಮಾಡ್ಯೂಲ್ ದೇಹದಲ್ಲಿ ಕಾರ್ಯಗತಗೊಳಿಸಬಲ್ಲ ಹೇಳಿಕೆಗಳನ್ನು ನಿಷೇಧಿಸಲಾಗಿದೆ
- **ಸರಿಪಡಿಸಲಾಗಿದೆ** `alias.CONST` ಈಗ ಸರಿಯಾಗಿ ಪರಿಹರಿಸುತ್ತದೆ; `#>` ಫಂಕ್ಷನ್ ವ್ಯಾಖ್ಯಾನಗಳ ನಂತರ ಬರಬಹುದು
- **ವಿಎಂ** ಸಂಪೂರ್ಣ ಸಮಾನತೆ: 393/393 ಪರೀಕ್ಷೆಗಳು ಉತ್ತೀರ್ಣ
