## ᑎᐯᒋᑲᐣ ᐊᑐᓯᑫᐃᐧᓇ

ᑎᐯᒋᑲᐣ UI ᐊᑐᓯᑫᐃᐧᓇ ᑲᑭᑑᒋᑫᐃᐧᐣ. ᒥᔦᒃ ᑲᑕᑯᑦᒥᑕᐠ `>>| { }` (alternate screen + raw mode).

```zymbol
>>| {
    >>!                             // ᑲᒥᑳᐣ ᑎᐯᒋᑲᐣ
    >>~ (1, 1, 0, 10) > "ᐃᑐᐦᑌᐤ"  // ᑐᑕᑫᐃᐧᐣ 1, ᓇᓂᑕᐃᐧᐣ 1, wacistakay=10 (ᓴᑭᑌᐤ)
    @~ 1000                         // ᑲᑕᑭᐦᑕᐣ 1 ᐃᑭᑑ (1000 ms)
    >>~ (2, 1) > "ᑭᑲᑕᐁᐧᐤ"
}
// ᑎᐯᒋᑲᐣ ᑲᑭᑐᐦᑌᐤ ᐃᔅᑿᑐᐦᑌᐤ
```

```zymbol
// ᒪᑯᔑᑎᑕᑫᐃᐧᐣ ᐁᑿ ᑎᐯᒋᑲᐣ ᒥᓯᐁᐧ
>>| {
    [tihtapiwin, askiwetapiwin] = >>?              // ᑭᒋᒋᑫᐃᐧᐣ ᑎᐯᒋᑲᐣ ᐊᔅᑭ
    >>~ (1, 1) > "ᑎᐯᒋᑲᐣ: " tihtapiwin " x " askiwetapiwin
    <<| maskimot                         // ᒪᑯᔑᑎᑕᑫᐃᐧᐣ ᑲᑕᑭᐦᑕᐣ
    >>~ (2, 1) > "ᑭᒋᐱᑕᒪᐊᐧᐣ: " maskimot
}
```

> `>>!` ᑲᒥᑳᐣ ᑎᐯᒋᑲᐣ. `>>?` ᑭᑐᐦᑌᐤ `[rows, cols]`. `@~ N` ᑲᑕᑭᐦᑕᐣ N milliseconds.
> `<<|` ᑲᑭᒋᑕᑕᐣ ᒪᑯᔑᑎᑕᑫᐃᐧᐣ; `<<|?` ᑲᐊᔅᑭᑯᑕᑫᐃᐧᐣ (ᑭᑐᐦᑌᐤ `'\0'` ᑲᑲᓇ).
> `(row, col, BKS, fg, bg)` — ᑲᓇ ᑲᑌᑦᑭᓯᑌᐤ ᑲᐊᑐᓯᑫᐃᐧᐣ (`>>~ (,,, 196) > "red"`).
> BKS: `1`=Bold, `2`=Italic, `4`=Underline. ANSI 256-color (`0`=terminal default).
> Tree-walker ᐊᑭ (ᑲᑲᓇ `>>!`, `>>?`, `@~`, `>>~` ᑲᑐᑕᑫᐃᐧᐣ `--vm`).

## ᑲᑰᒋᑲᑕᒥᑕᐠ

ᑲᑰᒋᑲᑕᒥᑕᐠ **ᑲᑎᐸᐦᑳᑌᐠ** ᐁᑿ ᑲᐊᔅᑭᑌᐤ ᑎᐸᐦᐃᑕᒧᐃᐧᓇ ᑕᑕᐤ **ᑲᑕᑯᑦᒥᑕᐠ**.

```zymbol
masinahikanak = [1, 2, 3, 4, 5]

x = masinahikanak[1]      // 1 — ᑲᐊᔅᑭᑎᑕᑫᐃᐧᐣ (1-ᐃᐧᑌᑦ: ᑲᐊᔅᑭᑎᑕᑫᐃᐧᐣ)
x = masinahikanak[-1]     // 5 — ᑲᑌᑦᑭᓯᑌᐤ ᑎᐸᐦᐃᑕᒧᐃᐧᐣ (ᑲᑌᑦᑭᓯᑌᐤ ᑭᑑᒋᑲᐣ)
x = masinahikanak$#       // 5 — ᒥᓯᐁᐧ (ᑲᐊᑐᓯᑫᐃᐧᐣ (masinahikanak$#) ᑭᑑᒋᑲᐣ >>)

masinahikanak = masinahikanak$+ 6            // ᑲᒋᑲᐣ → [1,2,3,4,5,6]
arr2 = masinahikanak$+[2] 99       // ᑲᐊᑐᓯᑫᐃᐧᐣ ᑎᐸᐦᐃᑕᒧᐃᐧᐣ 2 (1-ᐃᐧᑌᑦ)
arr3 = masinahikanak$- 3           // ᑲᑌᑦᑭᓯᑕᑫᐃᐧᐣ ᑲᐊᔅᑭᑌᐤ
arr4 = masinahikanak$-- 3          // ᑲᑌᑦᑭᓯᑕᑫᐃᐧᐣ ᑲᑕᑯᑦᒥᑕᐠ
arr5 = masinahikanak$-[1]          // ᑲᑌᑦᑭᓯᑕᑫᐃᐧᐣ ᑎᐸᐦᐃᑕᒧᐃᐧᐣ 1 (ᑲᐊᔅᑭᑎᑕᑫᐃᐧᐣ)
arr6 = masinahikanak$-[2..3]       // ᑲᑌᑦᑭᓯᑕᑫᐃᐧᐣ ᑲᑭᑎᐸᐦᐃᑕᒥᑕᐠ (1-ᐃᐧᑌᑦ)

ayaw = masinahikanak$? 3            // #1 — ᑲᐊᔅᑭᑌᐤ
nawat = masinahikanak$?? 3           // [3] — ᑲᑕᑯᑦᒥᑕᐠ (1-ᐃᐧᑌᑦ)
asiwisiwin = masinahikanak$[1..3]          // [1,2,3] — ᑲᒥᐣᒋᑲᑕᒥᑕᐠ (1-ᐃᐧᑌᑦ)
asiwisiwin2 = masinahikanak$[1:3]          // [1,2,3] — ᑲᐃᐧᔑᑳᑌᐠ

sihkihew = masinahikanak$^+             // ᑲᑲᑲᐃᐧᑲᑕᒥᑕᐠ ᑲᑲᑲᐃᐧᑭᐠ
sihkihewin = masinahikanak$^-            // ᑲᑲᑲᐃᐧᑲᑕᒥᑕᐠ ᑲᑌᑦᑭᓯᑕᑫᐃᐧᐣ

// ᑲᑎᑕᒥᑫᐃᐧᐣ ᑲᐱᑰᓱᒥᑕᐠ — ᑲᐊᑐᓯᑫᐃᐧᐣ $^ ᑲᐱᒥᐣᒋᑕᑫᐃᐧᓇ
tipahinamowin = [(ishinikaw: "ᑯᔅᑳ", kisiso: 28), (ishinikaw: "ᐊᓇ", kisiso: 25), (ishinikaw: "ᒫᑎᒻ", kisiso: 30)]
tipahikisew  = tipahinamowin$^ (a, b -> a.kisiso < b.kisiso)    // ᑲᑲᑲᐃᐧᑲᑕᒥᑕᐠ kisiso  (<)
tipahinikaw = tipahinamowin$^ (a, b -> a.ishinikaw > b.ishinikaw)  // ᑲᑲᑲᐃᐧᑲᑕᒥᑕᐠ ishinikaw (>)
>> tipahikisew[1].ishinikaw ¶     // → ᐊᓇ
>> tipahinikaw[1].ishinikaw ¶    // → ᑯᔅᑳ

// ᑲᑭᔓᒪᐦᑳᑌᐠ ᑲᑎᑕᒥᑫᐃᐧᐣ (ᑲᑰᒋᑲᑕᒥᑕᐠ ᐊᑭᐦᑕᓱᐃᐧᓇ ᐅᑕᑯ)
masinahikanak[1] = 99              // ᑲᐃᔑᑕᑫᐃᐧᐣ
masinahikanak[2] += 5              // ᑲᒥᓯᐁᐧ: +=  -=  *=  /=  %=  ^=

// ᑲᐊᑐᓯᑫᐃᐧᐣ ᑲᑎᐸᐦᑳᑌᐠ — ᑲᑭᑐᐦᑌᐤ ᑲᐊᔅᑭᑌᐤ ᑎᐸᐦᐃᑕᒧᐃᐧᐣ; ᐁᑿᐁᑿ ᑲᑕᑯᒋᑳᑌᐤ
arr2 = masinahikanak[2]$~ 99
```

> ᑲᑕᑯᑦᒥᑕᐠ ᐊᑐᓯᑫᐃᐧᓇ ᑭᑐᐦᑌᐤ **ᑲᑌᑦᑭᓯᑕᑫᐃᐧᐣ ᑲᐊᔅᑭᑌᐤ ᑲᑰᒋᑲᑕᒥᑕᐠ**.
> `$+` ᑲᑲᑲᐃᐧᑕᑫᐃᐧᐣ: `arr = arr$+ 5$+ 6$+ 7`. ᑯᑕᑯ ᑲᑕᑯᑦᒥᑕᐠ ᑲᐊᑐᓯᑫᐃᐧᐣ.
> **1-ᐃᐧᑌᑦ**: `masinahikanak[1]` ᐯᔭᐤ ᑭᑑᒋᑲᐣ; `masinahikanak[0]` ᑭᑑᒋᑲᐣ ᑲᑭᑐᐦᑌᐤ.
> `$^+` / `$^-` ᑲᑲᑲᐃᐧᑲᑕᒥᑕᐠ ᐊᑭᐦᑕᓱᐃᐧᓇ. ᑲᐱᑰᓱᒥᑕᐠ ᑲᐊᑐᓯᑫᐃᐧᐣ `$^` ᑲᐱᒥᐣᒋᑕᑫᐃᐧᓇ.

**ᑎᐸᐦᐃᑕᒧᐃᐧᓇ** — ᑲᑕᑯᑦᒥᑕᐠ ᑲᑰᒋᑲᑕᒥᑕᐠ ᑲᑭᑐᐦᑌᐤ ᑭᑐᐦᑌᐤ ᐁᑿᐁᑿ:

```zymbol
a = [1, 2, 3]
b = a
a[1] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b ᑲᑕᑯᒋᑳᑌᐤ ᐁᑿᐁᑿ
```

```zymbol
// ᑲᑲᑲᐃᐧᑭᐠ ᑲᑰᒋᑲᑕᒥᑕᐠ (1-ᐃᐧᑌᑦ)
tapikewin = [[1,2,3],[4,5,6],[7,8,9]]
>> tapikewin[2][3] ¶    // → 6  (ᑐᑕᑫᐃᐧᐣ 2, ᓇᓂᑕᐃᐧᐣ 3)
```

## ᑲᐱᑰᓱᒥᑕᐠ

ᑲᐱᑰᓱᒥᑕᐠ **ᑲᑲᓇ ᑲᑎᐸᐦᑳᑌᐠ** ᑲᐊᔅᑭᑌᐤ ᑎᐸᐦᐃᑕᒧᐃᐧᓇ **ᑯᑕᑯ ᑎᐸᐦᐃᑕᒧᐃᐧᓇ**.
ᑲᓇ ᑲᑰᒋᑲᑕᒥᑕᐠ, ᑲᑲᓇ ᑲᑎᐸᐦᑕᒧᐃᐧᐣ ᑲᑭᑐᐦᑌᐤ.

```zymbol
// ᑲᐊᔅᑭᑎᑕᑫᐃᐧᐣ — ᑲᓂᓱᐦᒥᑕᑭᐣ ᑎᐸᐦᐃᑕᒧᐃᐧᓇ
nawat = (10, 20)
>> nawat[1] ¶    // → 10

tipahinamowin = (42, "ᑕᓂᓯ", #1, 3.14)
>> tipahinamowin[3] ¶     // → #1

// ᑲᑎᑕᒥᑫᐃᐧᐣ
ayisiyin = (ishinikaw: "ᒥᑲᐧ", kisiso: 25)
>> ayisiyin.ishinikaw ¶    // → ᒥᑲᐧ
>> ayisiyin[1] ¶      // → ᒥᑲᐧ  (ᐊᑐᓯᑫᐃᐧᐣ ᑲᐃᑕᑯᑦᒥᑕᐠ, 1-ᐃᐧᑌᑦ)

// ᑲᑲᑲᐃᐧᑭᐠ
nawatokiwin = (x: 10, y: 20)
p = (nawatokiwin: nawatokiwin, ishinikaw: "ᐊᔅᑭᒪᓱᐃᐧᐣ")
>> p.nawatokiwin.x ¶        // → 10
```

**ᑲᑲᓇ ᑲᑎᐸᐦᑳᑌᐠ** — ᑲᑎᐸᐦᑳᑌᐠ ᑲᑎᑕᒥᑫᐃᐧᐣ ᑲᑐᑕᑫᐃᐧᐣ ᑭᑑᒋᑲᐣ ᑲᑭᑐᐦᑌᐤ:

```zymbol
t = (10, 20, 30)
// t[1] = 99    // ❌ ᑲᑐᑕᑫᐃᐧᐣ ᑲᑭᑐᐦᑌᐤ: ᑲᐱᑰᓱᐃᐧᓇ ᑲᑲᓇ ᑲᑎᐸᐦᑕᒧᐃᐧᐣ
// t[1] += 5    // ❌ ᑲᐃᐧᔑᑳᑌᐠ ᑲᑐᑕᑫᐃᐧᐣ
```

ᑲᐊᑐᓯᑫᐃᐧᐣ `$~` (ᑲᑎᐸᐦᑳᑌᐠ) — ᑭᑐᐦᑌᐤ **ᑲᑌᑦᑭᓯᑕᑫᐃᐧᐣ** ᑲᐱᑰᓱᒥᑕᐠ:

```zymbol
t = (10, 20, 30)
t2 = t[2]$~ 999
>> t ¶     // → (10, 20, 30)   ← ᐁᑿᐁᑿ ᑲᑕᑯᒋᑳᑌᐤ
>> t2 ¶    // → (10, 999, 30)

// ᑲᑎᑕᒥᑫᐃᐧᐣ ᑲᐱᑰᓱᒥᑕᐠ — ᑲᑲᑲᐃᐧᑕᑫᐃᐧᐣ ᑕᑕᐤ
ayisiyin = (ishinikaw: "ᒥᑲᐧ", kisiso: 25)
kiseyihtaw  = (ishinikaw: ayisiyin.ishinikaw, kisiso: 26)
>> ayisiyin.kisiso ¶    // → 25
>> kiseyihtaw.kisiso ¶     // → 26
```

## ᑲᑲᑐᐦᑌᑭᐠ ᐊᑐᓯᑫᐃᐧᐣ

ᑲᑐᑕᑫᐃᐧᐣ ᑲᑕᑯᑦᒥᑕᐠ `_` ᑲᐃᑐᐦᑌᐤ:

```zymbol
nisowatam = x -> x * 2
mamawi = (a, b) -> a + b

r1 = 5 |> nisowatam(_)        // → 10

r3 = 5 |> mamawi(2, _)        // → 7

// ᑲᑲᑲᐃᐧᑕᑫᐃᐧᐣ
r = 5 |> nisowatam(_) |> peyakwatam(_) |> nisowatam(_)
>> r ¶    // → 22  (5→10→11→22)
```

## ᐊᑭᐦᑕᓱᐃᐧᓇ ᑲᑎᑕᒥᑫᐃᐧᐣ

Zymbol ᑭᑑᒋᑲᐣ ᐊᑭᐦᑕᓱᐃᐧᓇ ᑲᑕᑯᑦᒥᑕᐠ **69 Unicode** — Devanagari, Arabic-Indic, Thai, Klingon pIqaD, ᐁᑿ ᑯᑕᑯ. ᑲᐊᔅᑭᑌᐤ `>>` ᐊᑭ; ᑲᐊᑭᐦᑕᒥᑕᐠ binary ᑕᑕᐤ.

### ᑲᑲᓇᑌᐦᑲᒋᒥᑕᐠ ᒫᑯᑦ ᑲᑕᑯᑦᒥᑕᐠ

ᑲᒪᓯᓇᐦᐃᑳᑌᒃ `0` ᐁᑿ `9` ᑲᑕᑯᑦᒥᑕᐠ `#…#`:

```zymbol
#०९#    // Devanagari   (U+0966–U+096F)
#٠٩#    // Arabic-Indic (U+0660–U+0669)
#๐๙#    // Thai         (U+0E50–U+0E59)
#09#    // ᑲᑭᑐᐦᑌᐤ ASCII
```

### ᑭᑑᒋᑲᐣ ᐁᑿ ᐊᐧᑳᐃᐧ ᓈᓂᑕᐃᐧᓇ

```zymbol
x = 42
>> x ¶          // → 42   (ASCII ᑲᑲᑲᐃᐧᑭᐠ)

#०९#
>> x ¶          // → ४२
>> 3.14 ¶       // → ३.१४   (ᑲᐊᑐᓯᑫᐃᐧᐣ ASCII ᑕᑕᐤ)
>> 1 + 2 ¶      // → ३

// ᐊᐧᑳᐃᐧ ᓈᓂᑕᐃᐧᓇ: # ᑕᑕᐤ ASCII, ᐊᑭᐦᑕᓱᐃᐧᐣ ᑲᑎᐸᐦᑕᒧᐃᐧᐣ
>> #1 ¶         // → #१   (ᑌᐱᐊᐧᐤ Devanagari)
>> #0 ¶         // → #०   (ᐸᐣ — ᐁᑿᐁᑿ ᑲᑕᑯᒋᑳᑌᐤ ᑭᓯᒋ ᐊᑭᐦᑕᓱᐃᐧᐣ ᑭᓯᒋ)

x = 28 > 4
>> x ¶          // → #१   (ᑲᑭᔓᒪᐦᑳᑌᐠ ᑲᑎᐸᐦᑕᒧᐃᐧᐣ)
```

### ᑎᐸᐦᐃᑕᒧᐃᐧᓇ ᑲᑐᑕᑫᐃᐧᐣ ᐅᒋ ᒫᑎᓂᓱᐃᐧᐣ

ᑲᑕᑯᑦᒥᑕᐠ ᑎᐸᐦᐃᑕᒧᐃᐧᓇ ᑲᑕᑯᑦᒥᑕᐠ — ᑲᑭᑎᐸᐦᐃᑕᒥᑕᐠ, modulo, ᑲᑭᔓᒪᐦᑳᑌᐠ:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### ᐊᐧᑳᐃᐧ ᓈᓂᑕᐃᐧᓇ ᑕᓂᑌ ᒫᑯᑦ ᑲᑕᑯᑦᒥᑕᐠ

`#` + ᐊᑭᐦᑕᓱᐃᐧᐣ `0` ᐁᑿ `1` ᑲᑕᑯᑦᒥᑕᐠ ᑕᑕᐤ ᐊᐧᑳᐃᐧ ᓈᓂᑕᐃᐧᐣ:

```zymbol
#٠٩#
kisisow = #١        // ᑲᐃᐧᔑᑳᑌᐠ #1
>> kisisow ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **ᑕᑕᐤ ASCII**. `#0` (ᐸᐣ) ᑕᑕᐤ ᑲᑕᑯᒋᑳᑌᐤ `0` (ᐊᑭᐦᑕᓱᐃᐧᐣ ᑭᓯᒋ) ᑕᓂᑌ.

## ᑎᐸᐦᐃᑕᒧᐃᐧᓇ ᐊᑐᓯᑫᐃᐧᓇ

```zymbol

f = ##.42         // → 42.0  (ᑲᑎᐸᐦᑕᒧᐃᐧᐣ Float)

t = ##!3.7        // → 3     (ᑲᑎᐸᐦᑕᒧᐃᐧᐣ Int, ᑲᑌᑦᑭᓯᑌᐤ)

// ᑲᑎᐸᐦᑳᑌᐠ ᒪᓯᓇᐦᐃᑫᐃᐧᓇ ᑲᐊᑭᐦᑕᓱᐃᐧᐣ
v1 = #|"42"|      // → 42  (Int)
v2 = #|"3.14"|    // → 3.14  (Float)
v3 = #|"abc"|     // → "abc"  (ᑲᑭᔓᒪᐦᑳᑌᐠ, ᐁᑿᐁᑿ ᑭᑑᒋᑲᐣ)

// ᑲᑲᑲᐃᐧᑭᐠ / ᑲᑌᑦᑭᓯᑌᐤ
pi = 3.14159265
r2 = #.2|pi|      // → 3.14  (ᑲᑲᑲᐃᐧᑭᐠ 2 ᑲᐊᑭᐦᑕᓱᐃᐧᐣ)
r4 = #.4|pi|      // → 3.1416
t2 = #!2|pi|      // → 3.14  (ᑲᑌᑦᑭᓯᑌᐤ)

// ᑲᑎᐸᐦᑳᑌᐠ ᐊᑭᐦᑕᓱᐃᐧᓇ
itwewin = #,|1234567|  // → 1,234,567  (ᑲᑎᐸᐦᐃᑕᒧᐃᐧᐣ)
akihtasiskahkew = #^|12345.678|    // → 1.2345678e4  (ᑲᐊᑭᐦᑕᓱᐃᐧᐣ)

// ᑎᐸᐦᐃᑕᒧᐃᐧᓇ ᑲᑐᑕᑫᐃᐧᐣ
a = 0x41         // → 'A'  (hex)
b = 0b01000001   // → 'A'  (binary)
c = 0o101        // → 'A'  (octal)

// ᑲᑎᐸᐦᑳᑌᐠ ᑎᐸᐦᐃᑕᒧᐃᐧᓇ
hex = 0x|255|    // → "0x00FF"
bin = 0b|65|     // → "0b1000001"
oct = 0o|8|      // → "0o10"
dec = 0d|255|    // → "0d0255"
```
