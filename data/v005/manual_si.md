## TUI මූලිකාංග

අන්තර්ක්‍රියාකාරී වැඩසටහන් සඳහා ටර්මිනල් පරිශීලක අතුරු මුහුණත ක්‍රියාකාරක. බොහෝමයකට `>>| { }` කොටස අවශ්‍ය වේ (විකල්ප තිරය + අමු මාදිලිය).

```zymbol
>>| {
    >>!                             // විකල්ප තිරය පිරිසිදු කරන්න
    >>~ (෧, ෧, ෦, ෧෦) > "ධාවනය වේ"   // පේළිය ෧, තීරුව ෧, fg=෧෦ (කොළ)
    @~ ෧෦෦෦                         // තත්පර ෧ ක් නවත්වන්න (෧෦෦෦ ms)
    >>~ (෨, ෧) > "සම්පූර්ණයි."
}
// පිටවන විට ටර්මිනලය ස්වයංක්‍රීයව ප්‍රතිෂ්ඨාපනය වේ
```

```zymbol
// යතුරු ඔබීම සහ ටර්මිනල් ප්‍රමාණය
>>| {
    [පේළි, තීරු] = >>?              // ටර්මිනලයේ මානයන් විමසන්න
    >>~ (෧, ෧) > "ටර්මිනලය: " පේළි " x " තීරු
    <<| යතුර                         // අවහිර කරන යතුරු ඔබීම කියවන්න
    >>~ (෨, ෧) > "ඔබන ලදී: " යතුර

```

> `>>!` තිරය පිරිසිදු කරයි. `>>?` `[පේළි, තීරු]` ආපසු ලබා දෙයි. `@~ N` N මිලිතත්පර නිදා ගනී.
> `<<|` එක් යතුරු ඔබීමක් කියවයි (අවහිර කරන); `<<|?` අවහිර නොකර පොල් කරයි (කිසිවක් නැතිනම් `'\0'` ආපසු ලබා දෙයි).
> ස්ථානීය ප්‍රතිදාන ටපලය: `(පේළිය, තීරුව, BKS, fg, bg)` — ඕනෑම ස්ලොට් එකක් කොමාවකින් අත්හැරිය හැක (`>>~ (,,, 196) > "රතු"`).
> BKS බිටු ආවරණය: `1`=තදකුරු, `2`=ඇලකුරු, `4`=යටිරේඛාව. ANSI 256 වර්ණ පුවරුව (`0`=ටර්මිනල් පෙරනිමිය).
> tree-walker පමණි (`>>!`, `>>?`, `@~`, `>>~` හැර ඒවා `--vm` තුළද ක්‍රියා කරයි).

```zymbol
ඇත = s$? "ලෝකය"          // #෧
කොටස් = "a,b,c,d"$/ ','   // [a, b, c, d]  (සීමාකාරකයෙන් බෙදන්න)
ප්‍රතිස්ථාපනය = s$~~["ල":"ර"]        // "හෙරෝ ලෝකය"
ප්‍රතිස්ථාපනය෧ = s$~~["ල":"ර":෧]     // "හෙරෝ ලෝකය"  (පළමු N පමණි)
රේඛාව = "─" $* ෨෦           // "────────────────────"  (N වාරයක් පුනරුච්චාරණය)
```

> `+` අදාළ වන්නේ සංඛ්‍යා සඳහා පමණි. තන්තු සඳහා `,`, juxtaposition, හෝ අන්තර්කේෂණය භාවිතා කරන්න.

## පාලන ප්‍රවාහය

```zymbol
x = ෭

? x > ෦ { >> "ධනාත්මක" ¶ }

? x > ෧෦෦ {
    >> "විශාල" ¶
} _? x > ෦ {
    >> "ධනාත්මක" ¶
} _? x == ෦ {
    >> "ශුන්‍ය" ¶
} _ {
    >> "ඍණාත්මක" ¶
}
```

> වරහන් `{ }` තනි ප්‍රකාශයක් වුවද **අවශ්‍ය** වේ.

## ගැලපීම

```zymbol
// පරාසයන්
ලකුණු = ෮෫
ශ්‍රේණිය = ?? ලකුණු {
    ෯෦..෧෦෦ => 'අ'
    ෮෦..෮෯  => 'ආ'
    ෭෦..෭෯  => 'ඉ'
    _       => 'ඊ'
}
>> ශ්‍රේණිය ¶    // → ආ

// තන්තු
වර්ණය = "රතු"
කේතය = ?? වර්ණය {
    "රතු"   => "#FF෦෦෦෦"
    "කොළ" => "#෦෦FF෦෦"
    _       => "#෦෦෦෦෦෦"
}

// සංසන්දන රටා
උෂ්ණත්වය = -෫
තත්ත්වය = ?? උෂ්ණත්වය {
    < ෦  => "අයිස්"
    < ෨෦ => "සීතල"
    < ෩෫ => "උණුසුම්"
    _    => "උණුසුම"
}
>> තත්ත්වය ¶    // → අයිස්

// ප්‍රකාශ ආකාරය (කොටස් අත්)
n = -෩
?? n {
    ෦    => { >> "ශුන්‍ය" ¶ }
    < ෦  => { >> "ඍණාත්මක" ¶ }
    _    => { >> "ධනාත්මක" ¶ }
}
```

## ලූප්

```zymbol
@ i:෦..෪  { >> i " " }        // පරාසය ඇතුළත්:  ෦ ෧ ෨ ෩ ෪
@ i:෧..෯:෨ { >> i " " }       // පියවර සමඟ:         ෧ ෩ ෫ ෭ ෯
@ i:෫..෦:෧ { >> i " " }       // ප්‍රතිලෝම:           ෫ ෪ ෩ ෨ ෧ ෦

n = ෧
@ n <= ෬෪ { n *= ෨ }
>> n ¶                        // → ෧෨෮  (while)

පලතුරු = ["ඇපල්", "පෙයාර්ස්", "මිදි"]
@ ප:පලතුරු { >> ප ¶ }         // අරාවේ සෑම මූලද්‍රව්‍යයක් සඳහා

@ අ:"හෙලෝ" { >> අ "-" }
>> ¶                          // → හ-ෙ-ල-ෝ-  (තන්තුවේ සෑම අක්ෂරයක් සඳහා)

@ i:෧..෧෦ {
    ? i % ෨ == ෦ { @> }       // @> දිගටම කරගෙන යන්න
    ? i > ෭ { @! }             // @! බිඳින්න
    >> i " "
}
>> ¶                          // → ෧ ෩ ෫ ෭

// අනන්ත ලූපය
i = ෦
@ {
    i++
    ? i >= ෫ { @! }
    >> i " "
}
>> ¶                          // → ෧ ෨ ෩ ෪

// ලේබල් කළ ලූපය (කැදලි බිඳීම)
ගණන = ෦
@:බාහිර {
    ගණන++
    ? ගණන >= ෩ { @:බාහිර! }
}
>> ගණන ¶                    // → ෩
```

## ශ්‍රිත

```zymbol
එකතුව(a, b) { <~ a + b }
>> එකතුව(෩, ෪) ¶    // → ෭

සාධකය(n) {
    ? n <= ෧ { <~ ෧ }
    <~ n * සාධකය(n - ෧)
}
>> සාධකය(෫) ¶    // → ෧෨෦
```

ශ්‍රිත සතුව **හුදකලා විෂය පථයක්** ඇත — ඒවාට බාහිර විචල්‍ය කියවිය නොහැක. ඇමතුම්කරුගේ විචල්‍ය වෙනස් කිරීමට ප්‍රතිදාන පරාමිතීන් `<~` භාවිතා කරන්න:

```zymbol
මාරු_කරන්න(a<~, b<~) {
    තාවකාලික = a
    a = b
    b = තාවකාලික
}
x = ෧෦
y = ෨෦
මාරු_කරන්න(x, y)
>> "x=" x " y=" y ¶    // → x=෨෦ y=෧෦
```

> නම් කළ ශ්‍රිත **ප්‍රථම-පෙළ අගයන්** වේ — කෙලින්ම පසුකරන්න: `සංඛ්‍යා$> දෙගුණය`. ඔතා ගැනීමට: `x -> fn(x)` ද වලංගු වේ.

## ලැම්ඩා සහ වැසීම්

```zymbol
දෙගුණය = x -> x * ෨
එකතුව = (a, b) -> a + b
>> දෙගුණය(෫) ¶    // → ෧෦
>> එකතුව(෩, ෭) ¶  // → ෧෦

// කොටස් ලැම්ඩා
වර්ගීකරණය = x -> {
    ? x > ෦ { <~ "ධනාත්මක" }
    _? x < ෦ { <~ "ඍණාත්මක" }
    <~ "ශුන්‍ය"
}

// වැසීම — බාහිර විෂය පථය අල්ලා ගනී
සාධකය = ෩
තෙගුණය = x -> x * සාධකය
>> තෙගුණය(෭) ¶    // → ෨෧

// කර්මාන්තශාලාව
එකතු_සාදන්නා(n) { <~ x -> x + n }
දහයේ_එකතුව = එකතු_සාදන්නා(෧෦)
>> දහයේ_එකතුව(෫) ¶    // → ෧෫

// අරාව තුළ
ක්‍රියාකාරක = [x -> x+෧, x -> x*෨, x -> x*x]
>> ක්‍රියාකාරක[෩](෫) ¶    // → ෨෫
```

## අරාව

අරා **වෙනස් කළ හැකි** වන අතර **එකම වර්ගයේ** මූලද්‍රව්‍ය රඳවා ගනී.

```zymbol
arr = [෧, ෨, ෩, ෪, ෫]

x = arr[෧]      // ෧ — ප්‍රවේශය (෧-පාදක: පළමු මූලද්‍රව්‍යය)
x = arr[-෧]     // ෫ — සෘණ දර්ශකය (අවසාන මූලද්‍රව්‍යය)
x = arr$#       // ෫ — දිග (>> හි (arr$#) භාවිතා කරන්න)

arr = arr$+ ෬            // එකතු කරන්න → [෧,෨,෩,෪,෫,෬]
arr෨ = arr$+[෨] ෯෯       // ස්ථානය ෨ හි ඇතුළු කරන්න (෧-පාදක)
arr෩ = arr$- ෩           // අගයේ පළමු සිදුවීම ඉවත් කරන්න
arr෪ = arr$-- ෩          // සියලු සිදුවීම් ඉවත් කරන්න
arr෫ = arr$-[෧]          // දර්ශකය ෧ හි ඉවත් කරන්න (පළමු මූලද්‍රව්‍යය)
arr෬ = arr$-[෨..෩]       // පරාසය ඉවත් කරන්න (෧-පාදක, අවසානය ඇතුළත්)

ඇත = arr$? ෩            // #෧ — අඩංගු වේ
ස්ථාන = arr$?? ෩           // [෩] — අගයේ සියලු දර්ශක (෧-පාදක)
උප_කොටස = arr$[෧..෩]          // [෧,෨,෩] — උප කොටස (෧-පාදක, අවසානය ඇතුළත්)
උප_කොටස෨ = arr$[෧:෩]          // [෧,෨,෩] — සමාන, ගණන් කිරීම-පාදක වාක්‍ය රීතිය

ආරෝහණ = arr$^+             // ආරෝහණ අනුපිළිවෙලට වර්ග කරන්න (ප්‍රාථමික පමණි)
අවරෝහණ = arr$^-            // අවරෝහණ අනුපිළිවෙලට වර්ග කරන්න (ප්‍රාථමික පමණි)

// නම් කළ/ස්ථානීය ටපල් අරාව — සංසන්දක ලැම්ඩා සමඟ $^ භාවිතා කරන්න
දත්ත_ගබඩාව = [(නම: "කාර්ලා", වයස: ෨෮), (නම: "ඇනා", වයස: ෨෫), (නම: "බොබ්", වයස: ෩෦)]
වයස_අනුව  = දත්ත_ගබඩාව$^ (a, b -> a.වයස < b.වයස)    // වයස අනුව ආරෝහණ (<)
නම_අනුව = දත්ත_ගබඩාව$^ (a, b -> a.නම > b.නම)   // නම අනුව අවරෝහණ (>)
>> වයස_අනුව[෧].නම ¶     // → ඇනා
>> නම_අනුව[෧].නම ¶    // → කාර්ලා

// සෘජු මූලද්‍රව්‍ය යාවත්කාලීන කිරීම (අරා පමණි)
arr[෧] = ෯෯              // පවරන්න
arr[෨] += ෫              // සංයුක්ත: +=  -=  *=  /=  %=  ^=

// කාර්යාත්මක යාවත්කාලීන කිරීම — නව අරාවක් ආපසු ලබා දෙයි; මුල් පිටපත නොවෙනස්ව
arr෨ = arr[෨]$~ ෯෯
```

> සියලු එකතු කිරීමේ ක්‍රියාකාරක **නව අරාවක්** ආපසු ලබා දෙයි. ආපසු පවරන්න: `arr = arr$+ 4`.
> `$+` දාමගත කළ හැක: `arr = arr$+ 5$+ 6$+ 7`. අනෙකුත් ක්‍රියාකාරක අතරමැදි පැවරීම් භාවිතා කරයි.
> **දර්ශකනය 1-පාදක**: `arr[1]` පළමු මූලද්‍රව්‍යය වේ; `arr[0]` ධාවන වේලාව දෝෂයකි.
> `$^+` / `$^-` **ප්‍රාථමික අරා** (සංඛ්‍යා, තන්තු) වර්ග කරයි. ටපල් අරා සඳහා සංසන්දක ලැම්ඩා සමඟ `$^` භාවිතා කරන්න — දිශාව ලැම්ඩාව තුළ කේතනය කර ඇත (`<` = ආරෝහණ, `>` = අවරෝහණ).

**අගය අර්ථකථනය** — අරාවක් වෙනත් විචල්‍යයකට පැවරීමෙන් ස්වාධීන පිටපතක් සාදයි:

```zymbol
a = [෧, ෨, ෩]
b = a
a[෧] = ෯෯
>> a ¶    // → [෯෯, ෨, ෩]
>> b ¶    // → [෧, ෨, ෩]   ← b බලපෑමට ලක් නොවේ
```

```zymbol
// කැදලි අරා (෧-පාදක දර්ශකනය)
matrix = [[෧,෨,෩],[෪,෫,෬],[෭,෮,෯]]
>> matrix[෨][෩] ¶    // → ෬  (පේළිය ෨, තීරුව ෩)
```

## ව්‍යුහ විසංයෝජනය

```zymbol
// අරාව
arr = [෧෦, ෨෦, ෩෦, ෪෦, ෫෦]
[a, b, c] = arr              // a=෧෦  b=෨෦  c=෩෦
[පළමු, *ඉතිරි] = arr         // පළමු=෧෦  ඉතිරි=[෨෦,෩෦,෪෦,෫෦]
[x, _, z] = [෧, ෨, ෩]        // _ ඉවතලයි

// ස්ථානීය ටපලය
ලක්ෂ්‍යය = (෧෦෦, ෨෦෦)
(px, py) = ලක්ෂ්‍යය             // px=෧෦෦  py=෨෦෦

// නම් කළ ටපලය
පුද්ගලයා = (නම: "ඇනා", වයස: ෨෫, නගරය: "මැඩ්‍රිඩ්")
(නම: n, වයස: v) = පුද්ගලයා   // n="ඇනා"  v=෨෫
```

## ටපලය

ටපල් යනු **වෙනස් කළ නොහැකි** අනුපිළිවෙලැති බහාලුම් වන අතර ඒවාට **විවිධ වර්ගයේ** අගයන් තබා ගත හැක.
අරාවලට ප්‍රතිවිරුද්ධව, නිර්මාණය කිරීමෙන් පසු මූලද්‍රව්‍ය වෙනස් කළ නොහැක.

```zymbol
// ස්ථානීය — මිශ්‍ර වර්ග වලට අවසර ඇත
ලක්ෂ්‍යය = (෧෦, ෨෦)
>> ලක්ෂ්‍යය[෧] ¶    // → ෧෦

දත්ත = (෪෨, "හෙලෝ", #෧, ෩.෧෪)
>> දත්ත[෩] ¶     // → #෧

// නම් කළ
පුද්ගලයා = (නම: "ඇලිස්", වයස: ෨෫)
>> පුද්ගලයා.නම ¶    // → ඇලිස්
>> පුද්ගලයා[෧] ¶      // → ඇලිස්  (දර්ශකය ද ක්‍රියා කරයි, ෧-පාදක)

// කැදලි
ස්ථානය = (x: ෧෦, y: ෨෦)
p = (ස්ථානය: ස්ථානය, ලේබලය: "මූලාරම්භය")
>> p.ස්ථානය.x ¶        // → ෧෦
```

**වෙනස් කළ නොහැකි බව** — ටපල් මූලද්‍රව්‍යයක් වෙනස් කිරීමට ගන්නා ඕනෑම උත්සාහයක් ධාවන වේලාව දෝෂයකි:

```zymbol
t = (෧෦, ෨෦, ෩෦)
// t[෧] = ෯෯    // ❌ ධාවන වේලාව දෝෂය: ටපල් වෙනස් කළ නොහැක
// t[෧] += ෫    // ❌ එකම දෝෂය
```

වෙනස් කළ අගයක් ලබා ගැනීමට `$~` (කාර්යාත්මක යාවත්කාලීන කිරීම) භාවිතා කරන්න — **නව** ටපලයක් ආපසු ලබා දෙයි:

```zymbol
t = (෧෦, ෨෦, ෩෦)
t෨ = t[෨]$~ ෯෯෯
>> t ¶     // → (෧෦, ෨෦, ෩෦)   ← මුල් පිටපත නොවෙනස්ව
>> t෨ ¶    // → (෧෦, ෯෯෯, ෩෦)

// නම් කළ ටපලය — පැහැදිලිව ප්‍රතිනිර්මාණය කරන්න
පුද්ගලයා = (නම: "ඇලිස්", වයස: ෨෫)
විශාල  = (නම: පුද්ගලයා.නම, වයස: ෨෬)
>> පුද්ගලයා.වයස ¶    // → ෨෫
>> විශාල.වයස ¶     // → ෨෬
```

## ඉහළ අනුපිළිවෙල ශ්‍රිත

```zymbol
සංඛ්‍යා = [෧, ෨, ෩, ෪, ෫, ෬, ෭, ෮, ෯, ෧෦]

දෙගුණ_කළ = සංඛ්‍යා$> (x -> x * ෨)                  // map  → [෨,෪,෬…෨෦]
ඉරට්ටේ    = සංඛ්‍යා$| (x -> x % ෨ == ෦)           // filter → [෨,෪,෬,෮,෧෦]
එකතුව    = සංඛ්‍යා$< (෦, (එකතුකාරක, x) -> එකතුකාරක + x)     // reduce → ෫෫

// අතරමැදියන් හරහා දාමගත කරන්න
පියවර෧ = සංඛ්‍යා$| (x -> x > ෩)
පියවර෨ = පියවර෧$> (x -> x * x)
>> පියවර෨ ¶    // → [෧෬, ෨෫, ෩෬, ෪෯, ෬෪, ෮෧, ෧෦෦]

// නම් කළ ශ්‍රිත කෙලින්ම HOF වෙත පසු කළ හැක
දෙගුණය(x) { <~ x * ෨ }
විශාල_ද(x) { <~ x > ෫ }
r = සංඛ්‍යා$> දෙගුණය       // ✅ සෘජු යොමුව
r = සංඛ්‍යා$| විශාල_ද       // ✅ සෘජු යොමුව
```

## නළ ක්‍රියාකාරකය

දකුණු පැත්තට සැමවිටම නළ කළ අගය සඳහා `_` ප්‍රතිස්ථාපකය අවශ්‍ය වේ:

```zymbol
දෙගුණය = x -> x * ෨
එකතුව = (a, b) -> a + b
වර්ධක = x -> x + ෧

ප්‍ර෧ = ෫ |> දෙගුණය(_)        // → ෧෦
ප්‍ර෨ = ෧෦ |> එකතුව(_, ෫)       // → ෧෫
ප්‍ර෩ = ෫ |> එකතුව(෨, _)        // → ෭

// දාමගත කළ
ප්‍ර = ෫ |> දෙගුණය(_) |> වර්ධක(_) |> දෙගුණය(_)
>> ප්‍ර ¶    // → ෨෨  (෫→෧෦→෧෧→෨෨)
```

## මොඩියුල

```zymbol
// lib/calc.zy — මොඩියුල වස්තුව වරහන් තුළ වට කර ඇත
# calc {
    #> { එකතුව, get_PI }

    _π := ෩.෧෪෧෫෯
    එකතුව(a, b) { <~ a + b }
    get_PI() { <~ _π }
}
```

```zymbol
// main.zy
<# ./lib/calc => c    // අන්වර්ථ නාමය අවශ්‍ය වේ

>> c::එකතුව(෫, ෩) ¶     // → ෮
π = c::get_PI()
>> π ¶               // → ෩.෧෪෧෫෯
```

```zymbol
// වෙනත් පොදු නමකින් අපනයනය කරන්න
# mylib {
    #> { _අභ්‍යන්තර_එකතුව => එකතුව }

    _අභ්‍යන්තර_එකතුව(a, b) { <~ a + b }
}
```

```zymbol
<# ./mylib => m

>> m::එකතුව(෩, ෪) ¶    // → ෭  (අභ්‍යන්තර නම _අභ්‍යන්තර_එකතුව සැඟවී ඇත)
```

> **මොඩියුල නීති**: `# නම { }` තුළ අවසර ඇත්තේ `#>`, ශ්‍රිත අර්ථ දැක්වීම්, සහ සාහිත්‍යමය විචල්‍ය/නියත ආරම්භක පමණි. ක්‍රියාත්මක කළ හැකි ප්‍රකාශ (`>>`, `<<`, ලූප්, ආදිය) E013 දෝෂයක් මතු කරයි.

## සංඛ්‍යා මාදිලි

Zymbol හට සංඛ්‍යා **69 යුනිකේත ඉලක්කම් අක්ෂර මාලාවල** ප්‍රදර්ශනය කළ හැක — දේවනාගරි, අරාබි-භාරතීය, තායි, ක්ලින්ගොන් pIqaD, ගණිතමය තදකුරු, LCD ඛණ්ඩ, සහ තවත්. සක්‍රීය මාදිලිය බලපාන්නේ `>>` ප්‍රතිදානයට පමණි; අභ්‍යන්තර අංක ගණිතය සැමවිටම ද්විමය වේ.

### අක්ෂර මාලාවක් සක්‍රීය කිරීම

ඉලක්ක අක්ෂර මාලාවේ `0` සහ `9` ඉලක්කම් `#…#` තුළ ලියන්න:

```zymbol
#०९#    // දේවනාගරි   (U+0966–U+096F)
#٠٩#    // අරාබි-භාරතීය (U+0660–U+0669)
#๐๙#    // තායි         (U+0E50–U+0E59)
#09#    // ASCII වෙත නැවත සකසන්න
```

### ප්‍රතිදානය සහ බූලියන්

```zymbol
x = ෪෨
>> x ¶          // → ෪෨   (ASCII පෙරනිමිය)

#०९#
>> x ¶          // → ४२
>> ෩.෧෪ ¶       // → ३.१४   (දශම ලක්ෂ්‍යය සැමවිටම ASCII)
>> ෧ + ෨ ¶      // → ३

// බූලියන්: # උපසර්ගය සැමවිටම ASCII, ඉලක්කම අනුගත වේ
>> #෧ ¶         // → #१   (දේවනාගරියේ සත්‍ය)
>> #෦ ¶         // → #०   (අසත්‍ය — ० පූර්ණ සංඛ්‍යා ශුන්‍යයෙන් වෙනස්)

x = ෨෮ > ෪
>> x ¶          // → #१   (සංසන්දන ප්‍රතිඵලය සක්‍රීය මාදිලිය අනුගමනය කරයි)
```

### මූලාශ්‍රයේ ස්වදේශීය ඉලක්කම් සාහිත්‍යම

ඕනෑම සහාය දක්වන අක්ෂර මාලාවක ඉලක්කම් වලංගු සාහිත්‍යම වේ — පරාසයන්හි, මොඩියුලෝවේ, සංසන්දනවල:

```zymbol
#०९#

@ i:१..१५ {
    ? i % १५ == ० { >> "FizzBuzz" ¶ }
    _? i % ३  == ० { >> "Fizz" ¶ }
    _? i % ५  == ० { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### ඕනෑම අක්ෂර මාලාවක බූලියන් සාහිත්‍යම

`#` + ඕනෑම කොටසකින් `0` හෝ `1` ඉලක්කම වලංගු බූලියන් සාහිත්‍යමයකි:

```zymbol
#٠٩#
සක්‍රිය = #١        // #෧ ට සමාන වේ
>> සක්‍රිය ¶        // → #१
>> (#١ && #٠) ¶ // → #०
```

> `#` **සැමවිටම ASCII** වේ. `#0` (අසත්‍ය) සැමවිටම සෑම අක්ෂර මාලාවකම `0` (පූර්ණ සංඛ්‍යා ශුන්‍ය) ට වඩා දෘශ්‍යමය වශයෙන් වෙනස් වේ.

## දත්ත ක්‍රියාකාරක

```zymbol
// වර්ග පරිවර්තන වාත්තු
f = ##.෪෨         // → ෪෨.෦  (පාවෙන ලක්ෂ්‍යයට)
i = ###෩.෭        // → ෪     (පූර්ණ සංඛ්‍යාවට, වට කරන්න)
t = ##!෩.෭        // → ෩     (පූර්ණ සංඛ්‍යාවට, කපා දමන්න)

// තන්තුව සංඛ්‍යාවකට ව්‍යාකරණ විග්‍රහ කරන්න
v෧ = #|"෪෨"|      // → ෪෨  (පූර්ණ සංඛ්‍යා)
v෨ = #|"෩.෧෪"|    // → ෩.෧෪  (පාවෙන ලක්ෂ්‍ය)
v෩ = #|"abc"|     // → "abc"  (ආරක්ෂිතයි, දෝෂයක් නැත)

// වට කරන්න / කපා දමන්න
π = ෩.෧෪෧෫෯෨෬෫
වට෨ = #.෨|π|      // → ෩.෧෪  (දශම ස්ථාන ෨ කට වට කරන්න)
වට෪ = #.෪|π|      // → ෩.෧෪෧෬
කපා෨ = #!෨|π|      // → ෩.෧෪  (කපා දමන්න)

// සංඛ්‍යා ආකෘතිකරණය
ආකෘතිය = #,|෧෨෩෪෫෬෭|  // → ෧,෨෩෪,෫෬෭  (කොමා-වෙන් කළ)
විද්‍යාත්මක = #^|෧෨෩෪෫.෬෭෮|    // → ෧.෨෩෪෫෬෭෮e෪  (විද්‍යාත්මක)

// පාදක සාහිත්‍යම
a = 0x41         // → 'A'  (ෂඩාස්‍රික)
b = 0b01000001   // → 'A'  (ද්විමය)
c = 0o101        // → 'A'  (අෂ්ටක)

// පාදක පරිවර්තන ප්‍රතිදානය
ෂඩාස්‍රික = 0x|෨෫෫|    // → "0x00FF"
ද්විමය = 0b|෬෫|     // → "0b1000001"
අෂ්ටක = 0o|෮|      // → "0o10"
දශම = 0d|෨෫෫|    // → "0d0255"
```

## ෂෙල් ඒකාබද්ධ කිරීම

```zymbol
දිනය = <\ date +%Y-%m-%d \>     // stdout ග්‍රහණය කරයි (අවසානයේ \n ඇතුළත්)
>> "අද: " දිනය

ගොනුව = "data.txt"
අන්තර්ගතය = <\ cat {ගොනුව} \>      // විධානවල අන්තර්කේෂණය

ප්‍රතිදානය = </"./subscript.zy"/>   // වෙනත් Zymbol ස්ක්‍රිප්ටයක් ක්‍රියාත්මක කරන්න, ප්‍රතිදානය ග්‍රහණය කරන්න
>> ප්‍රතිදානය
```

> `><` CLI තර්ක තන්තු අරාවක් ලෙස ග්‍රහණය කරයි (tree-walker පමණි).

## සම්පූර්ණ උදාහරණය: ෆිස්බස්

```zymbol
වර්ගීකරණය(සංඛ්‍යාව) {
    ? සංඛ්‍යාව % ෧෫ == ෦ { <~ "FizzBuzz" }
    _? සංඛ්‍යාව % ෩  == ෦ { <~ "Fizz" }
    _? සංඛ්‍යාව % ෫  == ෦ { <~ "Buzz" }
    _ { <~ සංඛ්‍යාව }
}

@ i:෧..෨෦ { >> වර්ගීකරණය(i) ¶ }
```
