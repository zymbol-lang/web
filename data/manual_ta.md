# Zymbol-Lang இன் சுருக்கமான வழிகாட்டி

**Zymbol-Lang** ஒரு குறியீட்டு நிரலாக்க மொழி. இதில் எந்த முக்கிய சொல்லும் இல்லை — எல்லாமே ஒரு குறியீடு. இது எந்த மனித மொழியிலும் ஒரே மாதிரி செயல்படுகிறது.

- எந்த முக்கிய சொல்லும் இல்லை (`if`, `while`, `return` இல்லை — குறியீடுகள் மட்டுமே: `?`, `@`, `<~`)
- முழு Unicode — எந்த மொழியிலும் அல்லது இமோஜியிலும் அடையாளங்காட்டிகள் 👋
- மனித-மொழி நடுநிலை — குறியீடு எல்லா மொழிகளிலும் ஒரே மாதிரி

## வெளியீடு மற்றும் உள்ளீடு

```zymbol
>> "வணக்கம்" ¶                      // ¶ அல்லது \\ வெளிப்படையான புதிய வரி
```

## சுழற்சி

```zymbol
@ i:௦..௪  { >> i " " }        // வரம்பு உள்ளடக்கியது:  ௦ ௧ ௨ ௩ ௪
```

## செயல்பாடுகள்

```zymbol
கூட்டு(a, b) { <~ a + b }
>> கூட்டு(௩, ௪) ¶    // → ௭

காரணி(ன) {
    ? ன <= ௧ { <~ ௧ }
    <~ ன * காரணி(ன - ௧)
}
>> காரணி(௫) ¶    // → ௧௨௦
```

செயல்பாடுகளுக்கு **தனிப்பட்ட scope** உள்ளது — வெளி மாறிகளை படிக்க முடியாது. வெளியீட்டு அளவுருக்கள் `<~` மூலம் அழைப்பாளர் மாறிகளை மாற்றலாம்:

```zymbol
swap(a<~, b<~) {
    tmp = a
    a = b
    b = tmp
}
x = ௧௦
y = ௨௦
swap(x, y)
>> "x=" x " y=" y ¶    // → x=௨௦ y=௧௦
```

> பெயரிட்ட செயல்பாடுகள் முதல்-வகுப்பு மதிப்புகள் அல்ல. வாதமாக அனுப்ப சுற்றவும்: `x -> கூட்டு(x)`.

## வரிசைகள்

Arrays are **mutable** and hold elements of the **same type**.

```zymbol
arr = [௧, ௨, ௩, ௪, ௫]

arr[௦]          // ௧ — அணுகல் (௦ முதல் குறியீடு)
arr[-௧]         // ௫ — எதிர்மறை குறியீடு (கடைசி)
arr$#           // ௫ — நீளம் (>> இல் (arr$#) பயன்படுத்தவும்)

arr = arr$+ ௬            // சேர் → [௧,௨,௩,௪,௫,௬]
arr௨ = arr$+[௨] ௯௯       // குறியீடு ௨ இல் செருகு
arr௩ = arr$- ௩           // மதிப்பின் முதல் நிகழ்வை நீக்கு
arr௪ = arr$-- ௩          // அனைத்து நிகழ்வுகளையும் நீக்கு
arr௫ = arr$-[௦]          // குறியீட்டில் நீக்கு
arr௬ = arr$-[௧..௩]       // வரம்பு நீக்கு (இறுதி தவிர்க்கப்படும்)

has = arr$? ௩            // #௧ — உள்ளதா
pos = arr$?? ௩           // [௨] — மதிப்பின் அனைத்து குறியீடுகள்
sl = arr$[௦..௩]          // [௧,௨,௩] — துண்டு (இறுதி தவிர்க்கப்படும்)
sl௨ = arr$[௦:௩]          // [௧,௨,௩] — எண்ணிக்கை-அடிப்படை தொடரியல்

asc = arr$^+             // ஏறு வரிசையில் வரிசைப்படுத்து  (அடிப்படை மட்டும்)
desc = arr$^-            // இறங்கு வரிசையில் வரிசைப்படுத்து (அடிப்படை மட்டும்)

// பெயரிட்ட/நிலையிட்ட டுபிள் வரிசைகள் — ஒப்பீட்டு லாம்டாவுடன் $^ பயன்படுத்தவும்
db = [(name: "Carla", age: ௨௮), (name: "Ana", age: ௨௫), (name: "Bob", age: ௩௦)]
by_age  = db$^ (a, b -> a.age < b.age)    // வயதின் படி ஏறு  (<)
by_name = db$^ (a, b -> a.name > b.name)  // பெயரின் படி இறங்கு (>)
>> by_age[௦].name ¶     // → Ana
>> by_name[௦].name ¶    // → Carla

// Direct element update (arrays only)
arr[௧] = ௯௯              // ஒதுக்கு
arr[௦] += ௫              // compound: +=  -=  *=  /=  %=  ^=

// Functional update — returns a new array; original unchanged
arr௨ = arr[௧]$~ ௯௯
```

> அனைத்து சேகரிப்பு ஆபரேட்டர்களும் **புதிய வரிசையை** திருப்பும். மீண்டும் ஒதுக்கவும்: `arr = arr$+ ௪`.
> ஆபரேட்டர்களை சங்கிலியாக இணைக்க முடியாது — இடைநிலை நியமனங்களை பயன்படுத்தவும்.
> `$^+` / `$^-` **அடிப்படை வரிசைகளை** (எண்கள், சரங்கள்) வரிசைப்படுத்தும். டுபிள் வரிசைகளுக்கு ஒப்பீட்டு லாம்டாவுடன் `$^` பயன்படுத்தவும் — திசை லாம்டாவில் குறிப்பிடப்படும் (`<` = ஏறு, `>` = இறங்கு).

**Value semantics** — assigning an array to another variable creates an independent copy:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b is unaffected
```

```zymbol
// உள்ளமை வரிசைகள்
matrix = [[௧,௨,௩],[௪,௫,௬],[௭,௮,௯]]
>> matrix[௧][௨] ¶    // → ௬
```

## டுபிள்

டுபிள்கள் **மாற்றமுடியாத** (immutable) வரிசைப்படுத்தப்பட்ட கொள்கலன்கள், அவை **வெவ்வேறு வகைகளின்** மதிப்புகளை கொண்டிருக்கலாம். வரிசைகளை விட வித்தியாசமாக, உறுப்புகளை உருவாக்கிய பிறகு மாற்ற முடியாது.

```zymbol
// நிலையிட்ட
point = (௧௦, ௨௦)
>> point[௦] ¶    // → ௧௦

தரவு = (42, "வணக்கம்", #1, 3.14)
>> தரவு[௨] ¶     // → #1

// பெயரிட்ட
நபர் = (name: "Alice", age: ௨௫)
>> நபர்.name ¶    // → Alice
>> நபர்[௦] ¶      // → Alice  (குறியீடும் வேலை செய்கிறது)

// உள்ளமை
pos = (x: ௧௦, y: ௨௦)
p = (pos: pos, label: "origin")
>> p.pos.x ¶      // → ௧௦
```

**மாற்றமுடியாமை (Inmutabilidad)** — டுபிள் உறுப்பை மாற்ற முயற்சிக்கும் போது இயக்க நேர பிழை ஏற்படும்:

```zymbol
t = (௧௦, ௨௦, ௩௦)
// t[௦] = ௯௯    // ❌ இயக்க நேர பிழை: டுபிள்கள் மாற்றமுடியாதவை
// t[௦] += ௫    // ❌ அதே பிழை
```

மாற்றப்பட்ட மதிப்பை பெற `$~` (செயல்முறை புதுப்பிப்பு) பயன்படுத்தவும் — **புதிய டுபிள்** திருப்பும்:

```zymbol
t = (௧௦, ௨௦, ௩௦)
t2 = t[௧]$~ 999
>> t ¶     // → (௧௦, ௨௦, ௩௦)   ← அசல் மாறவில்லை
>> t2 ¶    // → (௧௦, 999, ௩௦)

// பெயரிட்ட டுபிள் — வெளிப்படையாக மீண்டும் உருவாக்கவும்
நபர் = (name: "Alice", age: ௨௫)
நபர்2  = (name: நபர்.name, age: ௨௬)
>> நபர்.age ¶    // → ௨௫
>> நபர்2.age ¶   // → ௨௬
```

## உயர்-வரிசை செயல்பாடுகள்

> HOF ஆபரேட்டர்களுக்கு **இன்லைன் லாம்டா** தேவை — லாம்டா மாறிகளை நேரடியாக அனுப்ப முடியாது.

```zymbol
எண்கள் = [௧, ௨, ௩, ௪, ௫, ௬, ௭, ௮, ௯, ௧௦]

doubled  = எண்கள்$> (x -> x * ௨)                // map  → [௨,௪,௬…௨௦]
evens    = எண்கள்$| (x -> x % ௨ == ௦)           // filter → [௨,௪,௬,௮,௧௦]
total    = எண்கள்$< (௦, (acc, x) -> acc + x)     // reduce → ௫௫

// இடைநிலை மூலம் சங்கிலி
step௧ = எண்கள்$| (x -> x > ௩)
step௨ = step௧$> (x -> x * x)
>> step௨ ¶    // → [௧௬, ௨௫, ௩௬, ௪௯, ௬௪, ௮௧, ௧௦௦]

// HOF இல் பெயரிட்ட செயல்பாடுகள் — லாம்டாவில் சுற்றவும்
double(x) { <~ x * ௨ }
r = எண்கள்$> (x -> double(x))    // ✅
```


## எண் முறைமைகள்

Zymbol **69 Unicode இலக்க தொகுதிகளில்** எண்களை காட்ட முடியும் — தமிழ், தேவநாகரி, அரபிக்-இந்திக், தாய், கிளிங்கான் pIqaD மற்றும் பல. செயல்பாட்டு பாங்கு `>>` வெளியீட்டை மட்டுமே பாதிக்கிறது; உள்கணக்கு எப்போதும் இரும எண்.

### ஒரு முறைமையை இயக்கவும்

விரும்பிய எழுத்தின் `௦` மற்றும் `௯` இலக்கங்களை `#…#` இடையில் எழுதவும்:

```zymbol
#௦௯#    // TA script (௦..௯)
#٠٩#    // Arabic-Indic  (U+0660–U+0669)
#09#    // reset to ASCII
```

### வெளியீடு மற்றும் பூலியன்

```zymbol
x = ௪௨
>> x ¶          // → ௪௨   (ASCII இயல்புநிலை)

#௦௯#
>> x ¶          // → ௪௨
>> ௩.௧௪ ¶       // → ௩.௧௨௩௪   (தசம புள்ளி எப்போதும் ASCII)
>> ௧ + ௨ ¶      // → ௩

>> #௧ ¶         // → #௧   (உண்மை)
>> #௦ ¶         // → #௦   (பொய் — ௦ இலிருந்து வேறு)

x = ௨௮ > ௪
>> x ¶          // → #௧   (ஒப்பீட்டு முடிவு)
```

### மூலத்தில் சொந்த எழுத்துக்கூறுகள்

ஆதரிக்கப்படும் எந்த முறைமையிலும் உள்ள இலக்கங்கள் வரம்புகள், மாடுலோ, ஒப்பீடுகளில் செல்லுபடியாகும்:

```zymbol
#௦௯#

@ i:௧..௧௫ {
    ? i % ௧௫ == ௦ { >> "FizzBuzz" ¶ }
    _? i % ௩  == ௦ { >> "Fizz" ¶ }
    _? i % ௫  == ௦ { >> "Buzz" ¶ }
    _ { >> i ¶ }
}
```

### எந்த முறைமையிலும் பூலியன் எழுத்துக்கூறுகள்

ஆதரிக்கப்படும் எந்த தொகுதியிலும் `#` + `௦` அல்லது `௧` இலக்கம் ஒரு செல்லுபடியான பூலியன் எழுத்துக்கூறு:

```zymbol
#٠٩#
نشط = #١        // same as #௧
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **எப்போதும் ASCII**. `#௦` (பொய்) எந்த முறைமையிலும் `௦` (பூஜ்ய முழு எண்) இலிருந்து காட்சி ரீதியாக வேறுபட்டது.

## குறியீடு குறிப்பு

| குறியீடு | செயல் | குறியீடு | செயல் |
|---------|-------|---------|-------|
| `=` | மாறி | `$#` | நீளம் |
| `:=` | மாறிலி | `$+` | சேர் |
| `>>` | வெளியீடு | `$+[i]` | குறியீட்டில் செருகு |
| `<<` | உள்ளீடு | `$--` | மதிப்பால் அனைத்தையும் நீக்கு |
| `¶` / `\\` | புதிய வரி | `$-[i]` | குறியீட்டில் நீக்கு |
| `?` | if | `$-[i..j]` | வரம்பு நீக்கு |
| `_?` | else-if | `$?` | உள்ளதா |
| `_` | else / வைல்டுகார்டு | `$??` | அனைத்து குறியீடுகளையும் கண்டுபிடி |
| `??` | பொருத்தம் | `$[s..e]` | துண்டு |
| `@` | சுழற்சி | `$>` | map |
| `@!` | நிறுத்து | `$\|` | filter |
| `@>` | தொடர்க | `$<` | reduce |
| `->` | லாம்டா | `<~` | return |
| `arr[i] = val` | உறுப்பு புதுப்பி (வரிசை மட்டும்) | `arr[i] += val` | compound புதுப்பி |
| `arr[i]$~` | செயல்முறை புதுப்பிப்பு (புதிய நகல்) | `$^+` | ஏறு வரிசை |
| `$^-` | இறங்கு வரிசை | `$^` | ஒப்பீட்டுடன் வரிசை |
| `\|>` | pipe | | |
| `#௧` | உண்மை | `$!` | பிழையா |
| `#௦` | பொய் | `$!!` | பிழை பரப்பு |
| `!?` | try | `#` | தொகுதி அறிவி |
| `:!` | catch | `#>` | ஏற்றுமதி |
| `:>` | finally | `<#` | இறக்குமதி |
| `.` | புலம் அணுகல் | `::` | தொகுதி அழைப்பு |
| `#\|..\|` | எண் பாகுபடுத்து | `#.N\|..\|` | வட்டமிடு |
| `#!N\|..\|` | வெட்டு | `#,\|..\|` | காற்புள்ளி வடிவமைப்பு |
| `#d0d9#` | எண் பயன்முறை மாற்றி | | |

### v0.0.2_01 — செயற்றிகை பெயர் மாற்றம் _(30 மார்ச் 2026)_

- **மாற்றப்பட்டது** `c|..|` → `#,|..|` மற்றும் `e|..|` → `#^|..|` — `#` முன்னொட்டு குடும்பத்துடன் சீரானது
- **சேர்க்கப்பட்டது** ஏற்றுமதி மாற்றுப்பெயர்: தொகுதி உறுப்பினர்களை வேறு பெயரில் மறு-ஏற்றுமதி

### v0.0.2 — தொகுப்பு மறுவடிவமைப்பு மற்றும் நிறுவிகள் _(24 மார்ச் 2026)_

- **சேர்க்கப்பட்டது** வரிசைகள் மற்றும் சரங்களுக்கான ஒருங்கிணைந்த `$` செயற்றிகை குடும்பம் (`$#`, `$+`, `$?`, `$-`, `$[..]`)
- **சேர்க்கப்பட்டது** வரிசைகள், டப்பிள்கள் மற்றும் பெயரிட்ட டப்பிள்களுக்கான சிதைவு
- **சேர்க்கப்பட்டது** எதிர்மறை சுட்டிகள் (`arr[-௧]` = கடைசி உறுப்பு)
- **சேர்க்கப்பட்டது** சொந்த நிறுவிகள் — Linux (deb/rpm/pkg/musl), macOS (Intel + Apple Silicon), Windows (MSI, winget)

### v0.0.1-patch _(25 மார்ச் 2026)_

- **சேர்க்கப்பட்டது** கூட்டு ஒதுக்கீடு `^=`
- **சரிசெய்யப்பட்டது** பகுப்பாய்வி கணிதத்தில் விளிம்பு நிகழ்வுகள்; ஆவண திருத்தங்கள்

### v0.0.1 — முதல் பொது பதிப்பு _(22 மார்ச் 2026)_

- மரம்-நடந்து நடக்கும் மொழிபெயர்ப்பாளர் + பதிவேடு VM (`--vm`, ~4× வேகமானது, ~95% சமத்துவம்)
- அனைத்து அடிப்படை கட்டமைப்புகளும்: `?` `@` `<~` `->` `>>` `<<` `¶` `??`
- முழு Unicode அடையாளர்கள், தொகுதி அமைப்பு, lambda, மூடல்கள், பிழை கையாளுதல்
- REPL, LSP, VS Code நீட்டிப்பு, வடிவமைப்பாளர் (`zymbol fmt`)
