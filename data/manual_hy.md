## Փոփոխականներ և Հաստատուններ

```zymbol
x = 10              // փոփոխական
PI := 3.14159       // հաստատուն — վերա-վերագրումը ժամանակակտ սխալ է
անուն = "Alice"
ակտիվ = #1          // բուլյան ճշմարիտ
👋 := "Բարև"
```

```zymbol
x = 10    // 10
x += 5    // 15
x -= 3    // 12
x *= 2    // 24
x /= 3    // 8
x %= 3    // 2
x ^= 2    // 4
x++        // 5
x--        // 4
```

`°` (աստիճանի նշան, U+00B0) փոփոխականն ավտոմատ կերպով սկզբնորոշում է դրա չեզոք արժեքով առաջին օգտագործման ժամանակ.

```zymbol
թվեր = [3, 1, 4, 1, 5]
@ ն:թվեր {
    °գումար += ն    // հանգույցի վերևում 0-ի ավտո-սկզբնորոշում; @-ից հետո պահպանվում է
}
>> գումար ¶         // → 14
```

> `°x` (նախածանց) ամրացվում է հանգույցի վերևում — արդյունքը հասանելի է `@`-ից հետո։
> `x°` (վերջածանց) ամրացվում է հանգույցի ներսում — ավարտվում է հանգույցի ավարտին։
> Միայն ծառ-անցողիչ։

|-----|---------|------------|---------|
| Ամբողջ | `42`, `-7` | `###` | 64-բիթ նշաններով |
| Լողակ | `3.14`, `1.5e10` | `##.` | Գիտական նշումն ընդունված է |

| Նիշ | `'A'` | `##'` | Մեկ Unicode նիշ |
| Բուլյան | `#1`, `#0` | `##?` | ՉԻ թիվ — `#1 ≠ 1` |
| Զանգված | `[1, 2, 3]` | `##]` | Միատարրական տարրեր |

## Ելք և Մուտք

```zymbol
>> "Բարև" ¶                      // ¶ կամ \\ բացահայտ նոր տողի համար
>> "a=" a " b=" b ¶               // կցում — բազմաթիվ արժեքներ
>> (զանգված$#) ¶                  // postfix օպերատորները >> -ում () են պահանջում

<< անուն                          // կարդալ փոփոխականի մեջ (առանց հուշման)

```

> `¶` (Իսպանական ստեղնաշարի AltGr+R) և `\\` համարժեք նոր տողեր են։

## TUI Պրիմիտիվներ

Տերմինալ UI օպերատորներ ինտերակտիվ ծրագրերի համար։ Մեծ մասը պահանջում է `>>| { }` բլոկ (այլ էկրան + հում ռեժիմ)։

```zymbol
>>| {
    >>!                             // մաքրել այլ էկրանը
    >>~ (1, 1, 0, 10) > "Աշխատում"  // տող 1, սյուն 1, gg=10 (կանաչ)
    @~ 1000                         // դադար 1 վայրկյան (1000 ms)
    >>~ (2, 1) > "Ավարտված։"
}
// տերմինալն ավտոմատ վերականգնվում է ելքի ժամանակ
```

```zymbol
// Ստեղն սեղմել և տերմինալի չափ
>>| {
    [տողեր, սյուներ] = >>?              // հարցնել տերմինալի չափերը
    >>~ (1, 1) > "Տերմինալ. " տողեր " x " սյուներ
    <<| ստեղն                           // արգելափակող ստեղնի ընթերցում
    >>~ (2, 1) > "Սեղմված. " ստեղն
}
```

> `>>!` մաքրում է էկրանը։ `>>?` վերադարձնում է `[տողեր, սյուներ]`։ `@~ N` քնում է N մ/վ։
> `<<|` կարդում է մեկ ստեղնի սեղմում (արգելափակող); `<<|?` հարցնում է առանց արգելափակման (վերադարձնում `'\0'` եթե չկա)։
> Դիրքային ելքի կոռտեժ. `(տող, սյուն, BKS, fg, bg)` — ցանկացած բջիջ կարող է բաց թողնվել ստորակետով (`>>~ (,,, 196) > "կարմիր"`)։
> BKS բիթ-դիմակ. `1`=Հաստ, `2`=Շեղ, `4`=Ընդգծված։ ANSI 256-գույն պալիտրա (`0`=տերմինալի կանխադրված)։
> Միայն ծառ-անցողիչ (`>>!`, `>>?`, `@~`, `>>~` -ը նաև աշխատում են `--vm`-ում)։

## Հոսքի Կառավարում

```zymbol
x = 7
```

## Հանգույցներ

```zymbol
@ i:0..4  { >> i " " }        // ընդգրկող տիրույթ.  0 1 2 3 4
@ i:1..9:2 { >> i " " }       // քայլով.              1 3 5 7 9
@ i:5..0:1 { >> i " " }       // հակառակ.             5 4 3 2 1 0

ն = 1
@ ն <= 64 { ն *= 2 }
>> ն ¶                        // → 128  (while)

մրգեր = ["խնձոր", "տանձ", "խաղող"]
@ մ:մրգեր { >> մ ¶ }          // զանգվածի վրայով

@ ց:"բարև" { >> ց "-" }
>> ¶                          // → բ-ա-ր-և-  (տողի վրայով)

@ i:1..10 {
    ? i % 2 == 0 { @> }       // @> շարունակել
    ? i > 7 { @! }             // @! ելք
    >> i " "
}
>> ¶                          // → 1 3 5 7

// Անվերջ հանգույց
i = 0
@ {
    i++
    ? i >= 5 { @! }
    >> i " "
}
>> ¶                          // → 1 2 3 4

// Պիտակավոր հանգույց (ներդրված ելք)
հաշ = 0
@:արտ {
    հաշ++
    ? հաշ >= 3 { @:արտ! }
}
>> հաշ ¶                    // → 3
```

## Ֆունկցիաներ

```zymbol
գում(ա, բ) { <~ ա + բ }
>> գում(3, 4) ¶    // → 7

արտ(ն) {
    ? ն <= 1 { <~ 1 }
    <~ ն * արտ(ն - 1)
}
>> արտ(5) ¶    // → 120
```

Ֆունկցիաներն ունեն **մեկուսացված շրջանակ** — արտաքին փոփոխականները չեն կարող կարդալ։ Օգտագործեք ելքի պարամետրեր `<~` կանչողի փոփոխականները փոփոխելու համար.

```zymbol
փոխ(ա<~, բ<~) {
    ժամ = ա
    ա = բ
    բ = ժամ
}
x = 10
y = 20
փոխ(x, y)
>> "x=" x " y=" y ¶    // → x=20 y=10
```

> Անվանված ֆունկցիաները **առաջին դասի արժեքներ են** — ուղղակի փոխանցեք. `թվեր$> կրկ`։ Փաթաթելու համար. `x -> fn(x)` -ն ևս վավեր է։

## Զանգվածներ

Զանգվածները **փոփոխելի են** և պահում են **նույն տիպի** տարրեր։

```zymbol
զանգ = [1, 2, 3, 4, 5]

x = զանգ[1]      // 1 — մուտք (1-հիմնված. առաջին տարրը)
x = զանգ[-1]     // 5 — բացասական ինդեքս (վերջին տարրը)
x = զանգ$#       // 5 — երկարություն (>>-ում օգտ. (զանգ$#))

զանգ = զանգ$+ 6            // ավելացնել → [1,2,3,4,5,6]
զանգ2 = զանգ$+[2] 99       // տեղադրել 2-րդ դիրքում (1-հիմնված)
զանգ3 = զանգ$- 3           // հեռացնել արժեքի առաջին հայտնությունը
զանգ4 = զանգ$-- 3          // հեռացնել բոլոր հայտնությունները
զանգ5 = զանգ$-[1]          // հեռացնել 1-ին ինդեքսում (առաջին տարրը)
զանգ6 = զանգ$-[2..3]       // հեռացնել տիրույթ (1-հիմնված, վերջն ընդգրկված)

կա = զանգ$? 3              // #1 — պարունակում է

աճ = զանգ$^+               // աճման կարգ (միայն պրիմիտիվ)
նվազ = զանգ$^-             // նվազման կարգ (միայն պրիմիտիվ)

// Անվ./դիրք. կոռտեժ-զանգ. — օգտ. $^ համեմատողի լամբդայով

ըստ_ան = բդ$^ (ա, բ -> ա.անուն > բ.անուն)   // ըստ անուն-ի նվազ (>)

>> ըստ_ան[1].անուն ¶    // → Carla

// Ուղղակի տարրի թարմացում (միայն զանգ.)
զանգ[1] = 99              // վերագրել
զանգ[2] += 5              // բաղադրյալ. +=  -=  *=  /=  %=  ^=
```

> Բոլոր հավաքածու օպ. վերադ. **նոր զանգված**։ Վերա-վերագրեք. `զանգ = զանգ$+ 4`։
