# الدليل المختصر لـ Zymbol-Lang

**Zymbol-Lang** لغة برمجة رمزية. لا تستخدم أي كلمات مفتاحية — كل شيء رمز. تعمل بنفس الطريقة في أي لغة بشرية.

- بدون كلمات مفتاحية (`if`، `while`، `return` غير موجودة — فقط رموز `?`، `@`، `<~`)
- Unicode كامل — معرّفات بأي لغة أو رمز تعبيري 👋
- محايد للغة البشرية — الكود متطابق في كل اللغات

---

## المتغيرات والثوابت

```zymbol
س = ١٠           // متغير (قابل للتغيير)
ط := ٣.١٤١٥٩    // ثابت (غير قابل للتغيير — خطأ إذا أُعيد تعيينه)
اسم = "آنا"
نشط = #١        // منطقي صحيح
👋 := "مرحباً"
```

### الإسناد المركّب

```zymbol
س = ١٠    // ١٠
س += ٥    // ١٥
س -= ٣    // ١٢
س *= ٢    // ٢٤
س /= ٣    // ٨
س %= ٣    // ٢
س ^= ٢    // ٤
س++       // ٥
س--       // ٤
```

## أنواع البيانات

| النوع          | مثال                | رمز `#?`  | ملاحظات                                |
|----------------|---------------------|-----------|----------------------------------------|
| صحيح (Int)     | `٤٢`, `-٧`          | `###`     | 64-bit موقّع                          |
| عشري (Float)   | `٣.١٤`, `١.٥e١٠`    | `##.`     | يدعم الترميز العلمي                   |
| نص (String)    | `"مرحبا"`           | `##"`     | استيفاء: `"مرحباً {اسم}"`             |
| حرف (Char)     | `'أ'`               | `##'`     | حرف Unicode واحد                      |
| منطقي (Bool)   | `#١`, `#٠`          | `##?`     | ليس رقمياً 1 و 0                      |
| مصفوفة (Array) | `[١, ٢, ٣]`         | `##]`     | جميع العناصر من نفس النوع             |
| صف (Tuple)     | `(أ, ب)`            | `##)`     | موضعي                                 |
| صف مسمّى       | `(س: ١, ص: ٢)`      | `##)`     | وصول بالاسم أو الفهرس                 |

## الإخراج والإدخال

```zymbol
// الإخراج — لا يضيف سطراً جديداً تلقائياً
>> "مرحباً، العالم!" ¶               // ¶ أو \\ يعطي سطراً جديداً صريحاً
>> "أ=" أ " ب=" ب ¶                  // قيم متعددة بالتجاور
>> "المجموع=" مجموع(٢, ٣) ¶          // استدعاء دوال في أي موضع
>> (مصفوفة$#) ¶                      // عمليات اللاحقة تتطلب أقواساً

// الإدخال
<< اسم                               // بدون موجّه — يقرأ في متغير
<< "اسمك؟ " اسم                      // مع موجّه
```

> `¶` أو `\\` متكافئان كسطر جديد.

```zymbol
>> ١٠ - ٣ ¶    // → ٧
```

## السلاسل النصية

ثلاثة أشكال صحيحة — كلٌّ لسياقه:

```zymbol
اسم = "آنا"
ن = ٢٥

// ١. الفاصلة — في الإسنادات مع = أو :=
رسالة = "مرحباً "، اسم، "!"             // → مرحباً آنا!
العنوان := "المستخدم: "، اسم

// ٢. التجاور — في إخراج >>
>> "مرحباً " اسم " عمرك " ن ¶          // → مرحباً آنا عمرك ٢٥

// ٣. الاستيفاء — في أي سياق
وصف = "مرحباً {اسم}، عمرك {ن}"         // → مرحباً آنا، عمرك ٢٥
```

```zymbol
// عمليات على النصوص
ن = "العالم"
>> (ن$#) ¶                         // → ٦  (الطول — يتطلب أقواساً في >>)
قطعة = ن$[٠..٤]                    // "العال"  (قطع)
ن٢ = ن$~~["العالم":"الكون"]        // "الكون"  (استبدال)
ن٣ = ن$~~["ل":٢]                   // استبدل أول ٢ ظهور لـ"ل"
```

> **ملاحظة**: `+` للأرقام فقط. استخدامه مع النصوص يُنتج تحذيراً.

## التحكم في التدفق

```zymbol
س = ٧

// شرط بسيط
? س > ٠ { >> "موجب" ¶ }

// إذا / وإلا-إذا / وإلا
? س > ١٠٠ {
    >> "كبير" ¶
} _? س > ٠ {
    >> "موجب" ¶
} _? س == ٠ {
    >> "صفر" ¶
} _ {
    >> "سالب" ¶
}
```

الكتل `{ }` **مطلوبة** حتى لسطر واحد.

## الحلقات

```zymbol
// نطاق شامل: ٠..٤ يتكرر ٠,١,٢,٣,٤
@ م:٠..٤ { >> م " " }
>> ¶    // → ٠ ١ ٢ ٣ ٤

// نطاق بخطوة
@ م:١..٩:٢ { >> م " " }
>> ¶    // → ١ ٣ ٥ ٧ ٩

// نطاق عكسي
@ م:٥..٠:١ { >> م " " }
>> ¶    // → ٥ ٤ ٣ ٢ ١ ٠

// بينما (While)
ن = ١
@ ن <= ٦٤ { ن *= ٢ }
>> ن ¶    // → ١٢٨

// لكل عنصر في المصفوفة
فواكه = ["تفاح", "كمثرى", "عنب"]
@ ف:فواكه { >> ف ¶ }

// على أحرف النص
@ ح:"مرحبا" { >> ح "-" }
>> ¶    // → م-ر-ح-ب-ا-

// إيقاف ومتابعة
@ م:١..١٠ {
    ? م % ٢ == ٠ { @> }    // @> متابعة
    ? م > ٧ { @! }          // @! إيقاف
    >> م " "
}
>> ¶    // → ١ ٣ ٥ ٧
```

## الدوال

```zymbol
// التعريف والاستدعاء
مجموع(أ, ب) { <~ أ + ب }
>> مجموع(٣, ٤) ¶    // → ٧

// العودية
مضروب(ن) {
    ? ن <= ١ { <~ ١ }
    <~ ن * مضروب(ن - ١)
}
>> مضروب(٥) ¶    // → ١٢٠
```

الدوال لها **نطاق معزول** — لا وصول للمتغيرات الخارجية. استخدم معاملات الإخراج `<~` لتعديل متغيرات المستدعي:

```zymbol
مبادلة(أ<~, ب<~) {
    مؤقت = أ
    أ = ب
    ب = مؤقت
}
س = ١٠
ص = ٢٠
مبادلة(س, ص)
>> "س=" س " ص=" ص ¶    // → س=٢٠ ص=١٠
```

> الدوال المسمّاة ليست قيماً من الدرجة الأولى. لتمريرها كمعامل، لفّها: `س -> اختبار(س)`.

## Lambda والإغلاقات

```zymbol
// Lambda بسيطة (إرجاع ضمني)
مضاعف = س -> س * ٢
مجموع_لا = (أ, ب) -> أ + ب
>> مضاعف(٥) ¶    // → ١٠
>> مجموع_لا(٣, ٧) ¶  // → ١٠

// Lambda بلوك (إرجاع صريح)
تصنيف = س -> {
    ? س > ٠ { <~ "موجب" }
    _? س < ٠ { <~ "سالب" }
    <~ "صفر"
}
>> تصنيف(٥) ¶     // → موجب
>> تصنيف(٠) ¶     // → صفر
>> تصنيف(-٥) ¶    // → سالب

// الإغلاقات — lambdas تلتقط متغيرات النطاق الخارجي
معامل = ٣
ثلاثي = س -> س * معامل    // يلتقط 'معامل'
>> ثلاثي(٧) ¶    // → ٢١

// مصنع الدوال
صانع_جمع(ن) { <~ س -> س + ن }
جمع_عشرة = صانع_جمع(١٠)
>> جمع_عشرة(٥) ¶    // → ١٥

// Lambda كقيم: تخزينها في مصفوفات
عمليات = [س -> س+١, س -> س*٢, س -> س*س]
>> عمليات[٠](٥) ¶    // → ٦
>> عمليات[٢](٥) ¶    // → ٢٥
```

## المصفوفات

المصفوفات **قابلة للتعديل** وتحتوي على عناصر من **نفس النوع**.

```zymbol
arr = [١, ٢, ٣, ٤, ٥]

arr[٠]          // ١ — وصول (فهرس يبدأ من ٠)
arr[-١]         // ٥ — فهرس سالب (الأخير)
arr$#           // ٥ — الطول (يتطلب أقواساً في >>)

arr = arr$+ ٦            // إضافة → [١,٢,٣,٤,٥,٦]
arr٢ = arr$+[٢] ٩٩       // إدراج في الفهرس ٢
arr٣ = arr$- ٣           // حذف أول ظهور للقيمة
arr٤ = arr$-- ٣          // حذف جميع الظهورات
arr٥ = arr$-[٠]          // حذف عند الفهرس
arr٦ = arr$-[١..٣]       // حذف نطاق (النهاية حصرية)

يوجد = arr$? ٣           // #١ — يحتوي
مواضع = arr$?? ٣         // [٢] — كل فهارس القيمة
قطعة = arr$[٠..٣]        // [١,٢,٣] — قطع (النهاية حصرية)
قطعة٢ = arr$[٠:٣]        // [١,٢,٣] — بصيغة العدد

تصاعدي = arr$^+          // ترتيب تصاعدي (البدائيات فقط)
تنازلي = arr$^-          // ترتيب تنازلي (البدائيات فقط)

// مصفوفات الصفوف — استخدم $^ مع lambda مقارنة
db = [(name: "كارلا", age: ٢٨), (name: "آنا", age: ٢٥), (name: "بوب", age: ٣٠)]
حسب_العمر = db$^ (أ, ب -> أ.age < ب.age)
>> حسب_العمر[٠].name ¶    // → آنا

// تحديث مباشر للعنصر (المصفوفات فقط)
arr[١] = ٩٩              // إسناد
arr[٠] += ٥              // مركّب: +=  -=  *=  /=  %=  ^=

// تحديث وظيفي — يُرجع مصفوفة جديدة؛ الأصل لا يتغير
arr2 = arr[١]$~ ٩٩
```

> جميع عمليات المجموعات تُرجع **مصفوفة جديدة**. أعد التعيين: `arr = arr$+ ٤`.
> لا سلسلة: استخدم تعيينات وسيطة منفصلة.
> `$^+` / `$^-` يرتبان **مصفوفات البدائيات**. للصفوف استخدم `$^` مع lambda مقارنة.

**دلالات القيمة** — إسناد مصفوفة إلى متغير آخر يُنشئ نسخة مستقلة:

```zymbol
a = [1, 2, 3]
b = a
a[0] = 99
>> a ¶    // → [99, 2, 3]
>> b ¶    // → [1, 2, 3]   ← b غير متأثر
```

```zymbol
// مصفوفات متداخلة
مصفوفة_محورية = [[١,٢,٣],[٤,٥,٦],[٧,٨,٩]]
>> مصفوفة_محورية[١][٢] ¶    // → ٦
```

## الصفوف

الصفوف حاويات مرتبة **غير قابلة للتعديل** يمكنها احتواء قيم من **أنواع مختلفة**. على عكس المصفوفات، لا يمكن تغيير العناصر بعد الإنشاء.

```zymbol
// موضعي
نقطة = (١٠, ٢٠)
>> نقطة[٠] ¶    // → ١٠

بيانات = (42, "hello", #1, 3.14)
>> بيانات[2] ¶     // → #1

// مسمّى
شخص = (اسم: "أليس", عمر: ٢٥)
>> شخص.اسم ¶    // → أليس
>> شخص[٠] ¶     // → أليس (الفهرس يعمل أيضاً)

// متداخل
موضع = (س: ١٠, ص: ٢٠)
ن = (موضع: موضع, اسم: "الأصل")
>> ن.موضع.س ¶   // → ١٠
```

**اللاتغيّرية** — أي محاولة لتعديل عنصر في صف تُعدّ خطأ في وقت التشغيل:

```zymbol
t = (10, 20, 30)
// t[0] = 99    // ❌ خطأ في وقت التشغيل: الصفوف غير قابلة للتعديل
// t[0] += 5    // ❌ نفس الخطأ
```

لاشتقاق قيمة معدَّلة استخدم `$~` (تحديث وظيفي) — يُرجع صفاً **جديداً**:

```zymbol
t = (10, 20, 30)
t2 = t[1]$~ 999
>> t ¶     // → (10, 20, 30)   ← الأصل دون تغيير
>> t2 ¶    // → (10, 999, 30)

// صف مسمّى — إعادة بناء صريحة
شخص = (اسم: "Alice", عمر: 25)
أكبر  = (اسم: شخص.اسم, عمر: 26)
>> شخص.عمر ¶    // → 25
>> أكبر.عمر ¶     // → 26
```

## دوال الترتيب الأعلى

> عمليات HOF تتطلب **lambda مضمّنة** — ليس متغير lambda مباشر.

```zymbol
أرقام = [١, ٢, ٣, ٤, ٥, ٦, ٧, ٨, ٩, ١٠]

// تحويل ($>)
مضاعفات = أرقام$> (س -> س * ٢)
>> مضاعفات ¶    // → [٢, ٤, ٦, ٨, ١٠, ١٢, ١٤, ١٦, ١٨, ٢٠]

// تصفية ($|)
زوجية = أرقام$| (س -> س % ٢ == ٠)
>> زوجية ¶    // → [٢, ٤, ٦, ٨, ١٠]

// تقليص ($<) — (البداية، (مجمّع، عنصر) -> تعبير)
مجموع_كل = أرقام$< (٠, (تجميع, س) -> تجميع + س)
>> مجموع_كل ¶    // → ٥٥

// سلسلة عبر وسيطات
خطوة١ = أرقام$| (س -> س > ٣)
خطوة٢ = خطوة١$> (س -> س * س)
>> خطوة٢ ¶    // → [١٦, ٢٥, ٣٦, ٤٩, ٦٤, ٨١, ١٠٠]
```

## مُشغِّل الأنبوب

الجانب الأيمن يتطلب دائماً `_` كعنصر نائب للقيمة المُمرَّرة:

```zymbol
مضاعف = س -> س * ٢
جمع = (أ, ب) -> أ + ب
زيادة = س -> س + ١

٥ |> مضاعف(_)        // → ١٠
١٠ |> جمع(_, ٥)      // → ١٥
٥ |> جمع(٢, _)       // → ٧

// متسلسل
ن = ٥ |> مضاعف(_) |> زيادة(_) |> مضاعف(_)
>> ن ¶    // → ٢٢  (٥→١٠→١١→٢٢)
```

## أنظمة الأرقام

يمكن لـ Zymbol عرض الأرقام في **69 نظام أرقام Unicode** — عربي هندي، فارسي، تايلاندي، كلينغون pIqaD، الخط الرياضي الغامق، أرقام LCD، والمزيد. يؤثر الوضع النشط فقط على إخراج `>>`؛ العمليات الحسابية الداخلية دائمًا ثنائية.

### تفعيل نظام

اكتب الرقم `٠` والرقم `٩` من النظام المطلوب بين `#…#`:

```zymbol
#٠٩#    // عربي هندي   (U+0660–U+0669)
#०९#    // ديفاناغاري  (U+0966–U+096F)
#๐๙#    // تايلاندي    (U+0E50–U+0E59)
#09#    // إعادة ASCII
```

### الإخراج والقيم المنطقية

```zymbol
x = ٤٢
>> x ¶          // → ٤٢   (ASCII افتراضيًا)

#٠٩#
```

### قيم حرفية أصلية في المصدر

أرقام أي نظام مدعوم هي قيم حرفية صالحة في النطاقات والمودولو والمقارنات:

```zymbol
    _? م % ٥  == ٠ { >> "Buzz" ¶ }
```

### قيم منطقية حرفية في أي نظام

`#` + الرقم `٠` أو `١` من أي كتلة مدعومة هو قيمة منطقية حرفية صالحة:

```zymbol
#٠٩#
نشط = #١        // مثل #١
>> نشط ¶        // → #١
>> (#١ && #٠) ¶ // → #٠
```

> `#` **دائمًا ASCII**. `#٠` (خطأ) مختلف بصريًا عن `٠` (الصفر الصحيح) في أي نظام.
