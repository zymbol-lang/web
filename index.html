design principles

Built for every language.

Zymbol's core idea: the syntax belongs to no culture.

?@
No keywords — ever

Every construct is a symbol. ? is if. @ is loop. <~ is return. Nothing to translate per language.

∀
Full Unicode identifiers

Name variables in Arabic, write functions in Japanese, use emojis. The operators ? @ >> <~ stay the same.

#1
Language-agnostic types

#1 and #0 instead of true / false. No English words in the type system.
